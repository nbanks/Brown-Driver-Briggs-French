hebreu biblique

I. אַחֵר adjectif autre (proprement un qui vient apres), feminin אַחֶ֫רֶת (avec daghesh forte implicite) ; pluriel אֲחֵרִים (comme si du singulier אָחֵר), une fois Jb 31,10 אֲחֵרִין, אֲחֵרוֺת (= arabe , sabeen אאחֿר, assyrien a—ru futur feminin pluriel comme substantif a—rat ûmê futur des jours) Gn 4,25 זֶרַע אַחֵר autre semence Gn 8,10 שִׁבְעַת יָמִים אֲחֵרִים sept autres jours Ex 22,4 בִּשְׂדֵה אַחֵר dans le champ d'un autre + souvent ; אִישׁ אַחֵר autre homme (mari) Gn 29,19 ; Dt 24,2 ; Jr 3,1 חָצֵר הָאַחֶרֶת 1 R 7,8 l'autre cour, voir חָצֵר ; ׳ הַחוֺמָה א 2 Ch 32,5 l'autre mur, voir חוֺמָה . Ajoute a un nom propre pour distinction Esd 2,31= Ne 7,34 (voir Ne 7,7 = Nehemie Ne 7,12) Ne 7,33 (probablement ici erreur de texte, voir Be Ry18 : pas dans Esd 2,29). Souvent avec le sens collateral de different, comme ׳ בְּגָדִים א autres vetements Lv 6,4 ; 1 S 28,8 ; Ez 42,14 ; Ez 44,19 עָפָר, אֲבָנִים Lv 14,42 רוּחַ Nb 14,24 ׳ וְנֶהְפַּכְתָּ֫ לְאִישׁ אַ 1 S 10,6 לֵב 1 S 10,9 ; Ez 11,19 ( ᵐ5 Hi Sm) ; שֵׁם Es 65,15 (comparer Es 62,2 חָדָשׁ) ; avec celui de etrange, etranger, comme ׳ אִישׁ אַ Dt 20,5 ; Dt 20,6 ; Dt 20,7 ; Dt 28,30 (ainsi אַחֵר, אֲחֵרִים seul Ps 109,8 ; Jb 31,8 ; Jb 31,10 ; Es 65,22 ; Jr 6,12 ; Jr 8,10 et ailleurs) ; עַם Dt 28,32 אֶרֶץ Dt 29,27 ; Jr 22,26 אִשָּׁה Jg 11,2 לָשׁוֺן Es 28,11 (|| בְּלַעֲגֵי שָׂפָה) ; surtout dans l'expression אֱלהִים אֲחֵרִים autres dieux (63 f.) Ex 20,3 (= Dt 5,7) Dt 23,13 (les deux J E ) Jos 24,2 ; Jos 24,16 ( E ) 1 S 26,19 ; Os 3,1, & particulierement dans le Deuteronome ( Os 6,14 ; Os 8,19 + 15 f.) & les ecrivains deuteronomiques, comme Jos 23,16 ; Jg 2,12 ; Jg 2,17 ; Jg 2,19 Jeremie (18 f.) & le compilateur des Rois ; אֵל אַחֵר une fois Ex 34,14 ( J E ). Ainsi אַחֵר seul Es 42,8 ; Ps 16,4. De temps, suivant, prochain (rare) ׳ בַּשָּׁנָה הָאַ Gn 17,21 ( P ) dans l'annee suivante ; 2 R 6,29 ׳ בַּיּוֺם הָאַ ; ׳ דּוֺר אַ Jl 1,3 ; Ps 109,13 en poesie la prochaine generation (Jg 2,10 en prose = une autre generation).
