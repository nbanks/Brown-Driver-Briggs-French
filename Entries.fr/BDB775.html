hebreu biblique

אַרְנוֺן, אַרְנֹן nom propre [d'une riviere] Arnon, oued & cours d'eau en Moab ( MI ארנן, Thes Add Rob Ges MV derivent de רנן, c.-a-d. le torrent impetueux, rugissant) — אַרְנוֺן Nb 21,13 (deux fois dans le verset) + 12 f., אַרְנֹן Nb 21,14 + 10 f. — appele frontiere entre Moab & Amorites Nb 21,13 ; Jg 11,18 ; Jg 11,22 comparer גְּבוּלאַרְנֹן Nb 22,36 souvent נַחַל אֵרְנֹן Dt 2,24 ; Dt 2,36 ; Dt 3,8 ; Dt 3,12 ; Dt 3,16 ; Dt 4,48 ; 2 R 10,33 אַרְנוֺן ׳ נ Jos 12,1 ; Jos 12,2 ; Jos 13,9 ; Jos 13,16 הַנְּחָלִים אַרְנוֺן Nb 21,14, c.-a-d. les ravins-torrents qui s'unissent pour former l'Arnon, comparer Di ; aussi בָּמוֺת אַרְנֹן hauteurs de l'Arnon Nb 21,28 מעברות אַרְנוֺן gues de l'Arnon Es 16,2 ailleurs Nb 21,13 ; Nb 21,24 ; Nb 21,26 ; Jg 11,13 ; Jg 11,18 ; Jg 11,26 synonyme de Moab Jr 48,20 (comparer TristrMoab 125 f. ; moderne Môjib).
