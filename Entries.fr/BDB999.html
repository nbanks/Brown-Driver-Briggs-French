BDB999 [H974]

hebreu biblique

[בָּחַן] verbe examiner, eprouver (comparer arameen בְּחַן, syriaque , eprouver, examiner ; arabe I. VIII. tester, comparer WSG 65 ) —

Qal Accompli suffixe בְּחָנַ֫נִי Jb 23,10, בָּחַנְתָּ Ps 17,3 + 7 f. ; Inaccompli יִבְחָ֑ן Ps 11,5 + 5 f. ; Imperatif suffixe בְּחָנֵנִי Ps 26,2 + 2 f. ; Infinitif בְּחֹן Za 13,9 Participe בֹּחֵן Jr 11,20 + 5 f. (|| צרף, נסה) ; —
1. examiner, scruter, eprouver Ps 11,5 ; Ps 139,23 ; Jb 7,18 עַפְעַפָּיו יִבְחֲנוּ בְנֵי אָדָם Ses paupieres eprouvent les fils de l'homme (les scrutent de part en part) Ps 11,4.
2. prouver, tester, eprouver.
a. avec la metaphore de l'or Jb 23,10 וּבְחַנְתִּים כִּבְחֹן אֶתהַֿזָּהָב et je les eprouverai comme on eprouve l'or Za 13,9 .
b. sans metaphore, de Dieu testant les personnes Psaume Za 26,2 ; Za 66,10 ; Za 81,8 ; Jr 9,6 leurs voies Jr 6,27 le לֵב Ps 17,3 ; Jr 12,3 ; Pr 17,3 ; 1 Ch 29,17 les reins Jr 17,10 le coeur et les reins Ps 7,10 ; Jr 11,20 (= Jr 20,12).
c. de l'homme testant ou tentant Dieu Psaume Jr 95,9 ; Ml 3,10 ; Ml 3,15.
d. l'oreille eprouvant les paroles Jb 12,11 = Jb 34,3.
Niph`al Inaccompli יִבָּחֵן Jb 34,36 ; Gn 42,15 ; Gn 42,16, etre eprouve, prouve.
Pu`al בֹּחַן Ez 21,18 l'epreuve a ete faite ( ᵐ5 ᵑ6 ᵑ9 Ges MV Ew Ke, mais nom בֹּ֫חַן ᵑ0 Symm HaEv. ; lire בְּחֵן avec grace, faveur, Hi Co).
