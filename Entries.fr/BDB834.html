BDB834 [H833]

hebreu biblique

[אָשַׁר] verbe aller droit, avancer (assyrien ašâru ZimBP 11 ; arabe, arameen dans les derives) —

Qal Imperatif אִשְׁרוּ Pr 9,6 אִשְׁרוּ בְדֶרֶךְ בִּינָה allez droit dans le chemin de l'intelligence.
Pi`el Accompli אִשְּׁרוּ Ml 3,12 ; Gn 30,13 Inaccompli תְּאַשֵּׁר Pr 4,14 + 4 f. ; Imperatif אַשֵּׁר Pr 23,19 Participe pluriel מְאַשְּׁרִים Ml 3,15 + 2 f. ; —
1. intensif aller droit, avancer Pr 4,14.
2. causatif conduire Pr 23,19 ; Es 3,12 ; Es 9,15 .
3. redresser, rectifier Es 1,17.
4. declarer heureux, appeler beni Gn 30,13 ( J ) Jb 29,11 ; Ps 72,17 ; Pr 31,28 ; Ct 6,9 ; Ml 3,12 ; Ml 3,15.
Pu`al Inaccompli יְאֻשַּׁר Ps 41,3 Participe מְאֻשָּׁר Pr 3,18 ; Es 9,15.
1. etre conduit Es 9,15 .
2. etre rendu heureux, beni Ps 41,3 ; Pr 3,18.
