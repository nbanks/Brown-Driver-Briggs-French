hebreu biblique

אַחַר proprement substantif la partie posterieure ou suivante (comparer le pluriel)

1. adverbe
a. de lieu, derriere, deux fois Gn 22,13 (nombreux MSS . ⅏ ᵐ5 ᵑ6 Ol Ew lisent אֶחָד voir Di ) Ps 68,26 .
b. de temps, ensuite Gn 10,18 ; Gn 18,5 ; Gn 24,55 ; Gn 30,21 ; Jg 19,5 + ; וְאַחַר dans les lois de P , comme Lv 14,8 ; Lv 14,19 ; Lv 15,28 ; Lv 22,7 ; Nb 5,26 et ailleurs
2. preposition
a. de lieu, derriere, apres Ex 3,1 ; Ex 11,5 ; 2 R 11,6 ; Ct 2,9 ; Es 57,8 : הָלַח אַחַר aller apres, suivre Gn 37,17 ; 2 R 13,2 ; 2 R 23,3 ; Es 65,2 ; Ez 13,3 ; Jb 31,7 חָיָה אַחַר 1 S 12,14 מֵאַחַר de derriere 2 S 7,8 ; Ps 78,71 ; Es 59,13.
b. de temps, apres Gn 9,28 ; Lv 25,15 etc. ; הָאֵלֶּה אַחַר הַדְּבָרִים apres ces choses Gn 15,1 ; Gn 22,1 ; Gn 39,7 ; Gn 40,1 ; 1 R 17,17 ; 1 R 21,1 ; Esd 7,1 ; Est 2,1 ; Est 3,1 אַחַר כֵּן Lv 14,36 ; Dt 21,13 ; 1 S 10,5 אַחַר זֶה (tardif) 2 Ch 32,9 suivi de l'infinitif Nb 6,19 ; Jr 40,1 ; 1 Ch 2,24 ; Jb 21,3 עַד אַחַר jusqu'apres Ne 13,19. c. Ne 5,15 etrangement : Ew RV en plus de ; mais le texte est probablement corrompu, voir Be Ry.
3. conjonction אַחַר אֲשֶׁר apres que Ez 40,1 et sans אשׁר Lv 14,43 ; Jr 41,16 ; Jb 42,7. Comme preposition & conjonction le pluriel אַחֲרֵי est beaucoup plus frequent, qui en tout cas doit etre utilise devant les suffixes.
Pluriel seulement construit אַחֲרֵי avec suffixe אַחֲרַי , אַחֲרֶיךָ, etc.
