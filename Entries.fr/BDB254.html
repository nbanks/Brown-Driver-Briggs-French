BDB254 [H227]

hebreu biblique

אָז adverbe (comparer , aussi arameen biblique אֱדַיִן proprement un substantif = temps ; voir מֵאָז ) a ce moment-la, alors, exprimant soit la duree, soit l'inception (= la-dessus).

1. strictement temporel :
a. du temps passe : — sans verbe Gn 12,6 ; Gn 13,7 ; Jos 14,11 ; 2 S 23,14 avec un accompli Gn 4,26 ; Ex 4,26 ; Jos 10,33 ; Jg 8,3 ; Jg 13,21 ; Jr 22,15 אָז טוֺב לוֺ alors cela allait bien pour lui (comparer Jr 22,16 ; Os 2,9) ; plus habituellement (surtout quand = la-dessus ) avec un inaccompli (voir Dr§ 27) Ex 15,1 ; Nb 21,17 ; Jos 8,30 ; Jos 10,12 ; Jos 22,1 ; 1 R 3,16 ; 1 R 8,1 ; 1 R 9,11 (voir DrIntr.192) et ailleurs.
b. du temps futur (habituellement ou une emphase particuliere est voulue), avec inaccompli Es 35,5 ; Es 35,6 ; Es 60,5 ; Mi 3,4, assez differemment Lv 26,41 ; 1 S 20,12 rarement avec accompli 2 S 5,24 b (sera parti) Es 33,23 : avec une force logique accompagnante, impliquant l'accomplissement d'une condition, alors = si ou quand cela aura ete fait (avec l'inaccompli) Gn 24,41 ; Ex 12,44 ; Ex 12,48 ; Dt 29,19 ; Jos 1,8 ; 1 S 6,3 ; Es 58,8 ; Es 58,9 ; Jr 11,15 Hi Ke Ps 19,14 ; Ps 51,21 ; Pr 3,23 ; Jb 11,15 ; Jb 13,20 ; Jb 22,26.
c. en poesie אז est parfois employe pour mettre l'emphase sur un trait particulier de la description Gn 49,4 ; Ex 15,15 ; Jg 5,8 ; Jg 5,11 ; Jg 5,13 ; Jg 5,19 ; Jg 5,22 ; Es 33,23 ; Es 41,1 ; Ha 1,11 ; Ps 2,5 ; Ps 96,12.
d. il renvoie avec emphase a un infinitif avec בּ 2 S 5,24 a Ps 126,2 (deux fois dans le verset) ; Jb 28,27 ; Jb 33,16 a בַּיּוֺם הַהוּא (de maniere anomale) 1 Ch 16,7.
2. exprimant une consequence logique strictement dans l'apodose : — apres אם (rare) pour une emphase particuliere Es 58,14 ; Pr 2,5 ; Jb 9,31 = dans ce cas, apres לוּ ou לוּלֵא 2 S 2,27 si tu n'avais pas parle, כִּי אָז assurement alors le peuple aurait, etc. 2 S 19,7 ; Ps 119,92 apres אחלי 2 R 5,3 ; Ps 119,6 apres une protase supprimee 2 R 13,19 ; Jb 3,13 יָשַׁנְתִּי אָז יָנוּחַ לִי j'aurais dormi, alors il y aurait eu du repos pour moi ; Jos 22,31 (etrangement) = maintenant, en l'etat ; Qo 2,15 = cela etant. (122 f., outre מאז et מןאֿז. Rarement employe sauf quand une emphase particuliere est recherchee. 'Then' de AV RV represente plus communement וְ, surtout dans l'apodose.)
