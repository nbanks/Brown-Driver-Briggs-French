hebreu biblique

בְּלִי substantif usure ( , usure d'un vetement), d'ou

1. au figure destruction Es 38,17 שַׁחַת בְּלִי fosse de destruction (du Sh®°ôl).
2. defaut, manque, d'ou adverbe [de negation] (comparer אֶפֶס), principalement poetique pour לֹא, אֵין : —
a. avec verbe fini rare et seulement une fois en prose, Gn 31,20 ( E ) עַלבְּֿלִי הִגִּיד לוֺ parce qu'il ne lui dit pas, Es 14,6 ; Es 32,10 ; Os 8,7 ; Os 9,16 ( Kt ) Jb 41,18.
b. employe pour nier un adjectif ou participe 2 S 1,21 בְּלִי מָשִׁיחַ non oint, Ps 19,4 ; Os 7,8 : plus frequemment, surtout dans Job, joint a un substantif au sens de sans, Jb 8,11 le jonc poussera-t-il בְּלִימָֿ֑יִם sans eau ? Jb 24,10 ils vont nus בְּלִי לְבוּשׁ sans vetement, Jb 31,39 ; Jb 33,9 ; Jb 34,6 ; Jb 38,2 paroles בְּלִידַֿעַת sans connaissance, Jb 39,16 ; Jb 42,3 ; Ps 59,5 בְּלִיעָֿוֺן יְרוּצוּן sans (mon) iniquite ils courent (contre moi), comparer Ps 59,4 לֹא, Ps 63,2 une terre seche בְּלִי מָ֑יִם sans eau, Es 28,8 ; Jb 30,8 בְּנֵי בְלִישֿׁם fils d'(hommes) sans nom.

Avec prepositions
a. בִּבְלִי, dans בִּבְלִי דַעַת Dt 4,42 ; Dt 19,4 ; Jos 20,3 ; Jos 20,5 D (= sans le savoir : tous dans la loi de D sur l'homicide) ; sans connaissance Jb 35,16 ; Jb 36,12 .
b. לִבְלִי dans un etat de (voir ci-dessous לְ) non..., c.-a-d. sans,Es 5,14 לִבְלִיחֹֿק sans egard a, sans mesure, Jb 38,41 ; Jb 41,25 qui est fait לִבְלִיחֵֿת (pour etre) dans un etat de non crainte, c.-a-d. pour etre sans peur.
c. מִבְּלִי (a) faute de, suivi d'un substantif ou infinitif, מִן exprimant la causalite, Dt 9,28 ׳ מִבְּלִי יְכֹלֶת י a cause de l'incapacite de l'Eternel... (dans Nb 14,16 מִבִּלְתִּי), Nb 28,55 ( Ew238 d ; mais aussi RSJPh xvi. 72), Es 5,13 faute de connaissance, Os 4,6 ; Ez 34,5 ; Lm 1,4 מִבְּלִי בָּאֵי מוֺעֵד faute de participants a la fete fixee. Suivi d'un אֵין pleonastique dans l'expression הֲמִבְּלִי אֵין ֗֗֗ est-ce a cause de l'absence de...? (litteralement est-ce du manque de non...? comparer en syriaque ; PS528), Ex 14,11 ; 2 R 1,3 ; 2 R 1,6 ; 2 R 1,16. ( β) de sorte qu'il n'y a pas... (litteralement loin de il y avoir non..., מִן exprimant la negation, et בְּלִי etant pleonastique, comme dans מֵאֵין, voir ci-dessous אַיִן 6. d. β) ; Jr 2,15 ses villes sont brulees מִבְּלִי יוֺשֵׁב de sorte qu'il n'y a pas d'habitant, Jr 9,9-11 ; Ez 14,15 ; So 3,6. Employe une fois comme conjonction מִבְּלִי אֲשֶׁר, avec לֹא pleonastique, de sorte que ne pas... Qo 3,11. Dans Job מִבְּלִי est employe plus librement = sans, le lien avec un verbe precedent n'etant plus distinctement percu : Jb 4,20 sans que personne y prenne garde, ils perissent pour toujours, Jb 6,6 ; Jb 24,7 ; Jb 24,8 probablement aussi Jb 4,11 ; Jb 31,19 (bien qu'ici le sens (α) serait admissible). (γ) dans Jb 18,15 תִּשְׁכּוֺן בְּאָהֳלוֺ מִבְּלִילֿוֺ, מִן est probablement partitif (ainsi Hi) : il habitera dans sa tente ce qui n'est rien de lui : Ew§ 323a De moins probablement meme rien, comparer ci-dessous אַיִן 6.d. γ ; Ges 'la terreur (tiree de Jb 18,14) habitera dans sa tente de sorte qu'elle n'est plus sienne.'
d. עַדבְּֿלִי jusqu'a ce qu'il n'y ait plus de... Ps 72,7 ; Ml 3,10.
