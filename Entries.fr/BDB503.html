hebreu biblique

I. אַלּוּף adjectif apprivoise — ׳ א absolu Mi 7,5 + ; construit Pr 2,17 + ; suffixe אַלּוּפֵי Ps 55,14 pluriel אַלֻּפִים Jr 13,21 suffixe אַלֻּפֵינוּ Ps 144,14, etc. ; —

1. apprivoise, docile, כֶּבֶשׂ אַלּוּף un agneau (doux) docile Jr 11,19.
2. nom masculin ami, intime Ps 55,14 (|| מְיֻדָּע) Mi 7,5 (|| רֵעַ) Pr 16,28 ; Pr 17,9 ; Jr 13,21 du mari d'une femme נְעוּרֶיהָ ׳ א Pr 2,17 figure de ׳ י comme mari de Juda, id., Jr 3,4.
3. c.-a-d. I. אֶלֶף ; אַלּוּפֵינוּ מְסֻבָּלִים Ps 144,14 (c.-a-d. vaches).
