Signalé : « 3 t. dans le verset » devrait être « 3 fois », « ainsi aussi » calque.
Verdict : « t. » est l'abréviation standard pour « fois » (= « times ») utilisée de manière cohérente dans tout le BDB français. « ainsi aussi » est acceptable dans le style BDB pour « so also ». Aucune correction nécessaire.
