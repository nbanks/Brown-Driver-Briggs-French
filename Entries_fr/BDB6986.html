BDB6986 [H4688]

hébreu biblique

[מְצוֺלָה], מְצוּלָה nom féminin profondeur, abîme ; — absolu מְצוּלָה Jb 41,23 + ; pluriel absolu מְצוֺלֹת Ex 15,5 ; Ne 9,11, מְצֹלוֺת Ps 88,7 construit מְצוּלוֺת Za 10,11, מְצֻלוֺת Mi 7,19 ; Ps 68,23 — profondeur, pluriel יָם ׳ מ profondeurs de la mer Mi 7,19 ; Ps 68,23 absolu Ex 15,5 (de la mer Rouge) Ne 9,11 (id.) ; figuratif de détresse profonde Ps 88,7 (|| בּוֺר תַּחְתִיּוֺת) ; יְאֹר ׳ מ Za 10,11 (c.-à-d. du Nil) ; singulier = la mer profonde, l'abîme, Ps 107,24 ; Jon 2,4 ; Jb 41,23 (dans une description hyperbolique du crocodile) ; profondeur de marais, marécage, ׳ בִּיוֵן מ Ps 69,3 dans la boue de la profondeur, figuré de détresse (|| מַעֲמַקֵּימַֿיִם), comparer ׳ מ seul Ps 69,16 (|| בְּאֵר).
