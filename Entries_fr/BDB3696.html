BDB3696 [H3363]

hébreu biblique

יָקַע verbe être disloqué, aliéné (arabe est tomber, tomber sur, arriver, survenir) —

Qal Imparfait 3 féminin singulier תֵּקַע Jr 6,8 וַתֵּ֫קַע Gn 32,26 + 2 t. ; disloqué Gn 32,26 (la hanche de Jacob) ; ailleurs figuré de נפשׁ ; arraché, aliéné de quelqu'un, suivi de מִן Jr 6,8 ; Ez 23,17 suivi de מֵעַל Ez 23,18.

Semitic 398 (419) précipiter (arabe ) d'un rocher (comparer 2 Ch 25,12) : + ליהוה 2 S 21,6 ( ׳ לפני י 2 S 21,9), Nb 25,4.
