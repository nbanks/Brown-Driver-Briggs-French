hébreu biblique

אָרְנָן nom propre, masculin un Jébusien, dont l'aire de battage fut achetée par David pour ériger un autel 1 Ch 21,15 ; 1 Ch 21,18 ; 1 Ch 21,20 (deux fois dans le verset) ; 1 Ch 21,21 (deux fois dans le verset) ; 1 Ch 21,22 ; 1 Ch 21,23 ; 1 Ch 21,24 ; 1 Ch 21,25 ; 1 Ch 21,28, et selon 2 Ch 3,1 devint le site du temple ; appelé אֲרַוְנָה 2 S 24,16s. q. v.
