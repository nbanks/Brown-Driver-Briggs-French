hébreu biblique

[עָלַף] verbe couvrir (hébreu tardif id., couvrir, s'évanouir ; arabe enduire, II. couvrir, fourreau ; assyrien elpitu , langueur, épuisement) ; —

Pu`al Participe féminin couverte, incrustée, construct מְעֻלֶּפֶת Ct 5,14 son ventre est d'ivoire incrusté de saphirs. Parfait 3 pluriel בָּנַיִךְ עֻלְּפוּ Es 51,20 tes fils sont enveloppés, emmaillotés, c.-à-d. leurs sens obscurcis = se sont évanouis ; de même aussi Ez 31,15, lisant 3 féminin singulier עֻלָּ֑פָה pour le substantif עֻלְפֶּה (Köii. 1, 118), tous les arbres se sont évanouis pour (עַל) lui (ainsi ᵐ5 ᵑ6 ᵑ9 Hi Co et d'autres).
Hithpa`el s'envelopper, Imparfait 3 féminin singulier וַתִּתְעַלָּ֑ף Gn 38,14 ( J ) ; = s'évanouir, 3 masculin singulier וַיִּתְעַלָּ֑ף Jon 4,8 3 féminin pluriel תִּתְעַלַּפְנָה Am 8,13.
