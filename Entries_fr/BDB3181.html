BDB3181 [H2790 H2794]

hébreu biblique

I. [חָרַשׁ] verbe graver, labourer, machiner (hébreu tardif id., labourer, ainsi éthiopien phénicien חרש, araméen (rare) חֲרַת graver, fendre, labourer, arabe labourer) —

Qal Parfait חָֽרְשׁוּ Ps 129,3, חֲרַשְׁתֶּם Jg 14,18 ; Os 10,13 Imparfait יַחֲרשׁ Es 28,24 ; Pr 20,4 יַחֲרוֺשׁ Os 10,11+ Am 6,12 (voir ci-dessous) ; 2 masculin singulier תַּחֲרשׁ Dt 22,10 ; Pr 3,29 Infinitif construit חֲרשׁ 1 S 8,12 Participe חֹרֵשׁ Gn 4,22 + 5 t. ; חוֺרֵשׁ Am 9,13 pluriel חֹרְשִׁים Ps 129,3, חֹרְשֵׁי Jb 4,8 + 3 t., féminin pluriel absolu חֹרְשׁוֺת Jb 1,14 passif חֲרוּשָׁה Jr 17,1 : —
1. graver, d'un artisan des métaux נְחשֶׁת וּבַרְזֶל ׳ ח Gn 4,22 ( J ), נחשׁת ׳ ח 1 R 7,14 figuré עַללֿוּחַ לִבָּם ׳ ח Jr 17,1 gravé sur la tablette de leur cœur.
2. labourer, littéralement, sujet humain (animal habituellement avec בְּ), sans objet exprimé 1 R 19,19 ; Dt 22,10 ; Es 28,24, ainsi יַחֲרוֺשׁ בבקרים Am 6,12 (mais lire probablement יֵחָרֵשׁ בבקר יָם, voir We et d'autres) ; avec accusatif de sens apparenté au verbe חֲרִישׁוֺ ׳ ח 1 S 8,12 labourer son labourage (= faire son labourage), Pr 20,4 figuré de Juda Os 10,11 avec référence éthique רֶשַׁע ׳ ח Os 10,13 (|| קצר), אָוֶן ׳ ח Jb 4,8 (|| זרע, קצר) ; עַלגַּֿבִּי חָֽרְשׁוּ חֹרְשִׁים Ps 129,3 sur mon dos les laboureurs ont labouré (figuré de l'oppression par les méchants) ; חֹרֵשׁ = laboureur Es 28,24 ; Am 9,13 (|| קוֺצֵר) ; ׳ ח avec des bœufs comme sujet seulement Jb 1,14.
3. machiner (comme celui qui travaille dans, pratique), habituellement en mauvais sens, objet רָעָה Pr 3,29, רַע Pr 6,14 מַחְשְׁבוֺת אָוֶן Pr 6,18, חֹרְשֵׁירַע Pr 12,20 ; Pr 14,22 mais aussi חֹרְשֵׁי טוֺב Pr 14,22.
