BDB4451 [H3948]

hébreu biblique

לֶ֫קַח nom masculin Pr 4,2 instruction, enseignement ( concret ; probablement ce qui est reçu ) ; — ׳ ל absolu Es 29,24 + 5 t. ; suffixe לִקְחִי Dt 32,3 ; Jb 11,4 לִקְחָהּ Pr 7,21 —

1. instruction, subjectif, comme acquis personnel ; יִלְמְדוּ לֶ֑קַח Es 29,24, ils apprendront l'instruction (|| בִּינָה) ; וְיוֺסֶף לֶ֑קַח Pr 1,5 un homme sage entendra et augmentera (son) instruction, ainsi Pr 9,9.
2. enseignement, objectif, comme chose enseignée, יַעֲרֹף כַּמָּטָר לִקְחִי Dt 32,2 (|| אִמְרָתִי), Jb 11,4 טוֺב נָתַתִּי לָכֶם ׳ ל Pr 4,2 aussi = pouvoir d'enseignement, persuasion, ׳ מֶתֶק שְׂפָתַיִם יוֺסִיף ל Pr 16,21 la douceur des lèvres augmente la persuasion (voir מתק ), וְעַל שְׂפָתָיו יוֺסִיף לֶקַח Pr 16,23 en mauvais sens, des paroles séductrices de l'adultère, ׳ בְּרֹב ל Pr 7,21 (|| בְּחֵלֶק שְׂפָתֶיהָ).
