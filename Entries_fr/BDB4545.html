hébreu biblique

[מוּךְ] verbe être bas, abattu, s'appauvrir (hébreu tardif id., Niph`al ; araméen מוּךְ, s'enfoncer ou se courber ; comparer מָכַח, syriaque , être abaissé, humilié) ; —

Qal Parfait consécutif וּמָח Lv 27,8 ( P ) ; Imparfait יָמוּח Lv 25,25 ; Lv 25,35 ; Lv 25,39 (tous H ) ; Participe מָח Lv 25,47 ( P ) : tous d'Israélites appauvris.
