hébreu biblique

[שָׁפַר] verbe être beau, agréable, gracieux (arabe retirer le voile (des femmes), briller ; araméen שְׁפַר être plaisant (voir Dn 6,2), être beau, brillant, plaisant, beau) ; —

Qal Parfait 3 féminin singulier שָֽׁפְרָה עָלָ֑י (araméen) Ps 16,6 l'héritage est beau pour (plaisant à) moi (voir עַל II. 8).
