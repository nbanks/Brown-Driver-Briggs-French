hébreu biblique

[לָבַט] verbe renverser, jeter dehors ou repousser (hébreu tardif id. ; arabe frapper le sol avec une personne, c.-à-d. jeter quelqu'un à terre ; syriaque Pael incitavit, stimulavit) ; — seulement

Niph`al Imparfait être renversé, repoussé, c.-à-d. ruiné ; עָם לֹא יָבִין יִלָּבֵט Os 4,14 אֱוִיל שְׂפָתַיִם יִלָּבֵט Pr 10,8 ; Pr 10,10.
