BDB6537 [H6311]

hébreu biblique

פֹּה58, et (Ez 40,1-41,200, 23 sur 35 t.) פּוֺ, aussi Jb 38,11b פֹּא, adverbe [de lieu] ici, ici (vers) (probablement de la même racine démonstrative trouvée dans ainsi, alors (souvent dans l'apodose), אַף, אֵפוֺא : comparer Köii. l, 247 s., 243, 331 s.) : —

1. ici :
a. Gn 19,12 עֹד מִילְֿךָ פֹּה qui as-tu encore ici ? Gn 22,5 שְׁבוּ לָכֶם פֹּה, Gn 40,15 ; Nb 22,8 ; Nb 32,16 ; Dt 5,28 ; Dt 12,8 ; Jg 4,20 הֲיֵשׁ פֹּה אִישׁ, 2 R 3,11, etc. ; demandé avec quelque surprise, Jg 18,3 מַהלְּֿךָ פֹה = quelle est ton affaire ici ? de même 1 R 19,9 ; 1 R 19,13 ; Es 22,16 ; Es 52,5 וְעַתָּה מַהלִּֿיפֹֿה. comparer le synonyme בָּזֶה, כֹּה 2.
b. עַדפֹּֿה Jb 38,11 a.
c. מִפֹּה ou מִפּוֺ, seulement Ez 40-41, et toujours (sauf Ez 40,12 a, où il faut le suppléer avec ᵐ5 ᵑ6 ) répété, de ce côté … de ce côté (comparer מִזֶּה ֗֗֗ מִזֶּה ), Ez 40,10 (deux fois dans le verset) ; Ez 40,12b ; Ez 40,21 (deux fois dans le verset), etc.
d. avec l' interrogatif אֵי, אֵיפֹה où ?.
2. après un verbe de mouvement, ici (vers), 1 S 16,11 עַדבּֿאֹוֺ פֹה, Esd 4,2.
