hébreu biblique

חֵיק, חֵק nom [masculin] sein (hébreu tardif id.) — absolu חֵיק Ez 43,17 ; Pr 16,33 + Ez 43,13 ( Co חיקו, comparer ZKW1883, pp. 67ss. ; חֵיקֹה DrSm xxxi), חֵק Pr 21,14 construit חֵיק 1 R 22,35 + 3 t. + Ez 43,14 (Co חיקו) חֵק Pr 5,20 (omis par accident dans Baer, comparer Strsur le passage) Pr 17,23 suffixe חֵיקִי 1 R 3,20 ; Ps 89,51 חֵקִי Jb 19,27 etc. ; ailleurs toujours plene avec suffixe ; חוקף Kt Ps 74,11 ( Qr חֵיקְךָ) ; — sein, spécifiquement

1. du pli du vêtement, à la poitrine Ex 4,6 (deux fois dans le verset) ; Ex 4,7 (3 t. dans le verset) (tous J ), Pr 6,27 ; Pr 16,33 , métaphore de ׳ י Ps 74,11 ׳ שֹׁחַד בח Pr 21,14 est un présent caché (dissimulé dans le sein ; || מַתָּן בַּסֵּתֶר) ; שֹׁחַד טֵחֵק Pr 17,23 est un don donné secrètement ; figuré, rétribuer dans le sein Es 65,6 ; Es 65,7 ; Ps 35,13 ; Ps 79,12 ; Jr 32,18.
2. porter (נָשָׂא) dans le sein, Moïse le peuple Nb 11,12 ( J E ); Noémi, l'enfant de Ruth Rt 4,16, comparer 1 R 3,20 (deux fois dans le verset) ; 1 R 17,19 brebis 2 S 12,3 ; Es 40,11 (métaphore) ; Lm 2,12 d'enfants mourant dans les bras de leur mère ; figuré de porter des insultes dans son sein Ps 89,51 (lire probablement כְּלִמַּת pour כֹּל, voir Bi Che).
3. sein comme partie du corps :
a. externe, se coucher dans le sein, de l'épouse שׁככת Mi 7,5, comparer 2 S 12,8 concubine Gn 16,5 ( J ) comparer 1 R 1,2 אֵשֶׁת חיק(ךׅ Dt 13,7 ; Dt 28,54, comparer aussi Pr 5,20 תְּחַבֵּק חֵק נָכְּרִיָה ; אֹישׁ חֵיקָהּ Dt 28,56.
b. interne, la colère repose כסילים ׳ בה Qo 7,9 כָּלוּ כִלְֹיתַי בְּחֵקִי Jb 19,27 mes reins se consument en moi ; Jb 23,12 (lire בְּחֵקִי pour מֵחֻקִּי, ainsi ᵐ5 ᵑ9 Ol Me Stu Di) dans mon sein j'ai caché les paroles de sa bouche (comparer בְּלִבִּי Ps 119,11 ) ; figuré de l'intérieur du char 1 R 22,35 הרכב ׳ ח ; du fond creux de l'autel Ez 43,13 ; Ez 43,14 ; Ez 43,17 (Sm Rinne ; Co Grundeinfassung).
