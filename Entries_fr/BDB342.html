BDB342 [H319]

hébreu biblique

אַחֲרִית61 nom féminin partie postérieure, fin ; —

a. de lieu, seulement Ps 139,9 (tardif) יָם ׳ א.
b. de temps, dernière partie ou fin effective (selon le contexte), opposé à רֵאשִׁית ; — de l'année Dt 11,12 de la vie d'un homme Nb 23,10 ; Pr 5,11 ; Jb 8,7 ; Jb 42,12 de l'existence d'un peuple Nb 24,20 = sort final Dt 32,20 ; Dt 32,29 ; Jr 12,4 ; Jr 31,17 ; Ps 73,17 un avenir, c.-à-d. une fin heureuse de la vie, suggérant parfois l'idée d'une postérité, promise aux justes Pr 23,18 (|| תִּקְוָה espérance) Pr 24,14 ; Jr 29,11 (וְתִקְוָה ׳ לָתֵת לָכֶם א), refusée aux méchants Pr 24,20 (|| ׃ נֵר יִדְעָֽח ׃ voir ci-dessous) ; la fin ou l'issue ultime d'une ligne de conduite Jr 5,31 ; Pr 14,12 ; Pr 23,32 (du vin, c.-à-d. de l'excès de boisson) Pr 25,8 ; Es 46,10 (absolu, mais implicitement d'une phase de l'histoire) Es 47,7 (de la conduite décrite Es 47,6 b-7 a) Dn 12,8 ; Qo 7,8 d'une prédiction = l'événement Es 41,22. בְּאַחֲרִית הַיָּמִים à la fin des jours, expression prophétique désignant la période finale de l'histoire aussi loin que la perspective du locuteur atteint ; le sens varie donc selon le contexte, mais souvent = l'avenir idéal ou messianique ; Gn 49,1 (de la période de la possession de Canaan par Israël) Nb 24,14 ; Dt 4,30 (de la période du retour d'Israël à Dieu après l'adversité) Dt 31,29 (de la période de la rébellion d'Israël) Os 3,5 ; Es 2,2 ( = Mi 4,1) Jr 23,20 (voir Graf) = Jr 30,24 ; Jr 48,47 ; Jr 49,39 ; Ez 38,16 (de la période de l'attaque de Gog contre Israël restauré) Dn 2,28 (araméen) Dn 10,14 (de l'époque d'Antiochus Épiphane) : comparer הַשָּׁנִים ׳ כְּא Ez 38,8.
c. הַגּוֺיִם ׳ א Jr 50,12 la dernière, la plus reculée des nations (de Babylone), opposé à ראשׁית הגוים Am 6,1 (Israël) comparer Nb 24,20 (Amalec) chef des nations.
d. concret postérité (extension de l'usage noté ci-dessus dans Pr 24,20) Ps 37,37 ; Ps 37,38 (peut-être pas plus qu'« un avenir » ici) Ps 109,13 (|| אַחֵר יִמַּח שְׁמָם בְּדוֺר) Am 4,2 ; Am 9,1 ; Ez 23,25 (deux fois dans le verset) (selon d'autres, dans ces quatre passages, reste, résidu) Dn 11,4.
