II. אֶ֫לֶף nom masculin Jg 6,15 mille (אלף MI, SI, sabéen DHMZMG 1875, 615 ; arabe , araméen , אֲלַף, אלפיא) — ׳א (אָ֑לֶף Gn 20,16 + ; suffixe אַלְפִּי Jg 6,15 duel אַלְפַּיִם Nb 4,36 + ; pluriel אֲלָפִים Ex 18,21 + ; construit אַלְפֵי Ex 32,28 + ; אֲלָפֶיךָ Dt 7,13 + 2 t. ; אֲלָפָיו Qr 1 S 18,7 + 2 t. (Kt פֿו) — mille.

1. numéral :
a. employé avec le nom seul ; surtout avant le nom Nb 35,4 ; Jos 7,3 ; Ne 3,13 ; 1 Ch 18,4 + ; après le nom (tardif) 1 Ch 12,35 + 6 t. Chroniques Esdras Néhémie ; בָּאַפָּה׳א Nb 35,5 (4 t. dans le verset) ; Ez 47,3 (supprimer Co) ; le nom toujours pluriel quand il précède, parfois quand il suit, 1 S 25,2 ; 1 R 3,4 ; 2 R 18,23 = Es 36,8 ; 2 Ch 30,24 ; Ps 90,4 ; Jb 42,12 ; Qo 6,6 ailleurs singulier אישׁ Jg 9,49 ; Jg 15,15 ; Jg 15,16 ; Jos 7,3 + ; אַמָּה Nb 35,4 + (ainsi SI) ; דּוֺר Dt 7,9 +, comparer 1 Ch 18,4 ; 1 Ch 19,6 ; 2 R 15,19 ; Jb 42,12 ; Ct 4,4 ; Es 7,23, nom parfois collectif ; ׳א pluriel construit Mi 6,7 ; Ps 119,72 ; Gn 24,60 nom non exprimé (ou pas entièrement) Gn 20,16 ; Nb 31,5 ; Nb 31,6 + ; distributif אֶלֶף ֗֗֗ אֶלֶף Nb 31,4 multiplicatif Dt 1,11 פְּעָמִים׳א ; indéfini pour un grand nombre (pluriel) Ex 20,6 ; Ex 34,7 ; Dt 5,10 ; Jr 32,18, (singulier) Dt 32,30 ; Qo 6,6 +.
b.׳א + autre nombre le précède habituellement Ex 38,25 + souvent ; mais suivant רִבּוֺא Esd 2,64 + 2 t. ; il suit aussi un nombre plus petit Nb 3,50 ; 1 R 5,12 (ainsi SI) ; le nom suivant au singulier Ex 28,25 + 8 t. ; pluriel 2 S 8,4 + 2 t. ; le nom précède, au pluriel (tardif) Dn 12,12 + 2 t. ; nom non exprimé Ex 38,28 + souvent
c.׳א X autre nombre toujours suivant Ex 12,37 + ; l'ordre habituel est no. X ׳א + no. supplémentaire (le cas échéant) + nom (s'il est exprimé) Nb 31,52 ; Jg 20,35 comparer Ex 12,37 ; 2 R 3,4 (deux fois dans le verset) + ; moins souvent nom + no. X ׳א Nb 31,33 ; 1 R 8,63 + ; (autres combinaisons voir Ex 38,26 ; Nb 26,51 ; Nb 31,32 ; Ez 48,30 ; 1 Ch 29,7 etc.) ; ׳א habituellement singulier Nb 11,21 + ; sauf après les unités, où pluriel absolu Nb 1,46 ; Jg 20,34 + (ainsi MI) ; rarement pluriel construit Ex 32,28 ; Jg 4,10 ; Jb 1,3 (deux fois dans le verset) ; (nom surtout singulier quand il suit Jg 4,10 ; 1 R 12,21 +, pourtant pluriel Jos 4,13 ; 1 S 13,5 + ; quand il précède il est pluriel 1 Ch 5,21 +, ou collectif Nb 31,33 ; 1 S 25,2 ; 1 R 8,63 ; 2 Ch 7,5 +) ; אֶלֶף אֲלָפִים 1 Ch 21,5 ; 1 Ch 22,14 ; 2 Ch 14,8 אַלְפֵי שִׁנְאָן Ps 68,18, comparer שִׁנְאָן ; בְּהַרֲרֵי אָ֑לֶף Ps 50,10, lire אֵל comparer Ps 36,7, Ol Bi Che ; pourtant voir Hup De. —
Note. 10 000 = עֲשֶׂרֶת אֲלָפִים etc. Jg 1,4 ; Jg 3,29 ; 1 R 5,28 ; 1 Ch 29,7 + ; moins souvent רִבּוֺ, רְבָבָה etc. q. v.
2. un millier, une compagnie de 1000 hommes, comme unis sous un supérieur, ou chef, d'où (שָׂרֵי) שַׂר׳א Ex 18,21 ; Ex 18,25 comparer Nb 31,14 + ; comparer יִשְׂרָאֵל׳רָאשֵׁי א Nb 1,16 ; Nb 10,4 ; Jos 22,21 ; Jos 22,30 et voir ci-dessous ; comparer aussi 1 S 29,2 surtout famille, etc. Jg 6,15 ; 1 S 10,19 (|| שֵׁבֶט et comparer מִשְׁפַּחַת 1 S 10,21) ; comparer Mi 5,2 — Nb 10,36 (רִבְבוֺת אַלְפֵי יִשְׂרָאֵל) Nb 31,5 ; Jos 22,14 ; 1 S 23,23 montrent apparemment la transition vers cet usage technique.
