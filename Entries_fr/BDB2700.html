BDB2700 [H2404]

hébreu biblique

I. [חָטַב] verbe couper ou ramasser du bois, habituellement bois de chauffage (hébreu tardif id. ; arabe bois de chauffage, ramasser du bois de chauffage (voir Wetzst dans DePs 144,12) ; éthiopien couper (ou ramasser) bois de chauffage) —

Qal Imparfait יַחְטְבוּ Ez 39,10 Infinitif לַחְטֹב Dt 19,5 Participe חֹטֵב Dt 29,10 pluriel חֹטְבִים 2 Ch 2,9 construit חֹטְבֵי Jos 9,21 + 3 t. ; passif pluriel féminin חֲטֻבוֺת Pr 7,16 — couper ou ramasser du bois, suivi d'accusatif Dt 19,5 ; Dt 29,10 (|| שׁאב מים) ; comparer Ez 39,10 מןהֿיערים (objet dans la proposition précédente) ; 2 Ch 2,9 (|| כֹּרְתֵי הָעֵצִים) ; Participe construit עצים ׳ ח Jr 46,22, et (|| שֹׁאֲבֵי מַיִם) Jos 9,21 ; Jos 9,23 ; Jos 9,27.
Pu`al Participe מְחֻטָּבוֺת Ps 144,12 sens douteux ; peut-être taillé (ainsi, en substance, Thes MV SS Ew Che Bae ; voir cependant II. חטב), de piliers d'angle ( זָוִיּוֺת voir זָוִית ci-dessus), comparaison de filles belles et majestueuses.
