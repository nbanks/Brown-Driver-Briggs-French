BDB3126 [H2738]

hébreu biblique

חָרוּל nom [masculin] une sorte de mauvaise herbe, peut-être pois chiche (cicercula), voir Löwp. 153 (syriaque fourrage pour chevaux ; ᵑ7 Pr 24,31 donne חוּרְלָא) — absolu ׳ ח Jb 30,6 ; So 2,9 pluriel חֲרֻלִים Pr 24,31 — poussant en terre dévastée, collectif So 2,9 dans la vigne du paresseux, pluriel ׳ כִּסּוּ פניו ח Pr 24,31 (|| קִמְּשׂגִים) ; comme seul abri de certains peuples rejetés Jb 30,7 (שִׂיחִים).
