BDB5110 [H5155]

hébreu biblique

[נְחִילָה] nom féminin sens inconnu ; seulement au pluriel dans l'expression אֶלהַֿנְּחִילוֺת Ps 5,1 (titre) ; la plupart conjecturent = נְחִלּוֺת √חלל = חָלִיל flûte ; voir contre cela BaeEinl. xi, qui pense (comparer ᵐ5 ὑπὲρ τῆς Κληρονομούσης = אֶלהַֿנֹּחֶלֶת [comparer ᵑ9 ], Jérôme [ Aq Symm] pro hereditatibus = אֶלהַֿנְּחָלֹת) peut-être désignation d' une mélodie.
