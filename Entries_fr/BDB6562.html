II. פּוּק verbe ```

Hiph`il faire sortir, fournir, promouvoir (hébreu tardif id. ; apparemment apparenté à l'araméen נְפַק sortir, Aph`el faire sortir, produire) ; —
1. produire, fournir : Imparfait (jussif) וְתָפֵק לָרָעֵב Es 58,10 avec נַפְשְׁךָ (lire peut-être לַחְמְךָ voir CheHpt) ; Participe מְפִיקִים מִזַּן אֶלזַֿן Ps 144,13 (voir זַן ).
2. faire sortir, susciter, obtenir, avec accusatif ; יָפִיק Pr 3,13 , + מִן personne, וַיָּפֶק Pr 8,35 ; Pr 18,22 (tous || מצא), Pr 12,2.
3. זְמָמוֺ אַלתָּֿפֵק Ps 140,9 son dessein, ne le fais pas aboutir.
