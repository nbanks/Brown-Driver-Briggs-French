BDB7476 [H7068]

hébreu biblique

קִנְאָה nom féminin ardeur, zèle, jalousie (de la couleur produite au visage par une émotion profonde) ; — absolu ׳ ק Nb 5,14 + ; construit קִנְאַת Es 9,6 suffixe קִנְאָתִי Nb 25,11 +, etc. ; pluriel קְנָאֹת Nb 5,15 ; Nb 5,18 ; Nb 5,25 ; Nb 5,29 —

1. ardeur de jalousie de mari Pr 6,34 ; Pr 27,4 ׳ רוּחַ ק disposition jalouse Nb 5,14 (deux fois dans le verset) ; Nb 5,30 ( P ) ; offrande pour la jalousie, ׳ מנחת ק Nb 5,15.18.25 ( P ) ; ׳ תּוֺרַת הק Nb 5,29 ( P ) ; de la rivalité Qo 4,4 ; Qo 9,6 Éphraïm contre Juda Es 11,13 amour ardent, || אחכה Ct 8,6.
2. ardeur du zèle :
a. des hommes pour Dieu Nb 25,11 (deux fois dans le verset)( P ) 2 R 10,26 pour la maison de ׳ י Ps 69,10.
b. de Dieu pour son peuple, surtout dans la bataille Es 42,13 ; Es 63,15 ; Za 1,14 ; Za 8,2 ׳ מְעִיל ק Es 59,17 תעשׂה וצת ׳ י ׳ ק Es 9,6 ; Es 37,32 = 2 R 19,31 .
3. ardeur de colère :
a. des hommes contre les adversaires Ps 119,139 ; Jb 5,2 (|| כעשׂ), Pr 14,30 (opposé à לֵב מַרְפֵּא).
b. de Dieu contre les hommes, || חֵמָה Ez 5,13 ; Ez 16,38 ; Ez 16,42 ; Ez 23,25 ; Ez 36,6 || עֶבְרָה Ez 38,19 || אַף Dt 29,19 ; Ez 35,11 + אֵשׁ Es 26,11 ; Ez 36,5 ; So 1,18 ; So 3,8 ; Ps 79,5 הַמַּקְנֶה ׳ סֵמֶל הַק Ez 8,3 l'image de colère provoquant la colère ; ׳ הק ׳ ס seul Ez 8,5.
