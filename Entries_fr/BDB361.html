BDB361 [H346]

hébreu biblique

אַיֵּה45 (forme allongée de אַי, comparer הֵן & הִנֵּה) adverbe interrogatif Où ? Gn 18,9 ; Gn 19,5 ; Gn 22,7 le terme le plus général exprimant cette idée, employé pour les personnes & les choses (mais jamais avec un verbe [contraste אֵיפֹה]) ; souvent en poésie ou style élevé, où la réponse nulle part est attendue, Es 33,18 ; Es 36,19 ; Es 51,13 ; Jr 2,28 (comparer Dt 32,37 אֵי) Jr 17,15 ; Jr 37,19, Jb 15,23 il erre à la recherche de pain אַיֵּה (disant) Où est-il ? Jb 21,28 ; Za 1,5 (אַיֵּההֵֿם) ; dans la (iron.) phrase où est ton (leur) Dieu ? Ps 42,4 ; Ps 42,11 ; Ps 79,10 ; Ps 115,2 (אַיֵּהנָֿא) Jl 2,17 rhétoriquement, d'une demande pressante Jr 2,6 ; Jr 2,8 ; Jb 35,10, ou d'un désir ardent Es 63,11 ; Es 63,15 ; Jg 6,13.
