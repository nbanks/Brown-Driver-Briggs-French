hébreu biblique

קָאַת, קָאָת nom [féminin] un oiseau, habituellement pélican (comparer ᵐ5 Lévitique, Deutéronome, Psaumes, ᵑ9 Psaumes (dérivé par Thes de [קוא], קיא (voir Köii. 1, 173, et, sur ת, Ges«GKC:80g»§ 80g), comme régurgitant la nourriture de son jabot pour ses petits) ; mais un oiseau marin est improbable dans les Psaumes, Ésaïe ; hébreu tardif קָאָת, ᵑ7 קָאתָא id.) ; — absolu הַקָּאָת Lv 11,18 = Dt 14,17, comme impur ; קָאַת (van d. H. Gi ; Baer קָאָת) Es 34,11 ; So 2,14, comme habitant des ruines ; construit קְאַת מִדְכָּר Ps 102,7, comparaison de solitude.
