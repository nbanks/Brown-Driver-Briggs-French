BDB3380 [H2963]

hébreu biblique

טָרַף verbe déchirer, lacérer, arracher (hébreu tardif id., surtout des bêtes sauvages ; araméen טָרַף déchirer, saisir, surtout des créanciers ; טְרֵיפָא chair ou animal déchiré ; arabe paître, dit du chameau) —

Qal Parfait טָרַף Jb 16,9 טָרָ֑ף Os 6,1, etc. ; Imparfait יִטְרֹף Ps 7,3 וַיִּטְרֹף Am 1,11 (mais voir ci-dessous) ; יִטְרָ֑ף Gn 49,27 אֶטְרֹף Os 5,14 ; Ps 50,22 Infinitif לִטְרֹף Ps 17,12 Participe actif טֹרֵף Jb 18,4, etc. ; — déchirer, lacérer, des bêtes sauvages, Gn 37,33 ( J ), Gn 44,28 ( J ; infinitif absolu avec Pu`al q. v.), Ex 22,12 ( J E ; infinitif absolu avec Niph`al q. v.) Ailleurs seulement en comparaison et métaphore ; comparaison de la férocité de Gad Dt 33,20 (comme une lionne, poème) ; ainsi de Benjamin Gn 49,27 (comme un loup, poème) ; du reste de Jacob, comme un jeune lion parmi les brebis, רָמַס וְטָרַף Mi 5,7 des méchants, Ps 17,12 (lion), Ps 7,3 objet נַפְשִׁי (comme un lion) ; Ps 22,14 les ennemis du psalmiste comme un lion (|| שֹׁאֵג ) ; Ez 22,25 (טֹרֵף טָ֑רֶף), les princes comme des loups Ez 22,27 (id. ) ; métaphore des princes d'Israël, comme un jeune lion Ez 19,3 ; Ez 19,6 (avec טֶרֶף accusatif de sens apparenté avec le verbe) ; du roi de Ninive Na 2,13 (comme un lion) ; du traitement de Dieu envers les méchants Os 5,14 (comme un lion), Ps 50,22 sujet la colère de Dieu (אַף) conçue comme assaillant Job, Jb 16,9 sa colère déchire et me persécute ; comparer טָרָ֖ף וְיִרְפָּאֵנוּ Os 6,1 sujet ׳ י (|| יַךְ וְיַחְבְּשֵׁנוּ ; sur les temps voir Dr§ 84 β ; 171, 174 n. 1) ; sujet la colère d'Édom (absolu) Am 1,11, mais lire peut-être וַיִּטֹּר pour וַיִּטְרֹף et il garda sa colère à perpétuité, ainsi Ol sur Ps 103,9 We ; טֹרֵף נַפְשׁוֺ בְּאַפּוֺ, dit de Job par Bildad Jb 18,4.
Niph`al Imparfait d'un animal déchiré (par des bêtes sauvages) טָרֹף יִטָּרֵף Ex 22,12 ( J E ) ; יִטָּרֵף aussi dans la prédiction du jugement sur les habitants de Jérusalem Jr 5,6.
Pu`al Parfait טָרֹף טֹרַף יוֺסֵף Gn 37,33 ( J ) Joseph a certainement été déchiré en pièces ; comparer טֹרָ֑ף ׳ ט Gn 44,28 ( J ).
Hiph`il Impératif masculin singulier suffixe, noter surtout 2e accusatif לֶחֶם , הַטְרִפֵנִי לָחֶם חֻקִּי Pr 30,8 laisse-moi dévorer mon pain assigné (des hommes, tardif ; comparer טֶרֶף Pr 31,15, et Ps 111,5 ; Ml 3,10).
