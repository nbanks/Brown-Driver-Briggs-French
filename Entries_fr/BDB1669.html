hébreu biblique

[גָּלַע] verbe exposer, mettre à nu (hébreu tardif Piel divulguer, faire connaître ; comparer arabe (Frey) a. exuit (vestem), removit (praeputium) ; nuda fuit (femina) ; b. patuit (os) labiis non tegentibus dentes ; impudica fuit (mulier) ; comparer aussi syriaque circoncire (dans les lexiques) ; éthiopien cortex, crusta, testa ; GrMonatsschr. 1884, 24 f. ; Schult Thes De Pr 17,14 Fl dans De, Now Str et la plupart font dériver le sens hébreu de l'arabe b ; montrer les dents, puis grogner, quereller (comparer arabe conjugaison III), mais voir Grl.c.) —
