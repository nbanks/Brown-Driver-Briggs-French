BDB7970 [H7430]

hébreu biblique

רָמַשׂ verbe ramper, se mouvoir légèrement, se déplacer (surtout P ) (arabe toucher doucement (Kam Frey); hébreu tardif רְמָשִׂים reptiles (rare)) ; —

Qal Imparfait 3 féminin singulier תִּרָמשֹׁ Gn 9,2 + 2 t. ; Participe רֹמֵשׂ Gn 1,26 + ; féminin singulier רֹמֶ֫שֶׂת Lv 11,46 + 2 t. ; —
1. sujet sol, הָאֲדָמָה ׳ כֹּל אֲשֶׁר ת tout ce dont le sol fourmille (grouille), c.-à-d. tous les reptiles, Lv 20,55 ( H ), Gn 9,2 ( P ).
2. ailleurs sujet animal :
a. ramper, עַלהָֿאֲדָמָה Dt 4,18; Gn 7,8 ( P ), עַלהָֿאָרֶץ Lv 11,44 ( H ), Gn 1,30 ( P ); הָרֶמֶשׂ הָרֹמֵשׂ עַלהָֿאֲדָמָה Ez 38,20 , עַלהָֿאָרֶץ Gn 1,26; Gn 7,14; Gn 8,17 (tous P ).
b. se mouvoir légèrement, glisser, des animaux aquatiques, כָּלנֶֿפֶשׂ ( בַּמַּיִם) הַחַיָּה הָרֹמֶצֶת Lv 11,46 ( H ), Gn 1,21 ( P ), comparer Ps 69,35.
c. en général se déplacer , de tous les animaux terrestres, עַלהָֿאָרֶץ Gn 1,28 ; Gn 7,21 ; Gn 8,19 (tous P ); en particulier des bêtes sauvages rôdant la nuit Ps 104,20.
