hébreu biblique

מֹר, מוֺר nom masculin Ct 5,5 myrrhe (du goût amer ; assyrien murru MeissnSuppl. 60 ; arabe ; comparer LagBN 40) ; — absolu מֹר Ps 45,9 + 5 t., מוֺר Ct 4,6 + 3 t. Cantique ; construit מָרֿ Ex 30,23 suffixe מוֺרִי Ct 5,1 — myrrhe (tardif), une gomme d'Arabie, exsudant de l'écorce d'un arbre, Balsamodendron Myrrha (voir SigismundAromata 12 s. RiHWB sous le mot) : — comme coulante, c.-à-d. fine, de choix (soigneusement préparée par pression et mélange, comparer Di Ex 30,23 RiHWB) : מָרדְּֿרוֺר Ex 30,23 (voir I. דְּרוֺר ), ingrédient de l'huile sacrée ; מוֺר עֹבֵר Ct 5,5 ; Ct 5,13 (comme parfum) ; comparer (comme parfum) Ct 1,13 ; Ct 4,14 ; Ct 5,1 ; Ct 5,5 הַרהַֿמּוֺר Ct 4,6, c.-à-d. où on la récolte ; comme parfum aussi Ps 45,9 ; Pr 7,17, comme encens מְקֻטֶּרֶת מֹר וּלְבֹנָה Ct 3,6 שֶׁמֶן הַמּוֺר comme onguent Est 2,12.
