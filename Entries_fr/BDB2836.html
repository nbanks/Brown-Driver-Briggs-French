BDB2836 [H2529]

hébreu biblique

חֶמְאָה nom féminin lait caillé; — ׳ ח Gn 18,8 + 6 t. ; = חֵמָה Jb 29,6 construit חֶמְאַת Dt 32,14 lait caillé, lait fermenté, moderne labben ׳ מִיץ חָלָב יוֺצִיא ח Pr 30,33, ailleurs joint à חָלָב Gn 18,8( J ) Dt 32,14 (poème ; חֶמְאַת בָּקָר וַחֲלֵב צאֹן), Jg 5,25 (où servi בְּסֵפֶל אַדִּירִים, voir ספל ) ; avec דְּבַשׁ 2 S 17,29; Es 7,15; Es 7,22 (deux fois dans le verset) ; comparer ׳ נַחֲלֵי דבשׁ וח figuré de l'abondance Jb 20,17 בִּרְחֹץ חֲלִיכַי בְּחֵמָ֑ה Jb 29,6 (|| פַּלְגֵישָֽֿׁמֶן ׃ צוּר יָצוּק).

Sur la façon de faire le lebben, en secouant le lait dans une outre de lait aigre, voir DoughtyArab. Des. i. 221, 263 ; ii. 304, etc. ThomsonLand and Book ii. 418 Voir aussi ARSKEncy. Bib. MILK.
