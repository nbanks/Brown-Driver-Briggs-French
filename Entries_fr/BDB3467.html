(une autre cause doit donc être trouvée pour le malheur d'Israël), Jr 2,31 ; Jr 8,5 ; Jr 8,19 ; Jr 8,22 ; Jr 14,19 ; Jr 22,28 ; Jr 49,1, comparer Jr 30,6 ; Jb 3,12 ; Jb 18,8 (jamais dans les Psaumes).
