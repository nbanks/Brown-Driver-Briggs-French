BDB6211 [H6037 H6038]

hébreu biblique

עֲנָוָה nom féminin humilité : — Pr 15,33 + 3 t. ; עַנְוָהֿ (contracté à cause du Maqqeph, voir BrMp) Ps 45,5 עֶנְוָָֽתְךָ Ps 18,36 (= עֲנֹתְךָ 2 S 22,36 voir I. ענה ; lire עֶזְרָֽתְךְ Ol We) ; —

1. humilité, douceur Ps 45,5 (douteux), Pr 15,33 ; Pr 18,12 ; Pr 22,4 ; So 2,3 (probablement glose, voir We Now).
2. condescendance Ps 18,36 (douteux, voir ci-dessus).
