BDB2695 [H2399]

hébreu biblique

חֵטְא nom masculin Dt 15,9 péché (sur la formation voir Lag BN 142) — ׳ ח Lv 19,17 + 15 t. ; suffixe חָטְאוֺ Lv 24,15 + 5 t., חֲטָאָו֯ Lm 3,39, חֶטְאָם Lv 20,20 pluriel חֲטָאִים Qo 10,4 construit חֲטָאֵי 2 R 10,29 ; Ez 23,49 suffixe חֲטָאַי Gn 41,9, חֲטָאָ֑י Es 38,17 ; Ps 51,11, חֲטָאֵינוּ Dn 9,16 ; Ps 103,10, חֲטָאֵיכֶם Es 1,18 —

1. péché :
a. contre l'homme Gn 41,9 ( E ) Qo 10,4.
b. ailleurs contre Dieu, absolu Es 31,7 ; Os 12,9 חָטָא ׳ ח Lm 1,8 comparer Dt 19,15 ; 2 R 10,29 משׁפט מות ׳ ח péché digne de mort Dt 21,22 = מות ׳ ח Dt 22,26 avec בְּ, à cause de, Dn 9,16 Dieu est supplié : cache ta face de mes péchés Ps 51,11 et il est dit dans la foi, tu jetteras derrière ton dos tous mes péchés Es 38,17.
2. culpabilité du péché : היה חטא ב le péché (la culpabilité du péché) vient sur quelqu'un Dt 15,9 ; Dt 21,22 ; Dt 23,22 ; Dt 23,23 ; Dt 24,15 בחטאו מת dans son péché (culpabilité) mourir Nb 27,3 ( P ) ; אישׁ בחטאו chacun pour son propre péché Dt 24,16 = 2 R 14,16 = 2 Ch 25,4 בחטא יחמתני אמי Ps 51,7 dans le péché (condition de péché et de culpabilité) ma mère m'a conçu ; אם חטאיכם כשׁנים Es 1,18 quand vos péchés seraient comme l'écarlate (en culpabilité) ; לא כחטאינו עשׂה לנו Ps 103,10 ce n'est pas selon nos péchés (leur culpabilité) qu'il nous a traités.
3. châtiment du péché : ׳ נשׂא ח porter le péché (son châtiment) Lv 20,20 ; Lv 24,15 ( H ), Nb 9,13 ; Nb 18,22 ( P ) ; avec עַל, porter le péché à cause de Lv 19,17 ; Lv 22,9 ( H ) Nb 18,32 ( P ) ; porter les péchés des idoles (en adorant les idoles) Ez 23,49 le serviteur idéal de Yahvé a porté les péchés de beaucoup Es 53,12 sans נשׂא, seulement Lm 3,39 גבר על חטאו (pourquoi) un homme (se plaindrait-il) du châtiment de son péché ?
