BDB1461 [H1396]

hébreu biblique

גָּבַר verbe être fort, puissant (hébreu tardif id. araméen גְּבַר ; arabe (conjugaison I. dialecte ; habituellement en conjugaison dérivée) contraindre, forcer ; comportement autoritaire, contrainte ; éthiopien : I, 1, subigere ; II, 2, cogere ; syriaque jouer l'homme, est dénominatif de homme) —

a. absolu p. ex. ennemis Ex 17,11 (deux fois dans le verset) ( E ) 1 S 2,9 ; Lm 1,16, eaux Gn 7,18 ; Gn 7,19 ; Gn 7,20 ; Gn 7,24 ( P ), puissance Jr 9,2
