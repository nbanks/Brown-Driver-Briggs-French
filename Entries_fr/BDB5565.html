סִינַי nom propre [d'une montagne] Sinaï ; — nom de la montagne du don de la Loi dans J et surtout P (חֹרֵב dans E et surtout D ) : Jg 5,5 ; Ps 68,9 ; Dt 33,2 (poème), aussi Ex 16,1 ( P ) ; habituellement הַרסִֿינַי (סִֿינָ֑י Lv 7,38 +), rarement J , Ex 19,20 ; Ex 19,23 et peut-être Ex 34,2 ; Ex 34,4, ailleurs surtout P , Ex 24,16 ; Ex 31,18 ; Ex 34,29 ; Ex 34,32 ; Lv 7,38 ; Lv 25,1 ; Lv 27,34 et Lv 26,46 ( H ), Nb 3,1 ; Nb 28,6, aussi Ne 9,13 מִרְבַּר סִינַי (סִינָ֑י Nb 1,19 +), Ex 19,1 ; Ex 19,2 ; Lv 7,38 ; Nb 1,1 ; Nb 1,19 ; Nb 3,4 ; Nb 3,14 ; Nb 9,1 ; Nb 9,5 ; Nb 10,12 ; Nb 26,64 ; Nb 33,15 ; Nb 33,16 (tous P ) ; — au vu de ces faits, et de l'emploi par E de חֹרֵב (q. v.), il est probable que Ex 19,11 ; Ex 19,18 (dans les E passages) proviennent de R. — ᵐ5 Σ(ε)ινα. Contre les deux hypothèses, l'exploitation de mines égyptiennes dans la péninsule jusqu'à la XIXe dynastie (voir EbGS 135, 148 ss., 159, etc.), impliquant la présence de soldats sur place.
