I. סִיר nom masculin Jr 1,13 et féminin 2 R 4,38 marmite (probablement mot étranger (égyptien ?), arabe une grande jarre à eau Lane1276 est peut-être un mot emprunté ; sur les dérivés grecs σιρός, σίρινος voir LewyFremdw. 107) ; — ׳ ס absolu 2 R 4,38 +, construit Ex 16,3 + ; pluriel סִירֹת Ex 38,3, סִירוֺת 1 R 7,45 + ; suffixe סִירֹתָיו Ex 27,3, סִירֹתֵיכֶם Ps 58,10 — marmite :

1.
a. ustensile domestique pour faire bouillir 2 R 4,38 ; 2 R 4,39 ; 2 R 4,40 ; 2 R 4,41 (deux fois dans le verset) הַבָּשָׂר ׳ ס Ex 16,3 ( P ), Za 14,21 נָפוּחַ ׳ ס Jr 1,13 (comparer נפח dans la vision), symbole de Jérusalem Ez 11,3 ; Ez 11,7 ; Ez 11,11 ; Ez 24,3 ; Ez 24,6 figuré aussi Ps 58,10 (comparer Che); en comparaison Mi 3,3 ; Jb 41,23 ; Qo 7,6.
b. רַחְצִי ׳ ס Ps 60,10 ma marmite à laver, Ps 108,10.
2. marmites utilisées dans le sanctuaire, Ez 27,3 ; Ez 38,3 (les deux P ), 1 R 7,40 (lire סִירוֺת pour ᵑ0 כִּיִּרוֺת, ainsi manuscrits de HCT ᵐ5 ᵑ9 Th Ke Klo Kmp Benz), 1 R 7,45 = 2 Ch 4,11 ; 2 Ch 4,16 ; 2 R 25,14 = Jr 52,18 , aussi Jr 52,19 ; 2 Ch 35,13 ; Za 14,20. — Ici appartient probablement סִרָה dans בּוֺר הַסִּרָה.
