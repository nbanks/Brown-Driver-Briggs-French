I. כַּפְתּוֺר כַּפְתֹּר nom masculin

1. chapiteau,
2. bouton, bulbe
( 2 peut-être le sens primitif ; HoffmZAW iii. (1883), 124 , comparer syriaque poire (LöwNo. 153), d'après la forme) ; —
1. כַּפְתּוֺר Am 9,1 chapiteau de colonne ; ainsi pluriel suffixe כַּפְתֹּרֶיהָ So 2,14.
2. כַּפְתֹּר, bouton ou bulbe, ornement sur le chandelier d'or dans le tabernacle ; ᵐ5 σφαιρωτήρ, JosArchaeology iii, 6, 7 ρὁΐσκος petite grenade ; ᵑ9 . sphaerula : — Ex 25,33 (deux fois dans le verset) ; Ex 25,35 (3 t. dans le verset) = Ex 37,19 (deux fois dans le verset) ; Ex 37,21 (3 t. dans le verset) ; suffixe כַּפְתֹּרֶיהָ Ex 25,31 ; Ex 25,34 = Ex 37,17 ; Ex 37,20, כַּפְתֹּרֵיהֶם Ex 25,36 = Ex 37,22 (tous P ).
