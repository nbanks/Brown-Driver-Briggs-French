hébreu biblique

[פָּזַר] verbe disperser (habituellement considéré comme forme secondaire de בזר q. v. ; hébreu tardif id. Pi`el (opposé à כנס = כנשׁ ), Niph`al ; arabe déchirer, fendre, aussi séparer, disperser) ; —

Qal Participe passif féminin שֶׂה פְזוּרָה Jr 50,17 une brebis dispersée est Israël (c.-à-d. chassée, isolée).
Niph`al Parfait 3 pluriel נִפְזְרוּ Ps 141,7 nos os sont dispersés לִפִי שְׁאוֺל.
Pi`el Parfait 3 masculin singulier פִּזַּר Ps 53,6 ; Ps 112,9, etc. ; Imparfait 3 masculin singulier יְפַזֵּר Ps 147,16 2 féminin singulier וַתְּפַזְּרִי Jr 3,13 Participe מְפַזֵּר Pr 11,24 — disperser Ps 89,11 ; Ps 53,6 ; Ps 147,16 (tous sujet ׳ י), Jl 4,2 objet דְּרָכַיִךְ Jr 3,13, d' Israël courant çà et là vers des dieux étrangers ; absolu de dépenser de l'argent Ps 112,9 (aumônes), Pr 11,24.
