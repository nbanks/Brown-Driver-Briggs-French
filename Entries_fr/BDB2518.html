BDB2518 [H2270]

hébreu biblique

חָבֵר

1. adjectif uni (comme ce qui suit) ; — seulement חֲבֵרִים שְׁנֵיהֶם Ex 26,3 (deux fois dans le verset) ; Ex 26,6 (du tabernacle) ; חֲבֵרוֺת Ex 26,3 (deux fois dans le verset) Ex 26,9 וְהַבָּתִּים חֲבֵרִים אֶלאֶֿחָיו 1 Ch 4,11 où Ges1 Ch 4,11 l'explique comme שְׁאֵר בֵּיתוֺ (caeteri domestici ejus). au Qal, Jäger prend cela comme frères par alliance de Caleb (par le mariage de sa sœur Chelubah avec Shuhah 1 Ch 4,11, comparer v1 Ch 4,5) ; le verset est corrompu.
2. nom [masculin] associé, compagnon (peut-être araméen ; ainsi hébreu tardif, syriaque ; arabe ami, celui qui ressemble, assyrien ‡ibru voisin DlProl 85, puis ‡ubru ami, compagnon, associé en affaires ZimBP 23. 28) ; — חָבֵר Pr 28,24 + 5 t. ; חֲבֵרוֺ Jg 20,11 + 6 t. ; pluriel חֲבֵרִים Jb 40,30 + 3 t. ; —
a. associé d'un autre, compagnon = רֵעַ Qo 4,10 ; Es 1,23 (de voleurs) ; associé d'un maître Pr 28,24 ; associé, compagnon en affaires = רֵעַ, un contre l'autre Jb 40,30 (|| כְּנַעֲנִים).
b. = אָח 2 S 2,16 (ainsi de l'un des deux qui s'affrontaient en combat singulier) ; d'Israélites unis contre Benjamin Jg 20,11 (|| אֶחָד).
c. compagnon, camarade Ps 45,8 ; Pr 29,24 (en criminalité) ; חֲבֵרֵי קֶסֶם Es 47,9 ; Es 47,12 est difficile, compare LagBN 64 ; camarades Ct 1,7 ; Ct 8,13 חֲבֵרִים מַקְשִׁיבִים לְקוֺלֵךְ, il est possible que cela ait le sens de d.
d. un associé charmeur (araméen , arabe , comparer LagBN 64), ou métaphore, un danseur ? de serpent Ps 58,6, compare לחשׁ en Qo 10,11 ; Jr 8,17
e. associé, allié (Genosse) Os 6,9 (de bandes de prêtres).
f. selon Löwp. 105, 318, 346 charmeur, magicien, dans Es 47,9 ; Es 47,12 (voir c. ci-dessus)
3. nom [masculin] compagnon seulement חֲבֵר בַּעַל Jg 4,17 ; Jg 4,21 (deux fois dans le verset) Jg 5,24 ; חֶבֶר חָבֵר Jg 4,11 voir ci-dessous חֶבֶר nom propre, masculin
