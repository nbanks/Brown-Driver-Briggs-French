BDB6189 [H6026]

hébreu biblique

[עָנֹג, LagBN 31] verbe être doux, délicat, raffiné (hébreu tardif id. Pi`el rendre doux, souple, vivre ou passer dans la jouissance ; arabe avoir un comportement amoureux, affecter la langueur) ; —

Pu`al Participe féminin הַמְּעֻנָּגָה Jr 6,2 délicatement élevée, figuratif de Jérusalem.
Hithpa`el
1. être de manières délicates, Infinitif construit הִתְעַנֵּג Dt 28,56 (femme, || רֹךְ).
2. prendre un plaisir exquis, Parfait 3 pluriel וְהִתְעַנְּגוּ consécutif Ps 37,11 ( עַל de chose) ; 2 masculin pluriel וְהִתְעַנַּגְתֶּם consécutif Es 66,11 ( מִן de chose) ; Imparfait 3 masculin singulier יִתְעַנָּ֑ג Jb 27,10 2 masculin singulier תִּתְעַנַּג Es 58,14, תִּתְעַנָּ֑ג Jb 22,26 (tous avec [שַׁדַּי] ׳ עַליֿ) ; 3 féminin singulier תִּתְעַנַּג Es 55,2 ( ב de chose) ; Impératif masculin singulier ׳ הִתְעַנַּג עַליֿ Ps 37,4.
3. avec עַל, en mauvais sens, se réjouir de, se moquer de, Imparfait 2 masculin pluriel עַלמִֿי תִּתְעַנָּ֑גוּ Es 57,4.
