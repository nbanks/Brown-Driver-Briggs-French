BDB5771 [H5678]

hébreu biblique

עֶבְרָה nom féminin débordement, arrogance, fureur ; — absolu ׳ ע Es 13,9 + ; construit עֶבְרַת So 1,18 + ; suffixe עֶבְרָתִי Es 10,6 +, etc. ; pluriel עֲבָרוֺת Jb 21,30 construit עַבְרוֺת Ps 7,7 ; Jb 40,11 —

1. débordement, excès, emportement, זָדוֺן ׳ ע Pr 21,24 excès d'insolence ; עַבְרוֺת אַפֶּ֑ךָ Jb 40,11 emportements de ta colère.
2. arrogance, de Moab Es 16,6 ( + גַּאֲוָה), גַּא׃ן d'où Jr 48,20 (+ id. Jr 48,29).
3. rage débordante, fureur :
a. des hommes, Gn 49,7 (poème dans J ), Am 1,11 (tous deux || אַף), Es 14,6 ; Pr 14,35, שֵׁבֶט עֶבְרָתוֺ Pr 22,8 (c.-à-d. verge maniée par lui dans la fureur, voir aussi Lm 3,1 ci-dessous), עַבְרוֺת צוֺרְרָ֑י Ps 7,7 les emportements de fureur de mes ennemis
b. de ׳י Os 5,10 ; Os 13,11 (|| אַף), Ha 3,8 (|| id.), Es 9,18 ; Lm 2,2 ; Ps 90,9 ; Ps 90,11 + חֲרן אַף Es 13,9, || id. Es 13,13 ; Ps 78,49 (+ זַעַם), Ps 85,4 אֵשׁ עֶבְרָתִי Ez 21,36 (|| זַעַם), Ez 22,31 (|| id.), Ez 22,21 ; Ez 38,19 שֵׁבֶט עֶבְרָתוֺ Lm 3,1 (comparer Pr 22,8 ci-dessus) ; עַם עֶבְרָתִי Es 10,6 (c.-à-d. objet de ma rage), comparer דּוֺר עֶבְרָתוֺ Jr 7,29 ׳ יוֺם עֶבְרַת י So 1,18 jour de ׳ י's fureur (jugement à venir), ainsi Ez 7,19 (supprimer Co Berthol, d'après ᵐ5 , comme glose tirée de Sophonie) ; ainsi יוֺם עֶבְרָה So 1,15 ; Pr 11,4 יוֺם עֲבָרוֺת Jb 21,30 comparer עֶבְרָה Pr 11,23.
