hébreu biblique

הֵ֫מָּה et הֵם (sans distinction appréciable dans l'usage, sauf probablement dans la mesure où la forme plus longue ou plus courte s'adaptait mieux au rythme de phrases particulières : dans l'ensemble הֵמָּה (seul) est un peu plus fréquent que הֵם, הָהֵם au contraire se dit régulièrement, הָהֵמָּה n'apparaissant que 12 t., à savoir Nb 9,7 ( P ), Jr 14,15 ; Za 14,15, et dans l'expression בַּ(הַ)מִָים הָהֵמָּה 2 R 18,4 ; Jr 3,16 ; Jr 3,18 ; Jr 5,18 ; Jr 50,4 ; Jl 3,2 ; Jl 4,1 ; Za 8,23 ; Ne 13,15, bien que בַּיָּמִים הָהֵם soit beaucoup plus courant) pronom 3 pluriel masculin (הֵמָּה employé incorrectement pour le féminin Za 5,10 ; Rt 1,22 ; Ct 6,8), féminin הֵ֫נָּה [hébreu tardif aussi הֵן, qui n'apparaît cependant en hébreu biblique qu'après un préfixe : voir ci-dessous 8], pronom [3 pluriel] ils, elles (araméen biblique הִמּוֺ , q. v. ; , ; éthiopien (une forme) 'em¥n-t¥ , em¹n-t¥ ; comparer aussi pour la forme dissyllabique, le suffixe -œm¥ [féminin -œn ] ; peut-être aussi assyrien suffixe šunu , šina [ ni ], comparer références ci-dessous הוּא). Dans l'usage, pas différent en général de הוּא, הִיא (voir ces mots, sous les rubriques correspondantes, pour des explications plus complètes), ainsi : —

1.
a. Gn 6,4 ; Ex 5,7 הֵם יֵָֽלְכוּ qu'eux (emphatique) aillent ramasser de la paille, Nb 31,16 ; Es 24,14 ; Os 4,14 ; Os 7,3 etc. ; après כִּי Jr 5,5 ; Jr 34,7. (Pour l' usage dans les propositions circonstancielles voir Dr§ 160, 169.)
b. Jg 1,22 ויעלו ֗֗֗ גַם הֵם, Dt 2,11 (אַף הֵם), Dt 3,20 ; Jos 9,4 ; Ez 30,17.
c. Ex 18,22 toute affaire mineure יִשְׁפְּטוּהֵֿם qu'ils jugent eux-mêmes, Ex 18,26 ; Jr 15,19 ; Jr 17,18 ; Ps 109,28 ; Est 9,1.
d. Gn 33,6 ; Gn 44,3 ; Jg 6,5 etc.
e. comparer Ne 10,38.
2.
a. (Reprenant le sujet avec emphase) Gn 14,24 (noter les accents), Dt 1,39 ; Ps 23,4 ; Ps 27,2 (accents), Ps 37,9 ; Ps 107,24 ; Ez 36,7 ; Ez 44,15.
b. Es 30,7 ; Ez 27,13 ; Ez 27,17 ; Ez 27,21 comparer Es 44,11.
c. Nb 14,27 a Jr 27,9 ; Ez 43,19 אֲשֶׁר הֵם מִזֶּרַע צָדוֺק, Qo 4,2 ; Ne 2,13.
— הֵמָּה כִּדְוֵי לַחֻמִי Jb 6,7 lire peut-être ׳ וַהֲמָה וגו (ou זָָֽהֲמָה) c'est repoussant (√ זהס).
3.
a. Gn 3,7 ; Ex 5,8 כִּי נִרְפִּים הֵם car ils sont paresseux, Ex 14,3 ; Ex 15,23 ; Ex 29,33 etc. (l'ordre inverse rare : Nb 11,16 ; Es 61,9, comparer DePs 94,11).
b. Gn 34,21 ; Gn 34,23 ; Gn 41,26 ; Gn 48,5 שׁני בניך ֗֗֗ לִי הֵם tes deux fils . . ., ils sont à moi, Ex 32,16 + souvent : Es 41,22 הָרִאשֹׁנוֺת מָה הֵנָּה, Es 49,21 אֵלֶּה אֵיפֹה הֵם ceux-ci — où étaient-ils ? So 2,12 après אַתֶּם, de façon singulière (comparer Dr§ 198 Obs. 2).
c. Dt 20,15 אֲשֶׁר לֹאמֵֿעָרֵי הַגֹּויִם הָאֵלֶּה הֵנָּה, Jg 19,12 ; 1 R 9,20.
d. 2 R 7,10 כַּאֲשֶׁר הֵמָּה. (Pr 19,7 Kt לֹאהֵֿמָּה est probablement corrompu : voir les Commentaires)
4.
a. Pr 30,24 אַרְבָּעָה הֵם קְטַנֵּיאָֿ֑רֶץ quatre choses sont-elles, les petites de la terre, Pr 30,29 (comparer devant une proposition relative Pr 30,15 Pr 30,18 ; Pr 6,16 ) Ct 6,8 ; Es 51,19 שְׁתַּיִם הֵנָּה קֹרְאֹתַיִךְ, 1 Ch 9,26 (voir הוא 4.a.).
b. (β) Gn 21,29 מָה הֵנָּה שֶׁבַע ֗֗֗ littéralement Que sont-elles — ces sept brebis ? Za 1,9 ; Za 4,5. (γ) אֵלֶּה ֗֗֗ הֵם Gn 25,16 (= 1 Ch 1,36) Lv 23,2 ; Nb 3,20 ; Nb 3,21 ; Nb 3,27 ; Nb 3,33 ; 1 S 4,8 ; 1 Ch 8,6 ; 1 Ch 12,15 (voir Dr § 201, 3).
