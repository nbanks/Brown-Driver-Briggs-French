BDB4767 [H4603]

hébreu biblique

מָעַל verbe agir infidèlement, perfidement, mot sacerdotal ( P Esdras, Chroniques), surtout tardif (hébreu tardif id., agir infidèlement ; arabe chuchoter, médire, perfidie, fraude) ; —

Qal Parfait 3 masculin singulier מָעַל Jos 22,20 + 4 t. ; 3 féminin singulier מָֽעֲלָה Lv 5,21 ; Nb 5,12 + 13 t. Parfait ; Imparfait יִמְעַל Pr 16,10 ; 2 Ch 26,16 3 féminin singulier תִּמְעֹל Lv 5,15 ; Nb 5,27 3 masculin pluriel יִמְעֲלוּ Jos 7,1, וַיִּמְעֲלוּ 1 Ch 5,25 2 masculin pluriel תִּמְעָ֑לוּ Ne 1,8 Infinitif construit לִמְעֹל, lire ainsi pour HCT למסר Nb 31,16 Thes Di ; — agir infidèlement, perfidement, habituellement
1. מָעַל מַ֫עַל :
a. absolu בַּחֵרֶם concernant l'interdit (le péché d'Akân) Jos 7,1 ( J E ), Jos 22,20 ( P ), comparer Lv 5,15 ( P ), 2 Ch 36,14 ; Ez 14,13 ; Ez 15,8 ; Ez 18,24.
b. femme contre (בְּ) mari Nb 5,12 ; Nb 5,27 ( P ; infidélité conjugale).
c. ailleurs contre (בְּ) Dieu, Lv 26,40 ( H ), Lv 5,21 ; Nb 5,6 ; Nb 31,16 (?) Jos 22,16 ; Jos 22,31 (tous P ), 1 Ch 10,13 ; 1 Ch 28,19 ; Ez 17,20 ; Ez 20,27 ; Ez 39,26 ; Dn 9,7.
2. emploi sans מַעַל (seulement tardif) :
a. במשׁפט לא ימעל פיו Pr 16,10 que sa bouche (du roi) n'agisse pas perfidement envers la justice ( RV en jugement).
b. ailleurs contre Dieu, avec בְּ, Dt 32,51 ( P ), 1 Ch 5,25 ; 1 Ch 12,2 ; 1 Ch 26,22 ; 1 Ch 30,7 ; sans בְּ, 2 Ch 26,18 ; 2 Ch 29,6 ; Esd 10,10 ; Ne 1,8 בַּחֵרֶם 1 Ch 2,7.
