BDB5681 [H5601]

hébreu biblique

סַפִּיר nom [masculin] saphir, peut-être aussi lapis-lazuli (comparer Di Ex 24,10 NowArchaeology i. 131) (hébreu tardif id., et סַמְפִּירִינוֺן ; ᵑ7 סַפִּרִינוֺן ; ᵑ6 ; emprunt au sanskrit canipriya selon LagGes. Abh. 72 LewyFremdw. 56 comparer grec σάπφειρος) ; — absolu ׳ ס Ex 24,10 + 8 t. ; pluriel סַפִּירִים Es 54,11 ; Ct 5,14 — avec d'autres joyaux comme ornements du prince Ez 28,13 dans le pectoral du grand prêtre Ex 28,18 ; Ex 39,11 trouvé dans les mines Jb 28,6, précieux Jb 28,16 comparer Es 54,11 prenant un haut poli Lm 4,7 (au figuré) ; שֵׁן מְעֻלֶּפֶת סַפִּירִים Ct 5,14 ivoire couvert de saphirs (au figuré) ; lapis-lazuli, selon Hi BU Du et autres dans Jb 28,6 ( עַפְרֹת זָהָב לוֺ étant compris comme des cristaux étincelants de pyrite de fer dans celui-ci), mais voir Di ; le lapis-lazuli peut être visé dans ׳ לִבְנַת הַסּ Ex 24,10 ( J ) pavement de saphir (dans la théophanie), comparer ׳ אֶבֶןסֿ Ez 1,26 ; Ez 10,1.
