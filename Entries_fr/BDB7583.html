BDB7583 [H7176]

hébreu biblique

[קֶ֫רֶת] nom féminin ville, cité (poésie tardive) (comparer phénicien קרת חדשת nom propre, d'un lieu à Chypre, aussi = Carthage, et peut-être Cirta, Tigranocerta ; comparer Thes) ; — toujours קָ֑רֶת ; — de la ville de Job, Jb 29,7 indéfini Pr 8,3 ; Pr 9,3 ; Pr 9,14 ville comme corps constitué d'hommes Pr 11,11.
