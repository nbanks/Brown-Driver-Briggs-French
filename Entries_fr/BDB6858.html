hébreu biblique

פֶּ֫תִי [pour פָּתַי Lag BN 52 BaZMG xlii (1888), 353 ; NB 320] adjectif simple, peut-être au sens d'esprit ouvert ; — פֶּ֫תִי Pr 9,4 +, פֶּ֑תִי Ps 19,8 + ; pluriel פְּתָאיִם (Ges«GKC:93x»§ 93x) Ps 116,6 + 6 t. Proverbes ; פְּתָיִים Ps 119,130 ; Pr 22,3 פְּתָיִם Pr 1,22 ; Pr 1,32 — simple, comme substantif concret : ouvert à l'instruction de la sagesse ou de la folie, Pr 9,4 ; Pr 9,16 croyant toute parole Pr 14,15 manquant de עָרְמָה Pr 1,14 ; Pr 8,5 ; Pr 19,25 ayant besoin de בין Ps 119,130 , חכמה Ps 19,8 ; Pr 21,11 en sens favorable, שׁמר ׳ פתאים י Ps 116,8 ׳ י garde les simples ; mais généralement tendance au sens péjoratif ; פתאים aiment פתי Pr 1,22 héritent de אִוֶּלֶת Pr 14,18, sont facilement séduits, égarés et font marche arrière Pr 1,32 ; Pr 7,7 ; Pr 9,6 ; Pr 22,3 = Pr 27,12 ils ont besoin d'expiation Ez 45,20 (|| שׁגה).
