BDB4157 [H3727]

hébreu biblique

כַּפֹּרֶת nom propitiatoire, terme technique tardif de כּפר couvrir le péché : l'ancienne explication couvercle, couvercle n'a aucune justification dans l'usage ; ᵐ5 ιλαστήριον ; seulement P : Ex 25,17 ; Ex 25,18 ; Ex 25,19 ; Ex 25,20 (deux fois dans le verset) ; Ex 25,21 ; Ex 25,22 ; Ex 26,34 ; Ex 30,6 ; Ex 31,7 ; Ex 35,12 ; Ex 37,6 ; Ex 37,7 ; Ex 37,8 ; Ex 37,9 (deux fois dans le verset) ; Ex 39,35 ; Ex 40,20 ; Lv 16,2 (deux fois dans le verset) ; Lv 16,13 ; Lv 16,14 (deux fois dans le verset) ; Lv 16,15 (deux fois dans le verset) ; Nb 7,89, et 1 Ch 28,11 c'était une plaque d'or de 2 1/2 coudées x 1 1/2 coudée placée au sommet de l'arche du témoignage. Sur elle, et en faisant partie, se trouvaient deux chérubins d'or face à face, dont les ailes étendues se rejoignaient au-dessus et constituaient le trône de Yahvé. Quand le grand prêtre entrait dans le Saint des Saints au jour de l'expiation, il était nécessaire que ce lieu suprême de l'expiation soit enveloppé dans un nuage d'encens. Le sang du sacrifice pour le péché de l'expiation était alors aspergé sur la face et sept fois devant lui. Le temple proprement dit, distinct du porche etc., était appelé ׳ בֵּית הַכּ 1 Ch 28,11.
