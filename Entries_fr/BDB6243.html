BDB6243 [H6064]

hébreu biblique

[עָנַשׁ] verbe dénominatif (Gerber61 s.) infliger une amende, frapper d'amende ; —

Qal Parfait consécutif וְעָֽנְשׁוּ Dt 22,19 (2 accusatif) ; Imparfait וַיַּעֲנשׁ 2 Ch 36,3 (2 accusatif ; || 2 R 23,33 ci-dessus) ; Infinitif construit עֲנוֺשׁ Pr 17,26 ( לְ personne) ; = punir (en général) עֲנָשֿׁ Pr 21,11 Participe passif עֲנוּשִׁם Am 2,8 ceux frappés d'amende, taxés.
Niph`al Imparfait + Qal Infinitif absolu עָנוֺשׁ יֵעָנֵשׁ Ex 21,22 ( E ) il sera strictement frappé d'amende ; plus généralement Parfait נֶעֱנָ֑שׁוּ Pr 22,3 être frappé d'amende, puni = Pr 27,12.
