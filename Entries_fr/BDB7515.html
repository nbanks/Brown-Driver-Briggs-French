BDB7515 [H7092]

hébreu biblique

קָפַץ verbe resserrer, fermer (hébreu tardif = hébreu biblique ; arabe rassembler, conjoindre, lier ; syriaque resserrer, contracter, retirer (comparer NöM 47: ZMG xxxiii (1879), 516); ᵑ7 קְפַץ se hâter (se replier en courant), Pi`el sautiller, bondir); —

Qal Parfait 3 masculin singulier ׳ ק Ps 77,10, etc. ; Imparfait 2 masculin singulier תִּקְפֹּץ Dt 15,7 3 masculin pluriel יִקְפְּצוּ Es 52,15 — fermer la main Dt 15,7 (avec מִן loin de, pour ne pas prêter ; opposé פתח v Dt 15,80 ) ; la bouche, d'étonnement Es 52,15, dans un silence soumis Jb 5,16 ; Ps 107,42 figuré רַחֲמָיו ׳ ק Ps 77,10 a fermé sa compassion.
Niph`al Imparfait 3 masculin singulier יִקָּֽפְצוּן Jb 24,24 si vera lectio, ils se recroquevillent, de la contraction dans la mort (comparer Di Bu ; Ol יִקָּֽבְצוּן sont rassemblés).
Pi`el Participe מְקַפֵּץ עַלהַֿגְּבָעוֺת Ct 2,8 bondissant (c.-à-d. faisant des contractions répétées du corps, en sautant) sur les collines (|| מְרַלֵּג).
