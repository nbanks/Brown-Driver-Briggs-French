BDB1663 [H1538]

hébreu biblique

גֻּלְגֹּ֫לֶת nom féminin crâne, tête, personne (dénombrement) (sur la formation comparer BaNB 205 ; araméen גּוּלְגַּלְתָּא) — absolu ׳ ג 2 R 9,35 + 3 t. ; suffixe גֻּלְגָּלְתּוֺ Jg 9,53 ; 1 Ch 10,10 pluriel suffixe גֻּלְגְּלֹתָם Nb 1,2 + 5 t. ; —

1. crâne, brisé par une pierre Jg 9,53 séparé du corps 2 R 9,35 ; 1 Ch 10,10.
2. tête, personne (dans le dénombrement, la taxation, etc. ; seulement P et tardif) ׳ עֹמֶר לַגּ Ex 16,16 un omer par personne ; comparer Ex 38,26 ; Nb 1,2 ; Nb 1,18 ; Nb 1,20 ; Nb 1,22 ; Nb 3,47 ; 1 Ch 23,3 ; 1 Ch 23,24.
