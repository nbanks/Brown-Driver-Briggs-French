hébreu biblique

רָוֶה adjectif arrosé ; — masculin singulier ׳ ר, de jardin גַּן (dans une comparaison) Jr 31,12 ; Es 58,11 féminin singulier הָרָוָה Dt 29,18 (opposé à הַצְּמֵאָה ; apparemment de la végétation, dans un proverbe, expression pour tout ) ; peut-être aussi construit רְוֵה Jb 10,15 = rassasié d'affliction (pour רְאֵה), voir [ רָאֶה ].
