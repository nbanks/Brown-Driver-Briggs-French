BDB2960 [H2611]

hébreu biblique

חָנֵף adjectif profane, irréligieux (syriaque profane, d'où souvent païen, apostat ; arabe inclinant vers un état droit, surtout la vraie religion, un musulman) — absolu ׳ ח Jb 8,13 + 10 t. ; pluriel חֲנֵפִים Es 33,14 חַנְפֵי Jb 36,13 ; Ps 35,16 — profane, impie : des personnes, ׳ אָדָם ח Jb 34,30 ׳ גּוֺי ח Es 10,6 comme substantif homme impie Es 9,16 (|| מרע) Jb 8,13 ; Jb 13,16 ; Jb 17,8 ; Jb 20,5 ; Jb 27,8 ; Pr 11,9 collectif ׳ עֲדַת ח Jb 15,34 pluriel Es 33,14 חַנְפֵילֵֿב Jb 36,13 בְּחַנְפֵי לַעֲגֵי מָעוֺג Ps 35,16 comme des profanes, moqueurs pour un gâteau (c.-à-d. bouffons de table ; sur le construit voir Ges § 130. 5), mais texte douteux (voir Checritical note Bae).
