hébreu biblique

[נָאַף] verbe commettre l'adultère (hébreu tardif id. ; araméen נְאַף (rare)) : —

Qal Imparfait 3 masculin singulier יִנְאַף Lv 20,10 (deux fois dans le verset) ; 3 masculin pluriel וַיִּנְאָ֑פוּ Jr 5,7 + 3 t. Imparfait;Infinitif absolu נָאוֺף Jr 23,14 נָאֹף Os 4,2 ; Jr 7,9 Participe נֹאֵף Pr 6,32 ; Jb 24,15 féminin singulier נֹאָ֑פֶת Lv 20,10 féminin pluriel נֹאֲפוֺת Ez 16,38 + 2 t. ; —
1. littéralement commettre l'adultère :
a. habituellement de l'homme, toujours avec la femme d'un autre ; avec accusatif femme, Lv 20,10 (deux fois dans le verset) ( H ), Pr 6,32 ailleurs absolu Ex 20,14 = Dt 5,17 (Dix Paroles), Jr 5,7 ; Jr 7,9 ; Jr 23,14 ; Os 4,2 participe masculin Lv 20,10 ( H ), Jb 24,15.
b. de femmes, seulement participe Lv 20,10 ( H ), Ez 16,38 ; Ez 23,45 (deux fois dans le verset).
2. figuré du culte idolâtre, את האבן ואת העץ Jr 3,9.

1. littéralement commettre l'adultère :
a. de l'homme, avec accusatif femme, Jr 29,23 absolu Ez 23,37 pt pluriel Jr 9,2 ; Jr 23,10 ; Ps 50,18 .
b. de la femme, absolu Os 4,13 ; Os 4,14 participe Os 3,1 ; Pr 30,20 pluriel Ez 16,32.
