BDB185 [H188]

hébreu biblique

אוֺי interjection (onomatopée ; comparer , ) malheur ! expression passionnée de chagrin et de désespoir : habituellement avec datif אוֺי לִי Es 6,5 malheur à moi ! car je suis perdu, ainsi Es 24,16 ; Jr 10,19 ; Jr 15,10 אוֺי לָנוּ malheur à nous ! 1 S 4,7 ; 1 S 4,8 ; Jr 4,13 ; Jr 6,4 אוֺינָֿא לִי Jr 4,31 ; Jr 45,3 אוֺינָֿא לָנוּ Lm 5,16. Avec la 2e ou 3e personne impliquant souvent une dénonciation ; אוֺילְֿךָ מוֺאָב Nb 21,29 (= Jr 48,46) Jr 13,27 ; Ez 16,23 répété אוֺי אוֺי לָח ; Es 3,9 אוֺי לְנַפְשָׁם Es 3,11 ; Os 7,13 ; Os 9,12 (|| שֹׁד לָהֶם). Avec un voc. (ou accusatif implicite) Ez 24,6 ; Ez 24,9 אוֺי עִיר הַדָּמִים ; absolu Nb 24,23. Employé comme substantif Pr 23,29 לְמִי אוֺי (|| לְמִי אֲבוֺי).
