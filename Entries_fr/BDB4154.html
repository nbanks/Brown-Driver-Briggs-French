BDB4154 [H3724]

hébreu biblique

I. כֹּפֶר nom masculin le prix d'une vie, rançon (ποινή, wergild) ; — ׳ כ Ex 21,30 + 10 t. ; suffixe כָּפְרְךָ Es 43,3 כָפְרוֺ Ps 49,8 —

1. prix pour la rançon d'une vie Jb 33,24 ; Jb 36,18 עַל ׳ כ Ex 21,30 (code de l'alliance ; || פִּדְיֹן נַפְשׁוֺ) ; נפשׁ ׳ כ Pr 13,8 לְ ׳ כ rançon pour Pr 6,35 ; Pr 21,18 ; Nb 35,31 ; Nb 35,32 ( P ) ; כָּפְרוֺ sa rançon Ps 49,8 ׅ || ( פדה ; כָּפְרְךָ ta rançon Es 43,3 ׅ || ( תַּחְתֶּיךָ ; כֹּפֶר seul 1 S 12,3 ; Am 5,12.
2. dans le rituel de P נַפְשׁוֺ ׳ כ Ex 30,12 est un demi-sicle du sanctuaire payé par chaque mâle au-dessus de vingt ans lors du recensement afin qu'il n'y ait pas de fléau sur eux. Il était offert à Yahvé, לְכַפֵּר על pour expier pour eux.
