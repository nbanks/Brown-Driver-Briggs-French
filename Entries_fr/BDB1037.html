Quand l'espace séparant deux objets distincts doit être indiqué, cela se fait
a. le plus souvent en répétant בין, comme Gn 13,3 בֵּין בֵּיתְאֵל וּבֵין הָעָ֑י littéralement dans l'intervalle de Béthel, et dans l' intervalle de `Aï, c.-à-d. entre Béthel et `Aï, Gn 16,14 ; Gn 17,7 ; Gn 31,50 ; Gn 31,51 + souvent
b. plus rarement par בֵּין ֗֗֗ לְ, Gn 1,6 séparant בֵּין מַיִם לָמָ֑יִם littéralement dans l'intervalle de eaux par rapport aux eaux, c.-à-d. entre les eaux et les eaux, Lv 20,25 ; Lv 27,33 ; Dt 17,8 (3 t. dans le verset) ; 2 S 19,36 ; 1 R 3,9 ; Ez 41,18 ; Ez 42,20 ; Ez 44,23 (deux fois dans le verset) ; Jon 4,11 ; Ml 3,18 (deux fois dans le verset) ; 2 Ch 14,10.
c. par בֵּין ֗֗֗ וּלְ Jl 2,17 .
d. par בֵּין ֗֗֗ לְבֵין Es 59,2. בֵּין est utilisé non seulement de la localité réelle, mais aussi avec des verbes de division (figuré) Gn 1,14 ; Lv 10,10, et de jugement, connaissance, enseignement, etc., si l'idée de distinction est impliquée, comme Gn 16,5 ; 2 S 19,36 הַאֵדַע בֵּין טוֺב לְרָע puis-je discerner entre le bien et le mal ? 1 R 3,9 ; Jon 4,11 ; Gn 31,49 (surveiller entre), Jg 11,10 (entendre), Jos 22,27 (témoigner), Es 2,4 et il jugera (arbitrera) entre les nations, Es 5,3 ; Lv 27,33 ; Ez 44,23 ; Ml 3,18 ; 2 Ch 14,10 (voir RV ) ; et dans d'autres applications métaphoriques, comme d'une alliance ou d'un signe entre deux parties contractantes, Gn 9,12 ; Gn 9,15 ; Ex 31,13 + ; ou un serment Gn 26,28 ; 2 S 21,7 inimitié ou querelle Gn 3,15 ; Gn 13,7 ; Dt 25,1 ; Pr 6,19 paix 1 R 5,26 bienveillance Pr 14,9.
