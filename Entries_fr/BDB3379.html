BDB3379 [H2958 H2962]

hébreu biblique

טֶ֫רֶם54, une fois (Rt 3,14 Kt ) טְרוֺם, adverbe [de temps], pas encore, avant que (étymologie inconnue : ne se trouve pas dans les langues apparentées) — se construit surtout avec l'imparfait (avec le parfait seulement Gn 24,15 (Gn 24,45 imparfait), 1 S 3,7 a (v:b imparfait), et בְּטֶרֶם Ps 90,2 ; Pr 8,25) : —

1. טֶרֶם dans une proposition indépendante, pas encore, Gn 2,5 et toutes les plantes des champs טרם יהיה בארץ n'étaient pas encore sur la terre, etc., Gn 19,4 טרם ישׁכבו pas encore s'étaient-ils couchés, quand etc., Gn 24,15 ; Gn 24,45 ; Nb 11,33 ; Jos 2,8 ; 1 S 3,3 ; 1 S 3,7 (deux fois dans le verset) ; du temps présent, Ex 9,30 ; Ex 10,7 הֲטֶרֶם תֵּדַע ne sais-tu pas encore, etc. ? ; dans une proposition subordonnée, avant que, Ex 12,34 ils prirent leur pâte טרם יחמץ avant qu'elle fût levée, Jos 3,1 ; Ps 119,67 du temps futur Es 65,24. Plus fréquemment
2. בְּטֶרֶם39, avec la même valeur : du temps passé, Gn 27,33 j'ai mangé de tout בטרם תבא avant que tu vinsses, Gn 37,18 ; Gn 41,50 ; Jg 14,18 ; 1 S 2,15 ; Jr 1,5 ; Jr 47,1 ; Ez 16,57 ; Ps 90,2 ; Pr 8,25 (tous deux avec le parfait) ; plus souvent du présent ou du futur, comme Gn 27,4 בטרם אמות avant que je meure (ainsi Gn 45,28, comparer Ps 39,14 ; Jb 10,21), Lv 14,36 ; Dt 31,21 ; 1 S 9,13 ; 2 R 2,9 ; Es 7,16 ; Es 42,9 ; Es 48,5 ; Jr 13,16 l'imparfait avec une valeur fréquentative, Ex 1,19 avant que la sage-femme vienne, elles ont coutume d'accoucher, Rt 3,14 ; Pr 18,13. Pléonastique. בְּטֶרֶם לֹא So 2,2 (deux fois dans le verset). Construit avec un substantif au génitif, Es 17,14 בְּטֶרֶם בֹּקֶר = avant le matin, Es 28,4 בְּטֶרֶם קַיִץ ; avec un infinitif So 2,2a (mais lire ici avec ᵐ5 We בְּטֶרֶם לאֹ תֵהְיוּן כְּמוֺץ עֹבֵר, sans יוֺם, « avant que vous deveniez comme la paille qui passe »).
3. מִטֶּרֶם Ag 2,15 (suivi de l'infinitif) מִטֶּרֶם שׂוּם אֶבֶן depuis avant qu'on eût posé une pierre sur l'autre, etc.
