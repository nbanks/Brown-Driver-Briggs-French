BDB146 [H160]

hébreu biblique

אַהֲבָה nom féminin amour (= Infinitif de אהב q. v.) — absolu ׳ א Pr 10,12 + 18 t. ; construit אַהֲבַת Jr 2,2 + 3 t. ; suffixe אַהֲבָתִי Ps 109,4 ; Ps 109,5 אַהֲבָֽתְךָ 2 S 1,26 אַהֲבָתוֺ Es 63,9 ; So 3,17 אַהֲבָתָהּ Pr 5,19 אַהֲבָתָם Qo 9,6 — amour, surtout littérature sapientiale et tardif.

1. humain (envers objet humain) absolu Qo 9,1 ; Qo 9,6 (tous deux || שִׂנְאָה) ainsi Pr 10,12 ; Pr 15,17 comparer Pr 27,5 voir aussi Pr 17,9 de l'homme envers l'homme Ps 109,4 ; Ps 109,5 amour pour soi-même (נַפְשׁוֺ) 1 S 20,17 entre homme et femme Ct 2,4 ; Ct 2,5 ; Ct 5,8 ; Ct 8,6 ; Ct 8,7 (deux fois dans le verset) ; Pr 5,19 comparer aussi 2 S 1,26 (נָשִׁים ׳ א) ; personnifié Ct 2,7 ; Ct 3,5 ; Ct 7,7 ; Ct 8,4 comparer emploi figuré Ct 3,10 de simple désir sexuel 2 S 13,15 figuré de l'amour de Jérusalem pour ׳ י Jr 2,2 ( כְּלוּלֹתַיִח ׳ א) , et de l'amour de la Jérusalem adultère Jr 2,33.
2. amour de Dieu pour son peuple Os 11,4 (׳ עֲבֹתוֺת א) Jr 31,3 ; Es 63,9 ; So 3,17.
