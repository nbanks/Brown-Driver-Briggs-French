BDB3156 [H2742]

hébreu biblique

I. חָרוּץ adjectif tranchant, diligent (sur ceci et ce qui suit voir BaNB 173 ) — חָרוּץ Es 28,27 + 11 t. ; pluriel חֲרוּצִים Pr 10,4 ; Pr 12,24 חֲרֻצִים Pr 13,4 חֲרֻצוֺת Am 1,3 —

1. tranchant : de l'instrument de battage חָדָשׁ ׳ מוֺרָג ח Es 41,15 sans ׳ מ, comme substantif, Es 28,27 (où יוּדַשׁ) ; הַבַּרְזֶל ׳ ח Am 1,3 ; Jb 41,22 (figuré du crocodile).
2. figuré diligent : comme substantif Pr 21,5 opposé à רְמִיָּךְ Pr 10,4 ; Pr 12,24 ; Pr 12,27 opposé à עָצֵּל Pr 13,4. — Dn 9,25 voir IV. חרוץ .
