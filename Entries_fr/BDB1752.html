BDB1752 [H1639]

hébreu biblique

[גָּרַע] verbe diminuer, retenir, retirer (hébreu tardif id., araméen גְּרַע raser la tête, etc. ; comparer avaler) —

Qal Imparfait יִגְרַע Jb 36,7 ; יִגְרָ֑ע Ex 21,10 2 masculin singulier תִּגְרַע Dt 13,1 + 3 t. ; אֶגְרַע Ez 5,11 (mais comparer ci-dessous) ; וָאֶגְרַע Ez 16,27 תִּגְרְעוּ Ex 5,8 + 2 t. ; Infinitif לִגְרֹעַ Qo 3,14 Participe גְּרוּעָה Es 15,2 (ainsi Baer ; v. d. H. גְּדוּעָה voir גדע ) ; גְּרֻעָה Jr 48,37 —
1. diminuer, habituellement suivi de מִן Ex 5,8 ; Ex 5,19 (le nombre de briques) ; Dt 4,2 ; Dt 13,1 (dans les deux, parole de ׳ י ; opposé à הוסיף) comparer Qo 3,14 ; et aussi Jr 26,2 (avec accusatif) ; avec accusatif aussi Ex 21,10 (nourriture etc.) comparer Ez 16,27 Participe passif diminué, coupé, de la barbe ; Jr 48,37 et Es 15,2 (ainsi Baer, à juste titre, comparer sa note et voir גדע ).
2. retenir Jb 15,4 (avec accusatif שִׂיחָה méditation, dévotion) ; Jb 15,8 (avec accusatif חָכְמָה, suivi de אֵלֶיךָ, c.-à-d. vers, pour toi-même, monopoliser ; De Di Da rendent attirer à soi, s'approprier ; Schult. comparer arabe avaler ; comparer Ew saugst zu dir ein Weisheit).
3. retirer Jb 36,7 (objet עַיִן, œil, suivi de מִן) ; Ez 5,11 absolu (si vera lectio) ; Co. lit אֶתְגָּרֶה, pour אֶגְרַע, pour des raisons internes ; d'autres : retirer (tes besoins) Sm (comparer Ez 16,27) ; retirer (mon œil) Ges Ke.
