BDB2107 [H3077 H3111]

hébreu biblique

יְהוֺיָדָע, יוֺיָדָע nom propre, masculin (׳ י connaît ; comparer יְדַעְיָה, אֶלְיָדָע et références, בְּעֶלְיָדָע) —

1. ׳ יְהוֺ ; ᵐ5 habituellement Ἰωδαε, A parfois Ἰωιαδαε (2 S 23,30 + ), ᵐ5 L Ἰωαδ ( 2 S 20,23 ὠαδδαι), père de Benayahou (au temps de David), dans la combinaison ׳ בְּנָיָהוּ בֶּןיְֿהוֺ 2 S 8,18 ; 2 S 20,23 ; 2 S 23,20 ; 2 S 23,22 ; 1 R 1,8 ; 1 R 1,26 ; 1 R 1,32 ; 1 R 1,36 ; 1 R 1,38 ; 1 R 1,44 ; 1 R 2,25 ; 1 R 2,29 ; 1 R 2,34 ; 1 R 2,35 ; 1 R 2,46 ; 1 R 4,4 ; 1 Ch 11,22 ; 1 Ch 11,24 ; 1 Ch 18,17 ; 1 Ch 27,5 יְהוֺיָדָע בֶּןבְּֿנָיָהוּ 1 Ch 27,34 est probablement une inversion erronée de l'ordre habituel ; ׳ יְהוֺ seul seulement 1 Ch 12,28 (où il est appelé הַנָּגִיד לְאַחֲרֹן).
2. ׳ יְהוֺ, ᵐ5 habituellement Ἰωδαε A parfois Ἰωιαδαε etc., (grand) prêtre à Jérusalem au temps de Joas 2 R 11,4 ; 2 R 11,9 (deux fois dans le verset) ; 2 R 11,15 ; 2 R 11,17 ; 2 R 12,3 ; 2 R 12,8 ; 2 R 12,10 ; 2 Ch 22,11 ; 2 Ch 23,1 ; 2 Ch 23,8 (deux fois dans le verset) ; 2 Ch 23,9 ; 2 Ch 23,11 ; 2 Ch 23,14 ; 2 Ch 23,16 ; 2 Ch 23,18 ; 2 Ch 24,2 ; 2 Ch 24,3 ; 2 Ch 24,6 ; 2 Ch 24,12 ; 2 Ch 24,14 (deux fois dans le verset) ; 2 Ch 24,15 ; 2 Ch 24,17 ; 2 Ch 24,20 ; 2 Ch 24,22 ; 2 Ch 24,25 ; Jr 29,26.
3. יוֺיָדָע un bâtisseur au mur de Jérusalem, avec Néhémie, Ne 3,6 ( ᵐ5 Ἰοειαα, Ἰοειδα, Ἰωδαε etc.)
4. ׳ יוֺ fils du grand prêtre Éliashib Ne 12,10 ; Ne 12,11 ; Ne 12,22 ; Ne 13,28 ( ᵐ5 Ἰωδα, Ἰωαδαε, Ἰωιαδα).
