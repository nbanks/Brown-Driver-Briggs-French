BDB5211 [H5237]

hébreu biblique

נָכְרִי adjectif étranger, aliéné (hébreu tardif נָכְרִי = Gentil) ; — ׳ נ absolu masculin So 1,8 + ; féminin נָכְרִיָּה Ex 2,22 + ; masculin pluriel נָכְרִים Es 2,6 + 2 t. + Pr 20,16 Kt (> Qr נָכְרִיָּה), + Pr 27,13 (ainsi lire pour ᵑ0 נָכְרִיָּה) ; féminin pluriel נָכְרִיּוֺת Gn 31,15 + ; —

1.
a. étranger : עַם נָכְרִי Ex 21,8 ( E ) un peuple étranger (non israélite), ׳ אִישׁ נ Dt 17,15 (|| לֹא אָתִיךָ) ; ׳ מַלְבּוּשׁ נ So 1,8 אֶרֶץ נָכְרִיָּה Ex 2,22 ( J ) terre étrangère, ainsi Ex 18,3 ( E ) ; surtout נָשִׁים נָכְרִיּוֺת femmes étrangères (non israélites) 1 R 11,18 ; Esd 10,2 ; Esd 10,10 ; Esd 10,11 ; Esd 10,14 ; Esd 10,17 ; Esd 10,18 ; Esd 10,44 ; Ne 13,26 ; Ne 13,27.
b. comme substantif נָכְרִי étranger (non israélite) Jg 19,12 (+ אֲשֶׁר לֹא מִבְּנֵי יִשְׂרָאֵל), 1 R 8,41 (+ אֲשֶׁר לֹא מֵעַמְּךָ), = 2 Ch 6,32 (+ id.), 1 R 8,43 = 2 Ch 6,33 ; Dt 14,21 (|| גֵּר), Dt 15,3 ; Dt 23,21 (opposé à אָחִיךָ), Dt 29,21 (+ אֲשֶׁר יָבאֹ מֵאֶרֶץ רְחוֺקָה) ; comme prédicat, 2 S 15,19 (|| גֹּלֶה), נָכְרִיָּה Rt 2,10 pluriel נָכְרִים étrangers Lm 5,2 ( זָרִים), Ab 1,11 (|| id.) ; comparer יַלְדֵי נָכְרִים Es 2,6 féminin pluriel femmes étrangères Gn 31,15 ( E ; c.-à-d. n'appartenant pas à la famille du père).
2. נָכְרִיָּה femme étrangère, comme terme technique, dans les Proverbes, pour prostituée (peut-être parce que les prostituées étaient à l'origine principalement des étrangères) : Pr 2,16 (|| אִשָּׁה זָרָה), Pr 7,5 (|| id. ), Pr 5,20 (|| זָרָה), Pr 6,24 (|| אֵשֶׁת רָ֑ע), Pr 23,27 (|| זוֺנָה ). — Sur Pr 20,16 ; Pr 27,13 voir ci-dessous
3. figuré inconnu, non familier : נָכְרִי הָיִיתִי בְּעֵינֵיהֶם Jb 19,15 un étranger je suis devenu à leurs yeux ; לִבְנֵי אִמִּי ׳ נ Ps 69,9 (|| מוּזָר הָיִיתִי לְאֶחָ֑י) ; ׳ אִישׁ נ Qo 6,2 du jugement de ׳ י, נָכְרִיָּה עֲבֹדָתוֺ Es 28,21 étrange est sa tâche ! (|| זָר מַעֲשֵׂהוּ) ; גֶּפֶן נָכְרִיָּה Jr 2,21 une vigne étrangère (opposé à זֶרַע אֱמֶת), figuré d'un Israël dégénéré ; comme substantif, נָכְרִים Pr 20,16 Kt (> Qr נָכְרִיָּה), étrangers, personnes inconnues de lui (|| זָר ), ainsi lire aussi (pour ᵑ0 נָכְרִיָּה) || Pr 27,13.
