hébreu biblique

[קָהַל] verbe dénominatif (Gerber107 s.) s'assembler en tant que קָהָל ; —

Niph`al Parfait 3 masculin pluriel נִקְהֲלוּ Est 9,2 + 3 t. ; Imparfait 3 masculin singulier וַיִּקָּהֵל Ex 32,1 ; Jr 26,9 3 masculin pluriel וַיִּקָּֽהֲלוּ 2 S 20,14 Qr ( Kt ויקלהו, comparer HPS), etc. ; Infinitif construit הִקָּהֵל Nb 17,7 ; Est 8,11 Participe pluriel נִקְהָלִים Ez 38,7 — s'assembler en tant que קָהָל :
1.
a. pour le conflit ou la guerre 2 S 20,14 ( Qr ; voir ci-dessus), Est 8,11 ; Est 9,2 ; Est 9,15 ; Est 9,16 ; Est 9,18 avec עַל personne Ez 38,7, accusatif de lieu Jos 22,12 ; Jg 20,1.
b. rébellion, avec עַל personne Nb 16,3 ; Nb 17,7 ; Nb 20,2 ( P ), peut-être aussi Ex 32,1 ( J ; la plupart, vers Aaron).
2. à des fins religieuses : pour dresser la tente de la rencontre, accusatif de lieu Jos 18,1 ( P ); entendre la parole de ׳ י, avec אֶל lieu Lv 8,4 ( P ), ב lieu + אֶל personne Jr 26,9 pour une fête, אֶל personne 1 R 8,2 = 2 Ch 5,3 לְ local 2 Ch 20,26.

1.
a. pour le jugement Jb 11,10 .
b. pour la guerre 1 R 12,21 = 2 Ch 11,1 ; Ez 38,13.
2. à des fins religieuses : avec accusatif אֶתהַֿקָּהָל Nb 10,7, + אֶל lieu Nb 20,10, avec accusatif (כָּלֿ)הָעֵדָה Ex 35,1 ; Nb 1,18 ; Nb 8,9 ; Nb 20,8 ( P ), + אל lieu Lv 8,3 ; Nb 16,19 ( P ) ; avec accusatif הָעָם Dt 4,10 ; Dt 31,12 כָּליֿשׂראל 1 Ch 13,5 ; 1 Ch 15,3 anciens des tribus Dt 31,28, d'Israël 1 R 8,1 = 2 Ch 5,2 princes d'Israël 1 Ch 28,1.
