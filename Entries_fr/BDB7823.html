2. sonner un signal de guerre ou de marche : avec חֲצֹצְרוֺת Nb 10,7 ; Nb 10,9 ( P ) ; || תקע שׁוֺפָר Os 5,8, comparer Jl 2,1 avec על contre 2 Ch 13,12 ainsi aussi probablement Jb 30,5 (contre un voleur).

5. crier sous l'impulsion religieuse : תְּרוּעָה ׳ הר 1 S 4,5 ; Esd 3,11 ; Esd 3,13 ainsi dans le culte public avec musique et sacrifice, avec לְ, à Dieu Ps 47,2 ; Ps 66,1 ; Ps 81,2 ; Ps 95,1 ; Ps 95,2 ; Ps 98,4 ; Ps 100,1, לפני Ps 98,6 dans la joie, תַּחְתִּיּוֺת אֶרֶץ Es 44,23 (|| תָנּוּ שָׁמַיִם).
6. crier dans la détresse, Es 15,4 (mais Du Bu יָָֽרְעוּ tremblent) ; לָמָּה תָרִיעִי רֵ֑עַ Mi 4,9 (de Sion sous figure de femme, mais peut-être lire למה תָּרוֺעִי רֹעַ pourquoi es-tu brisée en morceaux ? comme Pr 11,15).
