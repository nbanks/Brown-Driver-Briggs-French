BDB4901 [H4985 H4988]

hébreu biblique

מָתֹק verbe devenir ou être doux, agréable (hébreu tardif מתק, surtout Pi`el ; assyrien [ matâ‡u ] être doux, dans les dérivés ; éthiopien doux ; araméen sucer (avec plaisir), מְתַק id., être doux ; arabe ( devant ) voir claquer les lèvres (avec plaisir), douceur) ; —

Qal Parfait 3 masculin singulier suffixe מְתָקוֺ Jb 24,20 3 pluriel מָֽתְקוּ Jb 21,33 Imparfait 3 masculin pluriel יִמְתָּ֑קוּ Pr 9,17, וַיִמְתְּקוּ Ex 15,25 —
1. littéralement devenir doux, de l'eau (opposé à מָרִים) Ex 15,25 ( J ) ; sont (c.-à-d. ont le goût) doux, ׳ מַיִם גְּנוּבִים יִמ Pr 9,17 (figuré des délices du plaisir illicite).
2. = être agréable, מָֽתְקוּ לוֺ Jb 21,33 doux (agréables) pour lui sont les mottes de la vallée (dit de celui qui repose dans la tombe).
3. sucer (sens araméen, comparer syriaque ci-dessus) מְתָקוֺ רִמָּה Jb 24,20 le ver le suce, se repaît de lui (sur le verbe masculin comparer Ges«GKC:145»§ 145. 7 KöSynt. § 345 a DaSynt. § 113 (b)), Di De Buhl (comparer Kau Da ; aussi SS qui rendent « angenehm finden »), mais ce sens en hébreu est douteux ; texte peut-être corrompu (comparer les conjectures de Bu Du).
Hiph`il Imparfait 3 féminin singulier אִם תַּמְתִּיק בְּפִיו רָעָה Jb 20,12 si le mal donne un goût doux dans sa bouche (déclaratif Hiph`il) ; 1 pluriel אֲשֶׁר יַחְדָּו נַמְתִּיק סוֺד Ps 55,15 nous qui rendions douce (notre) intimité.
