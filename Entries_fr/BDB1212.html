II.
seigneur, spécifiquement comme nom divin, Baal.
1. sans article: במות בעל Nb 22,41 (poésie Balaam) ; בעל פעור Nb 25,3 ; Nb 25,5 ( E ) Dt 4,3 (voir ci-dessous). Ce nom divin n'est pas employé ailleurs dans l'Hexateuque. Il provient probablement du sens de possession divine, plutôt que de souveraineté (RSSem 92). Il semble à avoir été utilisé dans le Nord d'Israël = אדון dans le Sud. C'était le nom spécial du dieu des Cananéens, des Philistins, etc., = babylonien בֵּל, comparer SchrSK 1874, 335 ff. Aux époques postérieures les scribes ont substitué בּשֶׁת, dans les noms propres (ירבשׁת = ירבעל, אשׁבשׁת = אשׁבעל , voir בּשֶׁת , GeiZMG 1862, 728 ff.), et aussi dans le texte pour בעל Os 9,10 ; Jr 11,13 (d'où ἡ βάαλ Jr 2,23 ; Jr 7,9 ; Jr 11,13 ; Jr 11,17 ; Jr 19,5 ; Os 2,10 ; Os 13,1 +, Rm 11,4, voir DiBaal mit d. weib. Art1 Kel, MBA 1881, June 16 Dr 2 S 4,4).
