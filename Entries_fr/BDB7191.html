hébreu biblique

צָפַן verbe cacher, thésauriser (hébreu tardif id. ; Tel Amarna ƒapânu ; coucher, du soleil) ; —

Qal Parfait 3 masculin singulier ׳ צ Pr 27,16 : 2 masculin singulier צָפַנְתָּ Jb 10,13 +, etc. ; Imparfait יִצְפֹּן Jb 21,19 + Pr 2,7 Qr ( Kt רצפן), suffixe יִצְפְּנֵנִי Ps 27,5, etc. ; Participe actif pluriel suffixe צֹפְנֶיהָ Pr 27,16 passif צָפוֺן Pr 13,22 +, Ps 17,14 Kt ; féminin צְפוּנָךְ Os 13,12, etc. ; —
1. transitif cacher, avec accusatif de personne Ex 2,2 ( E ), Jos 2,4 ( J E ), de cacher une femme querelleuse, comme cacher le vent Pr 27,16 (deux fois dans le verset) (si vera l. ; comparer Toy) ; de ׳ י cachant ses serviteurs (du mal) Ps 27,5 ; Ps 31,21 : = thésauriser une chose, Pr 10,14, + אִתְּךָ avec toi-même, en ta propre garde Pr 2,1 = Pr 7,1, dans (ב) le cœur Jb 10,13 (sujet ׳ י ; de desseins secrets), Ps 119,11, comparer Jb 23,12 + ל personne Pr 13,22 ; Ct 7,14, sujet ׳ י 21,19 réserver le châtiment ; bénédictions Ps 31,20 ; Pr 2,7 + מִן Jb 17,4 tu [Dieu] as tenu leur cœur éloigné de l'intelligence, l'en as préservé ; passif, du péché Os 13,12 (absolu ; || צָרוּר ) ; lire peut-être צָפוּן pour צָפוּ Jb 15,22 thésaurisé pour (אֱלֵי) l'épée (voir I. צפה ) ; צָפוּן = lieu thésaurisé, chéri (c.-à-d. Jérusalem) Ez 7,22 = trésor Ps 17,14 Qr ( Kt צפינך, voir צפין ), Jb 20,26 (sur le texte douteux voir Bu) ; pluriel des thésaurisés de ׳ י, ses saints Ps 83,4.
2. intransitif se cacher, guetter, absolu Ps 56,7, avec ל personne Pr 1,11 ; Pr 1,18 et (des yeux du méchant) Ps 10,8.
