BDB4894 [H4964]

hébreu biblique

מֶ֫תֶג nom masculin bride (araméen ( ᵑ7 ) מִתְגָּא ; hébreu tardif מֶתֶג de l'accent secondaire ָֽ ; מִתֵּג Pi`el dénominatif mettre une bride) ; — ```

1. littéralement bridle, pour les animaux ; מֶתֶג לַחֲמוֺר Pr 26,3 un fouet pour le cheval, une bride pour l'âne, etc. ; וָרֶסֶן ׳ בְּמ Ps 32,9 avec bride et licou (sur le texte du v. comparer Che ) ; de la bride de ׳ י pour Sennachérib (sous la figure d'une bête), וּמִתְגִּי בִּשְׂפָתֶיךָ 2 R 19,28 et je mettrai (וְשַׂמְתִּי) mon crochet dans ton nez et ma bride à tes lèvres = Es 37,29 (voir DrIs. éd. 2, 220).
