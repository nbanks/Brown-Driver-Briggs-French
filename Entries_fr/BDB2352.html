hébreu biblique

זָכָר nom masculin mâle (phénicien זכר ; arabe mâle, vieux persan zakara , sanskrit jâtáras id., selon MV1. 87, De compare éthiopien copuler) ; — זָכָר Lv 3,1 + 43 t. ; suffixe זְכָרָם Ex 34,19 (autres formes זְכוּר et זְכַר cidessous) ; construit זְכַר Lv 27,5 + 16 t. + (זְכוּר ci-dessous) ; —

1. mâle (homme ou animal), souvent opposé à נְקֵבָה (voir נְקֵבָה ) Lv 3,1 ; Lv 3,6 ; Lv 12,7 ; Lv 27,3 (tous P ) Gn 1,27 ; Gn 5,2 ( P ), Gn 6,19 ; Gn 7,3 ; Gn 7,9 ; Gn 7,16 ( J P ), Lv 27,5 ; Lv 27,6 ; Lv 27,7 ( P ) Qo 7,28 ; Ml 1,14 ; לְכָלהַֿזָּכָר לִבְנֵי אַהֲרֹן Lv 6,11 ; Lv 6,22 ; Lv 7,6 ( P ) Nb 18,10 ( P ) ; כָּלזָֿכָר בְּכֹהֲנִים Nb 3,15 ( P ) ; כָּלזֿכָר כָּלמַֿחֲלָה Nb 31,7 ; Nb 31,17 ( P ) Jos 5,4 ( E ?); כָּלזָֿכָר לְבֵית אָבִיו Jos 17,2 ( P ), 1 Ch 23,17 compare Nb 1,2 ; Nb 1,20 ; Nb 1,22 + 2 Chr 31, 16 + 2Chr 31,19 (10 t. P ) 1 Ch 5,7 ; 1 Ch 9,9 ; זְכָרִים מִבְּנֵי Esd 8,3 + 13 t. Ezra 8 ; כָּלזָֿכָר Jg 21,11 compare Jg 21,12.
2. spécifiquement, de premier-né, parmi les humains et les animaux, וַהֲקִדִּשׁ לִי כָּלבְֿכוֺר פֶּטֶר כָּלרֿחֶם בִּבְנֵי יִשְׂרָאֵל בָּאָדָם וּבַבְּהֵמָה לִי הוּא Ex 13,2 ( E ), compare Nb 3,40 ; Nb 3,43 ( P ); פֶּטֶר רֶחֶם הַזְּכָרִים Ex 13,12 ( E J ), Ex 34,19 ( J ) Nb 3,22 ; Nb 3,28 ; Nb 3,34 ; Nb 18,15 ; Nb 26,62 ( P ).
3. locution adverbiale מִזָּכָר וְעַד נְקֵבָה Lv 27,5 ; Lv 27,6 ; Lv 27,7 ( P ) 2 Ch 31,16 ; 2 Ch 31,19.
