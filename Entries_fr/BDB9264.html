araméen biblique

[ Dn 2,4-7,28 ; Esd 4,8-6,18 ; Esd 7,12-26 ; Jr 10,11 ; Gn 31,47aβ. ]

[Note]

Note. - Dans le cas de mots communs à l'araméen biblique et à l'hébreu biblique, les données étymologiques déjà fournies avec les mots hébreux ne sont pas répétées ici. Le signe est préfixé à tous les mots propres à l'araméen biblique. Dans les autres cas, sauf quand ᵑ7 ou le syriaque est mentionné seul, il faut comprendre que les mots se trouvent aussi dans l'araméen des ᵑ7 s et en syriaque, même lorsque cela n'est pas expressément indiqué.
