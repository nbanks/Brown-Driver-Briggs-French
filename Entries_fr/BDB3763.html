1. filet :
a. pour attraper (oiseaux, etc.) חִנָּם מְזֹרָה הָרָ֑שֶׁת Pr 1,17 en vain le filet est tendu aux yeux de tout oiseau ; ailleurs figuré (1) du jugement de ׳ י, tous avec פרשׁ : Os 7,12 (|| כְּעוֺף הַשָּׁמַיִם אוֺרִידֵם ), Ez 12,13 ; Ez 17,20 (les deux || מְצוּדָתִי), Ez 32,3 (de ׳ י attrapant Pharaon, sous la figure d'un monstre marin ; || II. חֵרֶם) ; (2) des nations capturant le roi d'Israël sous la figure d'un lion Ez 19,8 (avec פרשׁ ; || שַׁחְתָּם ) ; (3) des prêtres et des chefs (comme enlaçant le peuple dans le péché) Os 5,1 (|| פַּח).
b. tendu pour les pieds de l'homme (figuré) : (1) par ׳ י Lm 1,13 (avec פָּרַשׁ) ; (2) par les méchants (= complot) Pr 29,5 (avec פרשׁ ), Ps 140,6 (c. id. ; || טָמַן פַּךְ), Ps 10,9 ; Ps 25,15, שֻׁלַּח בְּדֶשֶׁת בְּרַגְלָיו Jb 18,8 (|| שְׂבָכָה) ; ailleurs avec טמן Ps 9,16 (|| שַׁחַת), Ps 31,5 ; Ps 35,7 (שַׁחַת רִשְׁתָּם), Ps 31,8 avec הֵכִין Ps 57,7 (|| שִׁיחָה).
2. grillage d'airain réseau pour l'autel du tabernacle הָרֶשֶׁת Ex 27,4 ; Ex 27,5 נְחשֶׁת ׳ מִכְבָּר מַעֲשֵׂה ר Ex 27,4 = Ex 38,4 (tous P ). — Sur l'Infinitif רֶשֶׁת, voir ירשׁ ci-dessus
