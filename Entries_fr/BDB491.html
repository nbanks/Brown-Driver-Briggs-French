BDB491 [H481]

hébreu biblique

[אָלַם] verbe lier (peut-être comparer assyrien [ al¹mu ], almattu , forteresse Dl dans ZimBP 114 & dans BDEzek.:xi ; arabe souffrir, araméen retenir la colère).

Niphal (être lié =) être muet, Parfait 3 féminin singulier נֶאֱלָ֑מָה Es 53,7 וְנֶאֱלַמְתָּ֫ Ez 3,26, etc. ; Imparfait 2 masculin singulier תֵּאָלֵם Ez 24,27 3 féminin pluriel תֵּאָלַמְנָה Ps 31,19. —
1. être muet, c.-à-d. silencieux Ps 39,3 דוּמִיָּה ׳ נ (|| הֶחֱשֵׁיתִי) ; comparer Ps 39,10 (|| לֹא אֶפְתַּחפִּֿי ) ; comparaison de la brebis Es 53,7 (|| id.) ; être muet, c.-à-d. incapable de parler Ez 3,26 (résultat de וּלְשׁוֺנְךָ אַדְבִּיק אֶלחִֿכָּ֑ךָ) comparer Ez 24,27 יִפָּתַח פִּיךָ ֗֗֗ וּתְדַבֵּר וְלֹא תֵאָלֵם עוֺד ; Ez 33,22 וַיִּפְתַּח פִּי וְלֹא נֶאֱלַמְתִּי עוֺד ; aussi Dn 10,15 (comparer Dn 10,16) ; être rendu muet sujet שִׂפְתֵי שָׁ֑קֶר lèvres menteuses Ps 31,19 (|| יִדְּמוּ לִשְׁאוֺל Ps 31,18).
Piel Participe lier Gn 37,7 ( E ) מְאַלְּמִים אֲלֻמִּים liant des gerbes.
