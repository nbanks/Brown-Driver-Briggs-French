BDB1936 [H1790]

hébreu biblique

דַּךְ adjectif écrasé, opprimé — דַּךְ Ps 74,21 דָּ֑ךְ Ps 9,10 ; Ps 10,18 pluriel suffixe דַּכָּ֑ו Pr 26,28 ( Baer ; et autres éditions דַּכָּיו) — en poésie ; toujours figuré, opprimé, affligé Ps 9,10 ; Ps 18 (|| יתום) Ps 74,21(|| עָנִי et אביון) ; Pr 26,28 דַּכָּו = ceux qu'elle écrase (une langue mensongère).
