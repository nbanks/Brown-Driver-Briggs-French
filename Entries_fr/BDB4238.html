BDB4238 [H3601]

hébreu biblique

כִּישׁוֺר nom [masculin] quenouille (étymologie douteuse ; peut-être, si le sens est correct, de כשׁר (comparer Sta§ 216 LagBN 182) = être droit, parce qu'elle se tient droite, De Str ; ᵑ6 industrie ; d'où ᵑ7 כּוּשְׁרָא, et de même ᵑ7 כּוּנְשְׁרָא, aussi Pr 3,8, (pour hébreu שׁר), où Levy wirbelsäule, Strsur le passage Jastr nombril ; mais Str doute du lien étymologique avec כישׁור) ; — seulement dans יָדֶיהָ שִׁלְּחָה בַכִּישׁוֺ֑ר Pr 31,19 (|| פָּֽלָךְ ׃ fusaïole). — Comme ci-dessus De Now Str RV SS et autres ; > Ki AW Thes Rob-Ges fusaïole ( AV fuseau) ; voir פֶּלֶךְ .
