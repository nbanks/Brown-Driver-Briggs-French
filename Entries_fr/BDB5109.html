BDB5109 [H5157]

hébreu biblique

נָחַל verbe dénominatif obtenir ou prendre comme possession (comparer Gerber236 ss. ; Ecclus נחל + ; hébreu tardif id. (rare)) ; —

Qal Parfait ׳ נ Za 2,16 2 masculin singulier נָחַלְתָּ Ex 23,20 + 8 t. Parfait ; Imparfait 3 masculin singulier יִנְחַלְ Es 57,13 3 masculin pluriel יִנְחֲלוּ Nb 18,23 +5 t. ; יִנְחָ֑לוּ Nb 26,55 + 2 t. ; + 8 t. Imparfait ; Infinitif נְחֹל Nb 34,18 ; Jos 19,49 —
1. prendre possession, hériter :
a. pays de Canaan, Ex 23,30 ( E ), Ex 32,13 ( J ), Es 57,13 ; Ez 47,14 Sion et Juda Ps 69,37.
b. sections particulières dans le pays, des tribus et des individus, נחל נחלה Jos 17,6 ( J ); ׳ נחלה אשׁר נ Dt 19,14 ( D ), Nb 35,8 ( P ), comparer Jos 14,1 ( P ); sans accusatif, avoir ou obtenir une (foncière) propriété , Nb 18,20 ; Nb 18,23 ; Nb 18,24 ; Nb 26,55 ; Nb 32,19 ; Jos 16,4; Jos 19,9 (tous P ), Jg 11,3 .
c. pays de Moab et d'Ammon par conquête So 2,9 propriété foncière בכל הנוים Ps 82,8.
d. Dieu prend possession d'Israël comme sa propriété privée Ex 34,9 ( J ); Juda Za 2,16.
2. figuré avoir ou obtenir comme possession, propriété : témoignages Ps 119,111, gloire Pr 3,35, bonnes choses Pr 28,10 , mensonges Jr 16,19 , vent Pr 11,29, naïveté Pr 14,18.
3. partager le pays pour en faire une possession, Nb 34,17 ; Nb 34,18 ; Jos 19,49 (tous P ; vocalisation incorrecte pour le Pi`el, ainsi SS).
