BDB2357 [H2143]

hébreu biblique

זֵ֫כֶר nom masculin Ps 9,7 souvenir, mémorial (comparer HptHebraica. 1885, 230 ) — ׳ ז absolu Ps 111,4 construit Dt 25,19 + ; זֶכֶר construit Ex 17,14 van d. H ; suffixe זִכְרִי Ex 3,15, זִכְרְךָ Es 26,8 +, זִכְרֶ֑ךָ Ps 6,6, זִכְרוֺ Os 12,6 +, זִכְרָם Dt 32,26 + : —

1. souvenir, mémoire :
a. de personnes ou d'un peuple, effacé par leur destruction, Ex 17,14 ( E ; avec מחה) = Dt 25,19 ; Dt 32,26 (השׁבית) Ps 34,17 ; Ps 109,15 (tous deux הכרית), Ps 9,7 (אבד), Jb 18,17 (אבד ; || שֵׁם), Es 26,14 ( אבּד) ; comparer d'autre part עוֺלָם ׳ ז comme part du juste.
b. de ׳ י Ps 6,6 ; Ps 102,13 (|| אַתָּה = יהוה).
c. du caractère et des œuvres de ׳ י Ps 30,5 ; Ps 97,12 ; Ps 111,4.
d. souvenir de jours particuliers, c.-à-d. leur observance Est 9,28 (הַפּוּרִים יְמֵי ).
2. mémorial, par lequel on est rappelé :
a. presque = שֵׁם, צַדִּיק לִבְרָכָה ׳ ז Pr 10,7 (|| שֵׁם), comparer aussi Qo 9,5 (avec נשׁכח) ; surtout de ׳ י Ex 3,15 ( E ), Os 12,6 ; Es 26,8 ; Ps 135,13 (tous || שֵׁם).
b. = renommée (d'Israël) Os 14,8 (comparer VB ).
