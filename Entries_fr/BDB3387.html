BDB3387 [H2973]

hébreu biblique

I. [יָאַל] verbe être insensé (comparer I. אול) —

Niph`al Parfait 3 masculin pluriel נוֺאֲלוּ Es 19,3 ; Jr 5,4 וְנֹאָ֑לוּ Jr 50,36 1 pluriel נוֺאַלְנוּ Nb 12,11 — agir follement : —
1. montrer une folie coupable = pécher חַטָּאת אֲשֶׁר נוֺאַלְנוּ וַאֲשֶׁר חָטָאנוּ Nb 12,11 ( J ) ; dans Jr 5,4 une manifestation de cette folie est attribuée à l'ignorance.
2. devenir insensés, manquant de discernement et de jugement : נוֺאֲלוּ שְׂרֵי צֹעַן Es 19,13 les princes de Tsoan sont devenus insensés (|| נִשְּׁאוּ שָׂרֵי נֹף, et, dans Es 19,11 אֱוִילִים שָׂרֵי צֹעַן, etc.) ; חֶדֶב אֶלהַֿבַּדִּים וְנֹאָ֑לוּ Jr 50,36 une épée est contre les vantards, et ils deviendront insensés — seront démasqués comme tels (|| חָֽתּוּ ׃).
