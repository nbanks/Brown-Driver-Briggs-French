BDB3988 [H3582]

hébreu biblique

[כָּחַד] verbe pas au Qal; Pi`el Hiph`il cacher, effacer; Niph`al (habituellement) passif (araméen אתכחדו être effacé Jb 4,7 est révérer, Participe vénérable, Pa`el couvrir de honte, Ethpa`al avoir honte ; éthiopien nier, apostasier) —

Niph`al Parfait 3 masculin singulier נִכְחַד Os 5,3 + 2 t. ; 3 pluriel נִכְחָ֑דוּ Jb 4,7 ; Ps 69,6 Imparfait 3 masculin singulier יִכָּחֵד 2 S 18,13, etc. ; Participe נִכְחֶדֶת Za 11,9 pluriel נִכְחָדוֺת Za 11,16 ; Jb 15,28 —
1. être caché, suivi de מִן personne, 2 S 18,13 ; Os 5,3 ; Ps 69,6 ; Ps 139,15 .
2. être effacé, détruit Za 11,9 ; Jb 4,7 ; Jb 15,28 (עָרִים נִכְחְדוֺת ), Jb 22,20, suivi de מִןֿ הָאָרֶץ Ex 9,15 ( J ) ; Participe intransitif d'un processus inachevé = allant à la ruine Za 11,9 ; Za 11,16.
Pi`el Parfait כִּחֵד 1 S 3,18 1 singulier כִּחַדְתִּי Jb 6,10 ; Ps 40,11 3 pluriel כִּחֲדוּ Jb 15,18 כִּחֵ֑דוּ Es 3,9 Imparfait 2 masculin singulier תְּכַחֵד Jos 7,19 + 4 t. ; 2 féminin singulier תְּכַחֲדִי 2 S 14,18 1 singulier אֲכַחֵד Jb 27,11 2 masculin pluriel תְּכַחֵ֑דוּ Jr 50,2 1 pluriel נְכַחֵד Gn 47,18 ; Ps 78,4 — cacher, suivi de מִן personne Gn 47,18 ( J ), Jos 7,19 ( J E ), 1 S 3,17 (deux fois dans le verset) ; 1 S 3,18 ; 2 S 14,18 ; Jr 38,14 ; Jr 38,25 ; Ps 78,4 suivi de לְ personne Ps 40,11 dissimuler envers, en ce qui concerne, c.-à-d. à ; sans préposition Es 3,9 ; Jr 50,2 ; Jb 15,18 ; Jb 27,11 = désavouer Jb 6,10.
Hiph`il Parfait 1 singulier consécutif suffixe וְהִכְחַדְתִּיו Ex 23,23 Imparfait 3 masculin singulier וַיַּכְחֵד 2 Ch 32,21 , suffixe יַכְחִידֶנָּה Jb 20,12 1 singulier וָאַכְחִד Za 11,8 1 pluriel suffixe וְנַכְחִידֵם Ps 83,5 Infinitif construit לְהַכְחִיד 1 R 13,34 —
1. cacher, au figuré יַכְחִידֶנָּה תַּחַת לְשֹׁנוֺ Jb 20,12 (objet la méchanceté).
2. effacer, anéantir Ex 23,23 ( E ), 1 R 13,34 (|| הַשְׁמִיד), Za 11,8 ; 2 Ch 32,21 suivi de מִגּוֺי Ps 83,5, c.-à-d. de sorte qu'ils ne soient plus (qu'ils ne soient pas) une nation.
