BDB193 [H191]

hébreu biblique

אֱוִיל26 adjectif insensé — ׳ א Jb 5,2 + ; pluriel אֱוִילִים Ps 107,17 +, etc. ; — insensé, Pr 29,9 (׳ אִישׁ א) Os 9,7 (prédicat du prophète) ; comparer Es 35,8, ailleurs nom masculin insensé (toujours moralement mauvais), qui méprise la sagesse et la discipline Pr 1,7 ; Pr 15,5 se moque de la culpabilité Pr 14,9 est querelleur Pr 20,3 licencieux Pr 7,22 c'est folie et inutile de l'instruire Pr 16,22 ; Pr 27,22 (19 t. Proverbes) ; comparer aussi Jr 4,22 ; Jb 5,2 ; Jb 5,3 ; Es 19,11 ; Ps 107,17.
