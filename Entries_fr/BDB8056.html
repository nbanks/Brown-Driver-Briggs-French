BDB8056 [H7503]

hébreu biblique

רָפה verbe faiblir, relâcher (hébreu tardif id., de même araméen רפא (espèce dérivée), Zinjirli רפה Aph`el, ; arabe une vie facile est peut-être apparenté) ; —

Qal Parfait 3 masculin singulier Jg 19,9 (pourtant voir ci-dessous) ; 3 féminin singulier רָֽפִ֑תָה Jg 8,3 ; Jr 49,24, etc. ; Imparfait 3 masculin singulier יִרְפֶּה Es 5,24, וַיִּרֶף Ex 4,26 (voir ci-dessous), etc. ; —
1. s'affaisser, du foin dans la flamme Es 5,24 décliner, du jour Jg 19,9 (si vera lectio ; ᵐ5 L κέκλικεν, GFM נָטָה [mais comment expliquer le difficile רפה ?]). Habituellement
2. tomber, retomber (des ailes, voir Pi`el), des mains, avec מִן Ne 6,9 leurs mains retomberont loin du travail (par peur) ; ailleurs absolu figuré = perdre courage, énergie, 2 S 4,1 ; Es 13,7 ; Jr 6,24 ; Jr 50,43 ; Ez 7,17 ; Ez 21,12 ; So 3,16 יָרַיִם omis Jr 49,24.
3. faiblir, se calmer, s'apaiser, du tempérament Jg 8,3 (+ מֵעַל personne).
4. relâcher, se retirer, sujet ׳ י, מִן personne Ex 4,26 ( J ), c.-à-d. laisser quelqu'un tranquille (mais lire peut-être וַיֶרֶף, voir Hiph`il).
Niph`al Participe pluriel נִרְפִּים oisifs Ex 5,8 ; Ex 5,17 ( J ).
Pi`el Parfait 3 masculin singulier רִפָּה Jb 12,21 Imparfait 3 féminin pluriel תְּרַפֶּינָה Ez 1,24 ; Ez 1,25 Participe מְרַפֵּא (Ges«GKC:75rr»§ 75rr) Jr 38,4 pluriel מְפִַּים Esd 4,4 — causatif : laisser retomber les ailes Ez 1,24 ; Ez 1,25 (lire Qal, כנפיהם sujet, ᵐ5 Co Toy Krae et autres, qui suppriment dans Ez 1,25) ; détacher et laisser retomber, ceinture du puissant, figuratif pour les affaiblir Jb 12,21 avec accusatif mains, figuratif pour décourager, démoraliser Jr 38,4 ; Esd 4,4.
Hiph`il Imparfait, 3 masculin singulier suffixe יַרְפְּךָ Dt 4,31 + ; jussif 2 masculin singulier תֶּ֫רֶף Jos 10,6 +, etc. ; Impératif masculin singulier הַרְפֵּה Jg 11,37 ; 2 R 4,27 הֶֶ֫רֶךָ Dt 9,14 + ; masculin pluriel הַרְפּוּ Ps 46,11 — causatif,
1. laisser retomber, la main 2 S 24,16 = 1 Ch 21,15 figuré, + מִן personne, = abandonner , Jos 10,6 ( J E ) ; peut-être aussi (יָד omis) = relâcher, s'abstenir Ex 4,26 (lisant וַיֶרֶף, voir Qal) ; accusatif de chose, = abandonner, délaisser, Ne 6,3, de même (sujet ׳ י) Ps 138,8 sujet ׳ י, avec accusatif de personne, Dt 4,31 ; Dt 31,6 ; Dt 31,8 ; Jos 1,5 ( D ), 1 Ch 28,20.
2. lâcher, suffixe personne, Ct 3,4 (opposé à אחז) ; figuratif, accusatif (omis) מוּסָר , Pr 4,13 (opposé à החזיק), צְדָקָה Jb 27,6 (opposé à id. ).
3. s'abstenir, avec מִן personne, = laisser quelqu'un tranquille Jg 11,37 ; Dt 9,14 , מִן de la colère Ps 37,8 avec ל personne 1 S 11,3 ; 2 R 4,27 accusatif de personne Jb 7,19 absolu laisser tranquille , c.-à-d. ne rien faire, rester tranquille, 1 S 15,16 ; Ps 46,11 (comparer Weir dans Dr 1 S 15,16 ). — Jr 3,22 ; Jb 5,18 ; Ps 60,4 voir רָפָא .
