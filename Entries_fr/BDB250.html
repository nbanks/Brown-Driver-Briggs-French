hébreu biblique

מָאוֺר nom masculin luminaire — absolu Gn 1,16 + ; construit מְאוֺר Pr 15,30 + ; pluriel מְאוֺרֹת Gn 1,15 מְאֹרֹת Gn 1,14 Gn 1,16 construit מְאוֺרֵי Ez 32,8 — lumière, porte-lumière, luminaire, lampe, du soleil & de la lune Gn 1,14 ; Gn 1,15 ; Gn 1,16 (3 t. dans le verset) ( P ) Ez 32,8 lune Ps 74,16 מְנֹרַת הַמָּאוֺר le candélabre du luminaire ou lumière (où ׳ הַמּ est la somme des sept lampes sacrées sur le candélabre d'or) Ex 35,14 (deux fois dans le verset) ; Ex 35,28 ; Ex 39,37 ; Nb 4,9 ; Nb 4,16, comparer Ex 25,6 ; Ex 27,20 ; Ex 35,8 ; Lv 24,2 (tous P ) ; מְאוֺרעֵֿינַיִם יְשַׂמַּחלֵֿב֑ Pr 15,30 le luminaire des yeux (= les yeux comme lampe) donne la lumière de la joie au cœur ; מְאוֺר פָּנֶיךָ Ps 90,8 le luminaire de ta face (ta face comme lampe) à la lumière de laquelle les secrets sont mis à nu.
