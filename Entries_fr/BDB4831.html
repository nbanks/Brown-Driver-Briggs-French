BDB4831 [H4751 H4755]

hébreu biblique

I. מַר adjectif et substantif amer, amertume ; — absolu ׳ מ Pr 27,7 +, הַמַּר Ha 1,6, מָ֑ר Ps 64,4 + ; construit מַר 1 S 22,2 + ; féminin מָרָה Gn 27,34 +, מָרָא Rt 1,20 construit מָרַת 1 S 1,10 masculin pluriel מָרִים Ex 15,23 + ; construit מָרֵי Jg 18,25 ; Pr 31,6 — amer :

1. littéralement, de l'eau Ex 15,23 ( J ) ; nourriture Pr 27,7 (opposé à מָתוֺק) ; absinthe (לַעֲנָה Pr 5,4 (en comparaison de la fin de la prostituée, comparer Qo 7,26 ci-dessous) ; comme substantif pluriel, de la douleur physique, résultat de la malédiction לְמָרִים Nb 5,24 ; Nb 5,27, d'où מֵי הַמָּרִים Nb 5,18 (comparer Di) Nb 5,19 Nb 5,23 Nb 5,24 du résultat nuisible d'un combat impitoyable 2 S 2,26 de la méchanceté Jr 2,19 ; Jr 4,18 = nuisible, pernicieux Es 5,20 (deux fois dans le verset) (opposé à מָתוֺק).
2. figuré,
a. d'un cri (צְעָקָה, זְעָקָה) Gn 27,34 ( J ), Est 4,1 מִסְפֵּד מָ֑ר Ez 27,31 (supprimer Co) ; דָּבָר מָ֑ר Ps 64,4 comme adverbe (crier, pleurer) amèrement Ez 27,30 ( מָרָה), So 1,14 ; Es 33,7 (les deux מַר).
b. de sentiment, tempérament, surtout avec נֶפֶשׁ : אֲנָשִׁים מָרֵי נֶפֶשׁ Jg 18,25 hommes féroces (GFM âpres) de tempérament, ainsi 2 S 17,8 (comme une ourse privée de ses petits) ; הַגּוֺי הַמַּר Ha 1,6 la nation féroce ; מַרנֶֿפֶשׁ = mécontent 1 S 22,2 מַרֵי נֶפֶשׁ = les amèrement misérables Jb 3,20 ; Pr 31,6, ainsi ׳ מָרַת נ 1 S 1,10 comparer יוֺם מָ֑ר Am 8,10 comme substantif abstrait מַרנֶֿפֶשׁ amertume de l'âme Es 38,15 ; Ez 27,31 (supprimer Co), Jb 7,11 ; Jb 10,1 = נפשׁ מָרָה Jb 21,25 comparer מָרָא Rt 1,20 (nom donné à Noémi par elle-même) ; מַרהַֿמָּוֶת 1 S 15,32 l'amertume de la mort ; מַרמִֿמָּוֶת Qo 7,26 (métaphore de la prostituée) ; en Ez 3,14 je m'en allai amèrement, supprimer מַר Co. — Es 38,17 voir מרר .
