BDB9153 [H8505]

hébreu biblique

[תָּכַן] verbe régler, mesurer, estimer (comparer araméen Pael תַּכֵּן (rare) préparer (= הֵכִין, כּוֺנֵן); selon RySyn. d. Wahren u. Guten 33 une racine secondaire de כון ); —

Qal Participe estimer, figuratif : ׳ וְתֹכֵן רוּחוֺת י Pr 16,2, ainsi ( לִבּוֺת) Pr 21,2 ; Pr 24,12.
