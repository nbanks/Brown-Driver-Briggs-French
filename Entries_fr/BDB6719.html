BDB6719 [H6472]

hébreu biblique

פַּעֲמֹן nom [masculin] clochette, sur la robe du grand prêtre (de son coup, battement) ; — ׳ פ absolu Ex 39,26 (deux fois dans le verset) ; construit זָחָב ׳ פ Ex 28,34 (deux fois dans le verset) ; pluriel הַפַּעֲמֹנִים Ex 39,25, construit פַּעֲמֹנֵי זָהָב Ex 28,33 ; Ex 39,25. פַּעֲנֵחַ voir ׳ צָֽפְנַת פ.
