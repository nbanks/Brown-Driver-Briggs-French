BDB3992 [H3584]

hébreu biblique

כָּחַשׁ verbe être décevant, tromper, faillir, maigrir (hébreu tardif כָּחַשׁ maigrir ; Hiph`il convaincre de fausseté, réfuter ; araméen כְּחַשׁ maigrir ; כַּחְשָׁא maigreur, כְּחִישׁוּתָא maigreur, faiblesse ; Aph`el comme hébreu tardif Hiph`il) —

Qal. Parfait 3 masculin singulier וּבְשָׂרִי כָּחַשׁ מִשָּׁ֑מֶן Ps 109,24 ma chair a maigri (loin) de la graisse (|| בִּרְכַּי כָּֽשְׁלוּ מִצּוֺם).
Niph`al Imparfait וְיִכָּֽחֲשׂוּ אֹיְבֶיךָ לָךְ Dt 33,29 tes ennemis ramperont (viendront en rampant, c.-à-d. à l'origine agir de manière trompeuse, feignant l'obéissance) devant toi (comparer aussi Pi`el 3, Hithpa`el)
Pi`el Parfait 3 masculin singulier כִּחֵשׁ 1 R 13,18 ; Ha 3,17 וְכִּ֫חֶשׁ consécutif (devant monosyllabe) Lv 5,22 ; Jb 8,18, etc. ; Imparfait יְכַ֫חֶשּׁ (בָּהּׅ Os 9,2 3 féminin singulier וַתְּכַחֵשׂ Gn 18,15, etc. ; Infinitif absolu כַּחֵשׁ Os 4,2 ; Es 59,13 construit Za 13,4 —
1. tromper, absolu Gn 18,15 ( J ), Jos 7,11 ( J E ), Os 4,2 (אלה וכחשׁ ensemble = faux serment selon WeKl. 2. agir de manière trompeuse envers (c.-à-d. sembler reconnaître, mais ne pas le faire vraiment), suivi de בְ personne Jos 24,27 ( E ), Jr 5,12 ; Es 59,13 ; Jb 8,18, apparemment aussi בָּהּ ׳ וְתִירוֺשׁ יכ Os 9,2 et le vin nouveau la reniera (Israël ; c.-à-d. refusera de la reconnaître comme sa maîtresse, ne se livrera pas à elle) ; absolu Pr 30,9.
3. ramper = venir en rampant (faire montre d'obéissance, voir Niph`al) suivi de בְ personne Ps 18,45 (Hithpa`el dans le || 2 S 22,45) ; suivi de לְ personne Ps 66,3 ; Ps 81,16.
4. décevoir, faillir, absolu כִּחֵשׁ מַעֲשֵׂה זַיִת Ha 3,17 le produit de l'olivier a fait défaut (pour le sens comparer Os 9,2 ci-dessus)
