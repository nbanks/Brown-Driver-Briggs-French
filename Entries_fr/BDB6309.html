BDB6309 [H6119]

hébreu biblique

I. עָקֵב nom masculin talon, empreinte de pas, arrière-partie (arabe talon, figuré fin (d'un mois), marque, signe, trace ; araméen עִקְבָּא, talon, empreinte de pas, figuré trace, marque (comparer Sagesse 2,4 ᵑ6 Qo 13,26 HCT ᵑ6 ), aussi fin, extrémité) ; — ׳ ע absolu Gn 3,15 +, construit עֲקֵב Gn 25,26, pluriel construit עִקָבֵי Ct 1,8, עִקְּבֵי Gn 49,17 ; Jg 5,22, עִקְּבוֺת Ps 77,20 ; Ps 89,52, etc. ; —

a. talon, de l'homme, Gn 25,26 וְיָדוֺ אֹחֶזֶת בַּעֲקֵב אָחִיו, comme objet d'attaque par derrière, Gn 3,15 וְאַתָּה תְּשׁוּפֶנּוּ עָקֵב, Jb 18,9 פַּח ׳ יאֹחֵז בְּע, Jr 13,22 נֶחְמְסוּ עֲקֵבָ֑יִךְ sont traités avec violence, c.-à-d. sont rudement exposés (|| נִגְלוּ שׁוּלַיִךְ) ; comme instrument d'attaque, Ps 41,10 ׳ הִגְדִּיל עָלַי עָ a levé le talon contre moi, c.-à-d. m'a donné insidieusement une grande chute (figuré pour a pris quelque avantage cruel sur moi ; comparer ὑποσκελίζω) ; d'un animal, Gn 49,17 הַנּשֵׁח עִקְּבֵי סוּם, Jg 5,22
b. marque du talon, empreinte de pas, Ct 1,8 צְאִילָֿח בְּעִקְבֵי הַצּאֹן, Ps 56,7 יִשְׁמְרוּ עֲקֵבַי ils guettent mes pas, c.-à-d. m'épient insidieusement partout où je vais Ps 89,52 אֲשֶׁר ֵהרְפוּ עִקְּבוֺת מְשִׁיחֶ֑ךָ c.-à-d. l'ont suivi en se moquant, de ׳ י (figuré) Ps 77,20 עִקְּבוֺתֶיךָ לֹא נוֺדָ֑עוּ tes traces n'ont pas été connues (les eaux se refermant sur elles).
c arrière-partie, arrière-garde (d'une troupe d'hommes), Gn 49,19 וְהוּא יָגוּב עָקֵב׃ (lire עֲקֵבָם ׃ אָשֵׁר) il s'élancera sur leur arrière-garde, Jos 8,13 עֲקֵבָם (comparer Di).
