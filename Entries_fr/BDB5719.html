BDB5719 [H5637]

hébreu biblique

סָרַר verbe être obstiné, rebelle (habituellement envers ׳ י) (hébreu tardif id., (rare) ; assyrien sarâru ) ; —

Qal Parfait 3 masculin singulier יִשְׂרָאֵל ׳ ס Os 4,16 Israël est obstiné ; Participe actif בֵּן סוֺרֵר וּמוֺרֶה Dt 21,18, comparer Dt 21,20 בָּנִים סוֺרְרִים Es 30,1, עַם סוֺרֵר Es 65,2, וּמֹרֶה ׳ דּוֺר ס Ps 78,8 (tous d' Israël) ; וּמוֺרֶה ׳ לֵב מ Jr 5,23 פָּרָה סֹרֵרָ֔ה Os 4,16 (comparaison d'Israël, voir ci-dessus) ; comme prédicat שָׂרֵיהֶם סוֺרְרִים Os 9,15 comparer Es 1,23 סָרֵי סוֺרְרִים Jr 6,28 rebelles parmi les récalcitrants (? comparer סור Qal vers la fin, et סַר ci-dessous) ; de la femme légère הֹמִיָּסה הִיא וְסוֺרָ֑רֶת Pr 7,11 comme substantif = les obstinés, Ps 66,7 ; Ps 68,7 ; Ps 68,19 féminin singulier וַיִּתְּנוּ כָתֵף סֹרֶ֫רֶת Ne 9,29 et ils présentèrent une épaule obstinée (d'Israël) = Za 7,11 (סֹרָ֑רֶת ).
