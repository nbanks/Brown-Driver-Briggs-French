BDB4240 [H3791]

hébreu biblique

כְּתָב nom masculin Esd 4,7 écriture (araméen tardif ; araméen biblique כְּתָב syriaque ) ; — ׳ כ absolu 1 Ch 28,19 + 4 t. ; construit Ez 13,9 + 4 t. ; suffixe כְּתָבָהּ Est 1,22 + 2 t. ; כְּתָבָם Esd 2,62 + 3 t. ; — écriture :

1. registre, enregistrement, Ez 13,9 ; Esd 2,62 = Ne 7,64.
2. = mode d'écriture, caractère, lettre, הִנִּשְׁתְּוָן כָּתוּב אֲרָמִית ׳ וּכ Esd 4,7 et l'écriture de la lettre était écrite en araméen (caractères ; ainsi probablement, voir BeRy Ryle, comparer hébreu tardif HoffmZAW i.(1881), 334 ss.) ; comparer Est 1,22 ; Est 3,12 ; Est 8,9 (deux fois dans le verset).
3. = lettre ; 2 Ch 2,10 ; Est 9,27.
4. d'un décret royal, édit 2 Ch 35,4 (|| מִכְתָּב), Est 3,14 ; Est 8,8 ; Est 8,13 plus complètement כְּתָבהַֿדָּת Est 4,8.
5. d'un écrit ayant une autorité divine 1 Ch 28,19 אֱמֶת ׳ כ Dn 10,21, c.-à-d. un livre de vérité.
