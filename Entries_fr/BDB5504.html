BDB5504 [H5465]

hébreu biblique

סַד nom [masculin] ceps, pour entraver les pieds des coupables (probablement mot d'emprunt araméen de סַדָּא, סַדָּנָא ; hébreu tardif סַד, pluriel סַדִּין) ; = latin nervus, grec ποδοκάκη ; — absolu רַגְלַיִם ׳ שִׂים בַּמּ Jb 13,27 ( ᵐ5 κώλύμα), Jb 33,11 ( ᵑ6 ξύλό) ; (comparer synonyme מַהְפֶּכֶת, et l'équivalent grec cité là).
