hébreu biblique

כִּלְאַיִם nom [masculin] duel deux espèces (arabe , les deux ; éthiopien deux ; peut-être aussi כלאי MI23 les deux, deux, NöLCB 1889, 61, cf DrSm xcii ; Sm u. SoMI, 1886 suggèrent réservoirs ; voir aussi כְּלִי ) — בְּהֶמְתְּךָ לֹא ׳ תַּרְבִּיעַ כ Lv 19,19 ne fais pas accoupler ton bétail (en) deux espèces ; ׳ שָֽׂדְךָ לֹא תִּזְרַע כ Lv 19,19 ( H ) ; לֹא תִּזְרַע ׳ כַּרְמְךָ כ Dt 22,9 (les deux כִּלְאָ֑יִם) ; ׳ בֶּגֶד כ Lv 19,19 ( H ) ; — seulement dans la loi de H et D contre les mélanges.
