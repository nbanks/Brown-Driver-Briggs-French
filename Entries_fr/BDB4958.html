BDB4958 [H5042]

hébreu biblique

[נָבַע] verbe couler, jaillir, bouillonner, en poésie et tardif (hébreu tardif id. (rare) ; assyrien nabû , id., DlHWB 442 ; arabe jaillir (de l'eau) ; araméen נְבַע ; éthiopien II. pleurer, larme(s)) ; —

Qal Participe נַ֫חַל נֹבֵעַ Pr 18,4 un oued qui coule, métaphore de מְקוֺר חָכְמָה.
Hiph`il Imparfait יַבִּיעַ Ps 19,3 + 3 t. ; אַבְּ֫יעָה Ps 78,2 ; Pr 1,23 יַבִּ֫יעוּ Ps 94,4 ; Ps 145,7 יַבִּיעוּן Ps 59,8 3 féminin pluriel תַּבַּ֫עְנָה Ps 119,171 — déverser, émettre, faire bouillonner, vomir :
1. déverser, figuré אַבִּיעָה לָכֶם רוּחִי Pr 1,23 (|| אוֺדִ֫יעָה דְבָרַי אֶתְכֶם).
2. faire bouillonner, fermenter זְבוּבֵי מָ֫וֶת יַבְאִישׁ יַכִּיעַ שֶׁמֶן רוֺקֵ֑חַ Qo 10,1 .
3. figuré, habituellement du discours, déverser, émettre, vomir, expression emphatique : פִּי רְשָׁעִים יַבִּיעַ רָעוֺת Pr 15,28 (opposé à לֵב צַדִּיק יֶהְגֶּה לַעֲנוֺת) c.-à-d. le juste réfléchit avant de parler, le méchant éclate en paroles imprudentes ; פִּי כְסִילִים יַבִּיעַ אִוֶּלֶת Pr 15,2 (opposé à לְשׁוֺן חֲכָמִים תֵּיטִיב דָּ֑עַת) ; יַבִּ֫יעוּ יְדַבְּרוּ עָתָק Ps 94,4 (|| יִתְאַמְּרוּ כָּלפֹּֿעֲלֵי אָ֑וֶן) ; בְּפִיהֶם ׳ יַבּ Ps 59,8 (|| חֲרָבוֺת בְּשִׂפְתוֺתֵיהֶם) ; en bon sens, אַבִּ֫יעָה חִידוֺת Ps 78,2 (|| אֶפְתֳּחה בְּמָשָׁל פִּי) ; תַּבַּעְנָה שְׂפָתַי תְּהִלָּה Ps 119,171 זֵכֶר רַבטֿוּבְךָ יִבִּ֑יעוּ Ps 145,7 (|| וְצִדְקָֽתְךָ יְרַנֵּ֑נוּ, comparer aussi Ps 145,6) ; métaphore אֹמֶר ׳ יוֺם לְיוֺם יַבּ Ps 19,3, c.-à-d. bien que silencieux, il déverse réellement la parole.
