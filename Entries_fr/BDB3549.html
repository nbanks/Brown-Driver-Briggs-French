1. parenté Gn 12,1 (|| ארצך, בית אביך), Gn 24,4 (|| ארץ), Nb 10,30 (|| id. ), Gn 31,3 (|| ארץ אבותיך), Gn 43,7 (tous J ) ; || עַם Est 2,10 ; Est 2,20 ; Est 8,6 surtout ׳ אֶרֶץ מ pays de sa parenté Gn 11,28 ; Gn 24,7 (les deux J ), Gn 31,13 ( E ), Jr 22,10 ; Jr 46,16 ; Ez 23,15 ; Rt 2,11.
