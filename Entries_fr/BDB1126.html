hébreu biblique

בַּל69 adverbe ne . . . pas (phénicien id. : p. ex. CISi. 165, 15 בל יכן לכהן ne sera pas pour le prêtre ; :3, 3 בל עתי = avant mon temps) synonyme poétique de לֹא, d'occurrence relativement rare, Os 7,2; Os 9,16 ( Qr ) Es 14,21 ; Es 35,9; Es 43,17; Pr 9,13; Pr 14,7; Pr 19,23; Pr 22,29; Pr 23,7; Pr 23,35 (deux fois dans le verset); Pr 24,23; 1 Ch 16,30 ( = Ps 96,10), seulement en outre, sauf dans les passages cités, dans d'autres Psaumes : souvent répété dans le même contexte, comme Es 26,10 (deux fois dans le verset); Es 26,11; Es 26,14 (deux fois dans le verset); Es 26,18 (deux fois dans le verset); Es 33,20 (deux fois dans le verset); Es 33,21; Es 33,23 (deux fois dans le verset); Es 33,24; Es 44,8; Es 44,9 (3 t. dans le verset); Ps 10,4; Ps 10,6; Ps 10,11; Ps 10,15; Ps 10,18; Ps 16,2; Ps 16,4 (deux fois dans le verset); Ps 16,8; Ps 17,3 (deux fois dans le verset); Ps 17,5; Ps 21,3; Ps 21,8; Ps 21,12 aussi souvent employé avec אֶמּוֺט, יִמּוֺט, תִּמּוֺט Ps 10,6; Ps 16,8; Ps 21,8; Ps 30,7; Ps 46,6; Ps 93,1; Ps 96,10; Ps 104,5; Pr 10,30; Pr 12,3; Jb 41,15. Dans Es 40,24 il est probable qu'il acquiert du contexte le sens de à peine : oui, à peine sont-ils plantés, oui, à peine sont-ils semés …, quand il souffle même sur eux, et ils se fanent ; comparer לֹא 2 R 20,4. Joint de manière anomale avec un infinitif, Ps 32,9 בַּל קְרוֺב אֵלֶיךָ (sinon) il n'y a pas d'approche vers toi (c.-à-d. sinon ils ne t'approcheront pas).
