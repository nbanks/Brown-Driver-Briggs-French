BDB5201 [H5230]

hébreu biblique

[נָכַל] verbe être rusé, fourbe, fripon (assyrien nakâlu, être rusé, astucieux ; ᵑ7 נְכֵל être rusé, Pa`el tromper ; syriaque tromper, comploter ; comparer peut-être agir perfidement, perfide, PräAmhar. Spr. 90) ; —

Qal Participe נוֺכֵל Ml 1,14 fripon.
Pi`el Parfait 3 masculin pluriel avec accusatif de sens apparenté avec le verbe נִכְלֵיהֶם אֲשֶׁר נִכְּלוּ לָכֶם Nb 25,18 leurs ruses par lesquelles ils vous ont trompés.
Hithpa`el Imparfait וַיִּתְנַכְּלוּ אֹתוֺ לַהֲמִיתוֺ Gn 37,18 ils complotèrent fourbeusement contre lui (Köiii. 9), etc. ; Infinitif construit לְחִתְנַכֵּל בְ Ps 105,25 agir fourbeusement envers.
