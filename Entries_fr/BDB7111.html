BDB7111 [H6779]

hébreu biblique

צָמַח verbe germer, pousser (hébreu tardif id., ainsi ᵑ7 צְמַח ; syriaque jaillir ou briller, brillant, aussi germer ; phénicien צמח postérité) ; —

Qal Parfait 3 masculin singulier צָֽמַחבּֿוֺ Lv 13,37 3 pluriel וְצָֽמְחוּ consécutif Es 44,4 Imparfait 3 masculin singulier יִצְמַח Jb 5,6, 3 féminin pluriel תִּצְמַחְנָה Es 42,9, etc. ; Participe צֹמֵחַ Ex 10,5, צוֺמֵחַ Qo 2,6 féminin pluriel צֹמְחוֺת Gn 41,6 ; Gn 41,23 — germer, pousser :
1. des plantes, arbres, etc. Gn 2,5 ; Ex 10,5 (les deux J ), Gn 41,6 ; Gn 41,23 ( E ), Ez 17,6 (dans une figure) ; une fois (tardif) יַעַר צוֺמֵחַ עֵצִים Qo 2,6 bois poussant avec des arbres (plein d'arbres en croissance) ; figuré du futur souverain (voir צֶמַח ) Za 6,12 , postérité Es 44,4, comparer Jb 8,19, peine (עָמָל) Es 5,6, vérité (אֱמֶת) Ps 85,12, restauration ( אֲרֻכָֽתְךָ) Es 58,8, événements futurs Es 42,9.
2. des cheveux Lv 13,37 ( P ; comparer Pi`el).
Pi`el croître abondamment, toujours des cheveux ; — Parfait 3 masculin singulier צִמֵּחַ Ez 16,7 (dans une figure) : Imparfait 3 masculin singulier יִצַמַּח 2 S 10,5 = 1 Ch 19,5 (de la barbe) ; Infinitif construit לְצַמֵּחַ Jg 16,22 (cheveux de Samson).
Hiph`il Parfait 3 masculin singulier suffixe וְהִצְמִיחָהּ Es 55,10 Imparfait 3 masculin singulier יַצְמִיחַ 2 S 23,5 ; Es 61,1l, וַיַּצְמַח Gn 2,9, etc. ; Infinitif construit לְהַצְמִיחַ Jb 38,27 Participe מַצְמִיחַ Ps 104,14 ; Ps 147,8 —
1. faire croître, avec accusatif de plante, sujet ׳ י Gn 2,9 ; Ps 104,14, comparer Jb 38,27, avec 2 accusatifs Ps 147,8 qui fait germer l'herbe sur les montagnes ; figuré, objet צֶמַח צְדָקָה Jr 33,15 corne (קֶרֶן) d'Israël Ez 29,21, de David Ps 132,17 justice, etc. Es 61,11b ; comparer aussi 2 S 23,5 (si interrogatif, voir Dr Bu ; sinon HPS) ; sujet sol Gn 3,18 comparer Es 61,11a (comparaison), Dt 29,22 (objet omis) ; aussi (figuré) Es 45,8 (comparer Di Kit).
2. de la pluie, faire la terre germer (absolu), Es 55,10.
