hébreu biblique

כַּף192 nom féminin 1 R 8,54 creux, ou plat de la main, paume, plante du pied, poêle (hébreu tardif id.; araméen id., assyrien kappu, main, poêle, kippatu, creux; arabe paume, main) — absolu ׳ כ Jb 29,9 +, de même lire probablement aussi Ez 29,7b pour ᵑ0 כָּתֵף ( ᵐ5 ᵑ6 Sm Co, comparer va) ; כָּ֑ף 2 R 11,12 + ; plus souvent construit כַּף Gn 40,11 + ; suffixe כַּפִּי Jg 12,3, כַּפְּךָ Jb 13,31, etc. ; dans Ez 29,7 lire כַּף Qr pour כפך Kt (ainsi Ew Hi Sm Co Da) ; duel כַּפַּיִם Es 49,16 + ; construit כַּפֵּי Ex 29,24 (deux fois dans le verset) + 3 t. ; suffixe כַּפַּי Gn 20,5 +, כַּפָּיו Nb 24,10 +, כַּפֵּיהֶם Es 59,6 + 2 t., כַּפֵּימוֺ Jb 27,23, etc. ; pluriel כַּפּוֺת (surtout des plantes de pied, et dans les sens métaphoriques) absolu Nb 4,7 + ; construit 2 R 9,35 +, suffixe כַּפֹתָיו Ex 25,29 ; Ex 37,16 —
