hébreu biblique

Dans Ex 4,4 ( J E ), Pr 31,19 ; Ez 21,16 ; Ps 129,7, etc., l'idée est celle de saisir.
b. anthropomorphique de ׳ י couvrant Moïse de sa paume, וְשַׂכֹּתִי כַפִּי עָלֶיךָ Ex 33,22 ( J ), et de même figuré (avec שׁית) Ps 139,5 retirant ( הֵסִירׅ sa main Ex 33,23 ( J ); écartant ( הִרְהִּיק ׅ sa main qui afflige Jb 13,21 עַלכּכַּפַּיִם כִּסָּה אוֺר Jb 36,32, c.-à-d. il (Dieu) a rempli ses paumes de lumière.
c. une fois d'animaux, הוֺלֵךְ עַלכּכַּפָּיו Lv 11,27 marchant sur leurs paumes, c.-à-d. pattes (chats, chiens, etc.).
d. expressions courantes : — (I) הִכָּה כַּף = frapper des mains, en applaudissement 2 R 11,12 de même ׳ תָּקַע כ Ps 47,2, ׳ מָחָא כ Es 55,12 (figuré des arbres), Ps 98,8 (figuré des fleuves) ; mais surtout par dérision, mépris, etc., סָפַק אֶתכַּֿפָּיו Nb 24,10 ( J ), יִשְׂפֹּק עָלֵימוֺ כַפֵּ֑ימוֺ Jb 27,23 de même תָּקַע כַּף עלֿ Na 3,19, הִכָּה כַפִּי אֶלֿ Ez 22,13 (de ׳ י) ; absolu, בְכַפְּךָהִכָּה Ez 6,11 (comme מחא יד Ez 25,6), הִכָּה כַּף אֶלכַּֿף Ez 21,19 , אַכֶּה כַפִּי אֶלכַּֿפִּי v.Ez 21,22 (de ׳ י). (2) תָּפַשׂ בַּכַּף saisir, empoigner avec la main Ez 29,7 comparer Ez 29,7 (sur texte voir ci-dessus) (3) de poignée de main comme gage לַזָּר כַּפֶּיךָתָּקַעְתָּ Pr 6,1 ( si) tu as frappé tes paumes (donné une double poignée de main), pour un autre (ainsi De ; || עָרַבְתָּ לְרֵעֶ֑ךָ ) ; de même absolu תקע כף Pr 17,18 ; Pr 22,26 (comparer תקע ליד Jb 17,3). (4) souvent de la main étendue dans la prière, comme signe du désir de recevoir, avec פָּרַשׂ אֶלֿ Ex 9,29 ; Ex 9,33 ( J ), 1 R 8,38 = 2 Ch 6,29 , etc. (voir יד Ps 143,6 ; Lm 1,17 et יָד 1.d., ci-dessus ; aussi פרשׂ, נשׂא, שׁטח ). (5) שׂים כַּף לְפֶה, en silence respectueux, Jb 29,9. (6) figuré בְכַפִּיוָאָשִׂימָה נַפְשִׁי Jg 12,3 = j'ai pris ma vie dans ma main (c.-à-d. l'ai risquée), de même 1 S 19,5 ; 1 S 28,21 ; Jb 13,14 et (sans verbe) Ps 119,109. (7) ׳ נִקְיֹן כ Gn 20,5 pureté des paumes est figuré pour pureté d'action, comparer Ps 26,6, aussi Jb 9,30 ; Jb 22,30 ; Ps 24,4 ; Ps 73,13 inversement, ( לֹא ) חָמָס בְּכַפָּ֑י Jb 16,17 ; 1 Ch 12,18 ; Es 59,6 ; Jon 3,8, comparer Es 59,3 (ainsi ידים Es 1,15 ), Jb 31,7 ; Ps 7,4. — Dans beaucoup de cas כַּף ne se distingue pas de יָד ; ainsi dans l'expression ׳ יְגִיעַ כ labeur des mains Gn 31,42 ; Ag 1,11 ; Jb 10,3 ; Ps 128,2 ׳ פֹּעַל כ Ps 9,17 , ׳ חֵפֶץ כ Pr 31,13 ׳ תְּבוּנוֺת כ Ps 78,72 , ׳ פְּרִי כ Pr 31,16 ; comparer Mi 7,3.
