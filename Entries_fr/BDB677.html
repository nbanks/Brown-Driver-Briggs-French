אָפַף verbe entourer, englober (araméen אֲפַף ) —

Qal Participe אֹפְפִים Jg 20,43 אֲפָפוּנִי Ps 17,9 ; אֲפָפוּנוּ Ps 118,11 (trois fois dans le verset) ; אֲפָפוּהוּ Ps 18,5 ; אֲפָפַ֫תְנִי Ps 40,13 ; אֲפָפוּנִי Jon 2,6 = Ps 18,6 (employé toujours dans le style poétique).
