[שָׁכַם] verbe

Hiph`il dénominatif se mettre en route, se lever, tôt (proprement charger le dos des bêtes pour le voyage du jour) ; — Parfait 3 masculin singulier consécutif וְהִשְׁכִּים 2 S 15,2 ; Jb 1,5, etc. ; Imparfait 3 masculin singulier וַיַּשְׁכֵּם Gn 19,27 +, 2 masculin singulier תְּשְׁכִּים Jg 9,33, etc. ; Impératif הַשְׁכֵּם Ex 8,16 + ; Infinitif absolu הַשְׁכֵּם Jr 7,25 +, כֵּים- Jr 44,4 ; Pr 27,14, erroné אַשְׁכֵּים Jr 25,3 (Ges«GKC:53»§ 53 K) ; Participe מַשְׁכִּים Os 6,4 +, etc., — se lever tôt, partir de bonne heure, pour le voyage (Hexateuque seulement J E ) ; לְדַרְכְּכֶם Jg 19,9 (+ מָחָר), comparer Ct 7,13 + verbe d'aller Gn 19,2 ( J ), se tenir debout 2 S 15,2 (comparer Dr), camper Jg 7,1, etc. ; + infinitif 1 S 15,12 (comparer Dr), 1 S 29,11 ; 2 R 6,15 (לָקוּם), מַשְׁכִּימֵי קוּם Ps 127,3 (opposé à מְאַחֲרֵי שֶׁבֶת) ; habituellement avec בַּבֹּקֶר Gn 19,27 ( J ; אֶלהַֿמָּקוֺם), 1 S 29,10 (deux fois dans le verset) + 27 t., + (hyperbole) שֵׁכָר ׳ מַשְׁכִּימֵי בַבּ יִרְדֹּ֑פוּ Es 5,11 (|| מְאַחֲרֵי בֵנֶּשֶׁף) ; avec מִמָּחֳרָת Ex 32,6 ( E ), Jg 6,28 + 3 t. ; avec בַּעֲלוֺת הַשַּׁחַר Jos 6,15 ( J ), 1 S 19,26 וַיְמַהֲרוּ וַיַּשְׁכִּימוּ וַיֵּצְאוּ Jos 8,14 ( J ) ils se hâtèrent et partirent de bonne heure ; et sortirent ; Infinitif absolu (Ges«GKC:113s»§ 113s) comme adverbe וְהַעֲרֵב ׳ הַשׁ 1 S 17,16 au petit matin et au soir ; surtout de ׳ י, dans Jérémie : ׳ הַשׁ וְדַבֵּר = parlant tôt et souvent Jr 7,13 ; Jr 35,14 et (de Jérémie) Jr 25,3 , comparer Jr 11,7 ; Jr 32,33 souvent וְשָׁלוֺחַ ׳ הַשׁ = envoyant tôt et souvent Jr 7,25 ; Jr 24,4 ; Jr 26,5 ; Jr 29,19 ; Jr 35,14 ; Jr 44,4, aussi 2 Ch 36,15 ainsi verbe fini הִשְׁכִּימוּ הִשְׁחִיתוּ, So 3,7 = avec empressement ils corrompirent (Ges«GKC:120g»§ 120g) ; participe הֹלֵךְ ׳ כַּטַּל מַשְׁ Os 6,4 comme la rosée qui s'en va tôt, Os 13,3 (Gesib.). — 1 S 9,26 lire וַיִּשְׁכַּב, voir שׁכב
