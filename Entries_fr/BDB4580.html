BDB4580 [H4229]

hébreu biblique

I. מָחָה verbe essuyer, effacer (hébreu tardif id. ; arabe , effacer, rayer, annuler, oblitérer ; araméen מְחָא (I), comparer phénicien למחת d'après la mesure = exactement HoffinPhoenician Inschr., Abh. GGW, xxxvi. mai 1889, 9). —

1. essuyer, la bouche Pr 30,20 les larmes de (מעל) la face Es 25,8 la malédiction écrite, dans l'eau (אלמֿים) pour boire Nb 5,23 ( P ) ; le nom de Moïse du livre (מספר) de Dieu Ex 32,32 ; Ex 32,33 ( J ) ; וּמָחִיתִי אֶתֿ יְרוּשָׁלִַם כַּאֲשֶׁר יִמְחֶה אֶתהַֿצַּלַּחַת מָחָה וְהָפַךְ עַלפָּֿנֶיהָ 2 R 21,13 et j'essuierai Jérusalem comme on essuie un plat, — il essuie et le retourne (mais lire probablement מָחֹה וְהָפֹךְ) sens dessus dessous.
2. effacer = oblitérer de la mémoire, מִתַּחַת הַשָּׁמַיִם de dessous les cieux p. ex. le nom Dt 9,14 ; Dt 29,19 ; 2 R 14,27 le souvenir Ex 17,14 (deux fois dans le verset) ( J ), Dt 25,19 le nom pour toujours Ps 9,6 les transgressions (פשׁע ; plus rappelées par Dieu contre le pécheur) Ps 51,3 ; Es 43,25 ; Es 44,22 עָון Ps 51,11.
3. effacer = exterminer, כלהֿיקוֺם tous les êtres existants Gn 7,23 ( J ) et l'humanité, מֵעַל פְּנֵי הָאֲדָמָה Gn 6,7 ; Gn 7,4 ( J ) ; מֹחוֺת מְלָכִין Pr 31,3 (avec la correction textuelle, voir ci-dessus) = destructrices de rois (c.-à-d. femmes impures), mais l'expression est étrange et douteuse.
