BDB2505 [H8458]

hébreu biblique

[תַּחְבֻּלָה] nom féminin direction, conseil (probablement à l'origine du tirage de corde, c.-à-d. pilotage, direction d'un navire, voir חֹבֵּל , ainsi Thes ; voir De Pr 1,5) ; — seulement pluriel et seulement littérature sapientielle. תַּחְבֻּלוֺת absolu Pr 1,5 + 3 t. ; construit Pr 12,5 suffixe תַּחבּוּלֹת֯ו Jb 37,12 — direction, conduite (c.-à-d. par Dieu) Jb 37,12 (du nuage de tonnerre) ; desseins des méchants (רשׁעים) Pr 12,5 (opposé à מַחְשְׁבוֺת צדיקים) ; généralement bons, sages conseils Pr 1,5 ; Pr 11,14 (|| רֹב יועֵץ), Pr 24,6 (|| id.) ; Pr 20,18 (|| עֵצָה).
