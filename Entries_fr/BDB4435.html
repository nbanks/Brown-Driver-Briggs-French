hébreu biblique

[לָעַג] verbe se moquer, railler ; aussi (Niph`al) bégayer, en poésie (hébreu tardif Hiph`il id. ; ᵑ7 לְעֵג Aph`el id. ; bégayer (comparer hébreu tardif לִגְלֵג se moquer, ᵑ7 לַגְלֵג id. ; syriaque bégayer, arabe id.)) ; —
