hébreu biblique

הִין nom masculin Lv 19,36 hin, une mesure de liquide ( ᵐ5 εἴν (Ἴν; ᵐ5 L Ἴν); mais χοῦς Lv 19,36 grec ἴυιον, mesure égyptienne, = sextarius, selon BöckhMetrol. Unters. 244 comparer BrandisMünz- Mass- & Gewichtswesen 29, 30 ; le hin hébreu (6,06 litres) était cependant bien plus grand que le hin égyptien (0,456 litre), voir HultschMetrol., Tab. xxi. p. 714, et pp. 368, 450 ss. ; voir en outre RiHWB SmithDict.Bib. EMeyGeschichte. i. 228) — absolu הִין Ex 30,24 + 4 t. Ez 45,46 הַהִין Ex 29,40 (deux fois dans le verset) + 14 t. ; construit הִין Lv 19,36 —

1. une mesure, utilisée pour la quantité d'eau à boire Ez 4,11 habituellement en rapport avec les offrandes :
a. d'huile utilisée avec l'offrande végétale Ex 29,40 ; Nb 15,4 ; Nb 15,6 ; Nb 15,9 ; Nb 28,5 (tous P ), Ez 45,24 ; Ez 46,5 ; Ez 46,6 ; Ez 46,11 ; Ez 46,14 — habituellement 1/4 de hin pour 1/10 d'épha (d'offrande végétale) ; Nb 15,6 1/3 ; v Nb 15,9 1/2 ; dans Ézéchiel un (entier) hin, sauf Ez 46,14 1/3 de hin pour 1/6 d'épha.
b. utilisé dans la préparation de l'huile d'onction Ex 30,24.
c. de vin comme libation Lv 23,13 ( H ); ailleurs P , Ex 29,40 ; Nb 15,5 ; Nb 28,14 (tous 1/4 de hin) Nb 15,7 ; Nb 28,14 (les deux 1/3 de hin), Nb 15,10 ; Nb 28,14 (les deux 1/2 hin ; tous avec 1/10 à 3/10 d'épha pour l'offrande végétale).
d. une fois de שֵׁכָר comme libation Nb 28,7 ( P ; 1/4 de hin ; comparer Di).
2. récipient d'une capacité d'un hin Lv 19,36 ( H ) הִיןצֶֿדֶק un hin juste (|| ׳ מאֹזְנֵי צ, ׳ אַבְנֵיצֿ, ׳ אֵיפַת צ). — Le hin = 1/6 de bath selon Joseph.Ant. «JOSEPHUS:Antiquities 3:8»iii. 8, 3 ; «JOSEPHUS:Antiquities 3:9»9, 4, = 6,06 litres (Hultschl.c.), voir II. בַּת . Sur l'ordre des mots dans les passages ci-dessus, noter que הִין (׳ רְבִיעִת ה etc.) est habituellement en apposition au liquide mesuré, et soit le précède Ex 29,40 ; Nb 15,4 ; Nb 28,14, soit, plus souvent, le suit Ex 30,24 + 10 t. ; Nb 28,7 ils sont dans des propositions différentes ; dans Ex 29,40 on trouve בְּשֶׁמֶןרֶ֗֗֗בַע הַהִין, comparer Nb 15,6 ; Nb 15,9 ; Nb 28,5 — Nb 28,14 est particulier : וְנִסְכֵּיהֶם חֲצִי הַהִין יִהְיֶה לַפָּר וּשְׁלִישִׁת הַהִין לָאַיִל וּרְבִיעִת הַהִין לַכֶּבֶשׂ יָ֑יִן (יין seulement à la fin).
