hébreu biblique

[עָרַץ] verbe faire trembler, trembler (de terreur ou de crainte) (comparer arabe trembler, vaciller ; syriaque fondre sur soudainement ou violemment, PS sous le mot passim ; comparer égyptien ±u-ra-¼-au(t) , ils terrifièrent (= ערצו) WMMAs.u.Eur.76) ; —

Qal Imparfait 2 masculin singulier תַּעֲרֹץ Dt 7,21 + ; 2 féminin singulier תַּעֲר֑וֺצִי Es 47,12 1 singulier אֶעֱרוֺץ Jb 31,34, etc. ; Infinitif construit עֲרֹץ Es 2,19 + ; —
1. faire trembler (c.-à-d. frapper de crainte) avec accusatif הָאָרֶץ Es 2,19 ; Es 2,21 (sujet ׳ י ; insérer probablement aussi Es 2,10) ; הֶעָלֶה נִדָּף ׳ תַּע Jb 13,25 frapperas-tu de crainte la feuille emportée ? absolu faire trembler = inspirer la crainte Es 47,12 (de la Babylonienne), Ps 10,18.
2. intransitif trembler, éprouver la terreur ( D ), Dt 1,29 ; Dt 7,21 ; Dt 20,3 ; Dt 31,6 ; Jos 1,9 avec accusatif Jb 31,34 parce que je tremblais devant la grande multitude.
