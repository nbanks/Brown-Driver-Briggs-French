BDB6398 [H6175]

hébreu biblique

עָרוּם adjectif rusé, avisé, sensé ; —

1. rusé, ׳ ע comme prédicat, du serpent Gn 3,1 pluriel comme substantif מַחְשְׁבוֺת עֲרוּמִים Jb 5,12, ׳ לְשׁוֺן ע Jb 15,5.
2. dans les Proverbes, en bonne part (opposé à אֱוִיל, כְּסִיל, פֶּתִי) ; comme attribut אָדָם עָרוּם Pr 12,23 un homme avisé ou sensé ; ׳ ע comme substantif = id., Pr 12,16 ; Pr 13,16, ainsi ׳ חָכְמַת ע Pr 13,8 = homme prudent Pr 14,15 ; Pr 22,3 ; Pr 27,12 pluriel עֲרוּמִים Pr 14,18.

עֵרָן, עֵרָנִי voir ci-dessous I. עור.
