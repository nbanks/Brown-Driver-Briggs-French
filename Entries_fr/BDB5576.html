BDB5576 [H4539]

hébreu biblique

מָסָךְ nom [masculin] couverture, écran ; — absolu ׳ מ 2 S 17,19 + 11 t. ; construit מָסַךְ Es 22,8 + 12 t. ; —

1. couverture, grande toile étendue (פרשׂ) sur un puits pour cacher des personnes à l'intérieur 2 S 17,19 figuré de protection, ou d'écran pour les yeux, יְהוּדָה ׳ וַיְגַל אֶתמֿ Es 22,8 écran, de nuage Ps 105,39 (פרשׂ).
2. nom donné (dans P ) à chacun des trois écrans, du tabernacle:
a. à la porte ( שַׁעַר) du parvis Ex 27,16 ; Ex 35,17 ; Ex 38,18 ; Ex 39,40 ; Ex 40,8 ; Ex 40,33 ; Nb 3,26 (פֶּתַח ), Nb 4,26 .
b. à l'entrée (פֶּתַח ) de la tente Ex 26,36 ; Ex 26,37 ; Ex 35,15 ; Ex 36,37 ; Ex 39,38 ; Ex 40,5 ; Ex 40,28 ; Nb 3,25 ; Nb 3,31 ; Nb 4,25,
c. ׳ פָּרֹכֶת הַמּ (voir ׳ פ), séparant le Lieu Très Saint à l'intérieur de la tente, Ex 35,12 ; Ex 39,34 ; Ex 40,21 ; Nb 4,5.
