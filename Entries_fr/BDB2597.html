BDB2597 [H2340]

hébreu biblique

חִוִּי adjectif [d'un peuple] (= villageois ?) —

1. שְׁכֶם ׳ בֶּןחֲֿמוֺר הַח Gn 34,2 ( P ), comparer ׳ צִבְעוֺן הַח Gn 36,2 (mais lire plutôt הַחֹרִי Ol Di comparer ᵐ5 Jos 9,7).
2. avec l'article comme nom propre collectif הַחִוִּי les Hivvites « engendrés » par Canaan Gn 10,17 = 1 Ch 1,15 surtout dans la liste des peuples cananéens déplacés par les Hébreux Ex 3,8 ; Ex 3,17 ; Ex 13,5 ; Ex 23,23 ; Ex 23,28 ; Ex 33,2 ; Ex 34,11 ; Dt 7,1 ; Dt 20,17 ; Jos 3,10 ; Jos 9,1 ; Jos 12,8 ; Jos 24,11 (tous J E D ), Jg 3,5 ; 1 R 9,20 = 2 Ch 8,7 comparer וְהַכְּנַעֲנִי ׳ עָרֵי הַח 2 S 24,7 (noter מִבְצַרצֹֿר, צִידוֺן juste avant) ; appliqué spécifiquement aux Gabaonites Jos 9,7 ( J E ), Jos 11,19 ( D ). — Dans Jos 11,3 ; Jg 3,3 lire probablement הַחִתִּי pour הַחִוִּי We Mey BuUrg 350 voir (en partie) ᵐ5 , Dr sur Dt 7,1.
