hébreu biblique

חֹרוֺנַיִם, חֹרֹנַיִם nom propre [d'un lieu] (possiblement deux creux, grottes, ravins, voir בֵּית חוֺרֹן ) — דֶּרֶךְ חרונים Es 15,5 ; Jr 48,3, מוֺרַד חֹרוֺנָ֑יִם Jr 48,5, חֹרֹנַיִם Jr 48,34, ville de Moab ; = MI31,32 חורנן (c.-à-d. probablement חוֺרֹנֵן) ; ᵐ5 Αρωνιειμ, Ωρωνσιμ. — Sur חֹרֹנַיִם ᵐ5 Jos 13,10 ; Jos 13,11 ; 2 S 13,34 voir We Dr et בֵּית חוֺרֹן.
