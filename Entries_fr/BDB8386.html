BDB8386 [H7607 H7608]

hébreu biblique

שְׁאֵר nom masculin Ps 73,26 chair (peut-être originellement la chair intérieure, pleine de sang, proche des os, comparer HoffmZAW iii (1883), 107, et בָּשָׂר = (originellement) chair proche de la peau) ; — ׳ שׁ absolu Ps 78,20 ; Ps 78,27, construit Mi 3,3 +, suffixe שְׁאֵרִי Jr 51,35 ; Ps 73,26, etc. ; —

1. chair :
a. comme nourriture, Ex 21,10 (HPS aussi 1 S 9,24, pour הַנִּשְׁאָר), Ps 78,20 (|| לֶחֶם), Ps 78,27 (|| עוֺף כָּנָף); figuré Mi 3,2 (|| עוֺר), Mi 3,3 (|| id.), Jr 51,35 (|| דָּם),
b. figuré pour la force physique Ps 73,26 (+ לֵבָב), Pr 5,11 (+ בָּשָׂר ).
2. dans HP = parent par le sang (c.-à-d. un proche parent):שְׁאֵר אָבִיךָ Lv 18,12 comparer Lv 18,13 ; Lv 20,19 , aussi Lv 18,17 (lire שְׁאֵרְךָ pour שַׁאֲרָה ᵐ5 Ew Dr-Wh ; comparer de. Di Baen) ; + אֵלָיו הַקָּרֹב Lv 21,2 ; Nb 27,11 בְשָׂרוֺ ׳ שׁ Lv 18,6 ; Lv 25,49.
3. = soi-même, Pr 11,17 (|| נֶפֶשׁ),
