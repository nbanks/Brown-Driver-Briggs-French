c. apparence, vue, vision Nb 8,4 ( P ).
2. en général ce qui est vu, עֵינָיו ׳ מ Es 11,3 c.-à-d. ce que ses yeux extérieurs voient (comparer 1 S 16,7 1 b), עֵינֶיהָ ׳ לִמ Ez 23,16, עְינֶיךָ אֲשֶׁר תִּרְאֶה ׳ מ Dt 28,34; Dt 28,67, ׳ לְכָלמֿ עֵינֵי הַכֹּהֵן Lv 13,12 ( P ).
3. spécifiquement une (surnaturelle) vision (dans Ézéchiel, Daniel ; souvent accusatif de sens apparenté au verbe avec ראה) : Ez 8,4 ; Ez 11,24 ; Ez 43,3 (3 t. dans le verset), lire aussi מַרְאֵה הָרֶכֶב Ez 43,3 (pour מַרְאוֺת voir I. מַרְאָה ci-dessus) ; ׳ בַּמּ Ez 11,24 ; Dn 8,16 ; Dn 8,27 ; Dn 9,23 ; Dn 10,1 ׳ מ הָעֶרֶב וְהַבֹּקֶר Dn 8,26.
4. vue, vision = faculté de voir (et de jouir), tardif : עֵינַיִם ׳ מ Qo 6,9, הַלֵךְ בְּדַרְּכֵי לִבְּךָ וּבְמַרְאֵי עֵינֶיךָ Qo 11,9 .
