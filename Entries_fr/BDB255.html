hébreu biblique

מֵאָז (comparer ), une fois (Jr 44,18) מִןאָֿז, littéralement depuis ce temps : employé

(a) absolument, comme adverbe = dans le passé, autrefois, que ce soit d'un passé proche 2 S 15,34 ; Es 16,13, ou d'un passé lointain Es 44,8 ; Es 45,21 ; Es 48,3 ; Es 48,5 ; Es 48,7 ; Es 48,8 ; Ps 93,2 ; Pr 8,22
(b) avec proposition générale ou relative suivante, comme préposition ou conjonction = depuis le temps de, depuis. Avec substantif Rt 2,7 מֵאָז הַבֹּקֶר depuis le matin Ps 76,8 avec infinitif Ex 4,10 מֵאָז דַּבֶּרְךָ depuis que tu parles à ton serviteur ; avec verbe fini Gn 39,5 ; Ex 5,23 ׳ וּמֵאָז בָּאתִי וג et depuis que je suis venu vers Pharaon, etc. Ex 9,24 ; Jos 14,10 ; Es 14,8 ; Jr 44,18.
