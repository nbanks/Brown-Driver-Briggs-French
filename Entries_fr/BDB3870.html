d. parfois כְּ est utilisé en particulier pour comparer un objet avec la classe à laquelle il appartient, et exprimer sa correspondance avec l'idée qu'il devrait réaliser : Es 13,6 (= Jl 1,15) du ׳ יום י, כְּשֹׁד מִשַּׁדֵּי יָבוֺא c.-à-d. il vient comme une véritable, ou idéale, destruction de Shaddaï, Es 29,2 d'Ariel (Es 29,1) והיתה לי כאריאל c.-à-d. sera pour moi un vrai Ariel (comparer Ges«GKC:118x»§ 118x) : ainsi aussi, selon Hi De, Es 10,13 ; Ez 26,10 ; Za 14,3 ; 2 S 9,8 ; Ps 122,3 ; Ct 8,10 (voir De), Qo 10,5 ; Ne 7,2. Les anciens grammairiens appelaient cela le Kaph veritatis (traduction de Ki's כף֞ הָאֲמִתּוּת, כף֞ לאמת הדבר, Michl45 a (Lyck), Comm Jos 3,4 ; 1 S 9,13 ; Es 1,7 ; Os 5,10, ou כף֞ הָאֲמִתִּי, Lexicon sous le mot יוֺם) : ils l'ont étendu indûment (p. ex. à Nb 11,1 ; Lm 1,20 ; Os 11,4).
3. devant un infinitif :
a. comme (littéralement avec ou dans la ressemblance de…), Jg 14,6 כְּשַׁסַּע הַגְּדִי comme le déchirement d'un chevreau, c.-à-d. comme quand un chevreau est déchiré (par un lion), 2 S 3,34 ; Es 5,24 ; Ps 66,10 ; Ps 68,3 ; Jb 2,10, etc. ; = comme si, Es 10,15 ou la scie se glorifiera-t-elle contre celui qui la manie כהניף שׁבט אתמֿרימיו avec le semblable de l'agitation d'un bâton (= comme si un bâton agitait) lui qui l'a levé !
b. de temps, environ, à, que ce soit du passé ou du futur : Gn 19,17 כהוציאם אותם à leur fait de les faire sortir, c.-à-d. quand ou lorsque ils les firent sortir Gn 24,30 ; Gn 29,13 ; Gn 39,10 ; Gn 39,13 + très souvent ; du futur, Gn 44,30 ; Gn 44,31 ; Dt 20,2 ; Dt 20,9 ; 2 S 13,28 + ; Es 28,20 b (différent de Es 28,29 a) et la couverture est (trop) étroite quand on se rassemble en. Occasionnellement avec le nom verbal, Os 13,6 כמרעיתם au temps de leur pâture, Es 30,19 כְּשָׁמְעָתוֺ ; וּכְחֶזְקָתוֺ 2 Ch 12,1 ; 2 Ch 26,16 ; Dn 11,2 ( Ew§ 238 a, 239 a) : comparer Es 23,5b כְּשֵׁמַע צֹר .
