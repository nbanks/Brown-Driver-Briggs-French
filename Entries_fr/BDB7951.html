BDB7951 [H7423]

hébreu biblique

II. רְמִיָּה nom féminin relâchement, nonchalance ; — toujours absolu ׳ ר ; — ׳ כַּף ר Pr 10,4 main relâchée (négligente, oisive) (opposé à יַד חָרוּצִים ) ; ׳ ר seul = nonchalance (abstrait pour concret = celui qui est relâché) Pr 12,24 (|| id.), Pr 12,27 (voir חרך) ; ׳ נֶפֶשׁ ר Pr 19,15 (|| עַצְלָה) ; comme adverbe עשֶֹׁה רְמִיָּה ׳ מְלֶאכֶת י Jr 48,10.
