BDB8622 [H7849]

hébreu biblique

[שָׁטַח] verbe étendre, étaler (hébreu tardif id. ; arabe étendre ; de même éthiopien araméen שְּׁטַח ) ; —

Qal Parfait 3 pluriel suffixe consécutif לַשֶּׁמֶשׁ וּטָחוּם Jr 8,2 ils les étaleront, etc., accusatif de nombreuses choses séparées, ainsi Imparfait 3 féminin singulier וַקתִּשְׁטַךְ 2 S 17,19 ( עַל lieu ; distingué de פָּרַשׂ étendre une couverture) ; 3 masculin pluriel + Infinitif absolu וַיִּשְׁטְחוּ לָהֶם שָׁטוֺחַ Nb 11,32 ( J ) ils étalèrent (c.-à-d. les cailles) pour eux tout alentour du camp (c.-à-d. pour les sécher ; ᵑ9 siccaverunt ; comparer Di Gray) ; Participe שֹׁטֵחַ לַנּוֺיִם Jb 12,23 élargissant les nations (comparer ל 3.b.).
Pi`el Parfait 1 singulier שִׁטַּחְתִּי אֵלֶיךָ כַפָּ֑י Ps 88,10 j'ai étendu vers toi mes paumes.
