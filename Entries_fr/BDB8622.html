BDB8622 [H7849]

hébreu biblique

[שָׁטַח] verbe spread, spread abroad (Late Hebrew id.; Arabic spread out; so Ethiopic Aramaic שְּׁטַח ); —

Parfait 3 pluriel suffixe consécutif לַשֶּׁמֶשׁ וּטָחוּם Perfect 3 plural suffix consecutive לַשֶּׁמֶשׁ וּטָחוּם ils les étaleront, etc., accusatif de nombreuses choses séparées, ainsi Imparfait they shall spread them abroad, etc., accusative of many separate things, so Imperfect 3 feminine singular וַקתִּשְׁטַךְ 2 S 17,19 ( עַל location; distinguished from פָּרַשׂ spread covering); 3 masculine plural + Infinitive absolute Nb 11,32 ( Num 11,32 ( ) ils étalèrent (c.-à-d. les cailles) pour eux tout alentour ) they spread (that is, the quails) for themselves all abroad siccaverunt; compare Di Gray); Participle Jb 12,23 ) ils étalèrent (c.-à-d. les cailles) pour eux tout alentourob 12,23 expanding the nations (compare ל 3.b.).
Ps 88,10 Perfect 1 singular שִׁטַּחְתִּי אֵלֶיךָ כַפָּ֑י I have spread out unto thee my palms.
