BDB3266 [H2844]

hébreu biblique

I. [חַת] nom masculin Gn 9,2 terreur, crainte ; — חָ֑ת Jb 41,25, חִתְּכֶם Gn 9,2 — avec suffixe comme génitif objectif, Gn 9,2 la terreur de vous (|| מוֺרַאֲכֶם) ; dans la description du crocodile avec négation, הֶעָשׂוּ לִבְלִיחָֿ֑ת Jb 41,25 un être fait pour ne pas connaître la peur.
