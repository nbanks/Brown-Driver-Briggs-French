hébreu biblique

תַּנִּין (erroné תַּנִּים) nom masculin Ez 29,3 serpent, dragon, monstre marin (hébreu tardif id. (rare) ; arabe , emprunt à l'araméen תַּנִּינָא, Frä123 ; éthiopien ) ; — absolu ׳ ת Ex 7,9 +, יִ֯ם Ez 29,3 ; Ez 32,2 (par confusion avec le pluriel de [תַּן]) ; pluriel תַּנִּינַם Gn 1,21, etc. ; —

1. serpent, Dt 32,33 (venimeux), Ps 91,13 (|| פֶּתֶן), Ex 7,9 ; Ex 7,10 ; Ex 7,12 ( P ).
2. dragon, en tant que dévoreur (comparaison) Jr 51,34 ׳ עֵין הַתּ Ne 2,13, près de Jérusalem
3. monstre marin (ou fluvial), Gn 1,21 ( P ) ; figuré Jb 7,12 ; Ps 74,13 (c.-à-d. les Égyptiens), Es 27,1 ; Es 51,9 (|| רַהַב, personnification mythologique du chaos) ; dans l'appel à louer ׳ י Ps 148,7 (|| תְּהֹמוֺת). — Voir CheEncy.
