hébreu biblique

חָלָב nom masculin lait (hébreu tardif id. (et verbe dénominatif) ; araméen חַלָבָּא , , phénicien חלב ; arabe nom lait frais, verbe traire ; assyrien alibu , lait, voir DlPr 174 ; éthiopien ) — ׳ ח absolu Gn 18,18 + 35 t. + Ez 34,3 (voir ci-dessous) ; construit חֲלֵב Ex 23,19 + 5 t. ; suffixe חֲלָבִי Ct 5,1, חֲלָבֵךְ Ez 25,4 — lait :

a. comme aliment courant Gn 18,8 ( J ; distinct de חֶמְאָה lait caillé, q. v.), חֲלֵב צאֹן Dt 32,14 (poésie ; distinct de חֶמְאַת בָּקָר) ; dans Jg 5,25 חָלָב est utilisé en parallèle avec חֶמְאָה ; ailleurs חֶמְאָה semble être produit à partir de חָלָב : — מִיץ חָלָב יוֺצִיא חֶמְאָה Pr 30,33 un barattage de lait produit du lait caillé ; מֵרֹב עֲשׂוֺת חָלָב יאֹכַל חֶמְאָה Es 7,22 à cause de l'abondante production de lait il mangera du lait caillé ; comparer aussi חֲרִצֵי הֶחָלָב 1 S 17,18 morceaux de lait, c.-à-d. fromages ; il était reçu dans des seaux ou baquets ( עֲטִינָיו) Jb 21,24 (voir Ew De Di VB ), et gardé dans des outres (׳ נאֹוד ח) Jg 4,19 les hommes le buvaient (שׁתה) Ez 25,4 (comparer השׁקה Jg 4,19), mais aussi le mangeaient (אכל), si l'émendation הֶחָלָב Ez 34,3 (pour הַחֵלֶב) est juste, ᵐ5 ᵑ9 Bo Hi-Sm Co, et autres ; il était versé, voir comparaison de la formation de l'embryon תַּתִּיכֵנִי ׳ הֲלֹא כֶח Jb 10,10 ne m'as-tu pas versé comme du lait ? spécifiquement (1) חֲלֵב עִזִּים לְלַחְמְךָ Pr 27,27 lait de chèvres pour ta nourriture (RSSemitic i. 204 ). (2) avec אֵם lait maternel : du chevreau לֹאתְֿבַשֵּׁל גְּדִי בַּחֲלֵב אִמּוֺ Ex 23,19 = Ex 34,26 ( J E ) = Dt 14,21, comparer ׳ טְלֵה ח 1 S 7,9 agneau de lait (pour le sacrifice) ; mais aussi d'êtres humains ׳ גְּמוּלֵי מֵח Es 28,9 sevrés du lait (|| עַתִּיקֵי מִשָּׁדָ֫יִם) ; figuré dans la promesse à Sion וְיָנַקְתְּ חֲלֵב גּוֺיִם Es 60,16 et tu suceras le lait des nations (|| תִּינָ֑קִי וְשֹׁד מְלָכִים). (3) ׳ ח avec le vin, comme mets particulièrement délicat Ct 5,1 (voir De), figuré des ׳ י's bénédictions les plus choisies Es 55,1. (4) souvent dans l'expression de la productivité de la terre de Canaan (דְּבַשׁ ) זָבַת חָלָב וּדְבָ֫שׁ ruisselant de lait et de miel Ex 3,8 ; Ex 3,17 ; Ex 13,5 ; Ex 33,3 ; Nb 13,27 ; Nb 14,8 ; Nb 16,13 ; Nb 16,14 (tous J E ), Lv 20,24 ( H ), Dt 6,3 ; Dt 11,9 ; Dt 26,9 ; Dt 26,15 ; Dt 27,3 ; Dt 31,20 ; Jos 5,6 ; Jr 11,5 ; Jr 32,22 ; Ez 20,6 ; Ez 20,15 au temps messianique les collines ׳ תֵּלַכְנָה ח Jl 4,18 (|| יִטְּפוּ הֶהָרִים עָסִיס) ; figuré des charmes de la bien-aimée תַּחַת לְשׁוֺנֵךְ ׳ דְּבַשׁ וְח Ct 4,12 miel et lait sont sous ta langue.
b. lait comme blanc ׳ לְבֶןשִֿׁנַּיִם מֵח Gn 49,12 blanc des dents, du lait ; ׳ צַחוּ מֵח Lm 4,7 ils étaient plus blancs que le lait (|| זַכּוּ מִשֶּׁלֶג) ; de même aussi probablement des yeux (iris) ׳ רֹחֲצוֺת בֶּח Ct 5,12 baignés de lait, c.-à-d. le blanc de l'œil. — Sur le lait comme non utilisé par Israël dans le sacrifice voir RSSemitic 204.
