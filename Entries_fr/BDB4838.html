BDB4838 [H4846]

hébreu biblique

[מְרֹרָה] nom féminin chose amère, fiel, poison ; construit מְרוֺרַת Jb 20,14 suffixe מְרֹרָתוֺ Jb 20,25 pluriel מְרֹרֹת Dt 32,32, מְרֹרוֺת Jb 13,36 —

1. fiel, c.-à-d. siège du fiel, vésicule biliaire Jb 20,25 (|| גֵּוָה dos ; comparer מְרֵרָה Jb 16,30 ).
2. = poison, פְּתָנִים ׳ מ Jb 20,14.
3. מְרֹרוֺת chose amère Jb 13,26 (de l'arrêt de jugement de Dieu).
4. pluriel abstrait = amertume ; אַשְׁכְּלֹת ׳ מ Dt 32,32 grappes d'amertume (|| עִנְּבְי רוֺשׁ).
