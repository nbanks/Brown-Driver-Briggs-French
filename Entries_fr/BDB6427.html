BDB6427 [H6212]

hébreu biblique

עֵ֫שֶׂב nom masculin Gn 1,11 herbe, végétation; — ׳ ע absolu Dt 11,15 + ; construit Gn 2,5 + ; suffixe עֶשְׂבָּם Es 42,15 pluriel construit עִשְּׂבוֺת (dagesh forte dirimens) Pr 27,25 — herbe, végétation, dans l'usage le plus ancien souvent général (incluant donc l'herbe [דֶּשֶׁא]): הַשָּׂדֶה ׳ ע Ex 9,22 ( E ), Gn 2,5 (+ שִׂיחַ) Ex 9,25 (+ עֵץ), Ex 10,15 (+ id; tous J ) Am 7,2, ainsi הָאָרֶץ ׳ ע Ex 10,12 ( E ), + פְּרִי הָעֵץ Ex 10,15 ( J ), d'où Ps 105,35 apparemment aussi de manière générale dans הארץ ׳ ע Jb 5,25; Ps 72,16, ׳ ע seul Ps 92,8 כָּלהַֿשָּׂדֶה ׳ ע Jr 12,14, ׳ ע seul Dt 32,2 (|| דֶּשֶׁא), Es 42,15; Ps 102,5; Ps 102,12, et Dt 29,22; Mi 5,6; Pr 19,12 d'où ׳ ע comme nourriture pour le bétail Dt 11,15; Ps 106,20 comparer Jr 14,6; Za 10,1 עִשְּׂבוֺת הָרִים Pr 27,25 (|| דֶּשֶׁא, חָצִיר); mais comme nourriture humaine (excluant donc דֶּשֶׁא) הַשָּׂדֶה ׳ ע Gn 3,18 ( J ); surtout P Gn 1,29 (+ עֶץ), ׳ יֶרֶק ע Gn 1,30; Gn 9,3 , d'où ׳ ע Ps 104,14 (|| חָצִיר pour le bétail); ׳ ע distingué de דֶּשֶׁא, עֵץ aussi Gn 1,11; Gn 1,12, שָׂדֶה ׳ ע + דֶּשֶׁא, חָצִיר 2 R 19,26 + Es 37,27.
