I. זֶ֫בַח162 nom masculin Jg 16,23 sacrifice

phénicien זבח d'une offrande de pain et d'huile CISi. 165, 12 ; 167. 8, comparer RSSemitic 205 (2e éd. 222). ׳ ז absolu Gn 31,54 + ; construit Ex 34,25 + ; זָ֑בַח Lv 17,8 + 16 t. ; suffixe זִבְחוֺ Lv 7,16 ; Ez 34,15, זִבְחֲכֶם Lv 19,6 pluriel זְבָחִים Gn 46,1 + ; construit זִבְחֵי Lv 17,5 + ; suffixe 3 masculin pluriel זִבְחֵיהֶם Lv 17,5, זְבָחֵימוֺ Dt 32,38, זִבְחֹתָם Os 4,19 etc. ; — le sacrifice commun et le plus ancien, dont le rite essentiel était de manger la chair de la victime lors d'un festin auquel le dieu du clan participait en recevant le sang et les morceaux de graisse. Dans la littérature ancienne il est distingué de מִנְחָה et עֹלָה, dans la littérature tardive de חַטָּאת et אָשָׁם.
I.
nom générique pour tous les sacrifices mangés lors de festins : —
1. du Dieu d'Israël Gn 46,1 ; Ex 10,25 ; Ex 18,12 ( E ) Os 3,4 ; Os 6,6 ; Os 9,4 ; Am 5,25 ; Es 1,11 ; Es 19,21 ; Dt 12,27 ; Dt 18,3 ; 1 S 2,13 ; 1 S 2,29 ; 1 S 3,14 ; 1 S 6,15 ; 1 S 9,12 ; 1 S 9,13 ; 1 S 15,22 (deux fois dans le verset) ; 1 S 16,3 ; 1 S 16,5 (deux fois dans le verset) ; 2 S 15,12 ; 1 R 8,62 ; 1 R 12,27 ; 2 R 5,17 ; 2 R 16,15 ; Ps 40,7 ; Ps 50,8 ; Ps 51,18 ; Ps 51,19 ; Pr 15,8 ; Pr 21,3 ; Pr 21,27 ; Es 43,23 ; Es 43,24 ; Es 56,7 ; Jr 6,20 ; Jr 7,21 ; Jr 7,22 ; Jr 17,26 ; Jr 33,18 ; Ez 40,42 ; Ez 44,11 ; 1 Ch 29,21 (deux fois dans le verset) ; 2 Ch 7,1 ; 2 Ch 7,4 ; 2 Ch 7,5 ; 2 Ch 7,12 ; Ne 12,43 ; Dn 9,27 ; Qo 4,17. Ils devraient être זִבְחֵי צֶדֶק sacrifices de justice (offerts en justice par les justes) Dt 33,19 ; Ps 4,6 ; Ps 51,21 זִבְחֵי הַבְהָבַי sacrifices de dons à moi (׳ י) Os 8,13 le temple est בֵּית זָבַ֑ח 2 Ch 7,12 ils ne devraient pas être רִיב ׳ ז sacrifices de querelle, où la querelle prévaut Pr 17,1.
2. d'autres divinités Ex 34,15 ; Nb 25,2 ; Dt 32,38 ( J E ) Jg 16,23 ; 2 R 10,19 ; 2 R 10,24 ; Os 4,19 ; Es 57,7 ; Ez 20,28 ceux-ci sont זִבְחֵי מֵתִים sacrifices aux morts (dieux sans vie, opposé à ׳ י comme vivant), Ps 106,28.
II.
il y a plusieurs sortes de זְבָחִים qui sont graduellement distingués :
1. le sacrifice d'alliance, entre Jacob et Laban Gn 31,54 ( E ), avec Dieu Ps 50,5 (comparer Ex 24,5 ; 1 S 11,15).
2. la pâque זֶבַח חַג הַפֶּסַח Ex 34,25 ( J E ), זבח פסח Ex 12,27 ( J ) ; et probablement aussi זבחי Ex 23,18 ( J E ) = Ex 34,25 (id.)
3. sacrifice annuel זֶבַח הַיָּמִים 1 S 1,21 ; 1 S 2,19 ; 1 S 20,6, comparer מִשְׁפָּחָה ׳ ז sacrifice du clan 1 S 20,29.
4. sacrifice d'action de grâce (הַ)תּוֺדָה ׳ ז Lv 7,12 ; Lv 22,29 ; Ps 107,22 ; Ps 116,17, qui est donné comme קָרְבָּנוֺ ׳ ז, זִבְחוֺ, הַזָּבַח Lv 7,16 ; Lv 7,17 et comme une variété de שְׁלָמִים Lv 7,13 ; Lv 7,15 il est impliqué dans זִבְחֵי תְרוּעָה Ps 27,6 pourtant זֶבַח est générique avec תּוֺדָה 2 Ch 29,31 (deux fois dans le verset).
5. dans le rituel de H et P זבחים sont définis par שׁלמים (q. v.) Lv 3,1 + 39 t. Lévitique, Nombres, 1 S 10,8 ; 1 R 8,63 ; 2 Ch 30,22 ; 2 Ch 33,16 ; Pr 7,14, mais parfois (rarement) ׳ ז seul est employé dans ce sens, avec ou sans suffixe, quand le sens est clair d'après le contexte : Lv 17,5 ; Lv 17,7 ; Lv 17,8 ; Lv 19,6 ; Lv 23,37 ( H ) Nb 15,3 ; Nb 15,5 ; Nb 15,8 ; Jos 22,26 ; Jos 22,29 (tous P ). Ces offrandes rituelles sont de trois variétés תּוֺדָה, נֶדֶר et נְדָבָה Lv 7,12 ; Lv 7,16 ( P ) ; dans les diverses listes זבחים sont distingués de תודות Am 2,31, de נדרים Nb 15,8 ( P ) Dt 12,6 ; Dt 12,11 ; Jon 1,16 , de נדבות Am 4,4s. ; Dt 12,6 et du plus englobant שׁלמים Nb 15,8 ; Jos 22,27 ( P ). Ainsi le sacrifice lors de l'institution de l'alliance à Horeb Ex 24,5 ( J E ) et le sacrifice lors de l'installation de Saül 1 S 11,5 sont définis comme זבחים שׁלמים. Ainsi ׳ ז semble non seulement être employé pour toutes ces formes spéciales mais aussi inclure d'autres sacrifices festifs non définis dans les codes de loi. Le rituel était le même pour la classe entière. C'étaient tous des sacrifices pour des festins dans lesquels la chair de la victime était mangée par les offrants, sauf dans la mesure où les prêtres officiants avaient certains morceaux de choix et le sang et les morceaux de graisse allaient à l'autel pour Dieu. Le sacrifice lors de l'institution de l'alliance à Horeb, la victime pascale et le bélier de consécration étaient spéciaux, en ce qu'il y avait une application d'une portion du sang aux personnes et aux choses qui devaient être consacrées, Ex 12,22 ; Ex 24,8 ; Lv 8,23.
6. le massacre des nations ennemies est aussi un זֶבַח offert par Dieu lui-même, dans lequel les vautours dévorent la chair des victimes Es 34,6 ; Jr 46,10 ; Ez 39,17 (deux fois dans le verset) ; Ez 39,19 ; So 1,7 ; So 1,8.
7. le verbe employé avec זֶבַח est ordinairement זָבַח mais d'autres aussi sont attestés, p. ex. עָשָׂה Nb 6,17 ; Nb 15,3 ; Nb 15,5 ; Nb 15,8 ; Jos 22,23 ( P ) 1 R 12,27 ; 2 R 5,17 ; 2 R 10,24 ; Jr 33,18 (comparer sabéen עסי דֿבח Hal148 DHMZMG 1888, 374) ; הֵבִיא Am 4,4 ; Dt 12,6 ; Dt 12,11 ; Jr 17,26, הִגִּישׁ Am 5,25, הִקְרִיב Lv 7,11 ; Lv 7,16 ; Lv 22,21 (tous P ), עָבַד Es 19,21, נָתַן Qo 4,17, יַעֲלָה עֹלָה אוֺ זֶבַח Lv 17,8 ( H ), ׳ בִּשֵּׁל ז Ez 46,24 , שָׁחַט אֶתהָֿעוֺלָה וְהַזָּבַ֑ח Ez 40,42 comparer Ez 44,11 .
