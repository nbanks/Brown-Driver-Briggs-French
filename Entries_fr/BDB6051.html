BDB6051 [H5908]

hébreu biblique

עַכָּבִישׁ nom masculin araignée ( ᵑ7 עַכּוּבִיתָא, עַכָּבִיתָא ; d'où arabe comme emprunt ; hébreu tardif עַכְשׁוּב probablement tarentule, selon LevyNHWB iii.648) ; — ׳ בֵּית ע Jb 8,14 c.-à-d. toile, ainsi probablement Jb 27,18 (lisant ׳ ע pour ᵑ0 עָשׁ) ; comparer ׳ קוּרֵי ע Es 59,5 filaments d'une araignée.
