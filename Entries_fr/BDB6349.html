hébreu biblique

II. עָרַב verbe prendre en gage, donner en gage, échanger (hébreu tardif se porter garant pour (rare) ; phénicien ערב garant ; ancien araméen ערבא gage ; ᵑ7 עֲרַב, syriaque , se porter garant pour ; habituellement identifié avec I. ערב — mélanger, échanger, mettre en gage — mais très incertain ; comparer BuhlLex LagBN 203) ; —

Qal Parfait 3 masculin singulier ׳ ע Gn 44,32 + 4 t. ; 2 masculin singulier עָרַבְתָּ Pr 6,1 Imparfait 1 singulier suffixe אֶעֶרְבֶנּוּ Gn 43,9 Impératif עֲרֹב Ps 119,122 , suffixe עָרְבֵנִי Es 38,14, ainsi Jb 17,3 (mais voir ci-dessous) ; Infinitif construit לַעֲרֹב Ez 27,9 Participe עֹרֵב Pr 17,18, etc. ; —
1. avec accusatif de la personne prise en gage, c.-à-d. se porter garant pour la sécurité de, Gn 43,9 ( J ), Gn 44,32 ( J ; + מֵעִם personne) ; pour les dettes de Pr 11,15 ; Pr 20,16 ; Pr 27,13 de Dieu Es 38,14 sois garant pour moi, Ps 119,122, ainsi Jb 17,3 (si vera lectio, voir עֵרָבוֺן ) ; accusatif de la personne omis, לְ ׳ ע Pr 6,1 se porter garant envers quelqu'un (en faveur d'un autre, || תָּקַע לַזָּר כַּפֶּ֑יךָ) ; avec accusatif de la chose Pr 22,26 (|| תֹּקְעֵיכָֿ֑ף).
2. donner en gage, avec accusatif de la chose Ne 5,3 (c.-à-d. hypothéquer), figuré אֶתלִֿבּוֺ ׳ ע Jr 30,21 ainsi עֲרֻבָּה לִפְנֵי רֵעֵהוּ ׳ ע Pr 17,18 (|| תֹּקֵעַ כָּ֑ף).
3. échanger (dans le commerce), avec accusatif de sens apparenté au verbe מָעֲרָבֵךְ Ez 27,9 ; Ez 27,27.
