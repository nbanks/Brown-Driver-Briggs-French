hébreu biblique

קָדִים69 nom masculin Os 13,15 est, vent d'est ; — absolu ׳ ק Gn 41,6 + ; avec ה locatif קָדִ֫ימָה Ez 11,1 +, קָדִ֫ימָה Ez 48,4 + ; —

1. (en dehors d'Ézéchiel) est, surtout ׳ רוּחַ (הַ)קּ vent de l'est, vent d'est, souvent violent et brûlant, venant du désert au sud-est, = sirocco arabe ( oriental), comparer Dr Am 4,9: Ex 10,13 (deux fois dans le verset) ( J E ), Ex 14,21 ( J ), Jr 18,17 (comparaison), Jon 4,8 ; Ps 48,8 aussi ׳ ק (רוּחַ omis) = vent d'est Gn 41,6 ; Gn 41,23 ; Gn 41,27 ( E ), Os 12,2 ; Os 13,15 ; Os 27,21 ; Os 38,24 ; Ps 78,26. — קָדִימָה Ha 1,9 est douteux, habituellement vers l'est, c.-à-d. en avant, mais texte probablement corrompu, comparer We Now Da, GASm propose מַקְדִּים (voir aussi [מְגַמָּה] p. 169 ci-dessus).
2. dans Ézéchiel (52 t.) :
a. est ׳ רוּחַ הַקּ vent d'est Ez 17,10 ; Ez 19,12 ; Ez 27,26 côté est Ez 42,16 (+ nord, sud, ouest) ; ׳ שַׁעַר הַקּ Ez 40,44 (+ nord, sud) ; ׳ פְּאַת ק Ez 47,18 , + ouest Ez 48,6 ; Ez 48,7 ; Ez 48,8 , comparer Ez 48,1 (׳ ק ׳ פ הַיָּם ; mais lire יָמָּה ׳ יְעַד פ ׳ ק ׳ מִפּ ᵐ5 Co Toy et autres), + nord, sud, ouest, Ez 48,16 ׳ דֶּרֶךְ הַ קּ c.-à-d. vers l'est Ez 40,10 + 7 t., ׳ הַקּ ׳ מִדּ Ez 43,2 ׳ לַקּ Ez 40,23 vers l'est (+ nord) Ez 41,14 ׳ מֵהַקּ Ez 42,9 à l'est.
b. קָדִ֫ימָה = est : ׳ דֶּרֶךְ הַקּ Ez 40,6 גְּבוּל קָדִימָה Ez 45,7 ; Ez 48,21 (+ ouest) ; פְּאַת קָדִימָה Ez 47,18 , + ouest Ez 48,3 + 8 t., + nord, sud, ouest, Ez 48,32.
c. comme adverbe קָדִ֫ימָה vers l'est Ez 11,1 ; Ez 44,1 ; Ez 47,1 + ouest Ez 45,7 (supprimer Berthol Krae), Ez 48,18 , + nord, sud, ouest, Ez 48,10 Ez 48,17 aussi (הַ)קָּדִים vers l'est Ez 40,19 ; Ez 43,17 ; Ez 46,1 ; Ez 46,12 ; Ez 47,1 ; Ez 47,3 , + nord Ez 47,2.
