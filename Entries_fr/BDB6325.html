BDB6325 [H6127]

hébreu biblique

[עָקַל] verbe. plier, tordre (hébreu tardif surtout dans les dérivés ; ᵑ7 dans les dérivés ; syriaque tordre, dans les espèces dérivées ; arabe lier ensemble la jambe antérieure pliée et le bras du chameau) ; —

Pu`al être déformé, tordu, Participe figuré מִשְׁפּט מְעֻקָּל Ha 1,4 justice tordue.
