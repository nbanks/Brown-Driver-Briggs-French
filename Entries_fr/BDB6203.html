BDB6203 [H4617]

hébreu biblique

מַעֲנֶה nom masculin Pr 15,1 réponse ; — absolu רַֿךְ ׳ מ Pr 15,1 une réponse douce ; construit מַעֲנִה פִּיו Pr 15,23, comparer לָשׁוֺן ׳ מ Pr 16,1 (c.-à-d. capacité de répondre) ; אֱלֹהִים ׳ מ Mi 3,7 c.-à-d. réponse pour direction ; absolu aussi Pr 29,19 réponse = obéissance ; Jb 32,3 ; Jb 32,5 réplique, réfutation ; suffixe לַמַּעֲנֵהוּ Jb 16,4 = pour son but, ce à quoi il répond (voir Toy).
