BDB8500 [H4875]

hébreu biblique

מְשׁוֺאָה, מְשֹׁאָה nom féminin désolation ; —

1. singulier seulement dans ׳ שֹׁאָה וּמ So 1,15 ; Jb 30,3 ; Jb 38,27, voir le précédent ; pluriel (מַשּׁוּאוֺת, lire) מְשׁוֺאוֺת, ainsi Klo Hup-Now CheComm. Bae Du Ps 74,3 ruines (du temple) et Ps 73,18 ruines (de la vie de quelqu'un, figuré ; mais Du ici dérive de נשׁא : tromperies ; pourtant comparer le synonyme שַׁמָּה Ps 73,19).
