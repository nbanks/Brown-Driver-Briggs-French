BDB4140 [H3708]

hébreu biblique

כַּ֫עַשׂ nom masculin (variante dialectale de כַּעַם, seulement dans Job) ; — ׳ כ Jb 17,7, כָּ֑עָשׂ Jb 5,2 suffixe כַּעְשִׂי Jb 6,2 כַּעַשְׂךָ Jb 10,17 —

1. vexation, chagrin des hommes Jb 5,2 (comparer Pr 12,16 ; Pr 27,3), Pr 6,2 ; Pr 17,7.
2. vexation, colère, de Dieu ; avec כַּעַשְׂךָ עִמָּדִי Jb 10,17 ta colère contre moi.
