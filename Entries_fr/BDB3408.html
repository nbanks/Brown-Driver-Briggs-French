BDB3408 [H2994]

hébreu biblique

[יְבֵ֫מֶת] nom féminin belle-sœur (hébreu tardif יְבָמָה, ᵑ7 יְבַמְתָּא, יְבִּימְתָּא ; syriaque ) — seulement suffixe יְבִמְתּו Dt 25,7 (deux fois dans le verset) ; Dt 25,9 la femme de son frère (veuve), (comparer Gn 38,8) dans la loi du lévirat ; יְבִמְתֵּךְ Rt 1,15 (deux fois dans le verset) la femme du frère de ton mari.
