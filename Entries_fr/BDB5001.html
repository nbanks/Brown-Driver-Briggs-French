BDB5001 [H5080]

hébreu biblique

[נָדַח] verbe pousser, chasser, bannir (surtout Deutéronome, Jérémie et plus tard) (hébreu tardif id., séduire, Hiph, Niph`al ; araméen נְדַח (rare) = hébreu biblique ; éthiopien ; pousser ; arabe est pousser un navire vers le rivage) ; —

Qal Imparfait יִדַּח 2 S 14,14 Infinitif לִנְדֹּחַ Dt 20,19 — transitif
1. pousser (c.-à-d. manier) une hache, suivi de עַלֿ Dt 20,19
2. repousser, bannir suivi de מִןֿ 2 S 14,14 .
Niph`al Parfait 3 féminin singulier נִדְּחָה Jb 6,13, etc. ; (Imparfait יִדַּ֫חוּ Jr 23,12 voir דחח ) ; Participe נִדָּח 2 S 14,14 נִדָּחָה Mi 4,6 + 2 t. ; נִדַּ֫חַת Ez 34,4 ; Ez 34,16, etc. Sur נִדְחֵי Es 11,12 ; Es 56,8 ; Ps 147,2 voir Ki Köi. 317 ; —
1. être poussé : וְנִדְּחָה יָרוֺ בַּגַּרְוֶן être poussé avec la hache Dt 19,5.
2. être chassé, banni : de la dispersion en exil, suivi de שָׁם Jr 40,12 ; Jr 43,5 (|| שׁוּב) ; suivi de לִפְנֵי Jr 49,5 (|| קִבֵּץ) ; comparer aussi figuré תֻּשִׁיָּה נִדְּחָה מִמֶּנִּי Jb 6,13 est bannie de moi, c.-à-d. est inexistante ; surtout Participe banni(s), exilé(s), comme accusatif de sens apparenté avec le verbe avec נָדַח 2 S 14,14 avec קִבֵּץ Mi 4,6 ; Dt 30,4 ; So 3,19 ; Ne 1,9 יִשִׂרָאֵל ׳ נ, avec אָסַף Es 11,12 (|| נְפֻצוֺת יְקַבֵּץ) ; avec קִבֵּץ Es 56,8 avec כִּנֵּס Ps 147,2 avec הֵשִׁיב 2 S 14,13 ; Ez 34,4 ; Ez 34,16 (tous deux || אֹבֵד) ; avec בּוֺא Es 27,13 (|| id.) ; voir aussi Es 16,3 ; Es 16,4 ; Jr 49,36 exilé = délaissé, figuré de Sion Jr 30,17 (|| דֹּרֵשׁ אֵין לָהּ).
3. chassé (du troupeau, c.-à-d.) égaré Dt 22,1 .
