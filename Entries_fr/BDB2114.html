BDB2114 [H3085]

hébreu biblique

יְהוֺעַדָּה nom propre, masculin (sens douteux ; Thes ׳ י a paré (comme hébreu עדה Hiphil ; verbe de forme Pa`el ) ; MV ׳ י est équipement, comparer arabe ; peut-être ׳ י a emporté le butin, ou a déposé ( ᵑ7 עדא Pe`al, Pa`el, Aph`el ; comparer Haph`el Dn 2,21) ; ou peut-être ׳ י a dénombré arabe ) — un descendant de Saül 1 Ch 8,36 (deux fois dans le verset) ( ᵐ5 Ἰωαδα, A Ἰωιαδα, ᵐ5 L Ἰωδα) = יַעְרָה 1 Ch 9,42 (deux fois dans le verset) (pour יֹעַדָּה ? comparer Be. ᵐ5 Ἰαδα, ᵐ5 L Ἰωδα.
