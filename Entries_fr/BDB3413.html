BDB3413 [H3004]

hébreu biblique

יַבָּשָׁה nom féminin terre ferme, sol sec (sabéen יבסם, opposé à בחרם mer, DHM dans MV ; palmyrénien יבשא terre (sèche) VogNo. 79) — toujours absolu singulier יַבָּשָׁה : Ex 4,9 ( J ) ; du sol sec comme chemin d'Israël à travers la mer Rouge Ex 14,16 ; Ex 14,22 ; Ex 14,29 ; Ex 15,19 (tous P ), Ne 9,11 ; Ps 66,6 à travers le Jourdain Jos 4,22 ( D ) ; de la terre ferme, opposé à la mer, lors de la création Gn 1,9 ; Gn 1,10 ( P ) ; comparer Jon 1,9 spécifiquement du rivage de la mer Jon 1,13 ; Jon 2,11 figuré d'Israël dans le besoin, devant être rafraîchi par l'esprit de ׳ י Es 44,3 (|| צָמֵא).
