BDB4440 [H3937]

hébreu biblique

[לָעַז] verbe parler de manière indistincte, inintelligible (hébreu tardif id., dans les dérivés ( לַעַז langue étrangère, לָעוּז étranger), aussi murmurer, protester ; syriaque parler de manière indistincte ; arabe déformer ; IV. parler obscurément, de manière ambiguë) ; — seulement

Qal Participe עַם לֹעֵז Ps 114,1 un peuple parlant de manière inintelligible (|| מִצְרַיִם).
