hébreu biblique

הָרָן nom propre, masculin (montagnard ; c.-à-d. peuple (ou pays) de la montagne ? comparer Di Gn 11,27) ; — ```

1. fils de Térah, frère d'Abram et père de Lot Gn 11,27 (deux fois dans le verset) ; Gn 11,31 (tous P ), Gn 11,28 ( J ) ; aussi père de Milka (femme de Nahor) et d'Iskah Gn 11,29 ( J ).
2. un Lévite (Guershonite) 1 Ch 23,9. — Sur הָרָן dans nom propre [d'un lieu] voir ci-dessous בית הרם, p. 111.
