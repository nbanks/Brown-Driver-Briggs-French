hébreu biblique

II. רְפָאִים nom propre [d'un peuple] ancienne race de géants (peut-être = I. ׳ ר, comme éteints et impuissants ; voir surtout WRS dans Dr Dt 2,11 ; ou comme ombrageux, vaguement connus, SchwZAW xviii (1898), 127 ss. voir aussi Stal.c.) ; — anciens habitants de Canaan [à l'ouest du Jourdain ?], Gn 15,20 ; Jos 17,15 ( J E ), comparer 1 Ch 20,4 (voir II. רָפָה ci-dessus) ; d'où ׳ עֶמֶמק ר, plaine au sud de Jérusalem 2 S 5,18 ; 2 S 5,22 ; 2 S 23,13 ; 1 Ch 11,15 ; 1 Ch 14,9 ; Es 17,5 ; Jos 15,8 ; Jos 18,16 ( P ) ; ׳ ר à l'est du Jourdain Dt 2,11 (|| עֲנָקִים ; grands, comparer Dt 2,10), Dt 2,20 (|| id. ; grands ; = Zamzummim) ; en Basan, Gn 14,5, Og le dernier d'entre eux Dt 3,11 (de taille énorme), Jos 12,4 ; Jos 13,12 ( D ) ; ׳ אֶרֶץ ר Dt 2,20 ; Dt 3,13. ᵐ5 Ραφαειν[μ] ; οἱ γίγαντες Gn 14,5 + 4 t. ; τῶν Τιτάνων 2 S 5,18 ; 2 S 5,22 + ( ᵐ5 L ) 2 S 23,13.

רִפוֺת voir רִיפָה ci-dessous ריף. p. 937.
