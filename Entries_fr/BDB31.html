BDB31 [H40]

hébreu biblique

אֲבִימֶ֫לֶךְ nom propre, masculin (Mélek (= Malik, Moloch) est père) — אֲבִימֶ֑לֶךְ Gn 20,18 + —

1. roi de Guérar Gn 20,2 ; Gn 20,3 ; Gn 20,4 +, Gn 21,22 ; Gn 21,25 (deux fois dans le verset) +, Gn 26,1 ; Gn 26,8 + (24 t. Genèse).
2. roi de Gath Ps 34,1 erreur pour אָכִישׁ, comparer 1 S 21,11 s. ; — un nom philistin mieux connu substitué à un moins connu (Hup3). > D'autres (Thes Ol De MV) pensent qu'il s'agit d'un titre des rois philistins, comparer Pharaon.
3. fils de Gédéon Jg 8,31 ; Jg 9,1 ; Jg 9,3 ; Jg 9,4 +, Jg 10,1 (40 t. Juges), 2 S 11,21.
4. prêtre, fils d'Abiathar 1 Ch 18,16 erreur pour אֲחִימֶלֶח q. v. (sabéen aussi nom propre, féminin OsianderZMG 1865, 209).
