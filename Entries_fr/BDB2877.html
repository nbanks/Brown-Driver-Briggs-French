BDB2877 [H2558]

hébreu biblique

חֹ֫מֶץ nom masculin vinaigre — ׳ ח absolu Pr 10,26 + 3 t. ; construit Nb 6,3 (deux fois dans le verset) ; — un condiment courant Rt 2,14, interdit au naziréen ( יין ׳ ח et שֵׁכָר ׳ ח) Nb 6,3 (deux fois dans le verset) ( P ), offert (par cruauté) à un homme assoiffé Ps 69,22 (figuré pour la dureté, le manque de sympathie) ; en comparaison du paresseux כַּחֹמֶץ לַשִּׁנַּיִם Pr 10,26, עַלנָֿ֑תֶר ׳ ח Pr 25,20 vinaigre sur le nitre (comparaison des chants joyeux pour le cœur accablé).
