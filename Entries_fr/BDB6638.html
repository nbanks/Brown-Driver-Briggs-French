BDB6638 [H6403]

hébreu biblique

[פָּלַט] verbe échapper (hébreu tardif id. ; phénicien dans nom propre ; arabe IV échapper, arabe vulgaire ; ancien araméen פלט Pa`el délivrer, ᵑ7 פְּלַט syriaque ) ; —

Qal Parfait 3 pluriel consécutif וּפָֽלְטוּ Ez 7,16 et (si) les rescapés d'entre eux échappent).
Pi`el causatif : Imparfait 3 masculin singulier suffixe יְפַלְּטֵם ; 2 masculin singulier תְּפַלֵּט Mi 6,14, etc. ; Impératif masculin singulier פַּלְּטָה Ps 17,13, etc. ; Participe suffixe מְפַלְּטִי Ps 18,3 + ; —
1. mettre en sécurité Mi 6,14 surtout de ׳ י, délivrer, avec accusatif de personne, Ps 22,5 ; Ps 22,9 ; Ps 31,2 ; Ps 37,40 ; Ps 71,2 (+ הִצִיל), Ps 82,4 ; Ps 91,14 + מִן de 2 S 22,44 = Ps 18,44 ; Ps 18,49 (מוֺצִיא || 2 S 22,49), 2 S 17,13 ; 2 S 37,40 ; 2 S 43,1 ; 2 S 71,4 participe avec suffixe,mon libérateur 2 S 22,2 = Ps 18,3 ; Ps 40,18 ; Ps 70,6 ; Ps 144,2.
2. faire échapper, mettre bas, d'une vache vêlant Jb 21,10 (comparer [מָלַט ] Hiph`il, aussi ᵑ7 vomir, et ainsi ).
3. apparemment intransitif être délivré, avec מִן Jb 23,7 , mais douteux ; Bu Qal ('peut-être') ; ᵐ5 Du lisent מִשְׁפָּטִי comme objet (mais ailleurs ׳ י sujet du Pi`el).
Hiph`il mettre en sécurité ; Imparfait 3 masculin singulier יַפְלִיט Es 5,29 de l'Assyrie, sous la figure du lion, emportant sa proie (opposé à הִצִּיל ) ; 2 masculin singulier תַּפְלִיט Mi 6,14 (comparer Pi`el 1).
