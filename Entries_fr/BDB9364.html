BDB9364 [H927 H926]

araméen biblique

[בְּהַל] verbe

Pa`el alarmer, épouvanter (ainsi habituellement ᵑ7 (pas syriaque) ; hébreu biblique [בָּהַל]) ; — Imparfait 3 masculin singulier suffixe יְבַהֲלָךְ Dn 4,16 et 3 masculin pluriel suffixe לֻגַּנִי- Dn 4,2 ; Dn 7,15 ; Dn 7,28 (les deux jussif Str§ 18 c M§ 52 a) ; לוּךְ- Dn 5,10, לֻנֵּ֑ךְ- Dn 4,16, לוּנֵהּ- Dn 5,6.
Hithpe`el se hâter (ainsi Pi`el en hébreu biblique tardif ; ᵑ7 Ithpe`el (rare)) ; Infinitif בְּהִתְבְּהָלָה = en hâte (à l'origine « dans l'alarme ») Dn 2,25 ; Dn 3,24 ; Dn 6,20.
Hithpa`el Participe passif מִתְבָּהַל Dn 5,9 fut grandement alarmé.
