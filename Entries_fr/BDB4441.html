hébreu biblique

[לָעַט] verbe avaler (goulûment) (hébreu tardif id., Hiph`il gaver le bétail de nourriture ; syriaque mâchoire ; comparer arabe parler confusément, émettre des sons indistincts) ; — seulement

Hiph`il Impératif masculin singulier suffixe causatif, הַלְעִיטֵנִי Gn 25,30 ( J ) laisse-moi avaler, suivi de מִן de chose.
