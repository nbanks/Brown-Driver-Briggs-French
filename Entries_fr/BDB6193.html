hébreu biblique

[עָנַד] verbe lier autour, sur (comparer arabe se détourner du chemin ; syriaque defecit, defuit ; comparer aussi עִמָּדִי (pour עִמִּי)) ; —

Qal Imparfait 1 singulier לִי אֶעֶנְרֶנּוּ Jb 31,36 (objet guirlande, dans le figuré) ; Impératif suffixe עָנְדֵם Pr 6,21 (figuratif ; || קָשַׁר).
