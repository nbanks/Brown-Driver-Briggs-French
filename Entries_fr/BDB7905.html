1. à Rephidim, ׳ מַסָּה וּמ Ex 17,7 ( J ), Λοιδόρησις.

( ᵐ5 ὕδωρ ἁντιλογίας) Dt 33,8 ; Nb 20,13 ( P ) Ps 81,8 ; Ps 106,32, aussi ( ᵐ5 ὕδωρ τῆς λοιδορίας ᵐ5 L comme ci-dessus) Nb 20,24 ( P ) ; ׳ מ construit dans מֵי מְרִיבַת קָדֵשׁ ( ᵐ5 ὕδωρ ἁντιλογίας [ἑν] Καδης) Nb 27,14 ; Dt 32,51 (tous deux P ), et ( ᵐ5 ὕδωρ βαριμωθ Καδης Ez 48,28 de même מֵי מְרִיבוֺת ׳ ק Ez 47,19 ( ᵐ5 ὕδωρ Μαριμωθ Καδης) ; Dt 33,2 voir קֹדֶשׁ vers la fin ; מְרִיבָה seul, seulement Ps 95,8 (παραπικρασμός ; || יוֺם מַסָּה ).
