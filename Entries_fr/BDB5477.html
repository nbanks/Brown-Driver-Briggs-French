BDB5477 [H5439 H4036 H5437]

hébreu biblique

סָבִיב336 substantif, employé surtout comme adverbe et préposition, circuit, tout autour : — construit סְבִיב Am 3,11 pluriel construit סְבִיבֵי Jr 32,44 ; Jr 33,13, suffixe סְבִיבֶיךָ etc. (10 t.), beaucoup plus souvent סְבִיבוֺת Ex 7,24 + 22 t., סְבִיבוֺתַי, תֶיךָ-, etc., Dt 17,14, etc. (48 t.) : —

1. au singulier —
a. comme substantif 1 Ch 11,8 מןהֿמלוא וְעַדהַֿסָּבִיב et jusqu'aux parties environnantes.
b. comme adverbe accusatif סָבִיב (dans) un circuit, c.-à-d. tout autour, Gn 23,17 בְּכָלגְּֿבֻלוֺ סָבִיב Ex 19,12 ; Ex 25,11 ; Ex 25,24 ; Ex 25,25 ; Lv 1,5 ; Lv 1,11 עַל הַמִּזְבֵּחַ סָבִיב, Jg 20,29 ; 1 R 3,1 ; 1 R 5,11 ; Ps 3,7 שָׁתוּ עָלָ֑י ׳ אֲשֶׁר ס, Ps 12,9 ; Ps 34,8 + souvent ; parfois redoublé, pour l'emphase, 2 Ch 4,3 ; Ez 8,10 ׳ ס ׳ עַל הַקִּיר ס, Ez 37,2 ; Ez 40,5 et souvent dans Ez 40-43.
c. comme préposition : (a) Psaumes, Am 3,11 צַר וּסְבִיב הָאָרֶץ détresse, et cela dans le circuit de (= tout autour ) le pays (mais lire probablement avec ᵑ6 יְסֹבֵב fera le tour). (b) ׳ סָבִיב לְ Ex 16,13 ; Ex 40,33 ; Nb 1,50 ; Nb 1,53 ; Nb 2,2 ; Jg 7,21 ; 1 R 6,5 ; 1 R 18,32 ; 1 R 18,35 ; Ez 41,10 ; Ez 14,16 ; Jb 19,12 ; Ps 34,8 ; Ps 78,28 ; Ps 125,2 (deux fois dans le verset) סָבִיב לְעַמּוֺ ׳ י, Ps 128,3 ; Ct 3,7 ; Na 3,8 . (c) étrangement, סָבִיב אֵת 1 R 6,5 (omis par ᵐ5 ), Ez 43,17.
d. מִסָּבִיב42, de tout autour, de chaque côté, Ez 16,33 ; Ez 16,37 ; Ez 23,22 ; Ez 37,21 , mais habituellement (מִן 1.c.) = de chaque côté, Jr 4,17 הָיוּ עָלֶיהָ מִסָּבִיב, Es 42,25 ׳ וַתְּלַהֲטֵהוּ מ, Jl 4,11 ; Jl 4,12 Psaumes מָגוֺר מִסָּבִיב terreur de chaque côté ! Jr 6,25 ; Jr 20,3 ; Jr 20,10 ; Jr 46,5 ; Jr 49,29 ; Ps 31,14, comparer Lm 2,22 surtout dans les expressions deutéronomiques ׳ אוֺיְבִים מ, ou ׳ הֵנִיחַ מ (parfois en combinaison), comme Dt 12,10 ׳ והניח לכם מכלאֿויביכם מ, Dt 25,19 ; Jos 21,42 ׳ לָהֶם מ ׳ וַיָּנַח י, Jos 23,1 ; Jg 2,14 ; Jg 8,34 ; 1 S 12,11 + ; מִסָּבִיב לְ de tout autour, Nb 16,24 ; Nb 16,27. — Sur 1 S 14,21 ; 2 S 24,6 ; Jb 10,8 voir סָבַב .
2. Au pluriel : —
a. סְבִיבִים : (a) au sens masculin, ceux qui sont autour, Ps 76,12 כָּלסְֿבִיבָיו יֹבִילוּ שַׁי, Ps 89,8 ; Jr 48,17 ; Jr 48,39. (b) au sens neutre, les parties environnantes, Jr 49,5 מִכָּלסְֿבִיבָ֑יִךְ, בִּסְבִיבֵי ירושׁלם Jr 32,44 ; Jr 33,13 ; Jr 21,14 וְאָֽכְלָה כָּלסְֿבִיבֶיהָ, Jr 46,14 avec la valeur d'une prép., Ps 50,3 וסביביו נשׂערה מאד, Ps 97,2 ; Lm 1,17.
b. סְבִיבוֺת (a) comme substantif (α) circuits, Qo 1,6 ועל סביבותיו שָׁב הָרוּחַ ; (β) les parties environnantes, Nb 22,4 ילחכו הקהל אתכֿלסְֿבִיבֹתֵינוּ, Jr 17,26 ; Jr 50,32 (comparer Jr 21,14 ci-dessus) ; avec référence à leurs habitants, Ps 44,19 = Ps 79,4 ׳ לעג וקלס לסב, Ez 16,57 ; Ez 28,24 ; Dn 9,16. (b) avec la valeur d'une préposition Ex 7,24 סְבִיבֹת הַיְאֹר littéralement (dans) les circuits du Nil = tout autour du Nil, Nb 11,24 ; Nb 11,31 ; Nb 11,32 ; Nb 35,2 סְבִיבֹתֵיהֶם, Jg 7,18 ; 1 S 26,5 ; 1 S 26,7 ; 2 R 6,17 סביבֹת אלישׁע, Ps 18,12 ; Ps 27,6 על אֹיְבַי סביבותי, etc. ; souvent idiomatique précédé de אשׁר, comme Gn 33,5 הערים אשׁר סביבותיהם Gn 41,48 ; Lv 25,44 ; Nb 16,34 ; Dt 6,14 ; Dt 13,8 ; Dt 17,14 ; Dt 21,2 +. Dans le même sens מִסְּבִיבוֺתָם Ez 28,26 .
