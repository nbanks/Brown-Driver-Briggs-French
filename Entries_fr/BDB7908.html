hébreu biblique

מְרִיב בַּ֫עַל nom propre, masculin (Baal est (notre, mon, son) avocat (?), comparer NesEg.120 Dr284, 4 ; mais GrauProp. N. 201 pense que מרי בעל est la forme originelle = héros de Baal) ; —

1. fils de Jonathan 1 Ch 8,34, בָּ֑עַל ׳ מ V:34 ; 1 Ch 9,40 = מְרִיבַֿעַל (err. ?) V:40 = מְפִיבשֶׁת (changement ultérieur, voir בּשֶׁת ci-dessous בושׁ) 2 S 4,4 ; 2 S 9,6 (deux fois dans le verset) ; 2 S 9,10 ; 2 S 9,11 ; 2 S 9,12 (deux fois dans le verset) ; 2 S 9,13 ; 2 S 16,1 ; 2 S 16,4 ; 2 S 19,26 ; 2 S 19,31 ; 2 S 21,7, מְפִבשֶׁת 2 S 16,25 (appelé בֶּןשָֿׁאוּל) ; ᵐ5 dans 2 Μεμφι Βοσθε, dans 1 Chroniques (habituellement) Μεριβααλ ; ᵐ5 L Mεμφιβααλ dans 2 Samuel (sauf 2 S 4,1 ; 2 S 4,2 etc., où absent de HCT).
2. מְפִבשֶׁת fils de Saül et Ritspa, selon 2 S 21,8, ᵐ5 (y compris ᵐ5 L ) Μεμφιβοσθε.
