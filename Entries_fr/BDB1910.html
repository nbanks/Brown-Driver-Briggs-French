BDB1910 [H1767]

hébreu biblique

דַּי substantif suffisance, assez (hébreu tardif, mais inconnu dans les autres langues apparentées) — construit דֵּי, avec suffixe (voir ci-dessous) דַּיֶּ֑ךָּ, דַּיָּם : —

1. absolu seulement trois fois Ml 3,10 je vous verserai une bénédiction עַדבְּֿלִידָֽֿי jusqu'à ce qu'il n'y ait plus assez, c.-à-d. jusqu'à ce que mon abondance puisse être épuisée, ou, comme cela ne peut jamais être, pour toujours (comparer Ps 72,7 ) ; Est 1,18 וּכְדַי בִּזָּיוֺן וָקָ֑צֶף et (il y aura) comme en abondance (c.-à-d. en abondance) mépris et colère ; 2 Ch 30,3 לְמַדַּי = לְמַהדַּֿי pour ce qui était suffisant. Avec un génitif de la personne ou de la chose pour laquelle quelque chose suffit : Ex 36,5 מִדֵּי הָעֲבֹדָה (plus) qu'assez pour l'ouvrage ; Lv 5,7 et si sa main n'atteint pas (s'il ne dispose pas de) דֵּי שֶׂה assez pour (c.-à-d. pour acheter) un agneau ; Lv 12,8 ; Lv 25,28 דֵּי הָשִׁיב לוֺ assez pour le recouvrer ; Dt 15,8 דֵּי מַחְסֹרוֺ assez pour son besoin ; Es 40,16 (deux fois dans le verset). Avec suffixe Pr 25,16 אֱכֹל דַּיֶּ֑ךָּ mange ce qui est suffisant pour toi ; דַּיָּם Ex 36,7 ; Jr 49,9 ; Ab 1,5. Une fois avec génitif de la chose qui est suffisante ; Pr 27,27 דֵּי חֲלֵב עִזִּים assez de lait de chèvres.
