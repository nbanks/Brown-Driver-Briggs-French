BDB6489 [H6275]

hébreu biblique

[עָתֵק] verbe se déplacer, avancer, progresser (hébreu tardif Hiph`il ; assyrien etêku comme H., aussi transitif amener, déplacer ; arabe , précéder, sortir, devenir libre, vieillir ; araméen , עֲתַק vieillir, palmyrénien adjectif עתק vieux) ; — ```

Qal Parfait 3 féminin singulier עָֽתְקָה Ps 6,8, 3 pluriel עָֽתְקוּ Jb 21,7 Imparfait 3 masculin singulier יֶעְתַּק Jb 14,18; Jb 18,4 —
1. se déplacer מִמְּקֹמוֺ ׳ וְצוּר יֶע Jb 14,18, de même Jb 18,4.
2. avancer (en années ; sens araméen) Jb 21,7 (|| יִחְיוּ) ; d'où = vieillir et s'affaiblir Ps 6,8 (de l'œil, || עָֽשְׁשָׁה).
Hiph`il Parfait 3 pluriel הֶעְתִּיקוּ Jb 32,16 ( Gi ; van d. H. Baer Jb 32,15), Pr 25,1 Imparfait 3 masculin singulier וַיַּעְתֵּק Gn 12,8 ; Gn 26,22 Participe מַעְתִּיק Jb 9,5 —
1. avancer (c.-à-d. la tente) = se mettre en route Gn 12,8 (מִשָּׁם, ה locatif), Gn 26,22 (מִשָּׁם) ; ainsi figuré Jb 32,16 (voir ci-dessus) les paroles se sont éloignées d'eux (מֵהֶם), mais voir probablement glose (Bu et autres).
2. Jb 9,5 celui qui déplace les montagnes.
3. très tardif Pr 25,1 transcrire (transférer d'un livre ou rouleau à un autre ; ᵐ5 ἐξεγράψαντο, ᵑ9 transtulerunt ; de même en hébreu médiéval : ZunzZMG xxv (1871), 447 s. = Ges. Schr. iii. 66 s.).
