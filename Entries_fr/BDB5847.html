II. עָב33 nom masculin Es 19,1 et (1 R 18,44 KöSynt. 103) féminin nuage sombre, masse nuageuse, fourré ; — ׳ ע absolu 1 R 18,44 +, construit Es 18,4 ; Pr 16,15 pluriel עָבִים Jg 5,4 +, עָבוֺת 2 S 23,4 + 2 S 77,18 construit עָבֵי 2 S 22,12 ; Ps 18,12 suffixe עָבָיו Ps 18,13 (mais probablement supprimer, voir עבר Qal vers la fin) ; —

a. nuage de pluie Jg 5,4 ; Es 5,6 ; 1 R 18,44 ; 1 R 18,45 + 9 t.
b. comme élevé Es 14,14 ; Jb 20,6 , + Ez 19,11 ; Ez 31,3 ; Ez 31,10 ; Ez 31,14 (lisant בְָוֺת, voir עֲבֹת ).
