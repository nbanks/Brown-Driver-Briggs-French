BDB821 [H816]

hébreu biblique

אָשַׁם, אָשֵׁם verbe offenser, être coupable (arabe , id., péché, offense, , , rétribution, faute, culpabilité, amende, comparer éthiopien ) — אָשַׁם Lv 5,19 ; Nb 5,7 אָשֵׁם Ha 1,11 + 7 t. ; אָֽשְׁמָה Nb 5,6 אָשָׁ֑מְתָּ Pr 30,10 אָשַׁמְתְּ Ez 22,4 אָשֵׁ֑מוּ Lv 4,13 Imparfait יֶאְשַׁם Os 4,15 + 13 t. ; Infinitif absolu אָשֹׁם Lv 5,19 אָשׁוֺם Ez 25,12 —

1. commettre une offense, une transgression, faire un tort ou un dommage, avec אָשֹׁם אָשַׁם ליהוה ׃לְ il a commis un grand tort envers Yahvé (en violant les commandements) Lv 5,19 ( P ) ; וְנָתַן לַאֲשֶׁר אָשַׁם לוֺ et il le donnera (restitution) à celui envers qui il a commis le tort Nb 5,7 ( P ), comparer 2 Ch 19,10 (deux fois dans le verset) ; וַיֶאְשְׁמוּ אָשׁוֺם et ils ont commis un tort durable (tort irréparable, les Édomites contre Juda) Ez 25,12 .
2. être ou devenir coupable Jg 21,22 ; Jr 50,7 ; Os 4,15 ; Ha 1,11 dans les offenses nécessitant un sacrifice pour le péché Lv 4,13 ; Lv 4,22 ; Lv 4,27 ( P ), du sacrifice de culpabilité Lv 5,2 ; Lv 5,3 ; Lv 5,17 ; Lv 5,23 ; Nb 5,6 ( P ) ; avec ל coupable de Lv 5,4 ; Lv 5,5 ( P ), avec ב dans ou par Ez 22,4 ; Os 13,1.
3. être tenu pour coupable, subir la punition Ps 34,22 ; Ps 34,23 ; Pr 30,10 ; Es 24,6 ; Jr 2,3 ; Os 5,15 ; Os 10,2 ; Os 14,1 ; Za 11,5 ; Ez 6,6 (mais comparer שָׁמֵם).
