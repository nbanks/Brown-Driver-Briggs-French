BDB4739 [H4542]

hébreu biblique

מִסְכֵּן adjectif pauvre, seulement Qohéleth (hébreu tardif id. ; araméen id. ; PS2632 ; arabe ; éthiopien Di382 ; √ douteux ; habituellement dérivé de III. סכן q. v. ; peut-être un emprunt ; selon JenZA iv (1889), 271 f Zimib. vii (1892), 353 = assyrien muškênu, mendiant, nécessiteux) ; — toujours absolu ׳ מ : — וְחָכָם ׳ יֶלֶד מ Qo 4,13 (opposé à מֶלֶח זֶקֵן וּכְסִיל) ; חָכָם ׳ אִישׁ מ Qo 9,15, ׳ הָאִישׁ הַמּ Qo 9,15 comme substantif ׳ הַמּ Qo 9,16 le pauvre.
