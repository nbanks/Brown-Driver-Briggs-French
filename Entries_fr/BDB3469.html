[יָהַב] verbe donner (araméen יְהַב, ; arabe ; éthiopien : sabéen והב DHMZMG xxix. 1875, 614 s. ול יהב Idib. 603) — seulement

Qal Impératif הַב Pr 30,15 (deux fois dans le verset) ; emphatique הָ֫בָה Gn 11,3 + 10 t. (6 t. suivi de monosyllabique) ; הָבָ֫ה Gn 29,21 devant gutturale (Di sur Gn 28,2 Köi. 418), féminin singulier הָבִ֫י Rt 3,15 masculin pluriel הָב֫וּ ; Gn 47,16 + 16 t. ; הָ֫בוּ לִ֑י Jb 6,22 (sur ces formes voir aussi Sta§606 a Böi. 225 Ges«GKC:69»§ 69. 2, R. 2) ; —
1. donner, suivi de l'accusatif Gn 29,21( E ) donne(-moi) ma femme, Gn 47,16 ( J ), Rt 3,15 ; Za 11,12 (prix) ; de donner (c.-à-d. faire sortir, en décidant par le sort) l'Urim et le Thummim 1 S 14,41 (deux fois dans le verset) ( ᵐ5 We Dr, voir אוּרִים ) ; suivi de l'accusatif de chose + לִי Gn 30,1( E ), Jg 1,15 accusatif de chose + לָנוּ Gn 47,15 ( J ), Ps 60,13 = Ps 108,13 שָהבוּ לִ֑י Jb 6,22 (pas d'accusatif exprimé) ; absolu שַה֫ב || הַ֤ב Pr 30,15 .
2. = placer, suivi de l'accusatif (Urie) + אֶלמֿוּל פְּנֵי הַמִּלְחָמָה 2 S 11,15 placer Urie au premier rang de la bataille.

5. = allons (à l'origine accorder, permettre), devant le volontatif : Gn 11,3 ; Gn 11,4 ; Gn 11,7 ; Gn 38,16 (tous J ), Ex 1,10 ( E ).
