פָּלִיט nom masculin Am 9,1 rescapé, fugitif ; — ׳ פ absolu 2 R 9,15 + ; pluriel construit פְּלִיטֵי Jg 12,4 + ; suffixe פְּלִיטָיו Ab 1,14, etc. ; — Am 9,1 ; 2 R 9,15 ; Gn 14,13 ; Ez 24,26 ; Ez 24,27 ; Ez 33,21 ( + מִן local), Ez 33,22 ׳ שָׂרִיד וּפ Jos 8,22 ; Jr 42,17 וְשָׂרִיד ׳ פ Jr 44,14 ; Lm 2,22 avec génitif sujet Jg 12,5 fugitifs d'Éphraïm (supprimer en Jg 12,4, voir GFM), ainsi avec suffixe Ez 6,9 ; Ez 7,16 ; Ab 1,14 avec génitif objet, rescapés de, c.-à-d. échappés de Es 45,20, ainsi פְּלִיטֵי חֶרֶב Jr 44,28 ; Ez 6,8.
