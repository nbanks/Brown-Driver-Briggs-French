BDB2217 [H5419]

hébreu biblique

[תַּהְפֻּכָה] nom féminin perversité, chose perverse (seulement Proverbes sauf Dt 32,20) — Pluriel absolu תַּהְפֻּכוֺת Dt 32,20 + 8 t. ; construit id. Pr 2,14 — choses perverses, surtout des paroles Pr 2,12 ; Pr 10,32 ; Pr 23,33, comparer ׳ פִּי ת Pr 8,13, ׳ לְשׁוֺן ת Pr 10,31, et même ׳ אִישׁ ת Pr 16,28 (|| נִרְגָּן calomniateur) ; mais aussi pensées, desseins Pr 6,14 ; Pr 16,30 ׳ דּוֺר ת Dt 32,20 = génération perverse, (|| לֹאאֵֿמֻן בָּם), רָ֑ע ׳ ת Pr 2,14 (|| עֲשׂוֺת רָ֑ע).
