BDB8628 [H7860]

hébreu biblique

שֹׁטֵר nom masculin officier, fonctionnaire (proprement scribe, secrétaire ? ou du sens originel organisateur, intendant ?) ; — absolu ׳ שׁ Pr 6,7, ׳ שׁו 2 Ch 26,11 pluriel שֹׁטְרִים Dt 1,15 +, etc. ; — apparemment officier subalterne officier, judiciaire, civil ou militaire (comparer Dr Dt 1. 15 Gray Nb 11,16 ) ; organisant le peuple pour la marche Jos 1,10 ; Jos 3,2 (tous deux E ), comparer Dt 20,5 ; Dt 20,8 ; Dt 20,9 établis sur Israël par les contremaîtres égyptiens Ex 5,14, comparer Ex 5,6 Ex 5,10 Ex 5,15 Ex 5,19 (tous J ) ; ailleurs + זְקֵנִים Nb 11,16 ( E ), Dt 29,9 ; Dt 31,28 + שֹׁפְטִים Dt 16,18 ; 1 Ch 23,4 ; 1 Ch 26,29, + ׳ ז et ׳ שֹׁפ Jos 8,33 ( R D ), et ( + רָאשִׁים) Jos 23,2 ( R D ), Jos 24,1 ( E , ou R D ) ; ׳ שֹׁט comme juges mineurs aussi Dt 1,15 הַמְשָֽׁרְתִים אֶתהַֿמֶּלֶךְ ׳ שֹׁט 1 Ch 27,1, choisis parmi les Lévites 2 Ch 19,11 ; 2 Ch 34,13 (+ סוֺפְרִים, שׁוֺעֲרִים) ; comme titre d'un individu 2 Ch 26,11 (|| הַסּוֺפֵר) ; de la fourmi, אֵיןלָֿהּ קָצִין שֹׁטֵר וּמשֵׁל Pr 6,7.
