BDB989 [H957 H897]

hébreu biblique

בַּז nom [masculin] pillage, vol ; butin — ׳ בּ absolu Nb 14,3 + ; ainsi aussi Ez 25,7 Qr ( Kt בג sans signification) ; בַּ֑ז Jr 2,14 + ; mais הַבָּ֔ז Nb 31,32 voir d. H ; suffixe בִּזָּהּ Ex 29,19 —

1. pillage, vol, ׳ לְב c.-à-d. être pillé, dépouillé Ez 7,21 (du temple || שָׁלָל) Ez 23,46 .
2. butin, pillage Es 10,6 ; Es 33,23 ; Ez 29,19 ; Ez 38,12 ; Ez 38,13 (tous accusatif de sens apparenté avec verbe || שָׁלָל ) ; comparer Nb 31,32 ; Jr 15,13 ; Jr 17,3 dans l'expression הָיָה לָבַז Jr 49,32 (de chameaux) Ez 36,4 comparer Ez 36,5 aussi d'êtres humains Nb 14,3 ; Nb 14,31 ; Dt 1,39 ; Jr 2,14 (tous ׳ היה לב) + 2 R 21,14 (|| מְשִׁסָּה) Es 42,22 (|| id. ) Jr 2,14, comparer Jr 30,16 (|| מְשִׁסָּה) ; aussi Ez 25,7 Qr , Ez 26,5 (׳ היה לב ; tous deux personnifiés) ; figuré d'Israël comme brebis Ez 34,8 ; Ez 34,22 (tous deux ׳ היה לב) Ez 34,28 היה בז (sans לְ). Sur Es 8,1 ; Es 8,3 מַהֵר שָׁלָל חָשׁ בַּז voir מהר .
