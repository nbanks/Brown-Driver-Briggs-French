BDB6402 [H6203]

hébreu biblique

עֹ֫רֶף nom masculin Dt 31,27 nuque, cou ; — ׳ ע absolu Jos 7,12 + ; construit Gn 49,8 + ; suffixe עָרְפִּי Jb 16,12, עָרְפְּךָ Dt 31,27, etc. ; —

1. nuque de l'ennemi en fuite ׳ וגו ׳ יָָֽדְךָ בְּע Gn 49,8 (poème dans J ), comparer וְאָחַז בְּעָרְפִּי Jb 16,12 (figuré) ; Ex 23,27 ( E ), Ps 18,41 = 2 S 22,41 tourner (הָפַח, פָּנָה) le dos devant (לִפְנֵי) un ennemi Jos 7,8 ; Jos 7,12 ( J E ), absolu Jr 48,39 figuré d'apostasie אֶל ׳ פָּנָה ע Jr 2,27 ; Jr 32,33 (opposé à פָּנָה פָנִים) absolu ׳ וַיִּתְּנּוּ ע 2 Ch 29,6 de ׳ י's défaveur וְלֹא פָנִים אֶרְאֵם ׳ ע Jr 18,17 c'est le dos et non la face que je leur montrerai.
