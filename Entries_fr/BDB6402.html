BDB6402 [H6203]

hébreu biblique

עֹ֫רֶף nom masculin Dt 31,27 nuque, cou ; — ׳ ע absolu Jos 7,12 + ; construit Gn 49,8 + ; suffixe עָרְפִּי Jb 16,12, עָרְפְּךָ Dt 31,27, etc. ; —

1. nuque de l'ennemi en fuite ׳ וגו ׳ יָָֽדְךָ בְּע Gn 49,8 (poème dans J ), comparer וְאָחַז בְּעָרְפִּי Jb 16,12 (figuré) ; Ex 23,27 ( E ), Ps 18,41 = 2 S 22,41 tourner (הָפַח, פָּנָה) le dos devant (לִפְנֵי) un ennemi Jos 7,8 ; Jos 7,12 ( J E ), absolu Jr 48,39 figuré d'apostasie אֶל ׳ פָּנָה ע Jr 2,27 ; Jr 32,33 (opposé à פָּנָה פָנִים) absolu ׳ וַיִּתְּנּוּ ע 2 Ch 29,6 de ׳ י de la défaveur וְלֹא פָנִים אֶרְאֵם ׳ ע Jr 18,17 c'est le dos et non la face que je leur montrerai.
2. figuré dans ׳ קְשֵׁהעֿ raide de cou, c.-à-d. obstiné, d'Israël Ex 32,8 ; Ex 33,3 ; Ex 33,5 ; Ex 34,9 (tous J E ), Dt 9,6 ; Dt 9,13 עָרְפְּךָ הַקָּשֶׁה Dt 31,27 avec verbe ׳ הִקְשָׁה ע c.-à-d. être obstiné, intraitable Dt 10,16 ; Jr 7,26 ; Jr 17,23 ; Jr 19,15 ; 2 R 17,14 (deux fois dans le verset) ; Ne 9,16 ; Ne 9,17 ; Ne 9,29 ; 2 Ch 30,8 comparer ׳ גִּיד בַּרְזֶל ע Es 48,4 (|| קָשֶׁה אָ֑תָּה) ; ׳ הִקְשָׁה ע d'un individu 2 Ch 36,13 ; Pr 29,1.
3. d'un oiseau : briser sa tête מִמּוּל עָרְפּוֺ Lv 5,8 ( P ).
