hébreu biblique

עִמָּנוּאֵל nom propre, masculin Emmanuel (avec nous est Dieu) ; — ׳ י Es 7,14 van d. H. Baer ; עִמָּנוּ אֵל Gi ; — nom d'enfant, symbolisant la présence de ׳ י pour délivrer son peuple (sur l'interprétation voir les commentaires) — עִמָּנוּ אֵל Es 8,8 ; Es 8,10 est une déclaration de confiance, avec nous est Dieu ! (comparer Ps 46,8 ; Ps 46,12) ; voir עִם .
