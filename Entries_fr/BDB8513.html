BDB8513 [H4878]

hébreu biblique

[מְשׁוּבָה], מְשֻׁבָה nom féminin retour en arrière, apostasie ; — absolu מְשֻׁבָה Jr 3,6 + ; construit מְשׁוּבַת Pr 1,32 suffixe מְשׁוּבָתָם Os 14,5 pluriel suffixe מְשׁוּבֹתֵיכֶם Jr 3,22, etc. ; — retour en arrière, apostasie, d'Israël, Os 14,5, מְשׁוּבָתִי Os 11,7 apostasie loin de moi (si vera lectio) ; de Juda Jr 2,19 ; Jr 3,22 ; Jr 5,6 ; Jr 8,5 ; Jr 14,7 (tous pluriel) + Ez 37,33 (lire מְשׁוּבֹתֵיהֶם pour ׳ מוֺשְׁ, ᵐ5 Comm.) ; נִצַחַת ׳ מ Jr 8,5 apostasie durable, perpétuelle ; aussi comme nom concret, apposition du mot suivant, ׳ מְשֻׁבָה יִשׂ Jr 3,6 ; Jr 3,8 ; Jr 3,11 ; Jr 3,12 l'apostate, Israël. D'un individu, מְשׁוּבַת פְּתָיִם Pr 1,32.
