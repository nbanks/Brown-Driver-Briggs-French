BDB3441 [H3039]

hébreu biblique

[יָדִיד] adjectif (poésie) bien-aimé (hébreu tardif id. , araméen ) — construit יְדִיד Dt 33,12 suffixe לִידִידִי Es 5,1 (deux fois dans le verset) ; Jr 11,15 לִידִידוֺ Ps 127,2 pluriel suffixe יְדִידֶיךָ Ps 60,7 ; Ps 108,7 féminin יְדִידֹת Ps 45,1 יְדִידוֺת Ps 84,2 —

1. mon bien-aimé, utilisé par les prophètes de ׳ י sous la figure du cultivateur Es 5,1 (deux fois dans le verset) ; ainsi mon (ton, son) bien-aimé Jr 11,15 ; Ps 60,7 = Ps 108,7 ; Ps 2 bien-aimé, de Benjamin comme aimé de ׳ י Dt 33,12.
2. charmant, מַהיְּֿדִידוֺת מִשְׁכְּנוֺתֶיךָ Ps 84,2 que tes demeures sont charmantes !
3. féminin pluriel comme substantif abstrait שִׁיר יְדִידֹת Ps 45,1 = un chant d'amour.
