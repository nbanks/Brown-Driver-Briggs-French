BDB8584 [H7817]

hébreu biblique

[שָׁחַח] verbe s'incliner, être courbé, se tapir (hébreu tardif id. ; Tel Amarna ša—âhu (WklTelAm. Vocab.), se prosterner, probablement cananéisme ; assyrien ša—â—u est opprimer, tourmenter) ; —

Qal Parfait 3 masculin singulier consécutif וְשַׁח Es 2,11 ; Es 2,17, 1 singulier שַׁחִ֫וֺתִי Ps 38,7, שַׁחֹ֫תִי Ps 35,14, 3 pluriel שׁחֲחוּ Jb 9,13, שַׁחוּ Ha 3,6 ; Pr 14,19 Imparfait 3 masculin singulier יָשֹׁחַ Ps 10,10, 3 masculin pluriel יָשֹׁ֫חוּ Jb 38,40, וַיָּשֹׁ֫חוּ Ps 107,39 Infinitif construit (= absolu, comme adverbe, Ges «GKC:118q»§ 118q Köii. 2, §§ 221, 402 d comparer aussi BaNB 164) שְׁחוֺחַ Es 60,14 —
1. être courbé, prosterné, humilié, par ׳ י, Es 2,11 ; Es 2,17 (tous deux || שָׁפֵל), Ha 3,6 (des collines), Jb 9,13 ( ׳ תַּחְתָּו שׁ), Ps 107,39 (+ מָעַט) ; par l'homme Ps 10,10.
2. s'incliner en hommage, לִפְנֵי personne Pr 14,19 וְהָֽלְבוּ אֵלַיִח שְׁחוֺחַ Es 60,14 (|| הִשְׁתַּחֲווּ).
3. se courber, d'un homme en deuil (קֹדֵר), Ps 35,14 ; Ps 38,7.
4. se tapir, d'une bête sauvage dans son repaire Jb 38,40.
