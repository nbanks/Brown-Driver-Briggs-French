BDB6338 [H6140]

hébreu biblique

[עָקַשׁ] verbe tordre (hébreu tardif id. (rare)) ; —

Niph`al Participe construit דְּרָכַיִם Pr 28,18 un tordu dans les voies (< pluriel, comparer Toy).
Pi`el Parfait 3 pluriel עִקְּשׁז לָהֶם Es 59,8 leurs sentiers ils les ont tordus . Imparfait 3 masculin pluriel יְעַקֵּ֑שׁזּ Mi 3,9 tout ce qui est droit ils le rendent tortueux ; Participe מְעַקֵּשׁ דְּרָכָיו Pr 10,9 (comparer Niph`al).
Hiph`il Imparfait וַיּעְקְשֵׁנִי Jb 9,20 ( Ges«GKC:53n»§ 53n) il m'a déclaré tortueux ; dévoyé (dans la vie).
