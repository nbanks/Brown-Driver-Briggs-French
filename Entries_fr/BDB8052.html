מַרְפֵּא, מַרְפֵּה nom masculin Qo 10,4 guérison, cure, santé (généralement figuré) ; — absolu מַרְפֵּא Jr 14,19 +, מַרְפֵּה Jr 8,15 construit מַרְפֵּא Pr 15,4 — ```

1. guérison, cure, des maux nationaux, Jr 8,15 ; Jr 14,19b (|| שָׁלוֺם), Jr 36,6 (+ אֲרֻכָה) ; surtout ׳ אֵין לָנוּ מ Jr 14,19a il n'y a pas de remède pour nous, ׳ עַד לְאֵין מ 2 Ch 36,16 .
2. figuré de personne, santé, profit, Pr 4,22 (|| חַיִּים), Pr 12,18 ; Pr 13,17 ; Pr 16,24 avec implication spirituelle Ml 3,20 ׳ אֵין מ Pr 6,15 = Pr 29,1 ׳ לֵב מ Pr 14,30 un esprit sain (esprit sain, composé), comparer ׳ מ seul = sérénité Qo 10,4 לְשׁוֺן ׳ מ Qo 15,4 guérison de la langue, = une langue apaisante (Toy).
3. littéralement, de maladie, ׳ לְאֵין מ 2 Ch 21,18.
