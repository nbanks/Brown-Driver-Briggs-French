BDB2995 [H2648]

hébreu biblique

[חָפַז] verbe être dans la frayeur, se hâter, ou être alarmé (arabe se hâter, inciter, presser) ; —

Qal Imparfait יַחְפּוֺז Jb 40,23 תַּחְפְּזוּ Dt 20,3 Infinitif construit suffixe בְּחָפְזִי Ps 31,28 ; Ps 116,11 בְּחָפְּזָהּ 2 S 4,4 בְּחָפְזָם 2 R 7,15 Qr ( Kt הֵחָֽפְזֵם) ; —
1. être pressé ou alarmé, de fuite précipitée 2 S 4,4 ; 2 R 7,15 Infinitif avec ב = nom, dans mon effroi Ps 31,23 ; Ps 116,11 être alarmé Dt 20,3 (|| תיראו), Jb 40,23 (de l'hippopotame)
Niph`al Parfait נֶחְפָּ֑זוּ Ps 48,6 (|| נִבְהֲלוּ) s'enfuir avec effroi ; ainsi Imparfait יֵחָפֵז֑וּן Ps 104,7 (|| יְּנוּסוּן) ; Infinitif הֵחָֽפְזֵם 2 R 7,15 Kt voir ci-dessus ; Participe נֶחְפָּז 1 S 23,26 pressé ללבת ׳ ויהי דוד נ et David se hâtait de partir.
