BDB423 [H411]

hébreu biblique

I. אֵל pronom pluriel masculin et féminin = le plus usuel אֵלֶּה, ceux-ci 1 Ch 20,8 avec l'article הָאֵל Gn 19,8 ; Gn 19,25 ; Gn 26,3 ; Gn 26,4 ; Gn 7,72 ; Dt 19,11. (Simplement une variante orthographique de אֵלֶּה, et sans doute prononcé de manière similaire ; les dialectes apparentés ont en général une forme dissyllabique : voir ci-dessous אֵלֶּה. Écrit de manière similaire en phénicien, p. ex. CIS Dt 3,22 ; Dt 14,5 ; Dt 93,3 ( האל), mais ZMG1875, 240 (néo-punique) אלא ; dans Plaut. Poen. voir I. 9 translittéré ily ; SchroedGr. phénicienne p. 81, 160, 286 ss.)
