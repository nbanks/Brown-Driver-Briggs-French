I. [עָתַר] verbe prier, supplier (toujours envers Dieu) (comparer arabe égorger pour un sacrifice DeGn (4) 381 WeSkizzen iii. 115, 166 ; Heid. (2). 118, 142 ; RSSem i. 210 s. ; 2e éd. ; 227 s.) ; —

Qal Imparfait 3 masculin singulier ׳ וַיֶעְתַּר לי Gn 25,21, ׳ אֶליֿ Ex 8,26 ; Ex 10,18 ; יֶעְתַּד אֶלאֱֿלוֺהַּ Jb 33,26.
Niph`al être supplié, exaucé (et accorder la supplication ; toujours de Dieu), habituellement avec ל du suppliant (voir ל 5.d.): Parfait 3 masculin singulier consécutif וְנֶעְתַּר לָהֶם Es 19,22 Imparfait 3 masculin singulier וַיֵּעָ֫תֶר לוֺ Gn 25,21 ; 2 Ch 33,13, וַיֵּעָתֵר לָנוּ Esd 8,23 avec לָאָרֶץ pour le pays 2 S 21,14 ; 2 S 24,25 Infinitif absolu לָהֶם וְנַעְתּוֺר 1 Ch 5,20 construit (comme substantif) הֵעָתֶרלֿוֺ 2 Ch 33,19 .
