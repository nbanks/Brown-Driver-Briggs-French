BDB206 [H362]

hébreu biblique

[אֵילִים] nom propre, d'un lieu Elim (grands arbres) — 2e station d'Israël dans le désert, avec 12 sources & 70 palmiers Ex 15,27 (dans les deux cas, aussi אֵילָ֑מָה) ; Ex 16,1 ; Nb 33,9 (dans les deux cas) ; Nb 33,10, tous P ; selon Di probablement = Wâdi Gharandel ; ainsi BäExod. p. 18, compare Kn. ; = אֵילָ֑מָה q.v.
