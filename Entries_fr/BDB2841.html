BDB2841 [H2532]

hébreu biblique

חֶמְדָּה nom féminin id. — ׳ ח Os 13,15 + 8 t. ; construit חֶמְדַּת 1 S 9,20 + 3 t. ; suffixe חֶמְדָּתִי Jr 12,10 חֶמְדָּתֵךְ Ez 26,12 חֶמְדָּתָם Dn 11,8 — désir d'Israël 1 S 9,20 suivi de לְ devant l'objet du désir ; ׳ בְּלֹא ח = sans désir 2 Ch 21,20 (c.-à-d. il vécut comme personne ne le désirait), voir Öttli ᵐ5 (pas ᵐ5 L ) ᵑ9 ; concret : ישׂראל ׳ כלחֿ 1 S 9,20 et pour qui est tout ce qui est désirable en Israël ? ainsi ᵐ5 ᵑ9 RV Dr ; (> AV et sur qui repose tout le désir d'Israël ?) ; comparer כלהֿגוים ׳ ובאו ח Ag 2,7 (c.-à-d. les choses désirables, précieuses de toutes les nations) ; ailleurs après construit comme חֶמֶד ; surtout ׳ כלי ח Os 13,15 ; Na 2,10 ; Jr 25,34 ; 2 Ch 32,27 ; 2 Ch 36,10 ; Dn 11,8 ׳ שְׂכִיּוֺת ח Es 2,16, ׳ אֶרֶץ ח Ps 106,24 ; Jr 3,19 ; Za 7,14, comparer ׳ חֶלְקַת ח Jr 12,10 ׳ בתי ח Ez 26,12 נשׁים ׳ ח Dn 11,37, apparemment référence à un objet de culte idolâtre, peut-être Adonis = Tammouz Ew Bev (Astarté, Meinh).
