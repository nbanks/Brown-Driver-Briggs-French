BDB1039 [H998]

hébreu biblique

בִּינָה nom féminin compréhension (forme avec בְּ préfixé, de בּוּן ) — ׳ בּ Jb 20,3 + 30 t. ; construit בִּינַת Es 29,24 + 6 t. ; suffixe בִּינָתִי Jb 32,8 + 7 t. —

1. compréhension, comme faculté mentale, au sens le plus large, d'un architecte habile, marin, chanteur, &c. Ex 31,3 ; Ex 35,31 ; Ex 36,1 ; 1 R 7,14 ; Ez 28,4 ; 1 Ch 15,22 ; 1 Ch 25,8 (|| מֵבִין) ; d'un homme sage Dt 32,28 ; Ps 49,4 ; Jb 32,8 (׳ נִשְׁמַת שׁ ׳ תְּבִינֵם), etc. ; en contraste avec דַּ֫עַת Jb 34,16 ; Pr 2,3 ; Pr 4,5 ; Pr 4,7 ; Pr 7,4 ; Pr 16,16 ; Pr 23,23 ; בִינָה personifié Pr 8,14 ; connaissance, acquise par l'expérience Jb 39,26 ; par enseignement, surtout divin Jb 32,11 ; Jb 38,36 ; Ps 136,5 ; Es 40,14 (bis) ; Pr 2,6 ; Pr 3,5 ; Pr 9,6 ; Pr 9,10 ; Dieu comme sujet Jb 38,4 ; Ps 147,5 ; Es 40,28.
2. compréhension ce qui est compris ou le résultat de la compréhension ; objet de connaissance Es 29,24 ; Dn 9,22.
3. pouvoir de compréhension Jb 20,3 ; Es 11,2 ; Es 44,19 ; Dn 10,1 ; Esd 8,18 ; Ne 8,2 ; Ne 8,3, etc. ; capacité de discernement 1 R 3,9 ; 1 R 3,11 ; comparer 1 R 4,29 ; 1 Ch 22,12 ; 2 Ch 2,11 ; Pr 23,4 ; Jb 28,28.
4. acte de compréhension Jb 28,12 ; Jb 28,20 ; comparer Pr 3,19.
