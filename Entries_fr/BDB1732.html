BDB1732 [H1622]

hébreu biblique

גִּרְגָּשִׁי adjectif [d'un peuple] seulement ׳ הַגּ comme nom propre collectif ; « engendré » par Canaan Gn 10,16 = 1 Ch 1,14 dans la liste des peuples cananéens déplacés par Israël ; territoire inconnu ; Gn 15,21 ; Dt 7,1 ; Jos 3,10 ; Jos 24,11 ; Ne 9,8 comparer Gn 10,16 ; 1 Ch 1,14 ci-dessus (Hexateuque seulement J E D ).
