BDB6444 [H6218]

hébreu biblique

עָשׂוֺר, et (Ex 12,3) עָשׂר nom [masculin] une dizaine, décade ; —

1. habituellement de jours :
a. une période de dix jours Gn 24,55 ( J ).
b. dans l'expression, בֶּעָשׂוֺר (עָשׂר) לַחֹדֶשׁ le dix du mois Lv 25,9 ( H ), Ez 20,1 ; Ez 24,1 ; Ez 40,1 ; Ex 12,3 ; Lv 16,29 ; Lv 23,27 ; Nb 29,7 ; Jos 4,19 (tous P ), 2 R 25,1 = Jr 52,4 ; Jr 52,12 (septième jour dans || 2 R 25,8).
2. de cordes d'un instrument : ׳ נֵבֶל ע harpe, à dix (cordes) Ps 33,2 ; Ps 144,9 ׳ ע seul Ps 92,4 sur un instrument à dix (cordes).
