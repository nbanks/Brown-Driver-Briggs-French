BDB9501 [H2255 H2254]

araméen biblique

ח

[חֲבַל] verbe

Pael détruire, faire du mal (voir hébreu biblique II. חבל ) ; — Parfait 3 masculin pluriel suffixe חַבְּלוּנִי Dn 6,23 (de lions) ; Impératif masculin pluriel suffixe חַבְּלוּהִי Dn 4,20 (suffixe de l'arbre) ; Infinitif לְחַבָּלָה Esd 6,12 (accusatif du temple).
Hithpa`al être détruit : Imparfait 3 féminin singulier תִּתְחַבַּל Dn 2,44 ; Dn 6,27 ; Dn 7,14 ( בַּ֑ל- ; tous du royaume de Dieu).
