BDB1547 [H1460]

hébreu biblique

I. גֵּו nom [masculin] dos — גֵּו absolu inusité, construit Pr 10,13 + 2 t., suffixe גֵּוִי Es 50,6, גֵּוְךָ Es 38,17, גֵּוֵךְ Es 51,23 — dos, comme frappé, fouetté Pr 10,13 ; Pr 19,29 ; Pr 26,3 comparer Es 50,6 dans l'expression ׳ הִשְׁלִיךְ אַחֲרֵי ג (comme גַּו) Es 38,17, de Yahvé rejetant les péchés du pénitent derrière son dos, les chassant de sa pensée ; comme foulé aux pieds Es 51,23, figuré d'une humiliation extrême.
