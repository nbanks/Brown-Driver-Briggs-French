III. [גּוּר] verbe redouter (comparer יָגֹר) —

Qal Imparfait וַיָּגָר Nb 22,3 ; 1 S 18,15 2 masculin singulier תָּגוּר Dt 18,22 אָגוּר Dt 32,27 pluriel יָגוּרוּ Ps 33,8 ; Jb 41,17 + Os 10,5, תָּגוּרוּ Dt 1,17 Impératif גּוּרוּ Ps 22,24 ; Jb 19,29 —
1. avoir peur de, suivi de מִפְּנֵי Nb 22,3 ; Dt 1,17 ; 1 S 18,15, גּוּרוּ לָכֶם מִפְּנֵי Jb 19,29 avec מִן Jb 41,17 ; Dt 18,22 avec accusatif Dt 32,27 (poésie) Os 10,5 lire probablement ינודו se lamenter (|| אבל, employé avec לְ p. ex. Na 3,7 ; Jr 15,5 et ailleurs ; comparer Che).
2. être dans la crainte de, suivi de מִן Ps 22,24 ; Ps 33,8 ( יָרֵא).
