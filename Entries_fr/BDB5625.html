BDB5625 [H5557]

hébreu biblique

סָלַף verbe tordre, pervertir, renverser ( ᵑ7 סְלַף tordre (rare) ; arabe est passer, s'évanouir, être réduit à néant Lane1407 s.) ; —

Pi`el Imparfait 3 masculin singulier יְסַלֵּף Jb 12,19 + 2 t., וַיְסַלֵּף Pr 22,12 3 féminin singulier תְּסַלֵּף Pr 13,6 ; Pr 19,3 Participe מְסַלֵּף Pr 21,12 —
1. pervertir, Ex 23,8 ( E ) un pot-de-vin pervertit la cause (דִּבְרֵי) du juste, = Dt 16,19.
2. subvertir, renverser, ruiner (seulement littérature sapientiale) : avec accusatif de personne Jb 12,19 ; Pr 13,6 (opposé à תִּצֹּר), Pr 21,12 ׳ אִוֶּלֶת אָדָם תּ דַּרְכּוֺ Pr 19,3 la folie d'un homme subvertit son chemin ; וַיְסַלֵּף דִּבְרֵי בֹגֵד Pr 22,12 et il (׳ י) subvertit les affaires d'un homme perfide.
