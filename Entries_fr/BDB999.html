BDB999 [H974]

hébreu biblique
בָּחַן verbe examiner, éprouver (comparer בָּחַר & בּוֺר , et , , , mais aussi araméen , , éthiopien Di 688 ; phénicien en nom propre בחן, et composant de nombreux noms propres africains (Punic) Hal 266) — Qal Parfait בָּחַן Jr 11,20 + 4 t. ; בָּחָ֑נְתָּ Ps 17,3 + 2 t. ; בָּחַ֫נְתִּי 1 Ch 29,17, etc. ; Imparfait יִבְחַן Jb 34,36 + 4 t. ; וָאֶבְחָ֑ן Ez 21,18, etc. ; Infinitif בְּחֹן Ml 3,10, בְּחֹן Ml 3,15 ; Participe actif בֹּחֵן Jr 11,20 + 8 t. —
1. examiner, métaux Jb 23,10 ; Za 13,9 ; Ml 3,10, donc tester, éprouver, avec accusatif seulement Dieu le faiseur et Israël le fait Ml 3,15 ( Qr ) ; mais beaucoup plus souvent :
2. essayer, en éprouvant ; avec accusatif, Dieu, ou ׳י, le faiseur, et le cœur d'homme, les reins, l'esprit, 1 Ch 29,17 (לֵבָב) ; Ps 7,10 (לִבּוֺת ׳ וּכ) ; Ps 11,5 ; Ps 17,3 (לִבִּי) ; Jr 11,20 ; Jr 12,3 (לִבִּי) ; Jr 17,10 ; Jr 20,12 (׳ כ ׳ וְל) ; Ps 26,2 (וְלִבִּי כִלְיוֺתַי) ; avec accusatif personnes, Ps 11,4 ; Ps 66,10 ; Ps 81,8 (Israël) ; Ps 95,9 ; Ps 139,23 (moi) ; Jr 9,6 (comparer Jr 6,27) ; Ez 21,18 ; Jb 34,36 (Elihu priant que Job puisse être essayé). Seulement une fois le faiseur est homme : Jb 7,18 (participe avec suffixe תִבְחָנֶּנּוּ).
Niph`al Participe נִבְחָן Es 28,16 = éprouvé, c.-à-d. pierre de choix. Pu`al Participe מְבֻחָן Pr 27,21 éprouvé ( Hi et autres ; Be De et autres préfèrent Qal comme collectif pluriel).
