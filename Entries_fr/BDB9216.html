BDB9216 [H8581]

hébreu biblique

[תָּעַב] verbe dénominatif Niph`al Pi`el :

Niph`al Parfait 3 masculin singulier נִתְעַב 1 Ch 21,6 Participe נִתְעָב Es 14,19 ; Jb 15,16 — être en abomination :
1. sens rituel : נֵצֶר נִתְעָב Es 14,19 un rameau abhorré (rejeté) ( < נֵץ vautour, l'oiseau charognard impur) ; נִתְעָב וֶנֶאֱלָךְ֑ Jb 15,16 abhorré et corrompu, par la maladie (נאלח ailleurs Ps 14,3 = Ps 53,4 || התעיב).
2. éthiquement : le recensement de David, selon 1 Ch 21,6.
Pi`el Parfait 3 pluriel suffixe תִּעֲבוּנִי Jb 9,31 + 2 t. ; Imparfait יְתָעֵב Ps 106,40 ; Ps 5,7 (lire 2 masculin singulier תְּתַעֵב) ; 3 féminin singulier תְּתַעֵב Ps 107,18 2 masculin singulier תְּתַעֵב Dt 23,8 (deux fois dans le verset), etc. ; Infinitif absolu תַּעֵב Dt 7,26 Participe מְתָעֵב Es 49,7 (mais voir infra), + Am 6,8 (ainsi lire pour מְתָאֵב GeiUrschrift 349 We Now Marti Harper) ; pluriel מְתַעֲבִים Mi 3,9 —
1. regarder comme une abomination, abhorrer :
a. sens rituel : (1) de Dieu ; avec accusatif d'Israël, à cause des idoles Ps 106,40. (2) de l'homme : avec accusatif, d'abomination Dt 7,26 (deux fois dans le verset), comparer Dt 23,8 (deux fois dans le verset) ; Jb Jb 19,19 ; Jb 30,10.
b. éthiquement : (1) de Dieu ; || שׂנא, avec accusatif : אישׁ דמים ומרמה Ps 5,7 (Am 6,8 voir II. תאב ). (2) de l'homme ; avec accusatif מִשְׁפָּט Mi 3,9 , דֹּבֵר תָּמִים Am 5,10 (|| שׂנא), שֶׁקֶר Ps 119,163 (|| שׂנא).
c. physiquement : avec accusatif כָּלאֹֿכֶל Ps 107,18.
2. rendre abominable : avec accusatif, sens rituel Ez 16,25 Job, des vêtements souillés Jb 9,31 מְתָעֵב גּוֺי Es 49,7 (|| בְּזִה נֶפֶשׁ) (lire probablement Pu`al מְתִעַב, comme Oort SS Bu Du).
Hiph`il Parfait 2 féminin singulier הִתְעַבְתְּ Ez 16,52 3 masculin pluriel הִתְעִיבוּ Ps 14,1 = Ps 53,2 Imparfait וַיַּתְעֵב 1 R 21,26 — rendre abominable, agir de manière abominable :
1. sens rituel : avec accusatif de chose, Ez 16,52 לְ avec infinitif 1 R 21,26.
2. éthiquement : avec accusatif עלילה Ps 14,1 = Ps 53,2 (עול).
