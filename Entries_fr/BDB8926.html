BDB8926 [H8191]

hébreu biblique

[שַׁעֲשֻׁעִים], שׁוּעִים- nom [masculin] pluriel intensif délice (BaNB 206); — absolu שׁוּעִים- Jr 31,20; Pr 8,30 suffixe שׁוּעָיו- Es 5,7, habituellement שֻׁעַי- Pr 8,31; Ps 119,24, שֻׁעָ֑י- Ps 119,77 + 3 t. ; — délice, dans l'expression ׳ נְטַע שׁ Es 5,7 la plantation de son délice (dans laquelle il prenait plaisir), ainsi ׳ יֶלֶד שׁ Jr 31,20 (|| בֵּן יַקִּיר); אֶתֿ ׳ שׁ Pr 8,31 mon délice (était) avec les fils de l'homme ; ailleurs = objet de délice, de la sagesse Pr 8,30 loi de Dieu, etc., Ps 119,24; Ps 119,77; Ps 119,92; Ps 119,143; Ps 119,174 .
