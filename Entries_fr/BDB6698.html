BDB6698 [H6455]

hébreu biblique

פִּסֵּחַ adjectif boiteux ; — absolu ׳ פ 2 S 9,13 + ; pluriel פִּסְחִים 2 S 5,6 + ; — comme attribut Lv 21,18 (+עִוֵּר) ; prédicat 2 S 19,27, שְׁתֵּי רַגְלָיו ׳ פ 2 S 9,13 boiteux des deux pieds ; comparer aussi (d'un animal) Dt 15,21 (+עִוֵּר) ; comme substantif 2 S 5,6 ; 2 S 5,8 (deux fois dans le verset) ; Jr 31,8 ; Ml 1,8 ; Ml 1,13 ; Jb 29,15 (tous + עִוֵּר), aussi Es 33,23 ; Es 35,6 ; Pr 26,7.
