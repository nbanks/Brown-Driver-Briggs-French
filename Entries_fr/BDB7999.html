BDB7999 [H4830]

hébreu biblique

[מַרְעִית] nom féminin pâture, conduite du troupeau, pâturage (toujours figuratif) ; — suffixe מַרְעִיתִי Jr 23,1 ; Ez 34,31, מַרְעִיתֶ֑ךָ Ps 74,1 ; Ps 79,13, etc. ; —

1. pâture, conduite du troupeau, צאֹן מַרְעִיתִי, d'Israël comme troupeau de ׳ י, Jr 23,1 ; Ez 34,31, comparer Ps 74,1 ; Ps 79,13 ; Ps 100,3 + Ps 95,7(lire וְעַם יָדוֺ ׳ מ ׳ צ, comparer Hup-Now Che).
2. pâturage, Os 13,6 (Now כִּרְעוֺתָם), Es 49,9 ; Jr 25,36.
3. par métonymie = troupeau Jr 10,21.
