BDB593 [H595]

hébreu biblique

אָֽנֹכִ֫י אָנֹ֑כִי (une fois Jb 33,9 אָֽנֹכִ֑י pronom [1 singulier, commun] je ; Gn 3,10 ; Gn 7,4 ; Gn 15,1 ; Gn 15,2 ; Gn 16,5 + souvent Avec הֲ הֶאָנֹכִי Nb 11,12 ; Jb 21,4. (Assyrien anâku , phénicien et moabite אנך : absent en arabe, araméen, éthiopien ; mais ku apparaît comme affixe de la 1 singulier dans le verbe éthiopien (p. ex. waladku = hébreu יָלַדְתִּי). אָנֹכִי et אֲנִי semblent être deux formations parallèles (contenant toutes deux l'élément ani [comparer le suffixe , נִי-] ou ana, et l'une renforcée par l'ajout de la base démonstrative Ku [probablement apparentée à כָּא, כֹּה ici] : comparer Sta§ 179 WSG 95 s. 98-101), dont, dans la plupart des langues sémitiques, l'une a prévalu à l'exclusion de l'autre, mais qui en hébreu ont maintenu leur place côte à côte.) Dans certains cas אני et אנכי semblent pouvoir être employés indifféremment ; dans d'autres le choix semble avoir été déterminé, en partie par des considérations rythmiques, en partie par une préférence croissante pour אֲנִי chez les auteurs tardifs. Ainsi quand il est postposé au verbe pour l'emphase (avec ou sans גַּם) la forme plus légère אֲנִי est presque toujours employée ( Lv 20,5 ; Lv 26,24 ; Lv 26,32 ; Dt 12,30 ; Jg 1,3 ; Jg 8,23 ; 2 S 12,28 ; 2 S 17,15 ; 2 S 18,2 ; 2 S 18,22 ; Jr 17,18 ; Jr 21,5 ; Ez 17,22 ; Jb 13,2 + ; comparer les cas Gn 27,34 ; 1 S 25,24 ; 2 S 19,1 ; 1 R 1,26 ; Pr 23,15) ; au contraire, dans le style rhétorique emphatique du Deutéronome, אָנֹכִי est préféré (dans les discours, uniformément, sauf Dt 12,30, à l'accusatif avec l'usage noté ci-dessus, et Dt 29,5 dans une expression fixe ; sur Dt 32,48 ; Dt 32,51 ( P ) comparer ci-dessous) Dans certaines locutions aussi, l'usage préfère tantôt אֶני tantôt אָנכי ainsi on trouve חַיאָֿנִי Nb 14,21 et toujours, sauf Dt 32,40 (אָ֑נִי) אִתְּךָ אֲנִי (Jr 1,9 ; Jr 1,19 ; Jr 30,11 ; Jr 46,18 ; Es 43,2 ; Es 43,5) ; אֲנִי יהוה Ex 6,2 ; Ex 6,6 ; Ex 6,8 et ailleurs dans P , et particulièrement fréquent dans H (Lv 18,2 ; Lv 18,4 ; Lv 18,6 etc.) et Ézéchiel, aussi Gn 15,7 ; Gn 28,13 ; Dt 29,5 ; Jg 6,10 + ; ( אנכי יהוה beaucoup moins fréquent ; seulement J E et auteurs prophétiques, Ex 20,2 = Dt 5,6 ; Os 12,10 ; Os 13,4 ; Ps 81,11 ; Ex 20,5 = Dt 5,9 ; Es 43,11 ; Es 44,24 ; Es 51,15 (Ex 4,11 est différent) ; comparer DrJPh xi. 224 s.) ; אני אמרתי Es 38,10 (Éz.) Es 49,4 ; Jr 5,4 ; Jr 10,19 (Jr 3,19 אנכי) Rt 4,4 ; Ps 30,7 ; Ps 31,23 ; Ps 41,5 ; Ps 82,6 + ; ויאמר אָ֫נִי (en réponse à une question) Gn 27,24 ; Jg 13,11 ; 2 S 20,17 ; 1 R 13,14 ; 1 R 18,8 (אנכי ׳ וי seulement 2 S 2,20 au contraire, avec un prédicat, אָנֹבִי est régulièrement employé, Gn 24,34 ; 1 S 30,13 ; 2 S 1,8 עֲמָלֵקִי אָנֹ֑בִי 2 S 11,5 ; 2 S 20,17 ; Es 6,5 ; Jr 1,6 ; Jon 1,9) ; ( הִנְנִי) וַאֲנִי הִנֵּה Gn 6,17 ; Gn 9,9 ; Ex 31,6 ; Nb 3,12 + (mais הִנֵּה אָנֹכִי Gn 24,14 ; Gn 24,43 ; Gn 25,32 ; Ex 3,13 ; Ex 19,9 + ; הִנֵּה אֲנִי est très rare ; voir ib.226). En ce qui concerne l'usage de livres particuliers, dans le Pentateuque (sauf le Deutéronome) אֲנִי est employé dans P (y compris H ) toujours (environ 130 t.) sauf Gn 23,4 (comparer Ézéchiel ci-dessous) ; dans J E אנכי est préféré, mais non exclusivement ( Gn 81,48). Dans Samuel il y a 50 exemples de chaque forme. Jérémie a quelque 54 exemples de אֶני, 37 de אנכי. Dans les livres tardifs la prédominance de אֲנִי est évidente. Ainsi dans Ézéchiel אני apparaît 138 t., אנכי une fois Gn 36,28 (peut-être une réminiscence de Jr 11,4 b Jr 24,7 ; Jr 30,22) ; dans Lamentations, Aggée, Esdras, Esther, Qohéleth אני 45 t., אנכי jamais ; dans Chroniques אני 30 t., אנכי une fois 1 Ch 17,1 (d'après 2 S 7,2) ; dans Daniel אני 23 t., אנכי une fois 2 S 10,11. Voir plus en détail GiesZAW 1881, 251-8 Drl.c. 222-7.
