BDB4047 [H3639]

hébreu biblique

כְּלִמָּה nom féminin insulte, opprobre, ignominie ; — absolu ׳ כ Es 45,16 + 9 t. ; construit כְּלִמַּת Jr 20,11 + 3 t. ; suffixe כְּלִמָּתִי Jb 20,3 + 3 t., etc., pluriel כְּלִמּוֺת Mi 2,6 ; Es 50,6 —

1. spécifiquement, insulte, opprobre, ׳ לֹא יִסַּג כ Mi 2,6 les opprobres ne cessent pas ; מוּסַר כְּלִמָּתִי Jb 20,3 = ma correction humiliante (insultante), c.-à-d. la correction qui m'insulte.
2. en général, opprobre, ignominie, opposé à כָּבוֺד Ps 4,3 ׳ הלך בַּכּ Es 45,16 aller dans l'ignominie (+ בּוֺשׁ, נִכְלַם) ; souvent || בּשֶׁת Es 30,3 ; Es 61,7 ; Ps 44,16, sujet de כִּסָּה Jr 3,25 notre ignominie nous couvre, ainsi sous la figure du vêtement, après לָבַשׁ Ps 35,26 (+ בושׁ, חפר dans va), Ps 109,29 || חֶרְפָּה Ps 69,8 (sujet de כִּסְּתָה פָּנָ֑י), Jr 51,51 (id. ; בושׁ dans la || proposition), Ps 71,13 (+ בושׁ dans la || proposition) ; || בּשֶׁת + חֶרְפָּה Ps 69,20 || אִוֶּלֶת Pr 18,13 || רֹק Es 50,6 עוֺלָם ׳ כ Jr 20,11 ( בושׁ dans la || proposition) ; כְּלִמָּתֵךְ Ez 16,63 ( id. ||) ; souvent dans Ez. ׳ נשׂא כ porter l'ignominie, Ez 16,52 (deux fois dans le verset) (|| בושׁ), Ez 16,54 (נִכְלַם dans la || proposition) ; Ez 32,24 ; Ez 32,25 ; Ez 32,30 (tous + אֶתיֿוֺרְדֵי בוֺר avec ceux qui descendent dans la fosse ; référence à une mort ignominieuse), Ez 36,7 ; Ez 39,26 (si vera l. = porter le sens humiliant d'une bonté imméritée de ׳ י ; mais texte douteux, Hi Co, q. v., [׳ כ] נשׁו ; Sm Da défendent) ; נאשׂ כְּלִמַּת הַגּוֺיִם, c.-à-d. causée par les nations, Ez 34,39 ; Ez 36,6 aussi v Ez 36,15 (|| חֶרְפַּת עַמִּים), probablement aussi Ps 89,51 (lire כְּלִמַּת, comparer VB Che Bae) ; כְּלִמָּתָם יְתוֺעֲבוֺתָם אֲשֶׁר עָשׂוּ ׳ וְנ Ez 44,13.
