BDB4427 [H3928]

hébreu biblique

לִמֻּד adjectif enseigné ; — ׳ ל Jr 2,24 pluriel לִמּוּדִים Es 50,4 (deux fois dans le verset) ; construit לִמּוּדֵי Es 54,13 לִמֻּדֵי Jr 13,23 suffixe לִמֻּדַי Es 8,16 —

1. enseigné, comme disciples Es 8,16 ; Es 50,4 (deux fois dans le verset) ; Es 54,13.
2. accoutumé à quelque chose, לִמֻּד מִדְבָּר Jr 2,24 (ânesse sauvage) accoutumée au désert ; לִמֻּדֵי הָרֵעַ Jr 13,23 accoutumés à faire le mal.
