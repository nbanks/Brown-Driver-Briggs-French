hébreu biblique

מָצוֺק nom [masculin] détresse, difficultés, angoisse ; — toujours absolu ׳ מ ; — ׳ כָּלאִֿישׁ מ 1 S 22,2 tout homme dans la détresse (en difficulté) ; ׳ בְּמָצוֺר וּבְמ Dt 28,53 ; Dt 28,55 ; Dt 28,57 ; Jr 19,9 (tous de difficultés nationales) ; מְצָאוּנִי ׳ צַרוּֿמ Ps 119,143 lire peut-être מָצוֺק aussi Ps 32,6, pour מצא רק, comparer Du Br.
