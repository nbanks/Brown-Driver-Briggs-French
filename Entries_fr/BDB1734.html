1. s'exciter contre (בְּ personne) ; s'engager dans la querelle avec, littéralement Dt 2,5 ; Dt 2,19 ; 2 R 14,19 = 2 Ch 25,19 (suivi de בְּרָעָה ; — רעה personnifié comme adversaire défié) ; aussi + accusatif de manière, מִלְחָמָה Dt 2,9 ; Dt 2,24 figuré וְשֹׁמְרֵי תוֺרָה יִתְגָּרוּ בָם Pr 28,4 tandis que ceux qui gardent la loi sont en querelle avec eux ; c.-à-d. avec ceux qui l'abandonnent.
2. absolu ; s'exciter (contre l'ennemi) ; faire la guerre ; seulement Dn 11 : יִתְגָּרוּ וְאָֽסְפוּ הֲמוֺן חֲיָלִים רַבִּים Pr 28,10 ils feront la guerre ; etc. ; וְיָשֹׁב וְיִתְגָּרֶ֯ו עַדמָֿעֻזֹּה v Pr 28,10 et il reviendra et fera la guerre, jusqu'à sa forteresse ; Pr 28,25 suivi de לַמִּלְחָמָה + בְּחַיִל גָּדוֺל (instrumental)
