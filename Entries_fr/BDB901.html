BDB901 [H898]

hébreu biblique

[בָּגַד] verbe agir ou se conduire traîtreusement —

Qal Parfait בָּֽגְדָה Jr 3,20 + 14 t. ; Imparfait יִבְגֹּד Ml 2,15 + 5 t. ; נִבְגַּד Ml 2,10 Infinitif absolu בָּגוֺד Es 48,8 ; Jr 5,11 construit בְּגֹד Es 33,1 בִּגְדוֺ Ex 21,8 Participe בֹּגֵד Pr 22,12 + 11 t. ; בּוֺגֵד Es 33,1 + 10 t. ; — agir ou se conduire traîtreusement, infidèlement, trompeusement, dans la relation conjugale, dans les questions de propriété ou de droit, dans les alliances, en parole et en conduite générale.
a. absolu 1 S 14,33 ; Jb 6,15 ; Ps 78,57 ; Es 24,16 ; Es 33,1 (deux fois dans le verset) ; Es 48,8 ; Ml 2,11. comparer les expressions אחי בגדו כמו נחל mes frères ont agi trompeusement comme un torrent Jb 6,15 בֹּגְדִים בָּנָ֑דוּ וּבֶגֶד בּוֺגְדִים בָּגָ֑דוּ les traîtres ont agi traîtreusement, oui dans la traîtrise les traîtres ont agi traîtreusement Es 24,16 (allitération frappante) ; בֹּגְדֵי אָוֶן traîtres dans la méchanceté Ps 59,6.
b. avec בְּ Ex 21,8 ( E ), Jg 9,23 ; Es 33,1 (deux fois dans le verset) ; Jr 3,20 ; Jr 5,11 ; Jr 12,6 ; Lm 1,2 ; Os 5,7 ; Os 6,7 ; Ml 2,10 ; Ml 2,14 ; Ml 2,15 ; Ml 2,16.
c. avec accusatif Ps 73,15.
d. avec מִן prégnant, בָּֽגְדָה אִשָּׁה מֵרֵעָהּ agit traîtreusement (en s'éloignant) de son ami Jr 3,20. Le participe est employé Pr 2,22 + 8 t., Ps 25,3 ; Ps 59,6 ; Ps 119,158 ; Es 21,2 ; Es 24,16 (deux fois dans le verset) ; Es 33,1 ; Jr 3,8 ; Jr 3,11 ; Jr 9,1 ; Ha 1,13 ; Ha 2,5 בֹּגְדֵי בָ֑גֶד agissant traîtreusement dans la traîtrise (très traîtreusement) Jr 12,1.
