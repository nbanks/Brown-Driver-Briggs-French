hébreu biblique

[קָלַל]82 verbe être léger, rapide, insignifiant (probablement à l'origine être léger ; hébreu tardif [ קָלַל], קַל, קִלֵּל = hébreu biblique ; assyrien ‡alâlu II. mépriser, déshonorer ; de même Tel Amarna (apparemment cananéisme) ; arabe être petit, insuffisant ; sabéen קללם insuffisant Hal 143.8f. (comparer FellZMG liv (1900), 246) ; éthiopien être léger, petit, facile ; II. mépriser ; targoum קְלַל, syriaque = hébreu biblique ; aussi (voir Pilpel, etc.) arabe secouer, : vibrer, tournoyer (d'une épée), : voir être secoué ; — SchwZAW xi (1891), 170 ss. pense que secouer est le sens originel, voir מַקֵּל ; —
