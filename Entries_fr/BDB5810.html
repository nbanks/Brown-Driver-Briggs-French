II. עָדָה verbe orner, se parer (araméen dans les dérivés עִידּוּת, ornement ; comparer peut-être עֲדִיתָא, croûte, gale ; arabe gale, etc. [comme enveloppe ou excroissance ?], aussi large pierre pour couvrir une tombe, etc., Lane1979 ) ; —

a. réfléchi, se parer de, avec accusatif de parures (de femme), de la fiancée Es 61,10 (comparaison ; comparer הִלְבִּישׁ va) ; figuré d'Israël Os 2,15 ; Jr 31,4 Jérusalem Jr 4,30 (avec accusatif de sens apparenté ; || לָבַשׁ), Ez 16,13 (מַלְבּוּשֵׁךְ dans la proposition ||), Ez 23,40 (avec accusatif de sens apparenté au verbe).
