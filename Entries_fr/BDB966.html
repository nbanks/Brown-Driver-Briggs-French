BDB966 [H2983]

hébreu biblique

יְבוּסִי, יְבֻסִי

1. adjectif [d'un peuple] Jébusite, Jébusites, singulier אורנה הַיְבֻסִי 2 S 24,16 ; 2 S 24,18 comparer 1 Ch 21,15 ; 1 Ch 21,18 ; 1 Ch 21,28 ; 2 Ch 3,1 comme substantif un Jébusite Za 9,7 habituellement avec article ׳ הַי collectif les Jébusites, dans une déclaration historique Gn 10,16 ( J ) = 1 Ch 1,14 ; Nb 13,29 ( E ) Jos 9,1 ; Jos 11,3 ; Jos 12,8 ; Jos 24,11 (tous D ) Jg 3,5 ; Ne 9,8, tous dans la liste des peuples cananéens ; ainsi aussi 1 R 9,20 = 2 Ch 8,7, où le reste de ces peuples est mentionné ; aussi Esd 9,1, qui semble montrer que cette liste était devenue une expression consacrée pour les premiers habitants ; comparer en outre dans les promesses ( J E D ) Gn 15,21 ; Ex 3,8 ; Ex 3,17 ; Ex 13,5 ; Ex 23,23 ; Ex 33,2 ; Ex 34,11 ; Dt 7,1 ; Dt 20,17 ; Jos 3,10 spécifiquement défini comme habitants de Jébus-Jérusalem Jos 15,63 (deux fois dans le verset) ( J ) Jg 1,21 (deux fois dans le verset) ; 2 S 5,6 ; 2 S 5,8 (sur quoi comparer Dr) 1 Ch 11,4 (appelé aussi יוֺשֵׁב הָאָרֶץ 2 S 5,6 & ישְׁבֵי הָאָרֶץ 1 Ch 11,4) ; voir aussi עִיר הַיְבוּסִי הַזּאֹת Jg 19,11 d'où
2. Jérusalem, Hexateuque seulement P , Jos 15,8 גֵּי בֶןהִֿנֹּם אֶלכֶּֿתֶף היבוסי מִנֶּגֶב הִיא יְרוּשָׁלָ֑םִ, comparer Jos 18,16 ; Jos 18,28 וְהַיְבוּסִי הִיא יְרוּשָׁלַםִ. — comparer כנען.
