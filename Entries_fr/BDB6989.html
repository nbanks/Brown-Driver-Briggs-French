צוֺם nom masculin Es 58,5 jeûne ; — ׳ צ absolu 2 S 12,16 +, construit Za 8,19 (4 t.) ; suffixe צֹמְכֶם Es 58,3 pluriel צוֺמוֺת Est 9,31 — jeûne, comme accusatif de sens apparenté avec le verbe 2 S 12,16 observance publique 1 R 21,9 ; 1 R 21,12 ; 2 Ch 20,3 ; Esd 8,21 ; Jr 36,9 ; Jon 3,5 (tous objet de קָרָא proclamer), comparer Jl 1,14 ; Jl 2,15 (tous deux objet de קַדְּשׁוּ) ; יוֺם צוֺם Jr 36,6, יוֺם צֹמְכֶם Es 58,3, comparer Es 58,5 (|| יוֺם עַנּוֺת אָדָם נַפְשׁוֺ), Es 58,5 Es 58,6 de jeûnes périodiques Za 8,19 (4 t. dans le verset) ; Est 9,31 acte ou état de jeûne, וּבְשַׂקִּים ׳ נֶאֶסְפוּ בְצ Ne 9,1 comparer Jl 2,12 ; Dn 9,3 ; Est 4,3 נַפְשִׁי ׳ עִנֵּיתִי בַצּ Ps 35,13 comparer Ps 69,11 (sur וָאֶבְכֶּה voir Che Bae) ; causant la faiblesse physique Ps 109,24.
