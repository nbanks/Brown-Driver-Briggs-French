BDB4897 [H4970]

hébreu biblique

מָתַי43 adverbe interrogatif quand ? (arabe ; assyrien mati : en hébreu tardif et araméen avec l'interrogatif אַי, אֵימָתִי , אֵימָת, ), dans l'AT seulement du temps futur :

a. seul, Gn 30,30 מָתַי אעשׂה גם אנכי לביתי quand travaillerai-je aussi, etc. ? Am 8,5 ; Ps 41,6 ; Ps 42,3 ; Ps 94,8 ; Ps 101,2 ; Ps 119,82 ; Ps 119,84 ; Pr 6,9 b Pr 23,35 ; Jb 7,4 מָתַי אָקוּם, Ne 2,6.
b. לְמָתַי pour quand ? Ex 8,5 לְמָתַי לְךָ אַעְתִּיר.
c. עַדמָֿתַי jusqu'à quand ? combien de temps ? suivi d'imparfait Ex 10,7 עד מתי יהיה זה לנו למוקשׁ, 1 S 1,14 ; Jr 4,14 ; Jr 4,21 ; Ps 74,10 ; Ps 82,2 ; Ps 94,3 ; Pr 1,22 +, suivi de participe 1 S Pr 16,1 ; 1 R 18,21 עַדמָֿתַי אַתֶּם פֹּסְחִים עַלשְֿׁתֵּי הַסְּעִפִּים, suivi de parfait Ex 10,3 ; Ps 80,5 עַד מָתַי לֹא 2 S 2,26 ; Os 8,5 ; Za 1,12 seul, Es 6,11 וָאֹמַר עַדמָֿתַי אֲדֹנָי, Ha 2,6 הוֺי הַמַּרְבֶּה לֹאלֿוֺ עַדמָֿתַי ; avec une aposiopèse, Ps 6,4 עַדמָֿתָ֑י ׳ וְאַתָּה י, Ps 90,13.
d. אַחֲרֵי מָתַי עוֺד, après combien de temps encore ? Jr 13,27.
