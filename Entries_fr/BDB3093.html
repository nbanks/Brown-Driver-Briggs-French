BDB3093 [H2723]

hébreu biblique

חָרְבָּה nom féminin désolation, ruine; — absolu ׳ ח Lv 26,31 + 15 t. + Ez 38,8 (supprimer ᵑ6 Co) + Ez 38,12 חרבה ᵐ5 Co pour חרבות; pluriel חֳרָבוֺת Ps 9,7 + 11 t. (Ez 38,12, comparer ci-dessus); construit חָרְבוֺת Es 5,17 + 7 t.; suffixe חָרְבֹתַיִךְ Es 49,19 חָרְבֹתָיו Esd 9,9 חָרְבֹתֶיהָ Es 51,3 חָרְבוֺתֶיהָ Es 44,26 חָרְבוֺתֵיהֶם Ps 109,10 + 2 Ch 34,6 (comparer חֶרֶב et ci-dessous); (principalement hébreu moyen et tardif Es2 8 t.; Jr 10 t., Ez 14 t. — y compris Ez 38,8, q. supprimer Co); —

1. désolation, ruine, des villes d'Israël Lv 26,31 ; Lv 26,33 ( H ) ; Juda Es 44,26 ; Es 49,19 (|| שֹׁמְמֹת) ; Es 58,12 (בנה ) ; Jr 25,18 ; Jr 27,17 ; Jr 44,2 ; Jr 44,6 ; Ez 5,14 ruines de Jérusalem Es 52,9 ; Dn 9,2 aussi Ez 35,4 comparer Ml 1,4 (avec בנה ) ; Ez 36,10 (avec בנה) Ez 36,33 (id.) ; Égypte Ez 29,9 ; Ez 29,10 (dans les deux || שְׁמָמָה ; חרבות חֹרֶב dans Ez 29,10) ; ruine du temple Esd 9,9 = habitations en ruine Ps 109,10 comparer 2 Ch 34,6, où lire peut-être בְּחָרְבֹתֵיהֶם pour בּחר בתיהם (voir חֶרֶב ci-dessous III. חרב) ; Jr 22,5 en général ruines (si vera 1., voir Di ; objet de בנה) Jb 3,14 ׳ כוס ח Ps 102,7 une chouette des ruines.
