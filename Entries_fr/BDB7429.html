BDB7429 [H6974]

hébreu biblique

I. [קִיץ] verbe

Hiph`il s'éveiller (|| forme de יקץ ; hébreu tardif au Hiph`il (rare)) ; — Parfait 3 masculin singulier הֵקִיץ 2 R 4,31 ; Ez 7,6 2 masculin singulier וַהֲקיצ֫וֺתָ consécutif Pr 6,22, etc. ; Imparfait 1 singulier אָקִיץ Pr 23,35 3 masculin pluriel יָקִ֫יצוּ Jr 51,39 + ; Impératif masculin singulier הָקִ֫יצָה Ha 2,19 + ; masculin pluriel הָקִ֫יצוּ Es 26,19 ; Jl 1,5 Infinitif construit הָקִיץ Ps 17,15 ; Ps 73,20 Participe מֵקִיץ 1 S 26,12 — montrer des signes d'éveil, s'éveiller :
1. du sommeil,
a. 1 S 26,12 ; Es 29,8 (deux fois dans le verset) ; Ps 3,6 ; Ps 73,20 ; Ps 139,18 ; Pr 6,22 (ligne douteuse, voir Toy).
b. de ׳ י Ps 44,24 s'éveille-toi [du sommeil] à l'action, ainsi Ps 35,23 (avec ל de chose), Ps 59,6 (c. infinitif de but).
c. du sommeil extatique des prophètes Jr 31,26.
2. du sommeil de la mort, avec négation 2 R 4,31 ; Jr 51,39 ; Jr 51,57 ; Jb 14,12 de résurrection Es 26,19 ; Dn 12,2
3. de la stupeur (de l'ivresse) Jl 1,5 ; Pr 23,35.
4. de chose inanimée Ha 2,19 comparer בָּא הַקֵּץ הֵקִיץ אֵלָ֑יִח Ez 7,6 la fin vient, elle s'est éveillée vers toi (Co supprimer הֵקִיץ, mais le jeu de mots le favorise). — Ps 17,15 est placé par Thes Ol Hup Du et autres sous 1. a. ; Ew De Che sous 2 ; nous sous 1 b. ; Calv Now de la nuit de détresse et d'impuissance, et ainsi Bae (le psalmiste représentant Israël).
