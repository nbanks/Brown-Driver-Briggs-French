BDB1679 [H1580]

hébreu biblique

גָּמַל verbe

1. traiter pleinement ou adéquatement, rendre à.
2. sevrer.
3. mûrir
(assyrien gamâlu, traiter , faire du bien, p. ex. VR 2 Ch 35,19 hébreu tardif גָּמַל, araméen גְּמַל ; voir palmyrénien nom propre אגמלא VogNo. 124, nabatéen nom propre, féminin גמלת VogNab. ci-dessous No. 7 ; peut-être à l'origine accomplir, achever, apparenté à גמר ; — arabe est rassembler, être beau, agréable) —
Qal Parfait Ps 13,6 + ; Imparfait יִגְמֹל Nb 17,23 + ; Impératif גְּמֹל Ps 119,17 Infinitif גָּמְלֵךְ, גָּמְלָהּ 1 S 1,23 Participe גֹּמֵל Pr 11,17 + ; passif גָּמוּל Es 11,8 גָּמֻל Ps 131,2 (deux fois dans le verset) ; —
1.
a. rendre à, faire à, avec 2 accusatifs גְּמָלַתְהוּ טוֺב וְלֹא רָע elle lui fait du bien et non du mal Pr 31,12 כִּי אַתָּה גְּמַלְתַּנִי הַטּוֺבָה וַאֲנִי גְּמַלְתִּיךָ הָֽרָעָה car tu m'as fait le bien mais moi je t'ai fait le mal 1 S 24,18 faire le mal (ה)רע à Gn 50,15 ; Gn 50,17 ; Pr 3,30 avec לְ personne Es 3,9 faire le bien à, 2 accusatifs Es 63,7 (deux fois dans le verset) ; comparer aussi גֹּמֵל נַפְשׁוֺ אִישׁ חֶסֶד Pr 11,17 .
b. avec עַל personne traiter avec bienveillance Ps 13,6 ; Ps 116,7 ; Ps 119,17 ; Ps 142,8 avec accusatif de personne récompenser 2 S 22,21 (= Ps 18,21), peut-être aussi לָמָּה יִגְמְלֵנִי הַגְּמוּלָה הַזּאֹת pourquoi me récompenserait-il de cette récompense ? 2 S 19,37 .
c. rétribuer, rendre, payer en retour, dans un sens défavorable, avec 2 accusatifs Ps 7,5 avec לְ personne Dt 32,6 ; Ps 137,8 avec עַל personne 2 Ch 20,11 ; Ps 103,10 ; Jl 4,4 .
2. sevrer un enfant (achever son allaitement) 1 S 1,23 (deux fois dans le verset) ; 1 S 1,24 ; 1 R 11,20 ; Os 1,8 גָּמוּל enfant sevré Ps 131,2 (deux fois dans le verset) ; Es 11,8 גְּמוּלֵי מֵחָלָב sevrés du lait Es 28,9 .
3. transitif mûrir, porter à maturité (amandes) Nb 17,23 intransitif devenir mûr (raisins) Es 18,5.
