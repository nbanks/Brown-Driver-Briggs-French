BDB4013 [H3611]

hébreu biblique

כֶּ֫לֶב nom masculin 1 S 24,15 chien (hébreu tardif id. ; phénicien כלב ; araméen כַּלְבָּא, , arabe ; éthiopien assyrien kalbu DlHWB 328) — absolu ׳ כ Jg 7,5 + 16 t., כָּ֑לֶב Ps 59,7 ; Ps 59,15 pluriel כְּלָבִים 1 R 14,11 + 12 t. ; construit כַּלְבֵי Jb 30,1 suffixe כְּלָבֶיךָ Ps 68,24 — chien :

a. littéral 1 R 14,11 ; 1 R 16,4 ; 1 R 21,19 (deux fois dans le verset) ; 1 R 21,23 ; 1 R 21,24 ; 1 R 22,38 ; 2 R 9,10 ; 2 R 9,36 (tous de chiens féroces, affamés, dévorant des cadavres et léchant le sang) ; comparer Jr 15,3 et (לְשׁוֺן כְּלָבֶיךָ) Ps 68,24 mangeant de la chair déchirée, Ex 22,30 ( E ), כַּלְבֵי צאֹנִי = mes chiens de berger, seulement Jb 30,1 (dans les deux avec une idée d' infériorité) ; le sacrifice de chien était un rite païen Es 66,3 (Di et d'autres) ; voir surtout RSSemitic i, 273 (291), 325 (343) ; dans diverses comparaisons Jg 7,5 ; Pr 26,11 ; Pr 26,17 ; Ps 59,7 ; Ps 59,15 dans des proverbes : לֹא לִשֹׁנוֺ ׳ יֶחֱרַץכֿ Ex 11,7 (voir I. חרץ ) ; מֵת טוֺב מִןֿ ׳ לְכ הָאַרְיֵה הַמֵּת Qo 9,4.
b. appliqué, figuré, aux hommes, par mépris 1 S 17,43, ainsi des ennemis du psalmiste Ps 22,17 ; Ps 22,21, ou par humilité excessive 2 R 8,13 encore plus emphatiquement מֵת ׳ כ un chien mort, הַמֵּת ׳ הַכּ 2 S 9,8 ; 2 S 16,9 aussi ׳ ראֹשׁ כ 2 S 3,8 כְּלָבִים אִלְּמִים Es 56,10 (des prophètes égarants), עַזֵּי נֶפֶשׁ ׳ הַכּ Es 56,11 (id.) ; ׳ כ était le nom donné aux prostitués sacrés masculins Dt 23,19 (voir Dr ; comparer קָדֵשׁ).
