BDB4338

hébreu biblique

[לִהְלֵהַּ] verbe quadrilitère stupéfier, effrayer (comparer syriaque Palpel confudit, obstupefecit ; Ethpalpel stupore percussus est ; consternatio, tous dans les lexiques, PS1894) —

Hithpalpel Participe מִתְלַהְלֵהַּ comme substantif = insensé, suivi de הַיֹּרֶה זִקִים Pr Gn 26,18 (voir Now).
