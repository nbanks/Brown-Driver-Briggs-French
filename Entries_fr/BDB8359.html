hébreu biblique

שָׁאַג verbe rugir (hébreu tardif id. ; arabe mugir, bêler, Frey) ; —

Qal Parfait 3 masculin singulier ׳ וְשׁ Es 5,29 Kt consécutif (> Qr יִשְׁאַג), שָׁאָ֖ג Am 3,8, etc. ; Imparfait 3 masculin singulier יִשְׁאַג Am 3,4 +, etc. ; Infinitif absolu שָׁאֹג Jr 25,30 Participe שֹׁאֵג Jg 14,5 ; Ps 22,14, שׁוֺאֵג Ez 22,25, שֹׁאֲגִים So 3,3 ; Ps 104,21 —
1. rugir, du lion Jg 14,5 (avec לִקְרָאתוֺ), Am 3,4 ; Am 3,8 ; Ps 104,21 (avec ל de proie), figuré des envahisseurs et ennemis Jr 2,15 (avec עַל personne) Ps 22,14 , comparer Ps 74,4, comparaison Es 5,29 ; Jr 51,38 figuré des gouvernants rapaces So 3,3 ; Ez 22,25 (lire נְשִׂיאֶיהָ pour נְבִיאֶתהָ ᵐ5 et modernes) ; de ׳ י (comme un lion) appelant Israël dispersé Os 11,10 (deux fois dans le verset) (sur le texte de voir voir Now) ; de ׳ י rugissant dans le tonnerre Jr 25,30a = Am 1,2 = Jl 3,16, aussi Jr 25,30 (שָׁאֹג יִשְׁאַג, suivi de עַלנָֿוֵהוּ) ; comparer יִשְׁאַגקֿוֺל Jb 37,4 (|| יַרְעֵם).
2. du cri humain dans la détresse Ps 38,9 (avec מִן causatif).
