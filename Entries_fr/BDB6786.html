hébreu biblique

[פָּרַם] verbe déchirer, lacérer un vêtement (hébreu tardif = hébreu biblique ; arabe hacher des oignons, etc. (LandbergProv.421 ); syriaque couper, déchirer); — toujours avec ׃בְּגָדִים ```

Qal Imparfait 3 masculin singulier יִפְרֹם Lv 21,10 ( H ), 2 masculin pluriel תִּפְרֹ֑מוּ Lv 10,6 ( P ) ; Participe passif pluriel פְּרֻמִים Lv 13,45 ( P ).
