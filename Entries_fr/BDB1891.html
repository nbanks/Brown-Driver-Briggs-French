BDB1891 [H1754]

hébreu biblique

דּוּר nom [masculin] cercle, boule —

1. cercle : Du Es 29,3 pense que le nom ici aussi est כַּדּוּר (ou כִּדּוֺר) = כִּידוֺר assaut de combattants ; Marti כְּדָוִד comme ᵐ5 Lo Brd et d'autres ; וְחָנִיתִי כַּדּוּר עליך Es 29,3 ( > ᵐ5 Brd כְּדָוִד).
2. boule צָנוֺף יִצְנָפְךָ צְנֵפָה כַּדּוּר Es 22,18 (ainsi la plupart ; mais BöNÄ ii. 134 De Or JastrPAOS 1888, xciv ss. considèrent le nom comme כַּדּוּר, boule, comme dans le Talmud, comparer aussi MV SS sous le mot ; voir כדר ). [
3. tas, amas Ez 24,5 Thes Ke MV comparer מְדוּרָה Ez 24,9 mais voir ci-dessus ci-dessous דּוּר verbe]
