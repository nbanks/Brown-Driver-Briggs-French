BDB8998 [H8245]

hébreu biblique

שָׁקַד verbe veiller, être éveillé (phénicien שקר être circonspect ; hébreu tardif שָׁקַד être insistant ; voir ᵑ7 שְׁקֵד = hébreu biblique) ; —

Qal Parfait 3 masculin singulier ׳ שׁ Ps 127,1, שָׁקַ֫דְתִּי Jr 31,28 ; Ps 102,8 Imparfait 3 masculin singulier יִשְׁק֑וֺד Jb 21,32 ( Me Bu יִשְׁוֹ֑דוּ), וַיִּשְׁקֹד Dn 9,14, 1 singulier אֶשְׁקֹד Jr 31,28 Impératif masculin pluriel שִׁקְדוּ Esd 8,29 Infinitif construit לִשְׁקֹד Pr 8,34 Participe actif שֹׁקֵד Jr 1,12 + ; pluriel construit שֹׁקְדֵי Es 29,20 —
1. veiller sur, être vigilant au sujet de, ׳ י sujet, עַל personne, pour le bien ou le mal Jr 31,28 (deux fois dans le verset) ; Jr 44,27 עַל chose, pour l'accomplir Jr 1,12 ; Dn 9,14 ainsi (hommes sujet) שֹׁקְדֵי אָוֶן Es 29,20 du léopard guettant לְַעָֿרֵיהֶם Jr 5,6 (pour saisir une proie, figuré) ; de l'homme veillant (pour être admis) aux (עַל) portes de la sagesse Pr 8,34 (|| שָׁמַר) ; = monter la garde sur (עַל ) Jb 21,32, comparer (absolu) וְשִׁמְרוּ ׳ שׁ Esd 8,29 et (du gardien, שֹׁמֵר) Ps 127,1.
2. être éveillé, veiller, comme pleureur, souffrant Ps 102,8. — Lm 1,14 lire peut-être Niph`al Parfait נִשְׁקַד עַלפְּֿשָׁעַי ( ᵐ5 Bu) on a veillé sur mes transgressions (comparer 1 ci-dessus), pour ᵑ0 ׳ נִשְׂקַד עֹל פ. —
