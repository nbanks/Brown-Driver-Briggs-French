hébreu biblique

גּ֫שֶׁן nom propre [d'un lieu] (sur l'étymologie comparer Di Gn 45,10) —

1. district en Égypte à l'est du bas Nil, voir EbGS 500 ss. ; NavilleGoshen, 1887, qui lit les hiéroglyphes Kesem, mais comparer GroffJAs xiv. 527 ( ᵐ5 Γεσεμ (Ἀραβιας)) — אֶרֶץ גּשֶׁן Gn 45,10 ( J E ) Gn 46,34 ; Gn 47,1 ; Gn 47,4 ; Gn 47,6 ; Gn 47,27 ; Gn 50,8 ; Ex 8,18 ; Ex 9,26 (tous J ) ; אַ֫רְצָה גּשֶׁן Gn 46,28 ( J ), גּ֑שְׁנָה Gn 46,28 Gn 46,29 (tous deux J ), dans les deux ᵐ5 καθ' Ἡρώων πόλιν comparer Di. ( P a le pays רעמסס Gn 47,11 etc., comparer Di Gn 45,10.
2. ארץ הַגּשֶׁן Jos 11,16 ( D ), אֶרֶץ גּשֶׁן Jos 10,41 ( D ), גּשֶׁן Jos 15,51 ( P ) ( ᵐ5 dans tous Γοσομ) ville nommée avec חֹלֹן et גִּלֹה comparer Di. — non clairement localisée, mais à la frontière méridionale de Juda.
