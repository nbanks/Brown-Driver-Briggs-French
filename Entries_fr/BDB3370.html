BDB3370 [H2945]

hébreu biblique

טַף42 nom masculin collectif enfants (comme marchant à petits pas rapides et sautillants ; éthiopien Di1251) — טַף Jr 41,16 + 11 t. ; טָ֑ף Jr 40,7 + 5 t. ; suffixe טַפֵּנוּ Gn 43,8 + 5 t. ; טַפְּכֶם Gn 45,19 + 10 t. ; טַפָּם Gn 34,29 + 6 t. ; jamais construit et jamais pluriel (dans Gn 47,24 omis par ᵐ5 , comparer Di) ; — enfants, petits, Gn 34,29 ( E ) + 18 t. J E (Genèse Exode Nombres), aussi Dt 1,39 + 8 t. D (y compris Jos 1,14 ; Jos 8,35 ) ; P seulement Nb 31,9 ; Nb 31,17 ; Nb 31,18 , où noter זָכָר בַּטָּ֑ף Nb 31,17 = jeunes garçons, et הַטַף בַּנָּשִׁים Nb 31,18 = jeunes filles ; aussi Jg 18,21 ; Jg 21,10 ; 2 S 15,22 ; Jr 40,7 ; Jr 41,16 ; Jr 43,6 ; Ez 9,6 ; 2 Ch 20,13 ; 2 Ch 31,18 ; Esd 8,21 ; Est 3,13 ; Est 8,11 les mots מְתִים וְנָשִׁים וְהַטָ֑ף Dt 2,34 ; Dt 3,6 incluant tous les habitants ; ainsi אֲנָשִׁים וְהַנָּשִׁים וְהַטָּף Dt 31,12 ; Jr 40,7 ; Jr 43,6 טַף distingué de בָּחוּר וּבְּתוּלָה Ez 9,6 aussi de בְּנֵיהֶם 2 Ch 20,13, et de בְנֵיהֶם וּבְנוֺתֵיהֶם 2 Ch 31,18.

Noter que le mot inclut (ou implique) les femmes aussi bien que les enfants, dans ce qui suit : Gn 47,12 ; Ex 12,37 (E), Ex 10,10 ; Ex 10,24 (J), Nb 32,16 ; Nb 32,17 (J E ; comparer Di sur Nb 32,26).
