BDB8272 [H8135]

hébreu biblique

שִׂנְאָה nom féminin haine ; — absolu ׳ שׂ Nb 35,20 +, construit שִׂנְאַת Dt 1,27 ; Pr 25,10 suffixe שִׂנְאָתֶיךָ Ez 35,11 (pluriel אֹתֶיךָ — probablement voulu, Köii. 1, § 88 comparer Ol§ 131 k, mais שִׂנְאָֽתְךָ ᵐ5 Co Berthol Toy, comparer Ges «GKC:91l»§ 91l), etc. ; — haine (parfois opposé à אַהֲבָה) :

1. humaine, Nb 35,20 ( P ), Ez 23,29 ; Ez 35,11 ; Pr 10,12 ; Pr 10,18 ; Pr 15,17 ; Pr 26,26 ; Ps 109,5 ; Qo 9,1 ; Qo 9,6 ׳ דִּבְרֵי שׂ Ps 109,3 emphatique, שִׂנְאַת חָמָס Ps 25,19, ׳ תַּכְלִית שׂ Ps 139,22 (contre les ennemis de ׳ י), aussi (de répulsion sexuelle) גְּדוֺלָה ׳ שׂ 2 S 13,15 (tous accusatif de sens apparenté avec le verbe).
2. ׳ י's haine, comme verbe, avec accusatif de personne Dt 1,27 ; Dt 9,28.
