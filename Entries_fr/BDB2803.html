BDB2803 [H2502]

hébreu biblique

I. חָלַץ verbe retirer, extraire, se retirer, (hébreu tardif id. ; araméen חֲלַץ ; dépouiller, habituellement Pa`el etc. ; phénicien nom propre חלצבעל Baal a délivré, et חלץ seul ; arabe se retirer, se replier) —

Qal Parfait Os 5,6 וְחָֽלְצָה consécutif Dt 25,9 חָֽלְצוּ Lm 4,3 Imparfait 2 masculin singulier תַּחֲלֹץ ; Participe passif construit חֲלוּץ Dt 25,10 —
1. retirer, ôter, avec accusatif de la sandale Dt 25,9 ; Es 20,2 (les deux suivis de רֶגֶל avec מִן), comparer בֵּית חֲלוּץ הַנָּ֑עַל Dt 25,10 sortir, présenter le sein, d'un animal allaitant Lm 4,3.
2. intransitif se retirer (lui-même) suivi de מִן Os 5,6.
Niph`al Parfait être délivré נֶחֱלָ֑ץ Pr 11,8 (suivi de מִן) ; Imparfait יֵחָלֵ֑צוּ Pr 11,9 יֵחָֽלְצוּן Ps 60,7 = Ps 108,7 ( Ges«GKC:51»§ 51. 2. R. 2).
Pi`el Parfait חִלֵּץ Lv 14,43 etc. ; Imparfait יְחַלֵּץ Jb 36,15 וָאֲחַלְּצָה Ps 7,5 (mais sur le texte voir ci-dessous vers la fin) ; suffixe אֲחַלָּצְךָ Ps 50,15 אֲחַלְּצֶ֫ךָּ Ps 81,8, etc. ; Impératif חַלְּצָה Ps 6,5 חַלְּצֵנִי Ps 119,153 ; Ps 140,2 —
1. arracher, extraire Lv 14,40 ; Lv 14,43 ( P ; pierres d'une maison infectée).
2. secourir, délivrer, avec objet personne, mais seulement en poésie ; — suivi de נפשׁי Ps 6,5 ; Ps 116,8 (מִמָּ֫וֶת) ; comparer Ps 140,2 (suivi de מִן) ; absolu avec accusatif Ps 18,20 = 2 S 22,20 ; Ps 34,8 ; Ps 50,15 ; Ps 81,8 ; Ps 91,15 ; Ps 119,153 ; Jb 36,15 וָאֲחַלְּצָה Ps 7,5 suivi d'accusatif, mais j'ai délivré AV Hup Ri, comparer Ew ; et dépouillé Ges Hi De Bae (mais cela seulement en araméen) ; < et opprimé (lire וָאֶלְחְצָה) Krochm Dy Gr Che ; (|| גָּמַלְתִּי רַע).
