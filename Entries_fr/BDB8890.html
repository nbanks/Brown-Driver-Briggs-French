BDB8890 [H8150]

hébreu biblique

[שָׁנַן] verbe aiguiser, affûter (comparer arabe id. ; araméen שְׁנִינָא, tranchant, etc. ; éthiopien [ ] III, 2, contester, plaider ; d'où שֵׁן dent, hébreu tardif id., assyrien šinnu , arabe , éthiopien araméen שִׁנָּא, ) ; —

Qal aiguiser, affûter : Parfait 1 singulier אִם שַׁנּוֺתִ֫י בְּרַק חַרְבִּי ( Ges«GKC:67ee»§ 67ee) Dt 32,41, de la préparation de ׳ י pour le jugement ; 3 pluriel (כַחֶרֶב) שָֽׁנְנוּ לְשׁוֺנָם Ps 64,4 ; Ps 140,4 de proférer des paroles acérées ; Participe passif שְׁנוּנִים aiguisées, de flèches, Es 5,28 ; Ps 45,6, figuré de la langue Ps 120,4, du faux témoin Pr 25,18.
Pi`el Parfait consécutif וְשִׁנַּנְתָּם לְבָנֶיךָ Dt 6,7 c.-à-d. enseigner les paroles de manière incisive, allemand « einschärfen ».
Hithpolel Imparfait 1 singulier Ps 73,21 dans mes reins אֶשְׁתּוֺנָ֑ן j'étais transpercé (d'envie poignante).
