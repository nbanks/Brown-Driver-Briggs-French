BDB8075 [H7522]

hébreu biblique

רָצוֺן nom [masculin] bienveillance, faveur, agrément, volonté ; — absolu ׳ ר Dt 33,23 + ; construit רְצוֺן Ps 145,19 + ; suffixe רְצוֺנִי Es 60,10, רְצֹנוֺ Dn 8,4 +, etc. ; —

1. bienveillance, faveur :
a. de Dieu, Dt 33,16 ; Es 60,10 ; Ps 5,13 ; Ps 30,6 ; Ps 30,8 ; Ps 51,20 ; Ps 89,18 ; Ps 106,4 ; Pr 8,35 ; Pr 12,2 ; Pr 18,22 ׳ עֵת ר Es 49,8 ; Ps 69,14 ׳ יוֺם ר Es 58,5 ׳ שְׁנַתרֿ Es 61,2 || בְּרָכָה Dt 33,23 opposé à תּוֺעֵבָה Pr 11,1 ; Pr 11,20 ; Pr 12,22 ; Pr 15,8.
b. des hommes : en général Pr 10,32 ; Pr 11,27 ; Pr 14,9 des rois Pr 14,35 ; Pr 16,13 ; Pr 16,15 ; Pr 19,12.
2. agrément, de personnes, offrande de sacrifice, לרצון לפני יהוה Ex 28,38 ( P ) ; avec ל personne pour son agrément devant ׳ י ; avec suffixe personne לרצנוֺ Lv 1,3 abrégé לְרָצוֺן לָכֶם Lv 22,20, לרצנכם Lv 19,5 ; Lv 22,19 ; Lv 22,29 ; Lv 23,11, לצֵּון Lv 22,21 ; Es 56,7 ; Jr 6,20 ; Ps 19,15 (de paroles ; comparer Os 14,3) ; על רצון Es 60,7 (lire לְ, ainsi Manuscrits, voir Gi, aussi Du), רצון Ml 2,13.
3. volonté, désir, bon plaisir :
a. de Dieu, ׳ עשׂה ר faire sa volonté Esd 10,11 ; Ps 40,9 ; Ps 103,21 ; Ps 143,10.
b. de l'homme, עשׂה כרצונוֺ faire selon sa volonté, exactement comme il plaisait, Dn 8,4 ; Dn 11,3 ; Dn 11,16 ; Dn 11,36 ; Ne 9,24 ; Est 1,8 ; Est 9,5, comparer Ne 9,37 désir Ps 145,16 ; Ps 145,19 ; 2 Ch 15,15 = volonté propre Gn 49,6 (poème).
