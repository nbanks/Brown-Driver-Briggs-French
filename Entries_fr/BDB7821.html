BDB7821 [H8642]

hébreu biblique

תְּרוּמִיָּה nom féminin dénominatif ce qui appartient à une contribution, contribution accessoire ou secondaire, subdivision (sur la forme, voir Köii. 1, 204 ; strictement adjectif (employé comme substantif), Ges«GKC:86h»§ 86h) ; — ׳ ת מִתְּרוּמַת הָאָרֶץ Ez 48,12, d'une bande de terrain pour les prêtres dans le plan d'Ézéchiel (mais lire probablement תְּרוּמָה).
