araméen biblique

הֵא particule démonstrative, préfixée idiomatiquement à כְּ pour plus de précision, littéralement voici comme (hébreu biblique הֵא , voir הָא ci-dessus), seulement dans הֵאכְּֿ דִי Dn 2,43 הֵאכְּֿ דִי פַרְזְלָא לָא מִתְעָרַב de même que le fer ne se mélange pas avec l'argile (comparer a. ᵑ7 Onk הָא כְמַיָּא Gn 49,4, ainsi ᵑ7 Jon ׳ הא כ Es 5,28 ; Es 9,4 + souvent ; ᵑ7 Jer ׳ הֵי כ Gn 26,4 ; Gn 26,18 ; Gn 26,22 + souvent ; הֵיח Ps 2,9 ; Ps 5,10 ; Ps 5,13 +, aussi pléonastique ׳ הֵיח כְּ Ps 22,14 ; Ps 22,15 ; Ps 22,17 + souvent b. palmyrénien הֵיח comme (Lzb259), comme היך בנמוסא comme (c'est) dans la loi (Cooke321 ; ainsi ᵑ7 Jer Ps 2,9, etc.), הֵיח דִּי conjecture (comme Daniel הֵא כְּ דִי : = hébreu כַּאֲשֶׁר), comme היך די גבא de même qu'il a prélevé, CookeTariff ii. b 12. 18, c 14. 17. 21. 26 [ ᵑ7 Onk Jon pour כַּאֲשֶׁר ont ׳ כְּמָא דְ, ᵑ7 Jer ׳ הֵיכְמָא דְ, ᵑ6 . ].
