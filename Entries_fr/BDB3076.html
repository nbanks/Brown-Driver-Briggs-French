BDB3076 [H2713]

hébreu biblique

[חָקַר] verbe chercher, sonder (hébreu tardif id., Niphal passif, araméen חֲקַר) —

Qal Parfait 3 masculin singulier suffixe וַחֲקָרוֺ consécutif Pr 18,17 חֲקָרָהּ Jb 28,27 וְחָֽקַרְתָּ֫ Dt 13,15 etc. ; Imparfait יַחְקֹר Jb 13,19 יַחֲקָרֿ Ps 44,22 יַחְקְרֶנּוּ Pr 28,11 אֶחְקֹר 1 S 20,12 תַּחְקְרוּן Jb 32,11 נַחְקֹ֑רָה Lm 3,40 etc. ; Impératif חָקְרֵ֫נִי Ps 139,23 חִקְרוּ Jg 18,2 Infinitif חֲקֹר 2 S 10,3 + 2 t. ; לַחְקוֺר Pr 23,30 לְחָקְרָהּ Jg 18,2 —
1. chercher, suivi de l'accusatif Pr 23,30 ; Jb 32,11 absolu וְדָרַשְׁתָּ֫ וְחָקַרְתָּ֫ וְשָׁאַלְתָּ֫ הֵיטֵיב Dt 13,15 littéral Ez 39,14 (c.-à-d. des ossements non ensevelis).
2.
a. fouiller, explorer, littéralement, objet הארץ Jg 18,2 (|| רַגֵּל) Jg 18,2, 1 Ch 19,3 (|| id.) ; ville העיר 2 S 10,3 (|| id.), de l'extraction minière Jb 28,3 scruter un sujet ou une affaire Jb 5,27 ; Jb 28,27 ; Pr 25,2 ; Ps 44,22.
b. sonder un homme, découvrir ses sentiments ( AV le sonder) 1 S 20,12.
c. de ׳ י sondant l'homme, avec accusatif Ps 139,1 (|| ידע), Ps 139,23 (|| id.) Jb 13,9, comparer Jr 17,10 (objet לֵב).
d. d'examiner à fond, de manière à exposer la faiblesse d'une cause Jb 29,16 ; Pr 18,17, comparer Pr 28,11 ainsi de l'examen de soi-même דרכים ׳ ח Lm 3,40 (|| חפשׂ).
