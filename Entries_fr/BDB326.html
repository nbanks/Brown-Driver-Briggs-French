hébreu biblique

II. אָח nom féminin brasero (arabe dans Thes, est une erreur) est un emprunt à l'égyptien &las— selon WMMOLZ 3, 51 ; Jr 36,22 lire וְאֵשׁ הָאָח ; toujours avec l'article Jr 36,22 לְפָנָיו ׳ הַא מְבֹעָ֑רֶת le brasero devant lui brûlait ; Jr 36,23 ׳ הָאֵשׁ אֲשֶׁר אֶלהָֿא ; Jr 36,23 ׳ הָאֵשׁ אֲשֶׁר עַלהָֿא (I. אָח, frère voir ci-dessous אחה ; III. אָח interjection, p. 25).
