רְכוּשׁ, רְכֻשׁ nom masculin Gn 13,6 propriété, biens ; — absolu רְכוּשׁ Nb 16,32 +, רְכֻשׁ Gn 14,16 ; Gn 14,21 construit רְכוּשׁ 2 Ch 35,7, רְכֻשׁ Gn 14,11 suffixe רְכֻשׁוֺ Gn 14,12 +, רְכוּשָׁם Gn 12,5 +, etc. ; — propriété, biens, Proverbes, Chroniques, Daniel, Gn 14 :

1. terme général pour les biens meubles de toute sorte (incluant souvent spécifiquement le bétail), Gn 12,5 ; Gn 15,4 ; Nb 16,32 ; 1 Ch 27,31 ; 1 Ch 28,1 ; 2 Ch 31,3 ; 2 Ch 32,29 ; Esd 8,21 ; Esd 10,8.
2. utilisé spécifiquement du bétail, des troupeaux, etc. Gn 13,6 ; Gn 31,18 ; Gn 36,7 ; Nb 35,3 ; 2 Ch 35,7.
3. de provisions, ustensiles, etc. Gn 46,6 ; Esd 1,4 ; Esd 1,6, comme bagages de camp Dn 11,13, surtout comme butin Gn 14,11 ; Gn 14,12 ; Gn 14,16 (deux fois dans le verset) ; Gn 14,21 ; 2 Ch 20,25 ; 2 Ch 21,14 ; 2 Ch 21,17 ; Dn 11,24 ; Dn 11,28.
