hébreu biblique

I. נְצִיב nom masculin colonne, préfet, aussi apparemment garnison, poste ( Qo 46,18 ויבנע נציבי צר et il soumit des garnisons, postes, de l'ennemi) ; — ׳ נ absolu 1 R 4,19 construit 1 S 13,3 + 3 t. ; pluriel נְצִיבִים 2 S 8,6 + 2 t. + 2 Ch 8,10 Kt , נְצִבִים 2 S 8,14 (deux fois dans le verset) ; construit נְצִבֵי 1 S 10,5 ᵑ0 , mais lire נְצִיב ᵐ5 ᵑ6 ᵑ9 Th We Dr Klo Kit Bu HPS ; — ```

1. colonne , מֶלַח ׳ נ Gn 19,26 ( J ).
2. préfet, député 1 R 4,19 (sur le texte comparer Klo Benz), ainsi peut-être aussi 1 S 10,5 (HPS ; colonne ᵐ5 Th Dr Klo Kit), et apparemment 1 S 13,3 ; 1 S 13,4 (> al. colonne) ; ce sens convient aussi pour 2 S 8,6 ; 2 S 8,14 (deux fois dans le verset) (ainsi Ki ; Th HPS garnison) 1 Ch 11,16 ; 1 Ch 18,13 ; 2 Ch 8,10 (où lire Kt ), 2 Ch 17,2 (Kau garnison dans 2 Ch 11,16 ; 2 Ch 17,2 ; Qo 46,18 — voir ci-dessus — favoriserait ce sens dans les autres passages).
