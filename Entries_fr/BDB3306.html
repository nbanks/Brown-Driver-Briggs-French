III. טוֺב nom masculin Jr 17,6 (comparer טוֺבָה nom féminin) une bonne chose, bienfait, bien-être : — ׳ ט Gn 26,29 + 134 t. ; —

1. bien-être, prospérité, bonheur : || שָׁלוֺם Es 52,7 ; Jr 8,15 ; Jr 14,19 אם טוב ואם רע Jr 42,6 que ce soit prospérité ou adversité ; יבוא טוב Jr 17,6 la prospérité vient ; ׳ שׂמח בט Dt 26,11 ; 2 Ch 6,41 ׳ ראה בט Qo 2,1 ; Jr 29,32 objet de ראה Ps 4,7 ; Ps 34,13 ; Qo 2,24 ; Qo 3,13 ; Jb 7,7 de דרשׁ Est 10,3 de בקשׁ Ps 122,9 מנע Ps 84,12 ; Jr 5,25 נתן Ps 85,13 קוה Jb 30,26 מצא Pr 16,20 ; Pr 17,20 ; Pr 18,22 ; Pr 19,8 נחל Pr 28,10 חסר Ps 34,11 בשׂר 1 R 1,42 ; Es 52,7 ׳ חול לט Mi 1,12 ׳ חכמה וט 1 R 10,7 טוב וחסד Ps 23,6 בטוב dans la prospérité Ps 25,13 ; Jb 21,13 ; Jb 36,11 ; Qo 7,14 מטוב à cause de la prospérité Za 1,17 loin du bonheur Ps 39,3 כטוב Os 10,1 לטוב Dt 30,9 ; Jr 15,11 לטוב לָנוּ Dt 6,24 pour notre bien, Dt 10,13 (לְךָ), Jr 32,39 (לָהֶם ; comparer לרע ל Jr 7,6 ; Jr 25,7).
2. bonnes choses, (collectif) ברכ(ו)ת ׳ ט bénédiction de bonnes choses Ps 21,4 ; Pr 24,25 מִלֵּא ׳ בָתֵּיהֶם ט Jb 22,18 a rempli leurs maisons de bonnes choses ; objet de שׂבע Jb 104,28 ; Pr 12,14 מלא Ps 107,9 אכל Pr 13,2 ; Es 55,2 לקח Os 14,3 ׳ השׂביע בט Ps 103,5 biens, possessions 1 S 15,9.
3. bon = bienfait : גְּמָלַתְהוּ טוֺב וְלֹארָֿע Pr 31,12 elle lui fait du bien et non du mal ; תַּאֲוַת צַדִּיקִים אַךְטֿוֺב Pr 11,23 le désir des justes n'est que du bien ; objet de קבּל Jb 2,10 מנע Pr 3,27 שׁחר Pr 11,27 שׁלּם Pr 13,21 עשׂה Gn 26,29 ( J ), Ps 119,65 היטיב Nb 10,32 ( J ); ׳ ערב לט Ps 119,122 ׳ חֹרְשֵׁי ט Pr 14,22 ceux qui méditent le bien.
