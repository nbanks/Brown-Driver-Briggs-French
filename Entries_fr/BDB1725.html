BDB1725 [H1610]

hébreu biblique

[גַּף] nom masculin corps, soi-même, hauteur, élévation (comparer hébreu tardif גּוּף) — singulier suffixe גַּפּוֺ Ex 21,3 (deux fois dans le verset) ; Ex 21,4 pluriel construit גַּפֵּי Pr 9,3 —

1. corps, soi-même, seulement dans l'expression יָבאֹ (יֵצֵא) בְּגַפּוֺ) il entrera (sortira) seul (avec son corps) Ex 21,3 (deux fois dans le verset) ; Ex 21,4 dans la loi sur l'esclave (comparer Di).
2. hauteur, élévation עַלגַּֿפֵּי מְרֹמֵי קָ֑רֶת Pr 9,3.
