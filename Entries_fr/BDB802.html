BDB802 [H794]

hébreu biblique

[אֲשֵׁדָה] nom féminin fondation, pente (de montagne) (comparer ci-dessus) — seulement pluriel absolu אֲשֵׁדוֺת Jos 10,40 ; Jos 12,8 construit אַשְׁדּוֺת Jos 12,3 ; Jos 13,20 אַשְׁדֹּת Dt 3,17 ; Dt 4,49 — pentes de montagne Jos 10,40 ׳ ההר והנגב והשׁפלה והא ; Jos 12,8 (|| id. + מדבר) ; ailleurs défini ׳ תַּחַת א הַפִּסְגָּה Dt 3,17 ; Dt 4,49 ; Jos 12,3 comparer Jos 13,20.
