BDB3287 [H2881]

hébreu biblique

I. טָבַל verbe tremper (hébreu tardif id. ; araméen טְבַל tremper, baigner ; comparer peut-être arabe saturavit tinctura vestem, Frey) —

Qal Parfait וְטָבַל consécutif Lv 4,6 + 5 t. ; וְטָבַלְתְּ consécutif Rt 2,14 וּטְבַלְתֶּם consécutif Ex 12,22 Imparfait וַיִּטְבֹּל Lv 9,9 + 3 t. ; 2 masculin singulier suffixe תִּטְבְּלֵנִי Jb 9,31 וַיִּטְבְּלוּ Gn 37,31 — tremper :
1. transitif, tremper une chose dans, avec accusatif de la chose + בְּ : dans le sang Gn 37,31 ( J ), ainsi surtout en lien avec les sacrifices Lv 4,6 ; Lv 9,9 ; Lv 14,6 ; Lv 14,51 (aussi dans l'eau vive) ; dans l'eau, pour la purification Nb 19,18 (tous P ) ; de tremper un bâton dans le miel 1 S 14,27, du pain dans le vinaigre Rt 2,24, le pied dans l'huile Dt 33,24 (poème) ; בַּשַּׁחַת תִּטְבְּלֵנִי Jb 9,31 dans la fosse tu me plonges ; accusatif omis Ex 12,22 ( J E ; dans le sang), 2 R 8,15 (dans l'eau) ; mais aussi + מִן = humecter avec, avec un peu de Lv 4,17 (sang), Lv 14,16 (huile).
2. intransitif, se tremper (soi-même), suivi de בְּ, 2 R 5,14 dans le Jourdain (|| רָחַץ בְּ 2 R 5,10 2 R 5,12).
Niph`al Parfait 3 masculin pluriel נִּטְבְּלוּ Jos 3,15 être trempé, בְּ de l'eau.
