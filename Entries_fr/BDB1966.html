BDB1966 [H1826]

hébreu biblique

I. [דָּמַם] verbe devenir muet, silencieux, immobile (hébreu tardif id. ; éthiopien en II. stupéfier ; III., 1 et 2 être stupéfait, hébété) —

Qal Parfait דַּ֫מּוּ Jb 30,27, דָּ֑מּוּ Ps 35,15 Imparfait יִדֹּם Am 5,13 + 2 t., וַיִּדֹּם Lv 10,3 ; Jos 10,13 3 féminin singulier תִּדֹּם Lm 2,18 2 féminin singulier תִּדֹּמִּי Jr 48,2 (ainsi Ki Ol et Kö327 q. v. ; d'autres Niph`al) ; וָאֶדֹּם Jb 31,34 : יִדְּמוּ Ex 15,16 + 3 t. ; 1 pluriel cohortatif נִדְּמָה Jr 8,14 (ainsi Thes Ol Bö Kö327 q. voir ; d'autres Niph`al) ; —
1. être silencieux Ps 4,5 ; Ps 30,13 ; Ps 35,15 (probablement, comparer De Che), Ez 24,17 ; Lv 10,3 ( P ) Am 5,13 dans le deuil Lm 2,10 ; Lm 3,28.
2. être immobile (opposé à la parole et au mouvement) Jb 31,34 = périr Jr 8,14 ; Jr 48,2 ; Ps 31,18 (לשׁאול ׳ ד) ; avec לְ être silencieux devant c.-à-d. se résigner à Ps 37,7 ; Ps 62,6, suivi de לְמוֺ עֲצָתִי Jb 29,21 (|| לִי שָֽׁמְעוּ) ; être immobile, ne pas bouger, s'arrêter Jos 10,12 ; Jos 10,13 (du soleil || עמד) ; 1 S 14,9 (hommes) ; Jr 47,6 (de l'épée || הֵרָֽגְעִי) ; ainsi aussi des entrailles, comme siège de l'agitation mentale Jb 30,27 ; Lm 2,18 אַלתִּֿדֹּם בַּתעֵֿינֵךְ, c.-à-d. ne cesse pas de pleurer (|| הוֺרִידִי כַנַּחַל דִּמְעָה ).
3. être frappé de mutisme, stupéfait, dans l'étonnement et la crainte Ex 15,16 (כאבן) Es 23,2 (mais comparer II. דמם).
