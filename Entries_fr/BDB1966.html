BDB1966 [H1820]

hébreu biblique

דָּמַם verbe être muet, silencieux, immobile (araméen דְּמַם ; syriaque être stupéfié ; éthiopien être stupéfait ; arabe détruire, périr ; Assyrien damâmu gémir, se lamenter, pleurer) —

Qal דָּֽמְמָה Lv 10,3 ; Jb 29,21 וַיִּדֹּם 1 S 14,9 + 4 t. ; Infinitif דֳּמִי Ps 37,7 ; Participe דֹמֵם Es 23,2 ; Jr 47,6 (Ges§ 72 R. 2) —
1. être réduit au silence, être muet Lv 10,3 ( H ) Jr 47,6 Ps 4,5 (דֹמּוּ = soyez muets).
2. être silencieux, le glaive וַיִּדֹּם Jos 10,13 ( JE ).
3. être immobile, rester tranquille 1 S 14,9 Es 23,2 Ps 4,5 ; Ps 30,13 ; Ps 37,7 Jb 29,21 ; Jb 31,34 souvent avec préposition לְ attendre, Ex 15,16 ( Cantique de la mer dans J E ) 2 S 6,16 Lm 3,28.
Niph`al Participe être abasourdi, être stupéfié Jr 8,14 (+ נֵאבַד).
Po`. Participe מְדוֺמֵם Ps 94,17 (|| שְׁאוֺל).
Hiph`il הִדְמִיתִי Ps 131,2 ; Imparfait 2 masculin singulier תַּדְמֶה Jr 48,2 ; Impératif הַדְמִי Lm 2,18 — cesser, être coupé court (métaphoriquement, des nations), Jr 48,2, également Lm 2,18 (de larmes, voir Löhr) ; faire taire Ps 131,2 (+ suivre).
