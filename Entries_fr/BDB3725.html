BDB3725 [H4175 H4176]

hébreu biblique

II. מוֺרֶה nom masculin Es 30,20 maître, enseignant ; — absolu dans אֵלוֺן מוֺרֶה Gn 12,6 ( J ) = אֵלוֺנֵי מֹרֶה Dt 11,30 ( ⅏ ᵐ5 אֵלוֺן), le térébinthe de l'enseignant (voir אֵלוֺן ) près de Sichem ; comparer גִּבְעַת הַמּוֺרֶה Jg 7,1 colline de l'enseignant près de la plaine de Jizréel, probablement le Petit Hermon, Nebî Daµî BdPal 244 ; le térébinthe étant un arbre sacré d'où l'enseignement divin était donné, et la colline de l'enseignant le siège d'un lieu saint d'où l'enseignement divin était donné ; voir aussi (de Dieu) Jb 36,22 ici appartiennent probablement aussi מוֺרָ֑י Pr 5,13, et מוֺרֶיךָ Es 30,20 (deux fois dans le verset).
