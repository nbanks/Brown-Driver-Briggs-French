BDB2428 [H2203]

hébreu biblique

זֶ֫פֶת nom féminin Es 34,9 poix (comparer sur le formatif Ol§ 164 c) Sta§ 187 a. Le mot est arménien selon LagArm. Stud. 1351, BN 219 ; sur les dérivés égyptiens comparer CookSpeaker's Comm. i. 484 ; arabe selon Frä151, si sémitique, est un emprunt araméen ; voir aussi éthiopien Di1068) — poix : — וַתַּחְמְרָה בַחֵמָר וּבַזָּ֑פֶת Ex 2,3 du panier de joncs ; Es 34,9 a וְנֶחֶפְכוּ נְחָלֶיהָ לְזֶפֶת וַעֲפָרָהּ לְגָפְרִ֑ית, Es 34,9 b והיתה אַרְצָהּ לְזֶפֶת בֹּעֵרָֽה׃.
