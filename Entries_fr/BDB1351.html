BDB1351 [H1249]

hébreu biblique

II. בַּר adjectif pur, propre, Jb 11,4 ; Ps 24,4 בָּר Pr 14,4 pluriel construit בָּרֵי Ps 73,1 féminin בָּרָה Ps 19,9 ; Ct 6,9 ; Ct 6,10 —

1. pur, clair : בר לבב pur de cœur Ps 24,4 ברי לבב Ps 73,1 une pure jeune fille Ct 6,9 ; Ct 6,10, homme Jb 11,4, commandements de Dieu Ps 19,9.
2. propre : אֵבוּס בָּר la crèche est propre Pr 14,4.
3. peut-être adverbe נַשְּׁקובַֿר embrasser purement, d'un sincère hommage Ps 2,12 mais comparer I. בַּר, p. 135.
