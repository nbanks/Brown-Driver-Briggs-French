hébreu biblique

1. épée,
a. comme arme de guerre Gn 48,22 ( E ), Jg 7,14 ; Jg 7,20 ; 1 S 21,9 , et ainsi à toutes les époques ; épée (courte) à deux tranchants וְלָהּ שְׁנֵי פֵיוֺ֖ת גֹּמֶד אָרְכָּהּ ׳ ח Jg 3,16, comparer פִּיּוֺת ׳ ח Pr 5,4 (en comparaison), פִּיפִיּוֺת ׳ ח Ps 149,6 voir aussi ׳ צוּר ח tranchant de l'épée Ps 89,44.
b. ceindre l'épée = ׳ חָגַר הח 1 S 17,39 ; 1 S 25,13 (3 t. dans le verset) ; Ps 45,4 (׳ הח ׳ ח עַל יָרֵך), שִׂים על ירך Ex 32,27 , comparer surtout 2 S 20,8.
c. tirer l'épée ׳ הֵרִיק הח Ex 15,9 ; Lv 26,33 ; Ez 5,2 ; Ez 5,12 ; Ez 12,14 ; Ez 28,7, ׳ הוציא הח Ez 21,8 ; Ez 21,10, ׳ פתח הח Ps 37,14 ; Ez 21,33, נטשׁ Es 21,15 (Gr Che לטשׁ voir ci-dessous) ; habituellement ׳ שׁלף הח Nb 22,23 ; Nb 22,31 ; Jos 5,13 ; Jg 3,22 ; Jg 8,10 ; Jg 8,20 ; Jg 9,54 ; 1 S 17,51 ; 1 S 31,4 = 1 Ch 10,4 ; 1 Ch 21,16, et comme caractéristique des guerriers ׳ שֹׁלֵף ח Jg 20,2 ; Jg 20,15 ; Jg 20,17 ; Jg 20,25 ; Jg 20,35 ; Jg 20,46 ; 2 S 24,9 ; 2 R 3,26 ; 1 Ch 21,5 (deux fois dans le verset) ; ׳ וַתִּדְבַּק יָדוֺ אֶלהַֿח 2 S 23,10, insérer aussi dans || 1 Ch 11,13 (Dr).
d. aiguiser l'épée ׳ לטשׁ הח Ps 7,13 (comparer Es 21,15 ci-dessus), mais aussi שׁנן Dt 32,41 comparer Ps 64,4, הוּחַדָּה Ez 21,14 ; Ez 21,16.
e. remettre l'épée dans le fourreau אֶלנְֿדָנָהּ ׳ הֵשִׁיב הח 1 Ch 21,27 (le fourreau de l'épée est ailleurs תַּעַר 1 S 7,51 ; 2 S 20,8 ; Jr 47,6 ; Ez 21,8).
f. tuer par l'épée הרג ׳ בח Jos 10,11 ; Jos 13,22 + ; dans ימותו אנשׁים 1 S 2,33 , insérer בְּחֶרֶב avant אנשׁים ᵐ5 We Dr ; rarement ׳ הכה בח Jos 11,10 ; 2 R 19,37 = Es 37,38 très souvent ׳ הִכָּה לְפִיחֿ frapper selon la bouche de l'épée c.-à-d. comme l'épée peut dévorer (2 S 2,26 ; 2 S 11,25) = sans quartier Nb 21,24 ( E ) Dt 13,16 a Dt 20,13 ; Jos 11,11 ; Jos 11,12 ; Jos 11,14 (tous D ) Jg 18,27 ; Jg 21,10 ; 1 R 10,25 ; Jb 1,15 ; Jb 1,17 aussi avec העיר comme seul objet Jos 8,24 ; Jos 10,28 ; Jos 10,30 ; Jos 10,32 ; Jos 10,35 ; Jos 10,37, comparer Jos 10,39 ; Jos 19,47 ; Jg 1,8 ; Jg 1,25 ; Jg 20,37 ; 1 S 22,19 (deux fois dans le verset) ; 2 S 15,14 ׳ הרג לפיחֿ Gn 34,26 ( J ) ; ׳ החרים לפיחֿ Dt 13,16b (incluant הָעִיר dans l'objet), Jos 6,21 ; 1 S 15,8 , ׳ חלשׁ לפיחֿ Ex 17,13 ( E ) ; ׳ נפל לפיחֿ Jos 8,24 ; Jg 4,16 ׳ ל֗֗֗פיחֿ ׳ וַיָּהָם י Jg 4,15 — (׳ לפיחֿ se trouve principalement dans Josué et Juges (21 t.), dans l'Hexateuque seulement J E D ; mais Jb post-exilique).
g. ׳ אכלה הח etc., de l'épée comme dévorant : — תּאֹכֵל חַרְבִּי בָשָׂר Dt 32,42 (poésie), 2 S 2,26 ; 2 S 11,25 ; 2 S 18,8 ; Es 1,10 ; Jr 2,30 ; Jr 46,10 ; Jr 46,14 ; Na 2,14.
h. tué par l'épée est ׳ חֲלַלחִֿ Nb 19,16 ( P ) et fréquemment Ézéchiel, ׳ חַלְלֵיחֿ Ez 31,17 ; Ez 31,18 ; Ez 32,21 ; Ez 32,25 ; Ez 32,28 ; Ez 32,29 ; Ez 32,30 ; Ez 32,32 ; Ez 35,8 ׳ מְחֻלְלֵי ח v Ez 35,26 < Co ׳ חלליחֿ (sur le texte d'Ez 32,20 ; Ez 32,22 ; Ez 32,31 voir Co).
i. ׳ שְׁבֻיוֺת ח Gn 31,26 sont des captifs de (pris par) l'épée .
j. l'épée fréquemment agent de ׳ י Jg 7,20 ; Es 34,6 ; Jr 12,12 ; Jr 47,6 etc., — ainsi Dt 28,22, mais lire חֹרֶב voir ci-dessous I. חרב (comparer d'autre part 1 S 17,47) ; noter surtout Gn 3,24 ׳ לַהַט הח הַמִּתְהַפֶּכֶת ; en poésie ׳ לַהַכ ח Na 3,3 , ׳ בְּרַק ח Dt 32,41.
k. ׳ ח figuré de la langue Ps 57,5 comparer Ps 59,8 figuré de la violence, guerre, etc., Gn 27,40 en comparaison Pr 5,4 (de la fin pénible des relations avec des femmes étrangères).
2. couteau צֻרִים ׳ ח couteaux de silex pour l'usage dans la circoncision Jos 5,2 ; Jos 5,3 (voir WeSkizzen iii, 166) ; Ez 5,1 ; Ez 5,2 חַדָּה תַּעַר ׳ ח הַגַּלָּבִים (selon Co l'usage de l'épée comme rasoir est significatif).
3. d'outils utilisés pour tailler la pierre חַרְבְּךָ הֵנַפְתָּ עליה Ex 20,25 ׳ יִתֹּץ בח Ez 26,9 , selon la plupart = outils, haches, « acier » (Co ' Eisen '), mais peut-être des épées comme instrument à portée de main ; 2 Ch 34,6 Qr בְּחַרְבֹתֵיהֶם avec leurs outils ; lire peut-être בְּחָרְבֹתֵיהֶם dans leurs ruines, Be, qui compare Ps 109,10 voir חָרְבָּה ci-dessous II. חרב ( ᵐ5 ᵑ9 omettent חרב).
