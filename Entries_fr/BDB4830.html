BDB4830 [H4843]

hébreu biblique

I. מָרַר verbe être amer (hébreu tardif id. ; assyrien marâru et dérivés DlHWB 427 ; arabe s'irriter (de la bile), II. rendre amer, IV. devenir amer ; amer ; éthiopien ; araméen , מְרַר) ; —

Qal Parfait 3 masculin singulier מַר Es 38,17 + 2 t. ; 3 féminin singulier מָ֫רָה 1 S 30,6 ; 2 R 4,27 Imparfait 3 masculin singulier יֵמַר Es 24,9 — être amer :
1. au sens propre יֵמַר שֵׁכָר לְשֹׁתָיו Es 24,9.
2. figuré מָ֫רָה נֶפֶשׁ 1 S 30,6 l'âme de tout le peuple fut amère contre (עַלֿ ) ; נַפְשָׁהּ מָ֫רָהלָֿהּ 2 R 4,27 son âme, elle est amère pour elle (c.-à-d. elle est dans une détresse amère) ; impersonnel מַרלִֿי מְאֹד מִכֶּם Rt 1,13 c'est très amer pour moi à cause de vous (je suis très affligée), comparer Lm 1,4. — לְשָׁלוֺם מַרלִֿי מָ֑ר Es 38,17 est douteux ; CheComm. pour (mon) bien-être c'était (si ) amer pour moi, (si) amer, comparer Brd ; ainsi De Kau (traitant מר apparemment comme adjectif ) ; < Drechsl. Di pour mon bien-être l'amer est devenu amer pour moi (מָ֑ר adjectif) ; Lo Gr, comparer Buhl (sous מַר adjectif ), lire מָר pour מַר (√ מור), l'amer s'est changé pour moi en bien-être ; CheHpt suppriment מָ֑ר ; Du (d'après ᵐ5 ) supprime la proposition comme glose.
Pi`el Imparfait 1 singulier אֲמָרֵר Es 22,4 3 masculin pluriel וַיְמָֽרֲרוּ Ex 1,14, suffixe וַיְמָֽרֲרֻהוּ Gn 49,23 — rendre amer, montrer de l'amertume : —
1. ׳ וַיְמ Gn 49,23 (poème) et les archers montrèrent de l'amertume (c.-à-d. une hostilité amère) envers lui (suffixe d'objet indirect) ; אֲמָרֵר בַּבֶּכִ֑י Es 22,4 je montrerai de l'amertume dans les pleurs (= pleurer amèrement).
2. אֶתחַֿיֵּיהֶם ׳ וַיְמ Ex 1,14 ( P ) et ils rendirent leur vie amère par l'esclavage.
