H. מ־ַן) nom propre, masculin ou [d'un lieu] (comparer palmyrénien nom propre שלמן GACooke299 ; phénicien nom propre, d'une divinité שלמן Lzb377 GACooke42 ; assyrien Salamanu , d'un prince moabite COT Os 10,14 ; nom propre, d'une divinité Šulmanu, voir le suivant) ; — בֵּית אַרְבֵּאל ׳ כְּשֹׁד שׁ Os 10,14, Σαλαμαν (comparer aussi FieldHex. ii. 957) ; — douteux, We Now pensent = le suivant (et proposition insérée après l'époque d'Osée), comparer aussi Marti ; d'autres comparent un prince moabite, voir ci-dessus ; SpiegelbZA xiii (1898), 120 s. suggère nom propre, d'un lieu, comparer Ša-ra-ma-na en égyptien
