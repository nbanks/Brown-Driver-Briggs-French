hébreu biblique

[נָדַף] verbe chasser, disperser ( hébreu tardif id., répandre, être diffusé, disséminé (d'une odeur) ; arabe frapper, battre, surtout la corde de l'arc avec un maillet, pour nettoyer le coton ; aussi jouer de la lyre ; araméen נְדַף = hébreu tardif ; éthiopien jeter, lancer) ; —
