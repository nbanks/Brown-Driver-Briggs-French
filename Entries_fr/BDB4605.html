BDB4605 [H4242]

hébreu biblique

I. מְחִיר nom masculin prix, salaire (comme ce qui est placé devant quelqu'un, présenté, offert ; assyrien ma—îru , prix ; ma—irânu , acheteur, DlHWB 404, Prol.94 ) ; — ׳ מ absolu Mi 3,11 + 9 t. ; construit Dt 23,19 + 2 t. ; suffixe מְחִירָהּ Jb 28,15 pluriel suffixe מְחִירֵיהֶם Ps 44,13 —

1. prix, souvent בִּמְחִיר pour un prix ; dans le commerce 2 S 24,24 ; 1 R 10,28 ; 2 Ch 1,16 comparer Lm 5,4 (|| כֶּסֶף), aussi (sans בְּ) 1 R 21,2 et שָׂדֶה עַתּוּדִים ׳ מ Pr 27,26 de la sagesse Jb 28,15 (|| סְגוֺר כֶּסֶף) comparer Pr 17,16 (sans בְּ ) ; figuré ׳ בְּלוֺאבֶֿסֶף וּבְלוֺא מ Es 55,1 dans l'achat des bénédictions de ׳ י ; du prix des gens, figuré Ps 44,13, comparer Jr 15,13.
2. salaire, כֶּלֶב ׳ מ (c.-à-d. du prostitué mâle) Dt 23,19 (|| אֶתְנַן זוֺנָה) voir Di Dr ; de prêtres vénaux Mi 3,11 (|| שֹׁחַד) ; = récompense, gain , nié de Cyrus Es 45,13 (|| id.) ; affirmé d'Ant. Épiphane Dn 11,39.
