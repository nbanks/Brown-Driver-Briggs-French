hébreu biblique

צִידֹנִי adjectif [d'un peuple] du précédent ; — ׳ צ Jg 3,3 ; Ez 32,30 ailleurs pluriel צִידֹנִים Dt 3,9 +, צִידוֺנִים Jg 10,12 צִדֹנִים Jg 18,7 +3 t. ; féminin pluriel צֵדְנִיֹּת 1 R 11,1 — comme substantif = Sidoniens, collectif avec article Ez 32,30 ; Jg 3,3 dans la littérature ancienne apparemment = Phéniciens Dt 3,9 ; Jos 13,4 ; Jos 13,6 ; Jg 3,3 ; Jg 10,12 ; Jg 18,7 (deux fois dans le verset) ; 1 R 5,20 (= Tyr v.1 R 5,15 ), 1 R 16,31 (si, [JosAnt. «JOSEPHUS:Antiquities 8:13»viii. 13, 1] Ethb. était aussi roi de Tyr), Ez 32,30 aussi ׳ עַשְׁתֹּרֶת אְֶהֵֹי צ 1 R 11,5 ; 1 R 11,33, comparer 2 R 23,13 nommé avec les Tyriens 1 Ch 22,4 ; Esd 3,7.
