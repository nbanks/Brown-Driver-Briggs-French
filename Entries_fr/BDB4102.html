BDB4102 [H3658]

hébreu biblique

כִּנּוֺר nom masculin Ps 81,3 lyre (sur la formation comparer LagBN 89, Anm. ; hébreu tardif id. ; araméen כִּנָּרָא, ; mandéen כינאר(אׅ NöM § 104 (qui met en doute l'origine sémitique) ; arabe , ; comme emprunt en grec hellénistique κινύρα ( ᵐ5 et Joseph.) ; en égyptien kn-an-aul selon Bondi79) — absolu ׳ כ Gn 4,21 + ; suffixe כִּנֹּרִי Jb 30,31 pluriel כִּנֹּרוֺת 1 R 10,12 + ; suffixe כִּנּוֺרַיִךְ Ez 26,13, כִּנֹּרוֺתֵינוּ Ps 137,2 — lyre, instrument à cordes utilisé pour la musique populaire aussi bien que sacrée (comparer BenzArchaeology 273 ss. NowArchaeology i. 273 ss.) ; — ׳ יְעוּגָבכ Gn 4,21, ׳ בְּתֹף וּבְכ Gn 31,27 ׳ אִישׁ יֹדֵעַ מְנַגֵּן בַּכּ 1 S 16,16 un homme habile à jouer de la lyre, comparer 1 S 16,23 (וִנִגֵּן בְּיָדוֺׅ) ; aux banquets Es 5,12 (+ נֶבֶל, תֹּף, חָלִיל), signe de réjouissance Es 24,8 (+ תֻּפִּים), Ez 26,13, comparer וַיְהִי לְאֵבֶל כִּנֹּרִ֑י וְעֻגָבִי לְקוֺל בֹּכִים Jb 30,31, וְיִשְׂמְחוּ לְקוֺל עוּגָב ׳ יִשְׂאוּ כְתֹף וְכ Jb 21,12 porté par une femme légère Es 23,16 (figuré de Tyr) ; en comparaison Es 16,11 (du son murmurant des entrailles = cœur, de pitié) ; utilisé aussi pour louer ׳ י, habituellement comme accompagnement du chant (comparer 1 Ch 15,16) ׳ נֵבֶל וְתֹף וְחָלִיל וְכ 1 S 10,5 ; Es 30,32 (|| תֻּפִּים) ; surtout devant l'arche 2 S 6,5 (+ נְבָלִים, תֻּפִּים, etc.) || 1 Ch 13,8, et dans le sanctuaire (habituellement + נֵבֶל, etc.) 1 R 10,12, mais surtout Chroniques, Psaumes : — 1 Ch 13,8 (pluriel, et ainsi principalement dans les Chroniques), 1 Ch 15,16 ; 1 Ch 15,21 ; 1 Ch 15,28 ; 1 Ch 16,5 ; 1 Ch 25,1 ; 1 Ch 25,3 ; 1 Ch 25,6 ; 2 Ch 5,12 ; 2 Ch 9,11 ; 2 Ch 20,28 ; 2 Ch 29,25 ; Ne 12,27 ; Ps 33,2 (singulier, et ainsi principalement dans les Psaumes), Ps 43,4 ; Ps 49,5 ; Ps 57,9 ; Ps 71,22 ; Ps 81,3 ; Ps 92,4 ; Ps 98,5 (deux fois dans le verset) ; Ps 108,3 ; Ps 147,7 ; Ps 149,3 ; Ps 150,3 comparer כִּנֹּרוֺתֵינוּ Ps 137,2. — Verbes utilisés avec ׳ כ sont : — תָּפַשׂ Gn 4,21, נִגֵּן בְּ 1 S 16,16, מְשַׂחֵק בְּ 2 S 6,5 = 1 Ch 13,8, מְשֹׁרֲרִים בּ 1 Ch 15,16, מַשְׁמִיעִים בְּ 1 Ch 15,28, נִבָּא בְּ 1 Ch 25,1 ; 1 Ch 25,3, הוֺדָה בְּ Ps 33,2 ; Ps 43,4, זמּר בְּ Ps 71,22 ; Ps 98,5 ; Ps 147,7 ; Ps 149,3, הַלֲלוּהוּ בְּ Ps 150,3. — Sur les lyres anciennes voir références dans DrAmos, p. 236 f.
