BDB1364 [H1315]

hébreu biblique

בָּֽשְׂמִת nom propre, féminin (parfum ?) —

1. femme hittite, une épouse d'Ésaü Gn 26,34 ( P ) ; appelée fille d'Ismaël, et sœur de Nebajoth Gn 36,3 (mais dû probablement à R ; cette fille d'Ismaël est מַחֲלַת en Gn 28,9 P ) ; voir aussi Gn 36,4 ; Gn 36,10 ; Gn 36,13 ; Gn 36,17 (tous P ) ( ⅏ a מחלת tout au long de Gn 36).
2. fille de Salomon, épouse d'Ahimaats 1 R 4,15.
