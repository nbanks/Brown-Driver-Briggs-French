BDB3839 [H3489]

hébreu biblique

יָתֵד nom féminin Dt 23,14 piquet, cheville (hébreu tardif id. ; arabe ) — יָתֵד Dt 23,14 + 9 t. ; construit יְתַד Jg 4,21 ; Jg 16,14 (où avec l'article mais voir Ges«GKC:127»§ 127, R. 4 a) ; pluriel יְתֵדֹת Ex 38,20 construit יִתְדֹת Ex 27,19 + 4 t. ; suffixe וִיתֵידֹתַיִךְ Es 54,2 יְתֵדֹתָיו Ex 27,19 ; Es 33,20 וִיתֵדֹתֶיהָ Ex 39,40 וִיתֵדֹתָם Nb 3,37 ; Nb 4,32 — cheville, piquet, surtout

a. piquet de tente Jg 5,26 ; Jg 4,21 יְתַד הָאֹהֶל Jg 4,22 lire peut-être aussi Jb 4,21 (figuré ; comparer II. יֶתֶר) ; souvent du tabernacle Ex 27,19 (deux fois dans le verset) ; Ex 35,18 (deux fois dans le verset) ; Ex 38,20 ; Ex 38,31 ; Ex 38,21 || מֵיתָרָיו Ex 39,40, comparer Nb 3,37 ; Nb 4,32 (tous P ) ; de Sion sous la figure d'une tente Es 33,20 ; Es 54,2 figuré d'un souverain comme soutien de l'État Za 10,4.
b. un piquet pour suspendre Ez 15,3 ainsi figuré d'Éliakim Es 22,23 ; Es 22,24 de position sûre Esd 9,8 (comparer arabe, voir Thes Hi Hariri361 Der., d'un roi).
c. pour creuser Dt 23,14 (= bêche).
d. la cheville ou la baguette utilisée pour tasser la trame dans le métier à tisser Jg 16,14 a, voir GFMPAOS, Oct. 1889, clxxvi. ff. ; dans היתד הּארג vb, suppr. היתד (Idib et dans comm. sur le passage)
