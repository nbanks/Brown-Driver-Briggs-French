BDB3682 [H3342]

hébreu biblique

יֶ֫קֶב nom masculin Pr 3,10 cuve à vin (une auge ou cavité creusée (חצב Es 5,2) dans le roc pour recevoir le jus foulé dans le גַּת : Benz212 f.), parfois aussi pressoir (l'auge dans laquelle les raisins étaient foulés) ; — absolu יֶ֫קֶב Es 5,2 + 4 t. ; יָ֑קֶב Nb 18,30 construit ֶ י֫קֶב Jg 7,25 suffixe יִקְבֶ֑ךָ Dt 15,14 ; Dt 16,13 pluriel יְקָבִּים Jb 24,11 + 4 t. ; יִקְבֵּי Za 14,10 יְקָבֶיךָ Pr 3,10 — cuve à vin, Es 5,2 ; Ag 2,16 ; Jl 4,13 (|| גַּת), Pr 3,10 souvent || גֹּרֶן, Nb 18,27 ; Nb 18,30 ; Dt 15,14 ; Dt 16,13 ; 2 R 6,27 ; Os 9,2 ; Jl 2,24 apparemment du pressoir Jb 24,11 יְקָבִים דָּֽרְכוּ, Es 16,10n (d'où Jr 48,33). Désignant des localités particulières, Jg 7,25 יֶקֶב זְאֵב, Za 14,10 יִקְבֵי הַמֶּלֶךְ (près de Jérusalem).
