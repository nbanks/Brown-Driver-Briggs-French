(Toutes les autres formes de participe sont utilisées comme noms, voir חֹזֶה.)
1. voir, contempler :
a. avec l'œil, accusatif de chose Ps 58,9 ; Ps 58,11 ; Pr 22,29 ; Pr 29,20 ; Es 33,20 ; Es 57,8 Dieu dans la théophanie Ex 24,11 ( J E ) ; Dieu, après la mort, hors de la chair Jb 19,26 ; Jb 19,27 la face de Dieu Ps 11,7 ; Ps 17,15 (après la mort).
b. sujet les yeux eux-mêmes ; les yeux de l'homme : תֶּחֱזֶינָה עֵינֶיךָ tes yeux contempleront Es 33,17 aussi des yeux de Dieu : Ps 11,4 ; Ps 17,2.
c. avec בְּ regarder (intensément, avec satisfaction) ; une jeune fille Ct 7,1 (deux fois dans le verset) ; Sion Mi 4,11 ׳ נֹעַם י dans le temple Ps 27,4 l'œuvre de Dieu Jb 36,25 avec investigation בַּכּוֺכָבִים sur les étoiles, comme astrologues, en astrologie Es 47,13.
2. voir comme un voyant dans l'état extatique, avec accusatif de la vision vue : חזון Es 1,1 ; Ez 12,27 ; Ez 13,16 מחזה Nb 24,4 ; Nb 24,16 ( J E ; poésie, Balaam) ; מחזה שׁוא Ez 13,7 משּׂא Es 13,1 ; Ha 1,1 משׂאות שׁוא Lm 2,14 שׁוא Ez 13,6 ; Ez 13,9 ; Ez 13,23 ; Ez 21,34 ; Ez 22,28 ; Lm 2,14 כזב Ez 13,8 שֶׁקֶר Za 10,2 objet choses trompeuses Es 30,10 דבר(יםׅ Es 2,1 ; Am 1,1 ; Mi 1,1 avec לְ être un voyant, pour quelqu'un Es 30,10 voir une vision pour quelqu'un Lm 2,14 (deux fois dans le verset).
3. voir, percevoir :
a. avec l'intelligence, absolu Jb 34,32 ; Pr 24,32 accusatif de chose Es 48,6 Yahvé dans son temple Ps 63,3 dans ses actions providentielles Jb 23,9 ; Ps 46,9 ; Es 26,11 (deux fois dans le verset).
b. voir, par expérience absolu Jb 27,12 accusatif de chose Jb 15,17 ; Jb 24,1 .
c. pourvoir, תחזה מכל העם אנשׁי חיל Ex 18,21 pourvois parmi tout le peuple des hommes capables ( E ).
