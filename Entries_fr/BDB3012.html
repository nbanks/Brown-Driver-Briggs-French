hébreu biblique

I. חָפַר verbe creuser, chercher (arabe id. ; araméen חֲפַר, , id. ; hébreu tardif חֵפֶר action de creuser) —

Qal Parfait ׳ ח Jb 39,29 וְחָפַרְתָּ֫ Jb 11,18 וְחָפַרְתָּ֫ה Dt 23,14 חָפַרְתִּי Gn 21,30 חָֽפְרוּ Gn 26,18 + 2 t. ; חָפָ֑רוּ Gn 26,32 חֲפָרוּהָ Nb 21,18 Imparfait וַיַּחְפֹּר Gn 26,18 ; Gn 26,22 וַיַּחְפְּרֵהוּ Ps 7,16 וָאֶחְפֹּר Jr 13,7 יַחְפְּרוּ Jb Jb 39,21 (mais lire יַחְפֹּר ainsi ᵐ5 ᵑ9 ᵑ6 Di) Dt 1,22 וַיַּחְפְּרוּ Gn 26,19 + 2 t. ; וַיַּחְפְּרֻהוּ Jb 3,21 Infinitif construit לַחְפֹּר Jos 2,2 ; Jos 2,3 + Es 2,20 (comparer ci-dessous) ; Participe חֹפֵר Qo 10,8 —
1. creuser :
a. un puits, avec accusatif בְּאֵר Gn 21,30 ( E ), Gn 26,15 ; Gn 26,18 (deux fois dans le verset) ; Gn 26,19 ; Gn 26,21 ; Gn 26,22 ; Gn 26,32 ( J ), Nb 21,18 ( E ) ; comparer Ex 7,24 ( E ) ; creuser (un trou, sous-entendu comme objet) Dt 23,14 une fosse, dans une intention hostile, comme piège ou trappe, figuré d'un plan malveillant, avec accusatif בּוֺר Ps 7,16 (|| כָּרָה) ; ainsi avec accusatif גומץ Qo 10,8 , comparer [שַׁחַת] לְנַפְשִׁי ׳ ח Ps 35,7 du cheval, creuser, c.-à-d. gratter le sol, יַחְפֹּר בָּעֵמֶק Jb 39,21 (ainsi lire, voir ci-dessus) il gratte dans la vallée (comparer arabe sabot).
b. creuser pour quelque chose de caché, avec accusatif Jb 3,21 (figuré du désir ardent de la mort), Jr 13,7 (pas d'objet exprimé)
2. chercher, explorer, avec accusatif הָאָרֶץ Jos 2,3 (deux fois dans le verset) ( J E ), Dt 1,22 de l'aigle, chercher de la nourriture, avec accusatif אֹכֶל ׳ ח Jb 39,29, avec référence particulière à la vue perçante (|| מֵרָחוֺק עֵינָיו יַבִּיטוּ) ; ainsi = regarder soigneusement autour de soi avant de se coucher Jb 11,18 (pas d'objet). — לַחְפֹּר Es 2,20 voir חֲפַרְפָּרָה ci-dessous
