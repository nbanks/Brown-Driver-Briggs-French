BDB4705 [H4471]

hébreu biblique

מַמְרֵא nom propre, masculin et [d'un lieu] Mamré, seulement dans la Genèse :

1. dans J seulement אֵלֹנֵי מַמְרֵא térébinthes de Mamré (nom propre, masculin ?), Gn 13,18 (défini par אֲשֶׁר בְּחֶבְרוֺן ), Gn 18,1 (dans les deux, lieu d'habitation d'Abraham).
2. nom propre, masculin dans Gn 14,13 (הָאֱמֹרִי ׳ אֵלֹנֵי מ), comparer Gn 14,24 (où allié d'Abraham)
3. nom propre [d'un lieu] dans P : près du lieu de sépulture d'Abraham Gn 23,17 ; Gn 25,9 ; Gn 49,30 ; Gn 50,3 apparemment identifié avec Hébron Gn 23,19 ; Gn 35,27, comparer Di. — ᵐ5 Μαμβρη.
