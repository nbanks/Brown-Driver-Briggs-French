BDB1836 [H1709]

hébreu biblique

דָּג nom masculin Jon 2,1 poisson (hébreu tardif id.) — דָּג Jon 2,1 (deux fois dans le verset) ; Jon 2,11 + Ne 13,16 Qr ( Kt דָּאג) ; דָּגִים 1 R 5,13 + 6 t. ; דְּגֵי Gn 9,2 + 7 t. ; — poisson de mer Jon 2,1 (deux fois dans le verset) ; Jon 2,11 (au v.Jon 1,1 גָּדוֺל ׳ ד) ; Ne 13,16 (דָּאג Kt דָּג article du commerce tyrien ; seulement ici collectif, lire peut-être דָּגָה ; ou considérer comme usage tardif, comparer l'inverse דָּגָה d'un individu Jon 2,2) ; — ailleurs toujours pluriel : comme sujet du discours de Salomon 1 R 5,13 harpon צִלְצַל דגים Jb 40,31 le plus souvent דְּגֵי הַיָּם Gn 9,2 ( P ), Ps 8,9 ; Os 4,3 ; So 1,3 ; Ez 38,20 ; Jb 12,8 (dans tous opposé aux bêtes et aux oiseaux, et apparemment utilisé avec eux, + רֶמֶשׂ Ézéchiel comparer verbe רמשׂ Genèse, pour la création animale en général ; aussi Ha 1,14 comparer ci-dessous) ; Nb 11,22 ( J E ) de poissons (avec les bêtes) comme nourriture (comparer Gn 9,2s.) en comparaison d'hommes pris au piège, comme des poissons pris dans un filet (מְצוֺדָה ) Qo 9,12 comparer Ha 1,14 (דְּגֵי הַיָּם) ; 4 t. שַׁעַר הַדָּגִים porte des poissons 2 Ch 33,14 ; So 1,10 ; Ne 3,3 ; Ne 12,39 (sur la situation comparer שַׁעַר).
