BDB3450 [H8426]

hébreu biblique

תּוֺדָה nom féminin action de grâce, ׳ תּ Lv 7,12 + 23 t. ; construit תּוֺדַת Lv 7,13 ; Lv 7,15 pluriel תּוֺדוֺת Ps 56,13 + 3 t. ; תּוֺדֹת Ne 12,31 ; Ne 12,40 —

1. rendre louange à ׳ י Jos 7,19 ( J E ; || ׳ שִׂים כָּבוֺד לְ), Esd 10,11, dans les deux cas louange rendue en reconnaissant et abandonnant le péché ; voir Di sur Josué, et sur Esdras Öttli et surtout Ryle ; comparer verbe ידה avec לְ ; ainsi RVm ; > Thes MV SS Be RV et autres confession.
2. action de grâce dans les chants du culte liturgique ׳ קול ת Ps 26,7 ; Ps 42,5 ; Jon 2,10 || שִׁיר Ps 69,31 ; Ne 12,27 || זְמִרוֺת Ps 95,2 || קוֺל זִמְרָה Es 51,3 || תְּהִלָּה Ps 100,4 || כִּנּוֺר Ps 147,7.
3. chœur d'action de grâce, procession, file, compagnie Ne 12,31 ; Ne 12,38 ; Ne 12,40, probablement aussi Jr 30,19 (|| קוֺל מְשַׂחֲקִים).
4. sacrifice d'action de grâce ׳ קטּר ת Am 4,5 (|| נְדָבוֺת) ; une subdivision de זֶבַח הַשְּׁלָמִים Lv 7,12 a ; שׁלמיו ׳ זבח ת Lv 7,13 Lv 7,15 ׳ זבח (ה)ת Lv 7,12 b (tous P ), Lv 22,29 ( H ) ; זבחים ותודות 2 Ch 29,31 (deux fois dans le verset) ; ׳ זבחי שְׁלָמִים ות 2 Ch 33,16 ׳ זבת זבח(י) ת Ps 107,22 ; Ps 116,17 ׳ מבאי(ם) ת Jr 17,26 ; Jr 33,11 לתודה Ps 100,1 (titre) ; dans les Psaumes || ׃נדרים ׳ זָבַח ת offrir un sacrifice d'action de grâce Ps 50,14 ; Ps 50,23 (voir BrMP. 239) ; ׳ שִׁלּם ת Ps 56,13.
