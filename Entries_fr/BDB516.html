BDB516 [H535]

hébreu biblique

[אָמַל] verbe être faible, languir (comparer arabe espérer, attendre).

Qal Participe passif féminin מָה אֲמֻלָה לִכָּתֵךְ comme ton cœur est faible ! Ez 16,30 (mais < Co מַהלִּֿי וְלִבְרִיתֵח) ;
Pu`lal Parfait אֻמְלַל Jl 1,10 + ; אֻמְלְלָה Es 33,9 + ; אֻמְלְלוּ Jr 14,2 etc. ; — être ou devenir faible, languir ; de perte de fertilité (femme) 1 S 2,5 ; Jr 15,19 du pêcheur dont le commerce échoue Es 19,8 || אנה, אבל) ; en gén. des habitants d'une terre frappée Os 4,3 (|| תֶּאֱבַל הָאָרֶץ) ; Es 24,4 comparer ci-dessous habituellement sujet inanimé, personnifié ; שַׁדְמוֺת חֶשְׁבּוֺן Es 16,8 comparer Na 1,4 (deux fois dans le verset) ; תֵּבֵל Es 24,4 (|| נבל, אבל) ; אֶרֶץ Es 33,9 (|| אבל) ; sujet גֶּפֶן Es 24,7 (|| אָבַל תִּירוֺשׁ ; יִצְּהָר Jl 1,10 (|| שׁדד, אבל, הובישׁ ; sujet apparenté) ; תְּאֵנָה Jl 1,12 (|| הַגֶּפֶן הוֺבִישָׁה) ; שְׁעָרִים Jr 14,2 (|| אָֽבְלָה יְהוּדָה) ; comparer Lm 2,8 וַיַּאֲבֶלחֵֿל וְחוֺמָה יַחְדָּו אֻמְלָ֑לוּ.
