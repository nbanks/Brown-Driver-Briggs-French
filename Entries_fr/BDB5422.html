BDB5422 [H5397]

hébreu biblique

נְשָׁמָה nom féminin souffle ; — ׳ נ Dt 20,16 + 9 t. ; construit נִשְׁמַת (comparer LagBN 82) Gn 2,7 + 10 t. ; suffixe נִשְׁמָתִי Jb 27,3 נִשְׁמָתוֺ Jb 34,14 pluriel נְשָׁמוֺת Es 57,16 —

1. souffle de Dieu comme vent brûlant allumant une flamme Es 30,33 comme vent destructeur 2 S 22,16 = Ps 18,16 ; Jb 4,9 comme vent froid produisant la glace Jb 32,8 ; Jb 33,4.
2. souffle de l'homme 1 R 17,17 ; Es 42,5 ; Jb 27,3 ; Dn 10,7 souffle de vie נשׁמת חיים Gn 2,7 ( J ) ; comparer נִשְׁמַת רוּחַ חַיִּים Gn 7,22 ( J ) ; en tant qu'insufflé par Dieu c'est le souffle de Dieu dans l'homme Jb 34,14 ; Jb 36,4 et c'est caractéristique de l'homme אדם אשׁר נשׁמה באפו Es 2,22 l'homme dans les narines duquel il n'y a qu'un souffle (glose tardive).
3. synonyme נפשׁ כָּל נְשָׁמָה tout être qui respire Dt 20,16 ; Jos 11,11 ; Jos 11,14 ; 1 R 15,29 כָּלהַֿנְּשָׁמָה Jos 10,14 ; Ps 150,6 נְשָׁמוֺת Es 57,16.
4. esprit de l'homme, || רוּחַ, אָדָם ׳ נ ׳ נֵר י Pr 20,27 l'esprit de l'homme est une lampe de ׳ י.
