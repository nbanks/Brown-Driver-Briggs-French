[אָסַר] verbe lier, attacher, emprisonner (arabe , assyrien asâru , comparer COTGloss, araméen אֲסַר , , éthiopien ) —

Qal Parfait suffixe אֲסָרָם Jb 36,13 3 féminin singulier אָֽסְרָה Nb 30,5 +, etc. ; Imparfait יֶאְסֹר 1 R 20,14 וַיֶּאֱסֹר Gn 42,24 וַיֶּאְסֹר Gn 46,29 + ; pluriel suffixe יַאַסְרֻנִי Jg 16,7 etc. ; Impératif אֱסֹר 1 R 18,44 ; 2 R 9,21 אִסְרוּ Jr 46,4 ; Ps 118,27 Infinitif absolu אָסֹר Jg 15,13 אָסוֺר Jg 16,11 construit לָאְסֹר Nb 30,3 + ; לָאֱסוֺר Jg 15,10 לֶאֱסָרְךָ Jg 15,12 אָסְרָם Os 10,10 Participe actif construit אֹסְרִי Gn 49,11 passif אָסור Gn 40,3 + ; pluriel אֲסוּרִים Gn 39,20 + ; aussi Jg 16,21 ; Jg 16,25 Qr ( Kt אסירים) ; הָסוּרִים Qo 4,14 (comparer Now) ; אֲסֻרוֺת 2 S 3,34 construit אסורי Gn 39,20 Kt ( Qr אֲסִירֵי) ; —
1. lier, attacher, pour la sécurité, ânon à la vigne (suivi de ל) Gn 49,11 (bénédiction de Jacob) ; chevaux et ânes 2 R 7,10 (deux fois dans le verset) ; Ps 118,27 est douteux De lier la victime de la fête avec des cordes ; Che lier la procession avec des branches, etc.
2. lier, atteler, vaches à (ב) un chariot 1 S 6,7 ; 1 S 6,10 ainsi (métaphore) Os 10,10 les atteler à (ל) leurs deux iniquités (mais Jerome Ew Now font de ׳ א ici = יסר châtier, comparer אֶסֳּרֵם va) ; aussi suivi de l'accusatif char Gn 46,29 ( J ) Ex 14,6 ( E ) 2 R 9,21 absolu 1 R 18,44 ; 2 R 9,21 même pour préparer les chevaux de bataille הַסּוּסִים ׳ א Jr 46,4 (|| עֲלוּ הַפָּרָשִׁים).
3. lier, avec des cordes, des fers, etc., comme prisonnier, Siméon Gn 42,24 ( E ), Samson Jg 15,10 ; Jg 15,12 ; Jg 15,13 (3 t. dans le verset) ; Jg 16,5 ; Jg 16,7 ; Jg 16,8 ; Jg 16,11 (deux fois dans le verset) ; Jg 16,12 בַּנְחֻשְׁתַּיִם ׳ א 2 R 25,7 = 2 Ch 36,6 = Jr 39,7 ; Jr 52,11 aussi 2 Ch 33,11 comparer (sans ׳ בַּנ) 2 R 17,4 (בֵּית בֶּלָא ׳ א || עָצַת), 2 R 23,33 comparer du châtiment divin Jb 36,13 et voir aussi Ez 3,25 ; Ps 149,8 figuré d'une autorité absolue Ps 105,22 surtout Participe passif 2 S 3,34 tes mains n'étaient pas liées (|| et tes pieds n'étaient pas mis aux fers) ; בָּאֿזִקִּים ׳ א Jr 40,1 comparer בַּזִּקִּים ׳ א Jb 36,8 (probablement figuré || חַבְלֵיעֹֿ֑נִי) ; métaphore du roi retenu captif par les tresses d'une femme Ct 7,6 peut-être = emprisonné (lié ou non) Gn 39,20 ; Gn 40,3 ; Gn 40,5 (tous J E ) ; comme substantif pluriel prisonniers Gn 39,20 ( Kt , voir ci-dessus) ; ainsi (tardif) comme affligés, et objet de la compassion divine Es 49,9 ; Es 61,1 ; Ps 146,7 בֵּית האסי֯רים prison Jg 16,21 ; Jg 16,25 comparer Qo 4,14 .
4. ceindre (rare et tardif) אֵזוֺר בְּמָתְנֵיהֶם ׳ א Jb 12,18 (אֵזוֺר = pagne d'esclave, RS p. 25 ci-dessus ; Hoffm lit וַיָּסַר pour וַיֶּאְסֹר) ; חַרְבּוֺ אֲסוּרִים עַלמָֿתְנָיו Ne 4,12.
5. suivi de מִלְחָמָה engager la bataille, lancer l'attaque (comparer allemand mit jemandem anbinden ) 1 R 20,14 ; 2 Ch 13,3.
6. figuré de l'obligation d'un serment ou d'un vœu (seulement Nb 30, P ) עַלנֶֿפֶשׁ ׳ א, habituellement suivi de l'accusatif de sens apparenté au verbe אֱסָר, אִסָּר Nb 30,3 ; Nb 30,5 (deux fois dans le verset) ; Nb 30,6 ; Nb 30,8 ; Nb 30,9 ; Nb 30,11 ; Nb 30,12 sans אֱסָר etc., v.Nb 30,7 v.Nb 30,10 comparer v.Nb 30,4 אִסָּר ׳ א (עַלנֶֿפֶשׁ omis)
