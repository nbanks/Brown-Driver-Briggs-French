BDB7804 [H7312]

hébreu biblique

רוּם, רֻם nom [masculin] hauteur, arrogance (proprement infinitif) ; — absolu ׳ ר,

1. hauteur, élévation, שָׁמַיִם וָאָרֶץ לָעֹמֶק ׳ לָר Pr 25,3 construit רוּם עֵינַיִם figuré d'arrogance Es 10,12 (|| גֹּדֶל לֵבָב), Pr 21,4 (|| רְחַבלֵֿב) ; ainsi רֻם לִבּוֺ Jr 48,29 (|| גַּאֲוָה, גָּאוֺן, גֹּבַהּ) ; puis seul :
2. arrogance, רוּם אֲנָשִׁים Es 2,11 ; Es 2,17 (tous deux || גַּבְהוּת).
