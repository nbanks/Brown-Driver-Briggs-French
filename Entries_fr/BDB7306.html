BDB7306 [H6927]

hébreu biblique

[קַדְמָה] nom féminin antiquité, état antérieur ; —

1. antiquité, commencement, suffixe קַדְמָתָהּ Es 23,7 (de Tyr).
2. état antérieur, suffixe קַדְמָתָן Ez 16,55 (deux fois dans le verset), קַדְמַתְכֶן Ez 16,55 pluriel suffixe קַדְמוֺתֵיכֶם Ez 36,11.
3. construit singulier = conjonction avant (temporel ; araméen) שֶׁקַּדְמַת שָׁלַף Ps 129,6.
