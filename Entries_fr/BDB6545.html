hébreu biblique

פּוּט nom propre [d'un peuple] probablement Libyens, ou tribu libyenne ; habituellement nommé avec des peuples africains : Na 3,9 ; Jr 46,9 ; Ez 27,10 ; Ez 30,5 ; Ez 38,5 ( ᵐ5 habituellement Λίβυες) ; Gn 10,6 ( P ) = 1 Ch 1,8 (Φουδ) ; + Es 66,19 (pour ᵑ0 פּוּל ; ᵐ5 φουδ), — voir Di Gn 10,6 JenZA x. 325 ss.
